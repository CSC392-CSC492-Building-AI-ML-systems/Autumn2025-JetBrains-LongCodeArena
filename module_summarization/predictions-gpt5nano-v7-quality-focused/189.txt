KernelRidge
============

Kernel ridge regression (KRR) combines ridge regression with the kernel trick to learn a linear function in a non-linear feature space. It behaves like a non-linear regressor in the input space when using non-linear kernels, while retaining a closed-form solution.

Key points
- Learns in the feature space induced by the chosen kernel.
- Equivalent in form to support vector regression with squared loss, but with L2 regularization and a closed-form solution.
- Suitable for medium-sized datasets; the learned model is dense (not sparse) at prediction time.
- Supports multi-output regression (y can have shape [n_samples, n_targets]).

Quick start
- Basic usage with an RBF kernel:
  - from sklearn.kernel_ridge import KernelRidge
  - krr = KernelRidge(alpha=1.0, kernel="rbf", gamma=0.1)
  - krr.fit(X, y)
  - y_pred = krr.predict(X)

- Multi-target regression:
  - Y has shape (n_samples, n_targets)
  - krr = KernelRidge(alpha=[1.0, 0.5], kernel="linear")
  - krr.fit(X, Y)
  - Y_pred = krr.predict(X)

- Precomputed kernel:
  - Set kernel="precomputed" and pass a kernel matrix K of shape (n_samples, n_samples) as X to fit/predict.

Parameters
- alpha: float or array-like of shape (n_targets,), default=1.0
  Regularization strength. Larger values increase regularization. If an array is provided, penalties apply per target.

- kernel: str or callable, default="linear"
  Kernel mapping used internally. If a string, it must be one of the kernels in pairwise PAIRWISE_KERNEL_FUNCTIONS or "precomputed". If "precomputed", X is treated as a kernel matrix. If a callable, it is invoked on pairs of samples. When the kernel is callable, kernel_params may be used to configure it.

- gamma: float, default=None
  Parameter for RBF, laplacian, polynomial, exponential chi2, and sigmoid kernels. Ignored by other kernels.

- degree: int, default=3
  Degree for the polynomial kernel. Ignored by non-polynomial kernels.

- coef0: float, default=1
  Independent term for polynomial and sigmoid kernels. Ignored by other kernels.

- kernel_params: dict, default=None
  Additional keyword arguments for the kernel function when kernel is callable or when calling the kernel function.

Attributes
- dual_coef_: ndarray of shape (n_samples,) or (n_samples, n_targets)
  Representation of the weight vector(s) in kernel space.

- X_fit_: {ndarray, sparse matrix} of shape (n_samples, n_features)
  Training data used for fitting. If kernel is "precomputed", this stores the training kernel matrix of shape (n_samples, n_samples).

- n_features_in_: int
  Number of features seen during fit.

- feature_names_in_: ndarray of shape (n_features_in_,)
  Names of features seen during fit, if feature names are provided as strings.

Estimator behavior
- Multi-output support: Y can be 2D with shape (n_samples, n_targets). dual_coef_ adapts accordingly.
- Kernel construction:
  - If kernel is callable, params = kernel_params or {} are passed to the kernel function.
  - If kernel is a string, params = {"gamma": gamma, "degree": degree, "coef0": coef0} are passed to pairwise_kernels.
  - The function pairwise_kernels(X, Y, metric=kernel, filter_params=True, **params) computes the kernel matrix.

Examples
- Fit and predict with a linear kernel:
  - from sklearn.kernel_ridge import KernelRidge
  - import numpy as np
  - X = np.random.randn(100, 5)
  - y = np.random.randn(100)
  - krr = KernelRidge(alpha=1.0, kernel="linear")
  - krr.fit(X, y)
  - y_pred = krr.predict(X)

- Use an RBF kernel with a non-default gamma:
  - krr = KernelRidge(alpha=1.0, kernel="rbf", gamma=0.5)
  - krr.fit(X, y)
  - y_pred = krr.predict(X)

Notes
- The model learns a function f such that f(x) = sum_i alpha_i k(x_i, x), where alpha_i are the dual coefficients.
- Prediction requires access to the training data (unless using "precomputed" kernels with a stored training kernel matrix).
- The closed-form solution provides a direct fit, but may become computationally intensive for very large datasets.
