Calibration of predicted probabilities
======================================

Overview
--------
CalibratedClassifierCV provides probability calibration for a classifier by fitting a calibration model to the classifier’s outputs. It supports two calibration methods:
- sigmoid (Platt scaling): fits a logistic regression model to map decision scores to probabilities
- isotonic: fits a non-parametric isotonic regression model

Calibration is performed via cross-validation. For each cross-validation split, a copy of the base estimator is trained on the training data and its outputs are calibrated using the testing data. Predictions can be obtained by averaging the calibrated probabilities across folds (ensemble=True) or by calibrating on cross-validated predictions (ensemble=False). A pre-fitted estimator can be calibrated by using cv="prefit".

Key points
- Estimator: the classifier whose output is calibrated (default is LinearSVC).
- Method: 'sigmoid' or 'isotonic'.
- cv: cross-validation strategy (int, CV splitter, iterable of train/test indices) or "prefit".
- ensemble: if True, averages probabilities from all CV folds; if False, uses cross_val_predict for unbiased calibration.
- n_jobs: number of parallel jobs for fitting across folds.
- Attributes include calibrated_classifiers_ and model metadata.

Getting started
---------------
Minimal example: calibrate a LinearSVC classifier with Platt scaling (sigmoid).

.. code-block:: python

   from sklearn.svm import LinearSVC
   from sklearn.datasets import make_classification
   from sklearn.model_selection import train_test_split
   from sklearn.calibration import CalibratedClassifierCV

   # Generate a binary classification problem
   X, y = make_classification(n_samples=1000, n_features=20, random_state=42)

   # Base estimator (default is LinearSVC)
   base_estimator = LinearSVC()

   # Calibrate probabilities with sigmoid (Platt scaling), 5-fold CV
   calibrated_clf = CalibratedClassifierCV(
       estimator=base_estimator,
       method='sigmoid',
       cv=5,
       n_jobs=None,
       ensemble=True
   )

   calibrated_clf.fit(X, y)

   # Predict calibrated probabilities for new samples
   probs = calibrated_clf.predict_proba(X[:5])
   print(probs)

Alternative: isotonic calibration
---------------------------------
Isotonic calibration is non-parametric and may overfit with small calibration sets. Use when you have enough calibration data.

.. code-block:: python

   calibrated_clf_iso = CalibratedClassifierCV(
       estimator=base_estimator,
       method='isotonic',
       cv=5,
       n_jobs=None,
       ensemble=True
   )

   calibrated_clf_iso.fit(X, y)
   probs_iso = calibrated_clf_iso.predict_proba(X[:5])
   print(probs_iso)

Prefit usage
------------
If you already have a fitted estimator, you can calibrate it with cv="prefit". All data is used for calibration in this case.

.. code-block:: python

   # Fit your estimator on the full training data
   base_estimator.fit(X_train, y_train)

   # Calibrate using all data (no further cross-validation)
   calibrated_prefit = CalibratedClassifierCV(
       estimator=base_estimator, method='sigmoid', cv="prefit"
   )
   calibrated_prefit.fit(X_train, y_train)
   probs_prefit = calibrated_prefit.predict_proba(X_test)

Cross-validation and ensemble behavior
--------------------------------------
- ensemble=True (default): for each CV split, fit a separate calibrated classifier and average their predicted probabilities at prediction time.
- ensemble=False: uses cross_val_predict to obtain unbiased predictions for calibration; at prediction time, the estimator trained on all data is used with its calibrated probabilities.
- cv can be an int, a CV splitter, an iterable of train/test indices, or "prefit".

APIs and attributes
--------------------
- estimator: base classifier to calibrate (e.g., LinearSVC).
- method: 'sigmoid' or 'isotonic'.
- cv: cross-validation strategy (int, splitter, iterable) or "prefit".
- n_jobs: number of parallel jobs.
- ensemble: whether to ensemble predictions across CV folds.
- calibrated_classifiers_: list of calibrated estimators (one per CV fold or a single one for "prefit"/ensemble=False).
- classes_: class labels learned from the data.
- n_features_in_: number of features seen during fit.
- feature_names_in_: feature names seen during fit (if supported by the base estimator).

Practical guidance
------------------
- Sigmoid calibration is compact and stable with moderate training data. It is often effective for many classifiers.
- Isotonic calibration is flexible but can overfit if the calibration dataset is small (often requires more than ~1000 calibration samples).
- Use cross-validation to obtain unbiased calibration scores. When deploying, you can either:
  - Use the ensemble-calibrated probabilities (default) or
  - Calibrate a pre-fitted estimator on all available data (cv="prefit").

Evaluating calibration quality
------------------------------
- Reliability diagrams (calibration curves) visualize how well predicted probabilities align with observed frequencies.
- Common metrics include Brier score (lower is better) and calibration curves per probability bin.

.. code-block:: python

   from sklearn.calibration import calibration_curve
   from sklearn.metrics import brier_score_loss

   # Get predicted probabilities for the positive class
   positive_probs = calibrated_clf.predict_proba(X)[:, 1]

   prob_true, prob_pred = calibration_curve(y, positive_probs, n_bins=10)

   brier = brier_score_loss(y, positive_probs)
   print("Brier score:", brier)

Notes
-----
- The default base estimator is a LinearSVC, which does not output probabilities without calibration.
- The cv parameter follows the same conventions as scikit-learn cross-validation utilities.
- The API mirrors classic calibration approaches but is designed to work robustly with modern cross-validation and parallel processing.
