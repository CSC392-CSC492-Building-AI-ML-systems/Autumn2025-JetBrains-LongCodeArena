Plothist
========

Overview
--------

The plothist command renders histogram data stored in an HDF5 file produced by WESTPA. It supports 1D and 2D plots, multiple y-axis scales, range specification, and an optional contour overlay for 2D plots. It writes the resulting plot to a file and can also produce auxiliary outputs.

Usage
-----

plothist [options] INPUT [DIMENSION]

Where:
- INPUT is the HDF5 file containing histogram data.
- DIMENSION is an optional dimension specifier describing which histogram dimension to plot. The DIMENSION syntax is:
  INT[:[LB,UB]:LABEL]
  - INT is the zero-based dimension index in the histogram.
  - LB and UB define the plotting range (lower/upper bounds) for the axis.
  - LABEL is the axis label for the plot.
  - If omitted, the command defaults to dimension 0 with the full range.

Options
-------

Output options
- -o, --output, --plot-output PLOT_OUTPUT
  Store the plot to PLOT_OUTPUT. If you pass an empty string, plotting is suppressed. The format is inferred from the filename extension (default: PDF).
- --hdf5-output HDF5_OUTPUT
  Store plot data in the specified HDF5 file.
- --plot-contour
  For 2D plots, superimpose a contour plot over the heatmap when enabled.

Plot options
- --title TITLE
  Include TITLE as the top-of-graph title.
- Mutually exclusive scale options (one may be selected):
  --linear
    Plot the histogram on a linear scale.
  --energy
    Plot on an inverted natural-log scale (interpreted as (free) energy). This is the default.
  --log10
    Plot on a base-10 logarithmic scale.
- --zero-energy E
  Set the zero of energy to E. E can be a scalar, "min", or "max". Default is "min".
- --range RANGE
  Plot ordinates over RANGE, given as "LB,UB". For 1D plots this is the Y axis; for 2D plots, this is the colorbar axis. Default is the full range.
- --postprocess-function FUNCTION
  Names a function (module.function) that runs just before saving the plot. The function is called as:
  postprocess(hist, midpoints, binbounds)
  - hist is the histogram being plotted.
  - midpoints are the bin midpoints for each dimension.
  - binbounds are the bin boundaries for each dimension (only for 2D plots).
  The function modifies the plot in place via the pyplot state.

Defaults
- The default plotting scale is energy (inverted natural log).
- If no DIMENSION is provided, the command uses dimension 0 with the full range.

Internal behavior (summary)
- Opens the input HDF5 file for reading.
- Determines the available iterations for averaging/evolution and for single-point plots.
- Applies the specified plot scale, energy zero, and range settings.
- Optionally overlays a contour on 2D plots.
- Writes the plot to the specified output file and, if requested, saves histogram data to an HDF5 file.
- If a postprocessing function is supplied, it runs just before saving the plot.

Notes
- The command uses a helper to reduce arrays along all axes except the selected ones when preparing data for plotting.
- The firstdim argument is parsed via a dimensions specification helper; if omitted, a default dimension 0 is used with its full range.

Examples
--------

Basic plot (dimension 0, full range)
- plothist input.h5

Plot with energy scale (default) and a simple title
- plothist input.h5 0 --title "Dimension 0"

Plot with a linear scale and explicit range
- plothist input.h5 0 --linear --range "0,10"

Plot 2D data with a contour overlay
- plothist input.h5 1 --plot-contour

Save the plot to PNG and also write histogram data to an HDF5 file
- plothist input.h5 0 --output plot.png --hdf5-output data.h5

Use a postprocessing function just before saving
- plothist input.h5 0 --postprocess-function mymodule.process_hist

Implementation notes
--------------------

- The command is implemented as a WestPA subcommand (PlotHistBase), handling argument parsing, option validation, and coordinating data loading from the input HDF5 file.
- Energy scaling supports both numeric and string-based zero-point specifications ("min" or "max").
- The postprocessing hook provides a mechanism to customize plots without altering the core plotting logic.
