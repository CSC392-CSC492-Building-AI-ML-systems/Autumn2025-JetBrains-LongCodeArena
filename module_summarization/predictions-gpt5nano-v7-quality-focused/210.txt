Empirical Likelihood (emplike)

Overview
--------
This module provides empirical likelihood inference for descriptive statistics. It supports hypothesis tests and confidence intervals for the mean, variance, skewness, kurtosis, and correlation. If matplotlib is available, it can also produce multivariate confidence region plots and mean–variance contour plots. The implementation relies on Lagrange-multiplier based optimization to maximize a weighted empirical likelihood under estimating equations.

Public API
----------
DescStat(endog)

- Description
  Returns an object that enables empirical likelihood inference on descriptive statistics.

- Parameters
  - endog: ndarray
    Array of data. If endog is 1-D, it is reshaped to (n, 1). If endog.shape[1] == 1, a univariate instance (DescStatUV) is returned. If endog.shape[1] > 1, a multivariate instance (DescStatMV) is returned.

- Returns
  DescStat instance
    - Univariate case (k = 1): DescStatUV
    - Multivariate case (k > 1): DescStatMV

Notes
-----
- The module imports numpy, scipy.optimize, and scipy.stats (chi-square, skew, kurtosis), and uses the StatsModels optimizer helper _fit_newton. It also uses itertools and StatsModels graphics utilities when plotting is available.
- The description and optimization framework are documented in the _OptFuncts class (see Internal API below) and in the associated reference material (Owen, 2001).

Internal API (for developers)
-----------------------------
DescStat and _OptFuncts form the core of the empirical likelihood machinery.

_descendStat selection_
- The DescStat(endog) function:
  - If endog is 1-D, reshapes to (n, 1).
  - If endog has one column, returns a univariate instance: DescStatUV(endog).
  - If endog has more than one column, returns a multivariate instance: DescStatMV(endog).

_OptFuncts class_
Purpose
- Encapsulates the optimization and solving routines used to obtain the empirical likelihood-based estimates and likelihood-ratio statistics.
- Building blocks for solving the Lagrange multiplier (eta) that parameterizes the optimal weights under the estimating equations.

Key methods
- _log_star(eta, est_vect, weights, nobs)
  - Transform the log-probabilities (log-star) in terms of the Lagrange multiplier eta.
  - Inputs:
    - eta: Lagrange multiplier (1 x m)
    - est_vect: estimating equations matrix (n x k)
    - weights: observation weights (n,)
    - nobs: number of observations
  - Returns: data_star, the transformed log-like probabilities used by the optimization.
  - Notes: This is a placeholder for _fit_newton; the value is not directly used in the final objective, but is essential for the Newton-based optimization.

- _hess(eta, est_vect, weights, nobs)
  - Computes the Hessian (second derivative) of the weighted empirical likelihood with respect to eta.
  - Returns: hess, an m x m matrix.

- _grad(eta, est_vect, weights, nobs)
  - Computes the gradient of the weighted empirical likelihood with respect to eta.
  - Returns: gradient, an m-dimensional vector.

- _modif_newton(eta, est_vect, weights)
  - Performs a Modified Newton method to maximize the log-star objective.
  - Uses:
    - f: negative sum of _log_star
    - grad: negative _grad
    - hess: negative _hess
  - Employs statsmodels.base.optimizer._fit_newton with tolerance 1e-8 and a maximum of 50 iterations.
  - Returns: the optimized eta (Lagrange multiplier).

- _find_eta(eta)
  - Initiates and coordinates the search for the optimal eta given current estimates.
  - (Implementation details not shown in the excerpt; this method integrates the Newton-based solver with the estimating equations.)

Data shapes and usage
- est_vect: ndarray of shape (n, k) containing the estimating equations evaluated at observations.
- weights: 1-D array of length n with observation weights.
- nobs: scalar, number of observations (n).

Algorithmic notes
- The empirical likelihood approach seeks weights p_i that maximize the log-likelihood subject to the estimating equations, parameterized by eta (Lagrange multipliers).
- The optimization proceeds via a Newton-type method on the transformed log-likelihood ("log-star"), with a Hessian and gradient computed as described above.
- The final inference (e.g., likelihood-ratio tests and confidence regions) relies on the profile likelihood constructed through eta optimization.

Usage example
- Basic usage (univariate or multivariate selection handled automatically):
  - from emplike import DescStat
  - data = np.asarray([/* your data */])
  - stat = DescStat(data)
  - # Use stat to perform empirical likelihood inference as provided by the concrete DescStatUV/DescStatMV implementations.

Plotting and visualization
- If matplotlib is installed, the module can generate:
  - Multivariate confidence region plots
  - Mean–variance contour plots
- Plotting depends on optional Matplotlib support; the core empirical likelihood logic remains independent of plotting.
