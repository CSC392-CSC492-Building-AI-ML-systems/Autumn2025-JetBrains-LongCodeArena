Using Timers
============

Overview
--------

Timers help measure the duration of operations, such as API calls or session setup, to aid in profiling and performance tuning. There is no dedicated timer primitive in the shown codebase; use Python’s standard timing facilities to create lightweight timers and, if needed, a small context manager to simplify repeated use.

Quickstart
----------

Measure a single operation with time.perf_counter:

.. code-block:: python

   import time

   start = time.perf_counter()
   # perform operation here (e.g., a session operation or API call)
   end = time.perf_counter()
   duration = end - start
   print("Operation took {:.6f} seconds".format(duration))

Context manager: a tiny Timer for blocks of code
-----------------------------------------------

Use a context manager to automate start/stop timing and optional logging.

.. code-block:: python

   import time
   import logging

   class Timer(object):
       def __init__(self, name, logger=None):
           self.name = name
           self.logger = logger

       def __enter__(self):
           self.start = time.perf_counter()
           return self

       def __exit__(self, exc_type, exc, tb):
           self.end = time.perf_counter()
           self.duration = self.end - self.start
           if self.logger:
               self.logger.info('%s took %.6f seconds', self.name, self.duration)

   logger = logging.getLogger(__name__)

   with Timer("Session initialization", logger=logger):
       # Insert real initialization code here
       pass

Practical example: timing a session-related operation
-----------------------------------------------------

Configure a timer around a real operation in your code. Replace the placeholder with your actual API usage (e.g., creating a Session, querying, or interacting with entities).

.. code-block:: python

   import time
   import logging
   from ftrack_api import Session

   logger = logging.getLogger(__name__)

   with Timer("Create and initialize session", logger=logger):
       s = Session(
           server_url="https://example.ftrackapp.com",
           api_key="your_api_key",
           api_user="your_username",
       )
       # e.g., perform a follow-up operation here
       time.sleep(0.05)  # replace with real work

Tips for effective timer usage
------------------------------

- Use time.perf_counter for durations; it provides the highest available resolution.
- Prefer monotonic timing when measuring across system sleep or clock adjustments.
- Log or report durations in human-friendly units (ms or s) as appropriate.
- When profiling, repeat measurements and report averages and variations.
