Multiclass and multioutput algorithms
=====================================

Overview
--------

This module provides meta-estimators that extend single-output estimators to handle multiple target outputs. The four estimators are:

- MultiOutputRegressor
- MultiOutputClassifier
- ClassifierChain
- RegressorChain

All of these are meta-estimators that require a base estimator in their constructor. They transparently fit separate models per target or chain models to capture dependencies among targets.

Key components
--------------

- _MultiOutputEstimator
  - Abstract base class for multioutput meta-estimators.
  - Requires an estimator and an optional n_jobs parameter to control parallelism.
  - Serves as a common foundation for the concrete classes.

- _fit_estimator(estimator, X, y, sample_weight=None, **fit_params)
  - Clones the provided estimator and fits it on (X, y).
  - If sample_weight is provided, it passes it to fit.

- _partial_fit_estimator(estimator, X, y, classes=None, partial_fit_params=None, first_time=True)
  - Supports incremental learning via partial_fit.
  - Clones the estimator on the first call and uses partial_fit with optional classes.

- _available_if_estimator_has(attr)
  - Helper to conditionally enable parameters based on whether sub-estimators expose a given attribute.
  - Useful for exposing or hiding parameters depending on the chosen base estimator.

Supported estimators and behavior
---------------------------------

- MultiOutputRegressor
  - Wraps a regressor and fits one regressor per target column.
  - Produces predictions by aggregating the individual regressors’ outputs.

- MultiOutputClassifier
  - Wraps a classifier and fits one classifier per target column.
  - Suitable for multioutput classification problems where each output is a separate label.

- ClassifierChain
  - Trains a chain of classifiers where each classifier is trained on the original features augmented with previous predictions.
  - Models dependencies between outputs via the chain order.
  - The order can be specified or set to random.

- RegressorChain
  - Analogous to ClassifierChain but for regression.
  - Each regressor is trained on augmented features that include previous targets’ predictions.

API and usage notes
-------------------

- All meta-estimators accept an estimator parameter (the base estimator) and an optional n_jobs to control parallel evaluation.

- For classification targets, import check_classification_targets to validate the target structure when needed.

- Partial fit is supported for estimators that implement partial_fit, via _partial_fit_estimator.

- The internal helpers ensure that models are cloned and fit independently per target or per chain step, providing isolated learning for each sub-estimator.

Examples
--------

MultiOutputClassifier example with a random forest base estimator

.. code-block:: python

    from sklearn.datasets import make_multilabel_classification
    from sklearn.ensemble import RandomForestClassifier
    from sklearn.multioutput import MultiOutputClassifier

    # Create a simple multioutput classification problem
    X, y = make_multilabel_classification(n_samples=200, n_features=20, n_classes=3, random_state=0)

    base_clf = RandomForestClassifier(n_estimators=100, random_state=1)
    clf = MultiOutputClassifier(base_clf, n_jobs=2)

    clf.fit(X, y)
    preds = clf.predict(X)

MultiOutputRegressor example with a random forest regressor

.. code-block:: python

    from sklearn.datasets import make_regression
    from sklearn.ensemble import RandomForestRegressor
    from sklearn.multioutput import MultiOutputRegressor

    X, y = make_regression(n_samples=200, n_features=20, n_targets=3, random_state=0)

    base_reg = RandomForestRegressor(n_estimators=100, random_state=1)
    mor = MultiOutputRegressor(base_reg, n_jobs=2)

    mor.fit(X, y)
    preds = mor.predict(X)

ClassifierChain example

.. code-block:: python

    from sklearn.linear_model import LogisticRegression
    from sklearn.multioutput import ClassifierChain
    from sklearn.datasets import make_multilabel_classification

    X, y = make_multilabel_classification(n_samples=300, n_features=20, n_classes=3, random_state=0)

    base_clf = LogisticRegression(max_iter=1000, solver="lbfgs")
    chain = ClassifierChain(base_clf, order='random', n_jobs=2)

    chain.fit(X, y)
    y_pred = chain.predict(X)

RegressorChain example

.. code-block:: python

    from sklearn.ensemble import RandomForestRegressor
    from sklearn.multioutput import RegressorChain
    from sklearn.datasets import make_regression

    X, y = make_regression(n_samples=300, n_features=20, n_targets=3, random_state=0)

    base_reg = RandomForestRegressor(n_estimators=100, random_state=1)
    chain = RegressorChain(base_reg, order=[0, 1, 2])

    chain.fit(X, y)
    y_pred = chain.predict(X)

Implementation notes
--------------------

- The module uses clone to create fresh copies of the base estimator for each target or chain step.
- When a sample_weight is provided, it is forwarded to the base estimator during fit.
- The partial_fit path enables online or incremental learning for estimators that support partial_fit.
- The _available_if_estimator_has helper enables or disables parameters depending on whether the base estimator supports certain attributes.
