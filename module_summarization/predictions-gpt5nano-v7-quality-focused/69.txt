Module: isotonic
================

Overview
--------

This module implements isotonic regression utilities used to fit monotone relationships
between a predictor and a target. It provides a lightweight function to assess monotonicity,
a utility to compute an isotonic fit for a 1-D target, and a scikit-learn compatible
IsotonicRegression estimator that follows the typical fit/predict API.

Public API
----------

- check_increasing(x, y)
- isotonic_regression(y, sample_weight=None, y_min=None, y_max=None, increasing=True)
- IsotonicRegression

check_increasing
----------------

Determine whether y is monotonically correlated with x.

What it does
- Computes the Spearman rank correlation between x and y.
- Returns True if the correlation is non-negative, otherwise False.
- If the correlation is not exactly -1 or 1 and there are more than 3 samples,
  it computes a Fisher transform-based 95% confidence interval for the correlation.
  If the interval spans zero, a warning is issued.

Parameters
- x: array-like of shape (n_samples,)
  Training data.

- y: array-like of shape (n_samples,)
  Training target.

Returns
- increasing_bool: bool
  True when y tends to increase with x, False otherwise.

Notes
- The decision is based on the sign of the Spearman correlation.
- A warning may be raised if the Fisher-transformed 95% confidence interval spans zero.

References
- Fisher transformation (for confidence intervals of correlation coefficients).

Example
- Check monotonicity direction:
  check_increasing([1, 2, 3, 4], [1.0, 2.1, 2.9, 4.0])  # True

isotonic_regression
-------------------

Solve the isotonic regression problem for a 1-D target.

What it does
- Produces an isotonic (monotone) fit of y.
- Honors optional bounds y_min and y_max.
- Can enforce increasing (default) or decreasing (by setting increasing to False).

Parameters
- y: array-like of shape (n_samples,)
  The data to fit.

- sample_weight: array-like of shape (n_samples,), default=None
  Weights for each point. If None, equal weights are used.

- y_min: float or None, default=None
  Lower bound on the fitted values. If None, the lower bound is -inf.

- y_max: float or None, default=None
  Upper bound on the fitted values. If None, the upper bound is +inf.

- increasing: bool, default=True
  If True, fits an increasing function; if False, fits a decreasing function.

Returns
- y_: ndarray of shape (n_samples,)
  Isotonic fit of y.

Notes
- Internally uses an in-place, contiguous isotonic regression routine.
- If y_min or y_max are specified, the result is clipped to the provided bounds.

Example
- Basic isotonic regression (increasing):
  y = [3.0, 1.0, 2.0, 4.0]
  y_iso = isotonic_regression(y)

- Decreasing fit:
  y_iso_dec = isotonic_regression(y, increasing=False)

- With bounds:
  isotonic_regression(y, y_min=0.0, y_max=3.0)

IsotonicRegression
------------------

Isotonic regression estimator compatible with the scikit-learn API.

Description
- Fits a monotone (increasing by default) function X -> y for 1-D inputs X and targets y.
- Supports an out_of_bounds policy for predictions made outside the training domain.

Parameters
- y_min: float, default=None
  Lower bound on the lowest predicted value (minimum may still be higher).

- y_max: float, default=None
  Upper bound on the highest predicted value (maximum may still be lower).

- increasing: bool or 'auto', default=True
  If True, enforces increasing predictions with X.
  If False, enforces decreasing predictions with X.
  If 'auto', decides direction based on the sign of the Spearman correlation estimate.

- out_of_bounds: {'nan', 'clip', 'raise'}, default='nan'
  How to handle X values outside the training domain during prediction:
  - 'nan'  => predictions are NaN
  - 'clip' => predictions are clipped to the nearest endpoint
  - 'raise'=> raises ValueError for out-of-bounds queries

Attributes
- X_min_: float
  Minimum value of input array X_ used to define the left bound for interpolation.

- X_max_: float
  Maximum value of input array X used to define the right bound for interpolation.

- X_thresholds_: ndarray of shape (n_thresholds,)
  Unique ascending X values used to interpolate the y = f(X) monotonic function.

- y_thresholds_: ndarray of shape (n_thresholds,)
  Fitted monotone values corresponding to X_thresholds_ (the y-values of the piecewise-constant/isotonic function).

Examples
- Simple usage with a 1-D X:
  from isotonic import IsotonicRegression
  import numpy as np

  X = np.array([0.0, 1.0, 2.0, 3.0])
  y = np.array([0.0, 1.2, 0.8, 3.0])

  ir = IsotonicRegression()
  ir.fit(X, y)
  y_pred = ir.predict(np.array([0.5, 1.5, 2.5]))

- With out_of_bounds handling and clipping:
  ir_clip = IsotonicRegression(out_of_bounds='clip')
  ir_clip.fit(X, y)
  ir_clip.predict(np.array([-1.0, 0.5, 4.0]))  # out-of-bounds handled per policy

- Auto direction:
  ir_auto = IsotonicRegression(increasing='auto')
  ir_auto.fit(X, y)

See Also
- Best, M. J., and Chakravarti, N. (1990s). Active set algorithms for isotonic regression; A unifying framework.
  (Reference for the underlying isotonic regression algorithm.)

Notes
- The implementation relies on an in-place, contiguous isotonic regression routine for efficiency.
- check_increasing can help decide the direction when using increasing='auto'.

References
- Fisher transformation and confidence intervals for correlation coefficients.
- Active set algorithms for isotonic regression; a unifying framework (Best & Chakravarti).

Usage tips
- Use check_increasing to determine whether to request an increasing or decreasing fit when using IsotonicRegression with increasing='auto'.
- When predicting outside the training X domain, choose an out_of_bounds policy that matches your application’s needs (nan, clip, or raise).
