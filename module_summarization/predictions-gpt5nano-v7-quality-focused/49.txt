Duration
==========

Overview
--------
This module provides utilities for survival analysis with right-censored data and competing risks. It includes low-level functions to compute the survival function and cumulative incidence function (CIF) and a class that encapsulates estimation and inference for CIF under right-censoring. Frequency weights are supported; when provided, the behavior of the estimators adapts accordingly.

Public API
----------

Functions
~~~~~~~~

_calc_survfunc_right(time, status, weights=None, entry=None, compress=True, retall=True)
    Calculate the survival function and its standard error for a single group.

    Purpose
        Computes the Kaplan–Meier-style survival function and its standard error
        for right-censored data, handling optional entry times and optional
        frequency weights.

    Parameters
        time : array_like
            Times (censoring or event times) for subjects.
        status : array_like
            Event indicator:
            - status >= 1 indicates an event of a specific type at the given time.
            - status = 0 indicates censoring at the given time.
        weights : array_like, optional
            Frequency weights for each observation.
        entry : array_like, optional
            Entry/left-truncation times. If provided, the risk set is adjusted.
        compress : bool, default True
            If True, retain only times at which events occur.
        retall : bool, default True
            If True, returns full set of intermediate values (sp, se, utime, rtime, n, d).
            If False, returns only (sp, utime, rtime, n, d).

    Returns
        sp, se, utime, rtime, n, d
            sp: survival probabilities at event times.
            se: standard errors for the survival probabilities.
            utime: unique event times.
            rtime: ranks corresponding to time values.
            n: size of the risk set just prior to each event time.
            d: number of deaths (events) at each unique time.

    Notes
        - Greenwood's formula is used for SE when weights is None.
        - Tsiatis' formula is used for SE when weights are provided.
        - Survival probabilities are clipped to a lower bound for numerical stability.
        - If retall is False, a tuple with (sp, utime, rtime, n, d) is returned.

_calc_incidence_right(time, status, weights=None)
    Calculate the cumulative incidence function (CIF) and its standard error.

    Purpose
        Computes CIFs for competing risks (all-cause survival is used as a basis).

    Parameters
        time : array_like
            Times (censoring or event times) for subjects.
        status : array_like
            Event indicator:
            - status >= 1 indicates an event of type (status value).
            - status = 0 indicates censoring.
        weights : array_like, optional
            Frequency weights for each observation.

    Returns
        ip, se, utime
            ip: list of CIF arrays, one per event type (k = 1..ngrp).
            se: list of standard error arrays corresponding to ip (or None if weights provided).
            utime: unique event times used for the CIFs.

    Notes
        - The all-cause survival is computed first via _calc_survfunc_right with
          status0 = (status >= 1).
        - If weights are provided, standard errors are not returned (se is None).

_checkargs(time, status, entry, freq_weights, exog)
    Validate input arguments and raise ValueError on invalid configurations.

    Behavior
        - Ensures time and status have matching lengths.
        - Validates entry times if provided (same length as time; must occur before events).
        - Verifies weights length matches time if provided.
        - Verifies exog shape aligns with time if provided.

Class
~~~~~

CumIncidenceRight
    Estimation and inference for a cumulative incidence function.

    Description
        Encapsulates the estimation of CIF under right censoring. If frequency
        weights are provided, the point estimate is returned without a standard
        error. This class is designed for practical use with real data where
        competing risks are present.

    Parameters
        time : array_like
            Times (censoring or event times).
        status : array_like
            Event indicator:
            - status >= 1 indicates which event occurred at time t.
            - status = 0 indicates censoring at time t.
        title : string
            Optional title used for plots and summary output.
        freq_weights : array_like, optional
            Frequency weights for each observation.
        exog : array_like, optional
            Optional exogenous variables to account for possible violation of
            independent censoring. If present, used to adjust estimation.
        bw_factor : float
            Bandwidth multiplier for kernel-based estimation. Used only if exog is provided.
        dimred : boolean
            If True, enables use of proportional hazards regression-type modeling
            for dimension reduction or related purposes.

    Behavior
        This class provides an interface for estimating CIF with optional
        adjustments for censoring mechanisms via exogenous information and
        weights. Concrete methods (not shown in the excerpt) perform fitting,
        inference, and potentially plotting.

    Example
        Example usage (illustrative; adapt to actual class methods in your environment):

        from duration import CumIncidenceRight

        cif = CumIncidenceRight(time, status, title="CIF Example")

Notes
-----
- The module uses NumPy for numerical operations and relies on array-based
  operations for speed and efficiency.
- The internal functions use event times (utime) and their ranks (rtime)
  to efficiently compute survival and CIFs.
- The implementation supports right censoring and competing risks through the
  CIF formulation.

Public usage pattern
--------------------

- Prepare time and status arrays with proper coding (0 for censoring, 1..J for event types).
- Call CumIncidenceRight(time, status, ...) to perform CIF estimation with optional weightings and covariate adjustments.
- Access results through the class's methods (as provided by your actual implementation).

Explanations of return values
------------------------------

- Survival function sp is the probability of being alive (not having any event) just prior to each event time.
- The standard error se reflects the uncertainty in the survival estimate:
  - Greenwood’s formula is used when no weights are provided.
  - Tsaiatis’ formula is used when weights are provided.
- CIF ip gives the probability of experiencing a specific type of event by time t.
- utime provides the timeline (unique times) used for the calculations.

Implementation notes
--------------------
- Time values are converted to ranks and unique times are computed for efficiency.
- The risk set size n at each event time is computed with and without left-truncation (entry times).
- When compress is True, calculations focus only on times where events occur.
- Numerical stability is addressed by clipping survival probabilities away from zero and handling NaNs where appropriate.
