RandAugment augmentation utility for NVIDIA DALI
===============================================

Overview
--------
This module provides the RandAugment-based augmentation entry point for DALI pipelines. It applies a configurable set of random augmentations to a batch of samples (images or videos) using the RandAugment scheme described in the referenced paper. The augmentation suite is built from NVIDIA DALI's auto_aug augmentations.

Public API
----------
- rand_augment(data, n, m, num_magnitude_bins=31, shape=None, fill_value=128, interp_type=None, max_translate_abs=None, max_translate_rel=None, seed=None, monotonic_mag=True, excluded=None) -> DataNode
- apply_rand_augment(augmentations, data, n, m, num_magnitude_bins=None, seed=None, **kwargs) -> DataNode
  - Note: apply_rand_augment is intended as a lower-level helper used by rand_augment.

rand_augment
------------
Applies the RandAugment scheme to a batch of samples.

Signature
- rand_augment(
    data: DataNode,
    n: int,
    m: int,
    num_magnitude_bins: int = 31,
    shape: DataNode or Tuple[int, int] = None,
    fill_value: int = 128,
    interp_type: types.DALIInterpType = None,
    max_translate_abs: int or Tuple[int, int] = None,
    max_translate_rel: float or Tuple[float, float] = None,
    seed: int = None,
    monotonic_mag: bool = True,
    excluded: List[str] = None,
) -> DataNode

Purpose
- Applies a randomly sampled set of n augmentations, each with magnitude determined by m, to the input batch.

Key behavior
- Chooses an augmentation suite:
  - If monotonic_mag is True, uses get_rand_augment_suite(use_shape, max_translate_abs, max_translate_rel).
  - If monotonic_mag is False, uses get_rand_augment_non_monotonic_suite(use_shape, max_translate_abs, max_translate_rel).
- Ensures all augmentations have unique names.
- Validates the excluded list: every name in excluded must exist in the suite; otherwise raises an exception.
- Filters augmentations by exclusions and passes the selected augmentations to apply_rand_augment along with data and sampling parameters.
- Returns a DataNode containing the transformed samples.

Parameters
- data (DataNode): Batch of samples to process. Supports images (HWC) and videos (FHWC) with dtype uint8.
- n (int): Number of random augmentations applied to each sample.
- m (int): Magnitude (strength) for each operation, in [0, num_magnitude_bins - 1].
- num_magnitude_bins (int): Number of magnitude bins to divide the magnitude ranges into. Default 31.
- shape (DataNode or Tuple[int, int], optional): Target height/width for the image or video frames. If provided, magnitudes for translation depend on the shape and span [0, max_translate_rel * shape]. Otherwise, magnitudes span [0, max_translate_abs].
- fill_value (int, optional): Padding value for warp_affine-based operations (translation, shear, rotate). If None, uses border clamping.
- interp_type (types.DALIInterpType, optional): Interpolation mode for warp_affine operations. Supports types.INTERP_LINEAR (default) and types.INTERP_NN.
- max_translate_abs (int or Tuple[int, int], optional): Max translation in pixels when shape is not provided. If a tuple, first element bounds height, second bounds width. Default 100.
- max_translate_rel (float or Tuple[float, float], optional): Max translation as a fraction of image shape when shape is provided. If a tuple, first bounds height, second width. Default ~0.45 (about 100/224).
- seed (int, optional): Seed to deterministically sample operations and magnitudes.
- monotonic_mag (bool, optional): If True (default), magnitude increases with the magnitude bin. If False, non-monotonic magnitudes are used for some color operations, leading to different strength behavior.
- excluded (List[str], optional): Names of augmentations to exclude from the default RandAugment suite. You can still use apply_rand_augment to introduce custom or fine-tuned augmentations.

Returns
- DataNode: The batch of transformed samples.

Usage example
- Basic usage with default settings:
  out = rand_augment(data, n=2, m=9)

- With custom translation limits and shape:
  out = rand_augment(
      data,
      n=3,
      m=12,
      shape=(224, 224),
      max_translate_rel=0.5,
      fill_value=0
  )

- Excluding specific augmentations:
  out = rand_augment(data, n=2, m=7, excluded=["ShearX", "Rotate"])

Notes
- The function validates exclusions against the available augmentation names, and raises an exception if an invalid name is provided.
- Magnitude interpretation depends on the shape argument as described above; without shape, max_translate_abs controls translation magnitude.
- The augmentation suite is derived from NVIDIA DALI's auto_aug augmentations.
