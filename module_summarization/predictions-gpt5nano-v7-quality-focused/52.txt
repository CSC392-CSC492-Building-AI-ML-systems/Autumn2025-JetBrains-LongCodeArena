Generalized Method of Moments (GMM) and Two-Stage Least Squares for IV2SLS
=========================================================================

Overview
--------

This module provides an instrumental variables estimator based on the
Two-Stage Least Squares (2SLS) approach. It is implemented as the IV2SLS class,
which performs the standard 2SLS procedure:

- First stage: regress each endogenous explanatory variable X on the instruments Z
  to obtain Xhat = Z (Z'Z)^{-1} Z' X.
- Second stage: regress the dependent variable y on Xhat to obtain the 2SLS estimates
  beta_hat = (Xhat'Xhat)^{-1} Xhat'y.

Key design notes:
- The implementation handles endog, exog, and instrument inputs as 1D or 2D arrays.
- The instrument matrix must include all exogenous variables that are instrumented
  as well as any additional instruments.
- Degrees of freedom use df_resid = nobs - k_vars, following the small sample
  adjustment convention (as used in the surrounding linear-model infrastructure).

This module is primarily focused on the IV2SLS estimator. A broader GMM interface is discussed in the module header, but the current implementation centers on IV2SLS and related results handling.

Classes
-------

IV2SLS
^^^^^^^

Instrumental variables estimation using Two-Stage Least Squares.

Constructor
- endog: Endogenous dependent variable (y). Shape can be 1D (nobs,) or 2D (nobs, 1).
- exog: Explanatory variables to be instrumented (X). Shape can be 1D or 2D (nobs, k).
- instrument: Instrument matrix (Z). Must be 2D. All exog variables that are instrumented
  and any additional instruments should be included here. Inputs are coerced to 2D.

Notes
- Internally, the class computes:
  - ztz = Z'Z, ztx = Z'X
  - Xhat = Z (Z'Z)^{-1} Z' X
  - beta_hat = (Xhat'Xhat)^{-1} Xhat'y
- Degrees of freedom: df_resid = nobs - k, df_model = k - k_constant, where k is the number
  of columns in exog.
- This class inherits from LikelihoodModel and returns a RegressionResultsWrapper
  around an IV-specific results object.

Methods
- initialize(): Prepare working copies of endog and exog (wendog, wexog).
- fit(): Perform the 2SLS estimation as described above. Returns a
  RegressionResultsWrapper containing the IV regression results.
  - Stores xhatparams (the first-stage coefficients for X on Z) and xhat (the fitted X)
    for potential use by Hausman-type tests or diagnostics.
  - Computes normalized_cov_params as a covariance estimate (see Notes) and stores
    the second-stage OLS results in _results_ols2nd.
- predict(params, exog=None): Predict values using the provided parameters and
  design matrix (exog). If exog is not supplied, uses the model’s exog.

Attributes
- df_resid: Degrees of freedom for residuals (nobs - k_vars).
- df_model: Model degrees of freedom (k_vars - k_constant).
- xhatparams: First-stage coefficients from Z'Z^{-1} Z'X.
- xhat: Fitted Xhat matrix (Z Xhatparams).
- normalized_cov_params: Covariance matrix for the parameter estimates (as computed).

IVRegressionResults
^^^^^^^^^^^^^^^^^^^

Results class for the IV/2SLS estimation. This class extends RegressionResults
and provides access to standard regression-style summaries and statistics
for IV2SLS fits.

API and Usage
--------------

Construction

- IV2SLS(endog, exog, instrument=None)

  - endog: array_like, shape (nobs,) or (nobs, 1)
  - exog: array_like, shape (nobs,) or (nobs, k)
  - instrument: array_like, shape (nobs, m)

Estimation

- fit() -> RegressionResultsWrapper
  - Returns a wrapper around an IVRegressionResults instance, containing:
    - estimated parameters (beta_hat)
    - covariance estimates (normalized_cov_params)
    - optional second-stage OLS results for diagnostic purposes

Prediction

- predict(params, exog=None) -> array
  - Returns predicted y values from the linear model using the given parameters.

Example
-------

The following example demonstrates a minimal usage pattern:

.. code-block:: python

    import numpy as np
    from statsmodels.gmm import IV2SLS

    # Synthetic data (n observations)
    np.random.seed(0)
    n = 200
    Z = np.random.normal(size=(n, 2))          # instruments
    X = 0.5 * Z[:, 0:1] + 0.4 * Z[:, 1:2] + 0.1 * np.random.normal(size=(n, 1))  # endogenous regressor
    y = 2.0 * X + 0.5 * np.random.normal(size=(n, 1))  # dependent variable

    model = IV2SLS(endog=y, exog=X, instrument=Z)
    res = model.fit()
    print(res.summary())

Notes and Limitations
---------------------

- The current implementation focuses on the IV2SLS estimator. A general GMM interface is discussed in the module header but is not fully implemented here.
- The covariance calculation in fit() is a simple (Xhat'Xhat)^{-1} formulation with a particular data-driven component; users should interpret the covariance with this context in mind.
- The "whiten" method is defined but not implemented; this is a placeholder for potential future refinements (e.g., robust or whitened calculations).
- Some advanced diagnostics (e.g., Hausman tests, J-tests, and external Hausman-type test statistics) are mentioned in the project notes but are not exposed as part of the current API.
