web3.miner
==========

Overview
--------
This document describes the Miner API wrapper in web3.py. The Miner class provides thin, direct wrappers around the Ethereum JSON-RPC miner_* endpoints. Each method issues a blocking RPC call through the Web3 request manager. The hashrate property returns the current mining hash rate as a decimal value.

Class: Miner
-------------

Initialization
--------------
- __init__(web3)
  - Purpose: Stores the Web3 instance used to issue JSON-RPC requests.
  - Parameter:
    - web3: The Web3 instance to use for RPC calls.

Properties
----------
hashrate
  - Description: Returns the current miner hash rate.
  - Details: The value is converted to a decimal number using to_decimal via the apply_formatters_to_return decorator.
  - Access: Read-only property.

API Methods
-----------

makeDAG(number)
  - Purpose: Triggers the creation of a mining DAG for the specified epoch/number.
  - Signature: makeDAG(self, number)
  - RPC: calls "miner_makeDag" with [number]
  - Return: The RPC result (as returned by the node).

setExtra(extra)
  - Purpose: Sets extra data to include in mined blocks.
  - Signature: setExtra(self, extra)
  - RPC: calls "miner_setExtra" with [extra]
  - Return: The RPC result.

setEtherBase(etherbase)
  - Purpose: Sets the etherbase (the address that receives mining rewards).
  - Signature: setEtherBase(self, etherbase)
  - RPC: calls "miner_setEtherbase" with [etherbase]
  - Return: The RPC result.

setGasPrice(gas_price)
  - Purpose: Sets the gas price used by the miner.
  - Signature: setGasPrice(self, gas_price)
  - RPC: calls "miner_setGasPrice" with [gas_price]
  - Return: The RPC result.

start(num_threads)
  - Purpose: Starts mining using the specified number of threads.
  - Signature: start(self, num_threads)
  - RPC: calls "miner_start" with [num_threads]
  - Return: The RPC result.

stop()
  - Purpose: Stops mining.
  - Signature: stop(self)
  - RPC: calls "miner_stop" with []
  - Return: The RPC result.

startAutoDAG()
  - Purpose: Enables automatic DAG generation.
  - Signature: startAutoDAG(self)
  - RPC: calls "miner_startAutoDag" with []
  - Return: The RPC result.

stopAutoDAG()
  - Purpose: Disables automatic DAG generation.
  - Signature: stopAutoDAG(self)
  - RPC: calls "miner_stopAutoDag" with []
  - Return: The RPC result.

Usage Examples
--------------

Accessing the Miner API
- Obtain the Miner wrapper from a Web3 instance:
  .. code-block:: python

     from web3 import Web3
     web3 = Web3(Web3.HTTPProvider("http://localhost:8545"))
     miner = web3.miner

Query current hashrate
- The hashrate is returned as a decimal value:
  .. code-block:: python

     current_hashrate = miner.hashrate
     print(current_hashrate)

Start mining with a specific number of threads
- Start mining using 4 threads:
  .. code-block:: python

     miner.start(4)

Set the etherbase address
- Assign the address that receives mining rewards:
  .. code-block:: python

     miner.setEtherBase("0xYourEtherbaseAddress")

Create a new DAG for a given epoch
- Trigger DAG creation for a specific epoch number:
  .. code-block:: python

     miner.makeDAG(0)

Configure mining parameters
- Set extra data and gas price:
  .. code-block:: python

     miner.setExtra("0xabc123")
     miner.setGasPrice(20000000000)  # 20 gwei

Manage DAG generation automatically
- Enable or disable automatic DAG generation:
  .. code-block:: python

     miner.startAutoDAG()
     miner.stopAutoDAG()

Return Values
-------------
- hashrate: Decimal value representing the current mining hash rate (due to to_decimal formatting).
- Other methods: Return the raw RPC response from the node. This is typically a boolean, string, or null value depending on the RPC method and node state.

Notes
-----
- All methods use the Web3 request manager to perform blocking RPC calls.
- The API mirrors the underlying Ethereum JSON-RPC miner_* endpoints with camelCase method names preserved.
