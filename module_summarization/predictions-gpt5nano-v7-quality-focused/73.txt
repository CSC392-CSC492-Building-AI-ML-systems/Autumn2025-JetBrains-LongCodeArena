The Naive Bayes estimators in scikit-learn

Overview
--------
The sklearn.naive_bayes module implements Naive Bayes classifiers. These are supervised
learning methods that apply Bayes' theorem with strong (naive) feature independence
assumptions. The estimators support both batch learning and online updates via partial_fit.

Public API
----------
The module exposes the following classifiers:

- BernoulliNB
- GaussianNB
- MultinomialNB
- ComplementNB
- CategoricalNB

Each estimator follows the scikit-learn estimator API (fit, predict, predict_proba, etc.) and exposes class-specific parameters and attributes.

Gaussian Naive Bayes
--------------------
GaussianNB implements the Gaussian Naive Bayes classifier. It supports online updates to model
parameters via partial_fit. See the user guide for details on the underlying algorithm.

Parameters
----------
priors : array-like of shape (n_classes,), default=None
    Prior probabilities of the classes. If specified, priors are not adjusted from the data.

var_smoothing : float, default=1e-9
    Portion of the largest variance across all features that is added to variances for calculation stability.

Attributes
----------
class_count_ : ndarray of shape (n_classes,)
    Number of training samples observed in each class.

class_prior_ : ndarray of shape (n_classes,)
    Probability of each class.

classes_ : ndarray of shape (n_classes,)
    Class labels known to the classifier.

epsilon_ : float
    Absolute additive value to variances (used for numerical stability).

n_features_in_ : int
    Number of features seen during fit.

feature_names_in_ : ndarray of shape (n_features_in_,)
    Names of features seen during fit (defined only if X has feature names that are all strings).

var_ : ndarray of shape (n_classes, n_features)
    Variance of each feature per class.

theta_ : ndarray of shape (n_classes, n_features)
    Mean of each feature per class.

Examples
--------
Basic usage with GaussianNB:

.. code-block:: python

   from sklearn.naive_bayes import GaussianNB
   from sklearn.datasets import make_classification

   X, y = make_classification(n_features=4, random_state=0)
   gnb = GaussianNB()
   gnb.fit(X, y)

   # Predict class labels
   y_pred = gnb.predict(X)

   # Predict class probabilities
   proba = gnb.predict_proba(X)

   # Predict log probabilities
   log_proba = gnb.predict_log_proba(X)

   # Online update (partial_fit)
   classes = np.unique(y)
   gnb.partial_fit(X[:20], y[:20], classes=classes)

How to interpret the results
----------------------------
- predict returns the most probable class for each sample.
- predict_proba returns the probability distribution over classes for each sample.
- predict_log_proba returns the log of the probabilities (useful for numerical stability in downstream computations).
- predict_joint_log_proba returns the joint log probabilities log P(x, y) for each sample and class.

Related estimators
------------------
- BernoulliNB: Naive Bayes classifier for binary/boolean features.
- MultinomialNB: Naive Bayes classifier for multinomial feature counts.
- ComplementNB: Variant designed for imbalanced datasets.
- CategoricalNB: Naive Bayes classifier for categorical features.

Internal base class
-------------------
All Naive Bayes estimators derive from _BaseNB, an abstract base class that provides common
inference utilities, including:
- predict_joint_log_proba(X): returns log P(x, y) for each sample and class.
- predict(X): returns the predicted class labels.
- predict_proba(X): returns class probabilities.
- predict_log_proba(X): returns log class probabilities.

Notes
-----
- Priors can be supplied to override the class distribution inferred from the data.
- The implementation uses log probabilities for numerical stability and computes logsumexp where appropriate.
