Resources
=========

This module provides support for plugins that manage access to resources such as schemas. It includes classes for wrapping and managing resource mappings with additional metadata.

Classes
-------

DirectoryResourceMapping
~~~~~~~~~~~~~~~~~~~~~~~~
A resource mapping that reads resource content from a directory or directory tree.

Inherits from :class:`~asdf_standard.resource.DirectoryResourceMapping`.

ResourceMappingProxy
~~~~~~~~~~~~~~~~~~~~
Wrapper around a resource mapping that carries additional information about the package that provided the mapping.

**Methods:**

- ``maybe_wrap(cls, delegate)``: Class method that wraps a mapping in a proxy if it isn't already wrapped.
- ``__init__(self, delegate, package_name=None, package_version=None)``: Initialize with a delegate mapping and optional package metadata.
- ``__getitem__(self, uri)``: Retrieve resource content by URI.
- ``__len__(self)``: Return number of resources in mapping.
- ``__iter__(self)``: Iterate over resource URIs.
- ``__eq__(self, other)``: Compare based on delegate identity.
- ``__hash__(self)``: Hash based on delegate identity.
- ``__repr__(self)``: Return string representation.

**Properties:**

- ``delegate``: Get the wrapped mapping instance (collections.abc.Mapping).
- ``package_name``: Get the name of the providing Python package (str or None).
- ``package_version``: Get the version of the providing package (str or None).
- ``class_name``: Get the fully qualified class name of the mapping (str).

ResourceManager
~~~~~~~~~~~~~~~
Wraps multiple resource mappings into a single interface with improved error handling.

**Parameters:**

- ``resource_mappings``: Iterable of collections.abc.Mapping objects. First mapping takes precedence for duplicate URIs.

**Methods:**

- ``__init__(self, resource_mappings)``: Initialize with collection of resource mappings.
- ``__getitem__(self, uri)``: Retrieve resource content by URI, raising KeyError if unavailable.
- ``__len__(self)``: Return total number of unique URIs.
- ``__iter__(self)``: Iterate over all available URIs.
- ``__contains__(self, uri)``: Check if URI is available (efficient implementation).
- ``__repr__(self)``: Return string representation.

JsonschemaResourceMapping
~~~~~~~~~~~~~~~~~~~~~~~~~
Resource mapping that fetches JSON Schema metaschemas from the jsonschema package.

**Methods:**

- ``__getitem__(self, uri)``: Fetch schema content for given URI.
- ``__len__(self)``: Return number of available schemas.
- ``__iter__(self)``: Iterate over available schema URIs.
- ``__repr__(self)``: Return string representation.
