w_succ
======

The ``w_succ`` command identifies and lists segments that successfully reach a target state (i.e., recycled segments) in a WESTPA simulation.

Purpose
-------
This tool extracts information about segments that have been recycled during a WESTPA simulation. It outputs details including iteration number, segment ID, weight, and final progress coordinate values for each successful segment.

Usage
-----
.. code-block:: bash

    w_succ [-h] [-o OUTPUT_FILE] [WESTPA_RC_ARGS]

Command-line Options
--------------------
.. option:: -h, --help

    Show help message and exit.

.. option:: -o OUTPUT_FILE, --output OUTPUT_FILE

    Store output in OUTPUT_FILE (default: write to standard output).

Output Format
-------------
The output consists of a header (if not suppressed) followed by data rows:

.. code-block:: text

    # successful (recycled) segments
    # column 0:    iteration
    # column 1:    seg_id
    # column 2:    weight
    # column>2:    final progress coordinate value

Each data row contains:
- Iteration number (8-character field)
- Segment ID (8-character field)
- Segment weight (20.14g format)
- Final progress coordinate values (formatted according to data type)

The progress coordinate values are formatted based on their data type:
- u8, i8: %20d
- u4, i4: %10d / %11d
- u2, i2: %5d / %6d
- f4: %14.7g
- f8: %23.15g

Implementation Details
----------------------
The tool inherits from:
- ``WESTAnalysisTool``
- ``WESTDataReaderMixin``
- ``CommonOutputMixin``

It processes WESTPA data by:
1. Iterating through all completed iterations
2. Identifying recycled segments using endpoint type ``SEG_ENDPOINT_RECYCLED``
3. Extracting final progress coordinate values for recycled segments
4. Formatting and outputting the results

Notes
-----
- Requires an active WESTPA simulation data directory
- The tool automatically skips iterations with no recycled segments
- Output can be redirected to a file using the ``-o/--output`` option
- Header output can be suppressed using appropriate options from CommonOutputMixin
