`numpy.lib.recfunctions` Module
==============================

.. currentmodule:: numpy.lib.recfunctions

The `recfunctions` module provides a collection of utilities for manipulating structured arrays and record arrays. Most functions were initially implemented for matplotlib and have been extended for general NumPy usage.

.. autofunction:: recursive_fill_fields

**Parameters:**

Utility Functions
----------------

The module also contains several internal utility functions including:

- ``_get_fieldspec``: Produces name/dtype pairs corresponding to dtype fields
- ``_zip_dtype``: Helper function for combining dtypes from multiple arrays
- ``_recursive_fill_fields_dispatcher``: Dispatch function for array function protocol

Notes
-----

This module is particularly useful for working with structured arrays containing nested fields or when needing to manipulate the structure of record arrays. The functions handle both regular arrays and masked arrays through NumPy's array function dispatch system.

Examples
--------

Basic usage of recursive field filling:

>>> import numpy as np
>>> from numpy.lib import recfunctions as rfn
>>> a = np.array([(1, 10.), (2, 20.)], dtype=[('A', np.int64), ('B', np.float64)])
>>> b = np.zeros((3,), dtype=a.dtype)
>>> rfn.recursive_fill_fields(a, b)
array([(1, 10.), (2, 20.), (0,  0.)], dtype=[('A', '<i8'), ('B', '<f8')])
