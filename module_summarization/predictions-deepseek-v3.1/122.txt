Unified Renderable Interface
============================

The unified renderable interface provides a consistent API for rendering components
in django_cradmin. It includes base classes for standard CSS class handling and
BEM (Block Element Modifier) methodology.

Helper Functions
----------------

.. py:function:: join_css_classes_list(css_classes_list)

   Join the provided list of CSS classes into a string.

   :param list css_classes_list: List of CSS class names
   :returns: Space-separated string of CSS classes
   :rtype: str

AbstractRenderable
------------------

.. py:class:: AbstractRenderable()

   An abstract class that implements an interface for rendering something.
   Everything is just helpers for the :meth:`render` method, which renders a 
   template with an object of this class as input.

   .. py:attribute:: template_name

      The default value for :meth:`get_template_names`. Should be set to a valid
      template path or overridden in subclasses.

   .. py:method:: get_template_names()

      Get the template name(s) for :meth:`render`.

      Defaults to :attr:`template_name`.

      :raises NotImplementedError: If :attr:`template_name` is not set.
      :returns: Template name or list of template names
      :rtype: str or list

   .. py:method:: get_context_data(request=None)

      Get context data for :meth:`render`.

      Defaults to: ``{'me': self}``

      :param request: HttpRequest object (optional)
      :returns: Context dictionary for template rendering
      :rtype: dict

   .. py:method:: render(request=None, extra_context_data=None)

      Render :meth:`get_template_names` with the context returned by 
      :meth:`get_context_data`.

      :param request: HttpRequest object (optional)
      :param extra_context_data: Additional context data to merge (optional)
      :returns: Rendered template content
      :rtype: str

AbstractRenderableWithCss
-------------------------

.. py:class:: AbstractRenderableWithCss()

   Extends :class:`AbstractRenderable` with a unified API for setting CSS classes.

   .. py:method:: get_base_css_classes_list()

      Get base CSS classes for the component.

      :returns: List of base CSS class names
      :rtype: list

   .. py:method:: get_extra_css_classes_list()

      Get extra CSS classes for the component.

      :returns: List of additional CSS class names
      :rtype: list

   .. py:method:: get_css_classes_list()

      Override this to define CSS classes for the component. Must return a list 
      of CSS classes. Combines base and extra CSS classes by default.

      :returns: Complete list of CSS class names
      :rtype: list

   .. py:method:: get_test_css_class_suffixes_list()

      List of CSS class suffixes to include when running automatic tests. These 
      suffixes are filtered through the 
      :func:`~django_cradmin.templatetags.cradmin_tags.cradmin_test_css_class` 
      template tag.

      :returns: List of test CSS class suffixes
      :rtype: list

   .. py:property:: css_classes

      Get CSS classes as a string. Joins :meth:`get_css_classes_list` into a 
      space-separated string. Includes test CSS classes when 
      ``DJANGO_CRADMIN_INCLUDE_TEST_CSS_CLASSES`` setting is enabled.

      :returns: Space-separated CSS classes string
      :rtype: str

   .. py:method:: get_css_classes_string()

      .. deprecated:: Use the :attr:`css_classes` property instead.

      Get CSS classes as a string.

      :returns: Space-separated CSS classes string
      :rtype: str

AbstractBemRenderable
---------------------

.. py:class:: AbstractBemRenderable(bem_block=None, bem_element=None, bem_variant_list=None, extra_css_classes_list=None)

   Base class for renderables that uses BEM (http://getbem.com/) for their CSS 
   class naming. This is an alternative to :class:`AbstractRenderableWithCss` 
   that makes it more natural to work with BEM methodology.

   :param str bem_block: The BEM block name (cannot be used with bem_element)
   :param str bem_element: The BEM element name (must contain '__', cannot be used with bem_block)
   :param list bem_variant_list: List of BEM variants (just the part after '--')
   :param list extra_css_classes_list: List of extra CSS classes

   .. py:method:: get_test_css_class_suffixes_list()

      List of CSS class suffixes to include when running automatic tests.

      :returns: List of test CSS class suffixes
      :rtype: list

   .. py:property:: bem_block_or_element

      Returns :meth:`get_bem_block` falling back to :meth:`get_bem_element`.

      :returns: BEM block or element name
      :rtype: str

   .. py:method:: get_bem_block()

      Get the BEM block string.

      :returns: BEM block name
      :rtype: str

   .. py:method:: get_bem_element()

      Get the BEM element string.

      :returns: BEM element name
      :rtype: str
