Gaussian Processes
=================

.. currentmodule:: sklearn.gaussian_process

Gaussian Processes (GP) are a powerful supervised learning method designed to solve regression and probabilistic classification problems. They are particularly useful for modeling complex, non-linear relationships in data while providing uncertainty estimates for predictions.

Kernel Engineering
------------------

The Gaussian process implementation supports kernel engineering through the :mod:`sklearn.gaussian_process.kernels` module. Kernels can be combined using:

- The ``+`` operator for additive combinations
- The ``*`` operator for multiplicative combinations  
- The ``**`` operator for exponentiation with scalars

Scalar values are automatically converted to constant kernels when used in these operations.

All kernels support analytic gradient-based hyperparameter optimization. The hyperparameter search space can be defined by specifying lower and upper bounds for each parameter (creating a rectangular search space), or hyperparameters can be declared as "fixed" to exclude them from optimization.

Hyperparameter Specification
----------------------------

The :class:`kernels.Hyperparameter` class provides a structured way to define kernel hyperparameters:

.. autoclass:: kernels.Hyperparameter
   :members:
   
   .. attribute:: name
      :type: str
      
      The name of the hyperparameter. Kernels using a hyperparameter named "x" must have attributes ``self.x`` and ``self.x_bounds``.

   .. attribute:: value_type
      :type: str
      
      Currently only "numeric" hyperparameters are supported.

   .. attribute:: bounds
      :type: pair of floats >= 0 or "fixed"
      
      Lower and upper bounds for the parameter. For vector-valued parameters, 1D arrays may be specified.

   .. attribute:: n_elements
      :type: int, default=1
      
      Number of elements in the hyperparameter value (1 for scalar, >1 for vector-valued).

   .. attribute:: fixed
      :type: bool
      
      Whether the hyperparameter value is fixed during optimization.

Base Kernel Class
-----------------

All kernels inherit from the abstract base class:

.. autoclass:: kernels.Kernel
   :members: get_params

      Get parameters for this kernel.
      
      :param deep: If True, returns parameters for contained subobjects
      :type deep: bool, default=True
      :returns: Parameter names mapped to their values
      :rtype: dict

Utility Functions
-----------------

.. autofunction:: kernels._check_length_scale

   Validates length scale parameters for anisotropic kernels, ensuring proper dimensionality matching with input data.
