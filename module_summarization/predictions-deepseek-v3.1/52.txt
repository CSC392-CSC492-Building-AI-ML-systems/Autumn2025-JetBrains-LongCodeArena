Generalized Method of Moments (GMM) and Instrumental Variables (IV2SLS)
======================================================================

Overview
--------

This module provides implementations of Generalized Method of Moments (GMM) and Two-Stage Least Squares (2SLS) for instrumental variables estimation. The module includes core estimation classes and supporting functionality for moment-based econometric models.

Main Classes
------------

IV2SLS
~~~~~~

The ``IV2SLS`` class implements instrumental variables estimation using Two-Stage Least Squares.

Parameters:
    - ``endog``: Endogenous variable (1D or 2D array, nobs × 1)
    - ``exog``: Explanatory variables (1D or 2D array, nobs × k)
    - ``instrument``: Instrumental variables (must include both non-instrumented exog variables and instruments)

Notes:
    - All variables in ``exog`` are treated as instrumented in calculations
    - Standard errors use degrees of freedom correction: ``df_resid = nobs - k_vars``
    - Returns a ``RegressionResults`` instance with parameter estimates and covariance

Key Methods:
    - ``fit()``: Performs 2SLS IV regression estimation
    - ``predict(params, exog=None)``: Returns linear predicted values from design matrix

Implementation Details
----------------------

The module includes several important implementation considerations:

Current Issues:
    - Parameter counting (nparams) and starting values handling
    - Optimal weighting matrix calculation limitations in certain cases (e.g., DistQuantilesGMM)
    - Need for identity matrix or analytical weighting matrix options
    - Integration with underlying model structures (regression, panel data models)

Unresolved Questions:
    - Degrees of freedom calculation in Hausman tests
    - J-test normalization and scaling
    - Standard error calculation for single-iteration fits

Dependencies
------------

- numpy
- scipy (optimize, stats)
- statsmodels base components (Model, LikelihoodModel)
- statsmodels regression modules
- statsmodels statistical tools
