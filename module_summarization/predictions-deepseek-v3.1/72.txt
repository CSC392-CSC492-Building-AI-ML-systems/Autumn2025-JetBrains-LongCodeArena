========
Metrics
========

This module provides functions to compute pairwise and paired distances and kernels between sets of data points. It includes utilities for validating input arrays and efficient computation of various distance metrics.

Distance Metrics
----------------

The module includes implementations for various distance metrics such as:

- Euclidean distance
- Manhattan distance
- Cosine distance
- Minkowski distance
- And many others

Kernel Functions
----------------

Kernel functions implemented include:

- Linear kernel
- Polynomial kernel
- RBF kernel
- Sigmoid kernel
- Chi-squared kernel

Efficient Computation
---------------------

The module uses optimized Cython implementations and parallel processing where applicable to ensure efficient computation of pairwise distances, especially for large datasets.

Parameters Validation
---------------------

Input arrays are rigorously validated using utility functions to ensure:
- Proper data types (float32 or float64)
- Consistent dimensions between arrays
- Handling of sparse matrices
- Management of missing and infinite values
