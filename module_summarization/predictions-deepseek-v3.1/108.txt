========

Polynomial Factorization Routines in Characteristic Zero
-------------------------------------------------------

This module provides polynomial factorization routines for polynomials over domains of characteristic zero, including integers and algebraic number fields.

Classes
-------

.. class:: _Factor

   Mixin class providing factorization methods for polynomial rings.

   .. method:: _trial_division(f, factors)
      
      Performs trial division of polynomial ``f`` by a list of factors.
      
      :param f: polynomial to factor
      :param factors: list of candidate factors
      :returns: list of (factor, multiplicity) pairs sorted by _sort_factors

   .. method:: _aa_factor_trager(f)
      
      Factors multivariate polynomials over algebraic number fields using Trager's algorithm.
      
      :param f: polynomial over algebraic number field
      :returns: tuple (leading coefficient, list of factors with multiplicities)

   .. method:: _zz_factor(f)
      
      Factors (non square-free) polynomials in ℤ[X]. Computes complete factorization into irreducibles over integers using a combination of:
      
      - Primitive part extraction
      - Square-free decomposition
      - Zassenhaus or Enhanced Extended Zassenhaus (EEZ) algorithm for univariate polynomials
      - Wang's algorithm for multivariate polynomials
      - Trial division for multiplicity recovery
      
      Returns tuple (content, [(factor₁, multiplicity₁), ..., (factorₙ, multiplicityₙ)])
      
      Examples:
      
      >>> _, x = ring('x', ZZ)
      >>> (2*x**4 - 2).factor_list()
      (2, [(x - 1, 1), (x + 1, 1), (x² + 1, 1)])
      
      >>> _, x, y = ring('x y', ZZ) 
      >>> (2*x² - 2*y²).factor_list()
      (2, [(x - y, 1), (x + y, 1)])

   .. method:: _zz_mignotte_bound(f)
      
      Computes Mignotte bound for multivariate polynomials in ℤ[X].
      
      :param f: multivariate polynomial
      :returns: bound value for coefficient sizes

   .. method:: _zz_zassenhaus(f)
      
      Factors primitive square-free univariate polynomials in ℤ[x] using Zassenhaus's algorithm.
      
      :param f: primitive square-free univariate polynomial
      :returns: list of irreducible factors

Algorithms
----------

The module implements several advanced factorization algorithms:

- **Trager's algorithm**: For algebraic number fields
- **Zassenhaus algorithm**: For univariate integer polynomials
- **Wang's algorithm**: For multivariate integer polynomials  
- **Hensel lifting**: Used for lifting modular factorizations
- **Mignotte bound**: Provides coefficient bounds for factorization

Dependencies
------------

- ``itertools``: For combinatorial operations
- ``math``: For mathematical functions
- ``factorint``, ``isprime``, ``nextprime``: From ntheory module
- ``symmetric_residue``: From modular arithmetic module
- Various polynomial error classes
- ``_sort_factors``: Utility function for factor sorting

Configuration
-------------

The factorization behavior can be configured through the query system:

- ``USE_IRREDUCIBLE_IN_FACTOR``: Whether to use irreducibility testing
- ``USE_CYCLOTOMIC_FACTOR``: Whether to use cyclotomic factorization
