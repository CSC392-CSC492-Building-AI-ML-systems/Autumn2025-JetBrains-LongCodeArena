===================
Create Cradmin Menu System
===================

Overview
========
The Cradmin menu system provides a flexible framework for creating and rendering navigation menus in Django CRadmin applications. The system includes base classes for menu items, link renderables, and expandable menus with support for BEM (Block Element Modifier) CSS methodology.

Core Components
===============

BaseMenuLinkRenderable
----------------------
The foundation for all menu link items.

**Parameters:**
- ``label``: Text displayed in the menu
- ``url``: Destination URL when clicked
- ``is_active``: Boolean indicating if item should be styled as active
- ``bem_variant_list``: List of BEM variants
- ``parent_bem_block``: Parent BEM block name

**Methods:**
- ``get_fallback_parent_bem_block()``: Returns 'adminui-page-header' by default
- ``get_parent_bem_block()``: Returns specified or fallback parent block
- ``get_bem_element()``: Generates BEM element name
- ``get_active_bem_variant_list()``: Returns ['active'] variant list
- ``get_bem_variant_list()``: Returns combined BEM variant list

Menu Item Types
---------------

NavLinkItemRenderable
~~~~~~~~~~~~~~~~~~~~~
Standard navigation link item for main menu.

NavLinkButtonItemRenderable
~~~~~~~~~~~~~~~~~~~~~~~~~~~
Navigation link styled as a button. Adds 'button' BEM variant.

ExpandableMenuItem
~~~~~~~~~~~~~~~~~~
Specialized link item for expandable menus.

**Features:**
- Uses template: ``django_cradmin/crmenu/menuitem/expandable-menu-item.django.html``
- Custom BEM block: 'expandable-menu'
- Additional method: ``get_list_element_bem_element()``

ExpandableMenuSeparator
~~~~~~~~~~~~~~~~~~~~~~~
Separator item for expandable menus.

**Features:**
- Uses template: ``django_cradmin/crmenu/menuitem/expandable-menu-separator.django.html``
- BEM element: 'expandable-menu__separator'

MenuToggleItemItemRenderable
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Toggle item for expandable menus.

**Features:**
- Uses template: ``django_cradmin/crmenu/menuitem/menutoggle.django.html``
- Translated label: "Menu" (context: 'cradmin default header menu toggle label')
- Methods for label and icon CSS classes

AbstractMenuRenderable
----------------------
Base class for creating custom menu renderables.

**Configuration:**
- ``link_renderable_class``: Class for link items
- ``separator_renderable_class``: Class for separator items  
- ``button_renderable_class``: Class for button items

**Methods:**
- ``iter_renderables()``: Returns iterator over menu items
- ``get_link_renderable_class()``: Returns link renderable class
- ``get_separator_renderable_class()``: Returns separator renderable class

Utility Functions
=================

get_default_expandable_menu_renderable()
----------------------------------------
Retrieves the default expandable menu renderable configured via Django settings.

**Parameters:**
- ``**kwargs``: Arguments for menu class constructor

**Returns:**
AbstractMenuRenderable instance or None if not configured

**Settings:**
- ``DJANGO_CRADMIN_DEFAULT_EXPANDABLE_MENU_CLASS``: Dot path to menu class

Templates
=========
The system uses Django templates located in ``django_cradmin/crmenu/menuitem/``:

- ``link.django.html``: Base link template
- ``expandable-menu-item.django.html``: Expandable menu item template  
- ``expandable-menu-separator.django.html``: Separator template
- ``menutoggle.django.html``: Menu toggle template

Internationalization
====================
Supports Django translation framework through ``pgettext_lazy`` for menu labels and text.
