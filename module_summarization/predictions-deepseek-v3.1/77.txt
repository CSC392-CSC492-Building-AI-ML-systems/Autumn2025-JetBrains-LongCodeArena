Transforms
==========

The transforms module provides common image transformations for both PIL Images and tensors. This module includes utility functions for image manipulation, conversion, and dimension extraction.

Interpolation Modes
-------------------

The module defines several interpolation modes for image transformations:

.. autoclass:: InterpolationMode
   :members:

Available interpolation methods include:
- ``NEAREST`` - Nearest neighbor interpolation
- ``NEAREST_EXACT`` - Exact nearest neighbor interpolation  
- ``BILINEAR`` - Bilinear interpolation
- ``BICUBIC`` - Bicubic interpolation
- ``BOX`` - Box interpolation (PIL compatibility)
- ``HAMMING`` - Hamming interpolation
- ``LANCZOS`` - Lanczos interpolation

Utility Functions
-----------------

.. autofunction:: get_dimensions
   :short-name: Returns image dimensions as [channels, height, width]

.. autofunction:: get_image_size
   :short-name: Returns image size as [width, height]

.. autofunction:: get_image_num_channels
   :short-name: Returns the number of image channels

Conversion Functions
--------------------

.. autofunction:: to_tensor
   :short-name: Converts PIL Image or numpy array to tensor

   This function converts images to tensors and normalizes pixel values to the default float dtype.
   For PIL Images, it handles various modes including "I", "I;16", "F", and "1".

.. autofunction:: pil_to_tensor
   :short-name: Converts PIL Image to tensor with same type

   .. note::
        Performs a deep copy of the underlying array.

Examples
--------

Basic usage examples:

.. code-block:: python

    from torchvision import transforms
    from PIL import Image
    import numpy as np
    
    # Load an image
    img = Image.open("image.jpg")
    
    # Get image dimensions
    channels, height, width = transforms.get_dimensions(img)
    print(f"Dimensions: {channels}C x {height}H x {width}W")
    
    # Convert to tensor
    tensor_img = transforms.to_tensor(img)
    print(f"Tensor shape: {tensor_img.shape}")
    print(f"Tensor dtype: {tensor_img.dtype}")
    
    # Using interpolation modes
    from torchvision.transforms import InterpolationMode
    print(f"Available modes: {list(InterpolationMode)}")

Note: Some functions in this module do not support torchscript and will raise appropriate warnings when used in scripting contexts.
