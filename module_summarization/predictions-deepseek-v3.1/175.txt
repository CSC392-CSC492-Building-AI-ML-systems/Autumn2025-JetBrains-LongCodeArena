Biclustering
===========

Biclustering algorithms simultaneously cluster rows and columns of a data matrix.
These algorithms can be used for tasks such as identifying gene groups that have
similar behavior under specific conditions in gene expression data.

Mathematical Formulation
------------------------

Biclustering aims to find submatrices where rows exhibit similar behavior across
columns, or vice versa. Given a data matrix A with n rows and m columns, a
biclustering algorithm tries to find a set of row clusters R and column clusters C
such that the submatrix A[R, C] has some desired property (e.g., constant values,
constant rows, or constant columns).

Example Usage
-------------

.. code-block:: python

   import numpy as np
   from sklearn.cluster import SpectralBiclustering

   X = np.array([[1, 1], [2, 1], [1, 0],
                 [4, 7], [3, 5], [3, 6]])
   model = SpectralBiclustering(n_clusters=2, random_state=0)
   model.fit(X)

   print(model.row_labels_)
   print(model.column_labels_)
