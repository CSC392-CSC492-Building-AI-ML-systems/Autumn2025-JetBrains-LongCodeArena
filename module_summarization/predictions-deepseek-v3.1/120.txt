Channel Options
===============

The Altair library provides a set of channel options that allow for detailed customization of visual encodings in Vega-Lite specifications. These options are available through various classes and functions that control how data is mapped to visual properties.

Available Channel Options
-------------------------

The following channel options are available for customizing visual encodings:

**Position Channels**
- ``X``: Horizontal position encoding
- ``Y``: Vertical position encoding

**Mark Property Channels**
- ``Color``: Color encoding for marks
- ``Opacity``: Transparency level encoding
- ``Size``: Size encoding for marks
- ``Shape``: Shape encoding for point marks
- ``StrokeDash``: Stroke dash pattern encoding

**Text Channels**
- ``Text``: Text content encoding
- ``Tooltip``: Tooltip content encoding

**Hyperlink Channel**
- ``Href``: URL encoding for hyperlinks

**Level of Detail Channel**
- ``Detail``: Grouping encoding for detailed views

**Order Channel**
- ``Order``: Order encoding for line and area marks

**Facet Channels**
- ``Row``: Row faceting for trellis plots
- ``Column``: Column faceting for trellis plots

**Interactive Channels**
- ``Href``: Interactive hyperlink encoding

Channel Option Properties
-------------------------

Each channel option supports various properties that can be configured:

- ``field``: The data field to encode
- ``type``: The data type (quantitative, ordinal, nominal, temporal)
- ``title``: Custom title for the axis or legend
- ``axis``: Axis configuration properties
- ``legend``: Legend configuration properties
- ``scale``: Scale configuration properties
- ``sort``: Sorting order for the encoding
- ``bin``: Binning configuration for quantitative data
- ``timeUnit``: Time unit transformation for temporal data
- ``aggregate``: Aggregation function for the data
- ``condition``: Conditional encoding rules

Usage Example
-------------

.. code-block:: python

    import altair as alt
    from vega_datasets import data
    
    chart = alt.Chart(data.cars()).mark_point().encode(
        x=alt.X('Horsepower:Q', title='Engine Horsepower'),
        y=alt.Y('Miles_per_Gallon:Q', scale=alt.Scale(zero=False)),
        color=alt.Color('Origin:N', legend=alt.Legend(title='Country of Origin')),
        size=alt.Size('Weight_in_lbs:Q', legend=None),
        tooltip=['Name:N', 'Horsepower:Q', 'Miles_per_Gallon:Q']
    )
    
    chart.display()

This example demonstrates how multiple channel options can be combined to create a rich visualization with customized axes, colors, sizes, and interactive tooltips.

Note: The channel options interface is part of Altair's declarative visualization grammar and follows the Vega-Lite specification standards.
