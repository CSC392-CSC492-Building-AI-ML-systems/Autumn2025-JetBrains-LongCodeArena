.. _gaussian_process_kernels:

Kernels
=======

The :mod:`sklearn.gaussian_process.kernels` module implements several kernels that can be combined to create powerful and flexible kernel functions for Gaussian process regression and classification.

Overview
--------

Kernels in this module support kernel engineering through algebraic operations:
- Addition via the ``+`` operator
- Multiplication via the ``*`` operator
- Exponentiation with a scalar via the ``**`` operator

These operations can include scalar values, which are automatically converted to constant kernels. All kernels support analytic gradient-based hyperparameter optimization with configurable search spaces.

Base Classes
------------

.. autoclass:: Hyperparameter
   :members:
   :inherited-members:
   
   A namedtuple subclass defining kernel hyperparameter specifications with attributes:
   - ``name``: The hyperparameter name
   - ``value_type``: Currently only "numeric" supported
   - ``bounds``: Lower/upper bounds or "fixed"
   - ``n_elements``: Number of elements (1 for scalar)
   - ``fixed``: Whether parameter is fixed during optimization

.. autoclass:: Kernel
   :members: get_params
   :inherited-members:
   
   Abstract base class for all kernels providing parameter management functionality.

Utility Functions
-----------------

.. autofunction:: _check_length_scale

   Validates length scale parameter compatibility with input data dimensions.

Notes
-----
- This module is strongly inspired by the kernel module of the George package
- All kernels support anisotropic parameterization through vector-valued length scales
- Hyperparameter search spaces are rectangular (axis-aligned) by design
- Fixed hyperparameters can be excluded from optimization by setting bounds="fixed"

Examples
--------
Basic kernel operations:

>>> from sklearn.gaussian_process.kernels import RBF, ConstantKernel
>>> kernel = ConstantKernel(1.0) + RBF(length_scale=2.0) * ConstantKernel(2.0)

Hyperparameter inspection:

>>> kernel = RBF(length_scale=1.0, length_scale_bounds=(0.1, 10.0))
>>> for hyperparam in kernel.hyperparameters:
...     print(hyperparam)
