Auto Augment Module
===================

The ``auto_augment`` module provides implementations of automatic data augmentation policies from the AutoAugment paper (https://arxiv.org/abs/1805.09501). These policies are designed to automatically apply a series of augmentations to image and video data in a way that improves model performance on various computer vision tasks.

Functions
---------

.. autofunction:: nvidia.dali.auto_aug.auto_augment.auto_augment

.. autofunction:: nvidia.dali.auto_aug.auto_augment.auto_augment_image_net

Overview
--------

The module contains predefined augmentation policies for various datasets:

- ``image_net``: Full ImageNet policy from the AutoAugment paper
- ``reduced_image_net``: Reduced ImageNet policy (contains no translation operations)
- ``svhn``: SVHN dataset policy
- ``reduced_cifar10``: Reduced CIFAR-10 policy

Key Features
------------

- Supports both images (HWC layout) and videos (FHWC layout) with uint8 data type
- Configurable translation magnitude based on either absolute pixels or relative image dimensions
- Customizable padding values and interpolation methods
- Seed-based reproducibility for random operations

Usage Example
-------------

.. code-block:: python

    import nvidia.dali as dali
    from nvidia.dali.auto_aug import auto_augment
    
    # Apply ImageNet policy to input data
    augmented = auto_augment.auto_augment(
        data=input_batch,
        policy_name='image_net',
        shape=(224, 224),
        fill_value=128,
        seed=42
    )

Note
----

This module requires numpy to be installed for its examples and internal operations.
