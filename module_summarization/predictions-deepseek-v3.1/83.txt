   .. note::
      This engine extends :class:`PostgresEngine <piccolo.engine.postgres.PostgresEngine>` and requires the ``asyncpg`` package.

   **Configuration**:
   
   The engine accepts a configuration dictionary with standard PostgreSQL connection parameters (host, port, user, password, database).

   **Warning**:
   
   If the database user lacks sufficient privileges to set cluster settings, certain CockroachDB functionality may not work as expected.
