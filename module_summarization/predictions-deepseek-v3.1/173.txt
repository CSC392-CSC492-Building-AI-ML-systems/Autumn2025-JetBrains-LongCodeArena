Fitting SESANS Data
===================

Introduction
------------
Spin-Echo Small Angle Neutron Scattering (SESANS) is a technique that measures the correlation function in real space directly. This document describes how to fit SESANS data using the modeling framework.

Parameter Listing
-----------------
The system provides a utility to list all parameters used across different models. This can be helpful when preparing SESANS fits to understand which parameters are available.

To list all parameter names:

.. code-block:: bash

    python -m sasmodels.list_pars

To list parameters along with the models that use them:

.. code-block:: bash

    python -m sasmodels.list_pars -v

Parameters can be filtered by type (volume, orientation, sld, etc.):

.. code-block:: bash

    python -m sasmodels.list_pars [volume|orientation|sld|none|any]

Coordinate Transformation
-------------------------
The fitting process may involve coordinate transformations between latitude-longitude and Guyou map coordinates using elliptic functions. The implementation uses:

- `ellipticJi()` for complex arguments
- Special function handling via SciPy's `ellipj` and `ellipkinc`
- Numerically stable evaluation for real and imaginary components

Example usage:

.. code-block:: python

    from sasmodels import ellipticJi
    sn, cn, dn = ellipticJi(u, v, m)

Note: The coordinate transformation functions are implemented with NumPy for efficient array operations and include special case handling for real and imaginary inputs.

Fitting Procedure
-----------------
1. Load SESANS data and appropriate model
2. Set initial parameters and constraints
3. Perform fit using maximum likelihood or least squares minimization
4. Validate results using appropriate goodness-of-fit metrics

See the individual model documentation for specific parameter definitions and usage guidelines.
