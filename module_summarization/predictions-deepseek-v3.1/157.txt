nextcord: A Python wrapper for the Discord API
==============================================

Introduction
------------

nextcord is a modern, easy-to-use, feature-rich, and async-ready API wrapper for Discord written in Python.

Key Features
------------

- Modern Pythonic API using ``async``/``await`` syntax
- Proper rate limit handling
- Command extension to aid with bot creation
- Type annotations throughout the codebase for better development experience

Widget Support
--------------

nextcord provides comprehensive support for Discord widgets through several key classes:

WidgetChannel
~~~~~~~~~~~~~
Represents a partial widget channel with basic information including:

- Channel ID
- Channel name
- Channel position

WidgetMember
~~~~~~~~~~~~
Represents a partial member visible in the widget with attributes such as:

- User information (ID, username, discriminator, bot status)
- Online status and activity
- Voice state information (deafened, muted, suppressed)
- Connected voice channel

Widget
~~~~~~
Represents a guild widget containing:

- Guild ID and name
- Accessible voice channels
- Online members (offline members are not shown due to Discord limitations)

Getting Started
---------------

To start using nextcord, install it via pip:

.. code-block:: bash

    pip install nextcord

Example usage:

.. code-block:: python

    import nextcord
    from nextcord.ext import commands

    bot = commands.Bot(command_prefix='!')

    @bot.event
    async def on_ready():
        print(f'Logged in as {bot.user}')

    bot.run('your_token_here')
