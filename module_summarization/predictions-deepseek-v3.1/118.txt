.. _Configuration:

Configuration
=============

The Configuration module provides classes and methods for managing import and export options for various design elements in PyAEDT. It allows users to customize which components are included when exporting or importing design configurations.

Classes
-------

ConfigurationsOptions
^^^^^^^^^^^^^^^^^^^^^

The ``ConfigurationsOptions`` class defines a set of boolean flags and properties that control the export and import behavior of different design elements. Each property can be enabled or disabled to include or exclude specific components during configuration operations.

Properties
~~~~~~~~~~

- **object_mapping_tolerance** (float): Gets or sets the tolerance value used in object mapping (e.g., for boundaries).
- **export_variables** (bool): If ``True``, variables are exported to the JSON file. Default is ``True``.
- **export_setups** (bool): If ``True``, setups are exported to the JSON file. Default is ``True``.
- **export_optimizations** (bool): If ``True``, optimizations are exported to the JSON file. Default is ``True``.
- **export_parametrics** (bool): If ``True``, parametrics are exported to the JSON file. Default is ``True``.
- **export_boundaries** (bool): If ``True``, boundaries are exported to the JSON file. Default is ``True``.
- **export_mesh_operations** (bool): If ``True``, mesh operations are exported to the JSON file. Default is ``True``.
- **export_coordinate_systems** (bool): If ``True``, coordinate systems are exported to the JSON file. Default is ``True``.
- **export_materials** (bool): If ``True``, materials are exported to the JSON file. Default is ``True``.
- **export_object_properties** (bool): If ``True``, object properties are exported to the JSON file. Default is ``True``.
- **export_datasets** (bool): If ``True``, datasets are exported to the JSON file. Default is ``True``.
- **import_datasets** (bool): If ``True``, datasets are imported from the JSON file. Default is ``True``.
- **import_variables** (bool): If ``True``, variables are imported from the JSON file. Default is ``True``.
- **import_setups** (bool): If ``True``, setups are imported from the JSON file. Default is ``True``.
- **import_optimizations** (bool): If ``True``, optimizations are imported from the JSON file. Default is ``True``.
- **import_parametrics** (bool): If ``True``, parametrics are imported from the JSON file. Default is ``True``.
- **import_boundaries** (bool): If ``True``, boundaries are imported from the JSON file. Default is ``True``.
- **import_mesh_operations** (bool): If ``True``, mesh operations are imported from the JSON file. Default is ``True``.
- **import_coordinate_systems** (bool): If ``True``, coordinate systems are imported from the JSON file. Default is ``True``.
- **import_materials** (bool): If ``True``, materials are imported from the JSON file. Default is ``True``.
- **import_object_properties** (bool): If ``True``, object properties are imported from the JSON file. Default is ``True``.
- **skip_import_if_exists** (bool): If ``True``, skips import if the component already exists. Default is ``False``.

Each property includes a setter method to update its value and a getter method to retrieve the current value.

Helper Functions
----------------

_find_datasets
^^^^^^^^^^^^^^

A helper function that recursively searches through a dictionary or ordered dictionary to find and extract dataset references (specifically those containing "pwl" in their "free_form_value"). The extracted dataset names are appended to the provided output list.

Usage Example
-------------

To customize export options:

.. code-block:: python

    from pyaedt import Hfss
    hfss = Hfss()
    hfss.configurations.options.export_variables = False  # Disable variables export
    hfss.configurations.options.export_setups = False     # Disable setups export

This allows fine-grained control over which design components are included in configuration import/export operations.
