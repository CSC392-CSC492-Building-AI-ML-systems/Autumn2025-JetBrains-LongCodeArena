.. _multiclass:

Multiclass and multioutput algorithms
=====================================

This module implements multioutput regression and classification algorithms that extend single output estimators to handle multiple target variables.

Overview
--------

The estimators provided in this module are meta-estimators that require a base estimator to be provided in their constructor. These meta-estimators enable single output estimators to handle multioutput problems by employing strategies such as fitting one estimator per target variable or using classifier chains.

Base Class
----------

The module provides an abstract base class ``_MultiOutputEstimator`` that serves as the foundation for the multioutput estimators. This class ensures that all subclasses implement the required interface and provides common functionality.

Implementation Details
---------------------

The module includes utility functions for fitting and partial fitting estimators:

- ``_fit_estimator``: Clones and fits an estimator with optional sample weights
- ``_partial_fit_estimator``: Handles partial fitting with optional class information
- ``_available_if_estimator_has``: Helper function for checking attribute availability in sub-estimators

Dependencies
------------

The multioutput module depends on several core scikit-learn components:

- Base classes and mixins from ``sklearn.base``
- Model selection utilities from ``sklearn.model_selection``
- Various utility functions for validation, multiclass handling, parallel processing, and metadata routing

The module is designed to work with any estimator that implements the standard scikit-learn interface with ``fit`` and ``predict`` methods.

Note: This module extends the capabilities of single output estimators to handle multi-target problems, making it possible to solve complex regression and classification tasks with multiple dependent variables.
