Isotonic Regression
-------------------

Isotonic regression fits a non-decreasing (or non-increasing) function to data. It is a powerful nonparametric technique for transforming data to satisfy monotonicity constraints.

Examples
--------

Using `isotonic_regression` to fit a simple monotonic function:

>>> from sklearn.isotonic import isotonic_regression
>>> y = [3, 7, 5, 8, 9, 6]
>>> result = isotonic_regression(y)
>>> result
array([3., 5., 5., 6., 6., 6.])

Using the `IsotonicRegression` estimator:

>>> from sklearn.isotonic import IsotonicRegression
>>> import numpy as np
>>> X = np.arange(len(y))
>>> ir = IsotonicRegression().fit(X, y)
>>> ir.predict([0.5, 1.5, 2.5])
array([3.5, 4.5, 5.5])
