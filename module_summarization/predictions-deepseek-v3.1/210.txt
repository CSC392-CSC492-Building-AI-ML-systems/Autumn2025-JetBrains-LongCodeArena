============

Empirical Likelihood Inference
==============================

Overview
--------
The `emplike` module provides empirical likelihood-based inference for descriptive statistics, including hypothesis tests and confidence intervals for the mean, variance, skewness, kurtosis, and correlation. If matplotlib is installed, it also supports multivariate confidence region plots and mean-variance contour plots.

Key Features
------------
- Univariate and multivariate descriptive statistic inference
- Hypothesis testing and confidence interval construction
- Visualization capabilities for multivariate confidence regions and mean-variance contours

References
----------
- Owen, A. (2001). "Empirical Likelihood." Chapman and Hall

Classes
-------

_OptFuncts
~~~~~~~~~~
Internal class containing optimization functions for empirical likelihood calculations.

.. autoclass:: emplike._OptFuncts
   :members:
   :private-members:

   Methods
   -------
   _log_star(eta, est_vect, weights, nobs)
        Transforms log probabilities in terms of Lagrange multipliers.

   _hess(eta, est_vect, weights, nobs)
        Calculates the Hessian for weighted empirical likelihood problems.

   _grad(eta, est_vect, weights, nobs)
        Calculates the gradient for weighted empirical likelihood problems.

   _modif_newton(eta, est_vect, weights)
        Modified Newton's method for maximizing the log 'star' equation.

Technical Notes
---------------
- The module uses estimating equations and Lagrange multipliers for optimization.
- Nuisance parameters are profiled out during optimization.
- The `_fit_newton` optimizer from `statsmodels` is used for numerical optimization.

Dependencies
------------
- numpy
- scipy.optimize
- scipy.stats (chi2, skew, kurtosis)
- statsmodels.base.optimizer._fit_newton
- statsmodels.graphics.utils
- itertools (for combinatorial utilities)

Visualization
-------------
Multivariate confidence region plots and mean-variance contour plots require matplotlib.
