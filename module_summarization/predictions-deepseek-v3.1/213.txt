Nonparametric Methods
====================

This module provides tools for nonparametric estimation, including kernel density estimation for univariate data.

Univariate Kernel Density Estimation
------------------------------------

The `KDEUnivariate` class implements univariate kernel density estimation using various kernel functions and bandwidth selection methods.

Class Reference
---------------

.. autoclass:: KDEUnivariate
   :members:
   :inherited-members:

The `KDEUnivariate` class supports several kernel types:

- Gaussian ("gau")
- Epanechnikov ("epa") 
- Uniform ("uni")
- Triangular ("tri")
- Biweight ("biw")
- Triweight ("triw")
- Cosine ("cos")
- Cosine2 ("cos2")

Bandwidth selection methods include:

- "scott" - Scott's rule of thumb
- "silverman" - Silverman's rule of thumb  
- "normal_reference" - Normal reference rule
- Custom float values
- Callable functions

The implementation supports both FFT-based computation (faster, but only for Gaussian kernel) and direct computation methods.

Examples
--------

.. code-block:: python

    import statsmodels.api as sm
    import matplotlib.pyplot as plt
    
    # Generate sample data
    nobs = 300
    np.random.seed(1234)
    data = np.random.normal(size=nobs)
    
    # Create and fit KDE
    dens = sm.nonparametric.KDEUnivariate(data)
    dens.fit()
    
    # Plot results
    plt.plot(dens.support, dens.density)
    plt.title('Kernel Density Estimate')
    plt.show()
