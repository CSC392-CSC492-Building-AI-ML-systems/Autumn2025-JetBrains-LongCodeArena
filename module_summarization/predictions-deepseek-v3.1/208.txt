```rst
Utilities for Developers
========================

This module provides compatibility fixes and utility functions for handling differences between Python, NumPy, and SciPy versions.

Compatibility Functions
-----------------------

ThreadPool Control
~~~~~~~~~~~~~~~~~~

.. autofunction:: sklearn.utils.fixes.threadpool_limits
.. autofunction:: sklearn.utils.fixes.threadpool_info

These functions provide compatibility with threadpoolctl >= 3.0.0, allowing global threadpool controller management.

NumPy Compatibility
~~~~~~~~~~~~~~~~~~~

.. autofunction:: sklearn.utils.fixes.percentile

Provides backward compatibility for NumPy's percentile function, handling the transition from `interpolation` to `method` parameter in NumPy >= 1.22.

SciPy Compatibility
~~~~~~~~~~~~~~~~~~~

Sparse Matrix Containers
^^^^^^^^^^^^^^^^^^^^^^^^

The module defines container lists for various sparse matrix types that include both traditional sparse matrices and newer sparse arrays (available from SciPy 1.8+):

- ``CSR_CONTAINERS``
- ``CSC_CONTAINERS``
- ``COO_CONTAINERS``
- ``LIL_CONTAINERS``
- ``DOK_CONTAINERS``
- ``BSR_CONTAINERS``

Optimization Functions
^^^^^^^^^^^^^^^^^^^^^^

.. autofunction:: sklearn.utils.fixes._sparse_linalg_cg

Provides compatibility wrapper for ``scipy.sparse.linalg.cg`` to handle parameter changes between SciPy versions.

Statistical Functions
^^^^^^^^^^^^^^^^^^^^^

.. autofunction:: sklearn.utils.fixes._mode

Compatibility wrapper for ``scipy.stats.mode`` with proper handling of ``keepdims`` parameter across SciPy versions.

Python Importlib Resources Backport
-----------------------------------

.. autofunction:: sklearn.utils.fixes._open_text
.. autofunction:: sklearn.utils.fixes._open_binary
.. autofunction:: sklearn.utils.fixes._read_text
.. autofunction:: sklearn.utils.fixes._path

These functions provide backward compatibility for Python's importlib.resources functionality, supporting both pre-3.9 and post-3.9 Python versions.

Deprecated Functions
--------------------

.. autofunction:: sklearn.utils.fixes.delayed

Note: This function has been moved to ``sklearn.utils.parallel`` and is deprecated in this location.

Helper Functions
----------------

.. autofunction:: sklearn.utils.fixes._object_dtype_isnan

Utility function for detecting NaN values in object dtype arrays.

Version Information
-------------------

The module maintains version information for NumPy and SciPy to enable version-specific compatibility fixes:

- ``np_version``: Parsed NumPy version
- ``sp_version``: Parsed SciPy version
- ``sp_base_version``: Base version of SciPy (without pre/post-release tags)

Note: When adding new compatibility fixes, please specify the package version at which the fix is no longer needed.
```
