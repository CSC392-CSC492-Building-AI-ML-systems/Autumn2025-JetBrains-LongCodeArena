Quantile Regression
==================

The ``QuantReg`` class implements quantile regression using iterated reweighted least squares (IRLS) for parameter estimation. The asymptotic covariance matrix is estimated using kernel density estimation methods.

Model Overview
--------------
Quantile regression models the relationship between a set of predictor variables and specific quantiles of the response variable. Unlike ordinary least squares regression that models the conditional mean, quantile regression provides a more complete view of the conditional distribution of the response variable.

The Least Absolute Deviation (LAD) estimator is a special case where the quantile is set to 0.5.

Key Features
------------
- Supports estimation for any quantile between 0 and 1
- Multiple kernel options for asymptotic covariance estimation
- Several bandwidth selection methods
- Heteroskedasticity-robust standard errors
- Convergence control with maximum iterations and tolerance parameters

Parameters
----------
**endog** : array or dataframe
    Endogenous/response variable

**exog** : array or dataframe  
    Exogenous/explanatory variable(s)

Fit Method Parameters
---------------------
**q** : float, default 0.5
    Quantile to estimate (must be between 0 and 1)

**vcov** : string, default 'robust'
    Variance-covariance matrix calculation method:
    - 'robust': heteroskedasticity robust standard errors
    - 'iid': iid errors (as in Stata 12)

**kernel** : string, default 'epa'
    Kernel for density estimation:
    - 'epa': Epanechnikov
    - 'cos': Cosine  
    - 'gau': Gaussian
    - 'par': Parzene

**bandwidth** : string, default 'hsheather'
    Bandwidth selection method:
    - 'hsheather': Hall-Sheather (1988)
    - 'bofinger': Bofinger (1975)
    - 'chamberlain': Chamberlain (1994)

**max_iter** : int, default 1000
    Maximum number of iterations

**p_tol** : float, default 1e-6  
    Convergence tolerance
