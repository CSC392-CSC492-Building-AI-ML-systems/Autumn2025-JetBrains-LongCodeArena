# General Plotting Routines

## Overview

This module provides general plotting routines using the Bokeh library for creating interactive visualizations. The routines are built around the `BokehPlotter` class, which manages plot parameters and defaults.

## Default Configuration Management

### Setting and Resetting Defaults

The module provides several functions to manage default plotting parameters:

- `set_bokeh_plot_defaults(**kwargs)`: Set default parameters for Bokeh plots
- `reset_bokeh_plot_defaults()`: Reset parameters to hardcoded defaults
- `show_bokeh_plot_defaults()`: Display current default settings
- `get_bokeh_help(key)`: Get description of a specific parameter
- `load_bokeh_defaults(filename)`: Load defaults from JSON file
- `save_bokeh_defaults(filename, save_complete=False)`: Save defaults to JSON file

## Plotting Functions

### Scatter Plot

`bokeh_scatter(x, y=None, *, xlabel='x', ylabel='y', title='', figure=None, data=None, saveas='scatter', copy_data=False, **kwargs)`

Creates an interactive scatter plot with Bokeh.

**Parameters:**
- `x`: Array-like data or key for x-axis data
- `y`: Array-like data or key for y-axis data
- `data`: Data source (pandas DataFrame, ColumnDataSource, etc.)
- `xlabel`: Label for x-axis (default: 'x')
- `ylabel`: Label for y-axis (default: 'y')
- `title`: Figure title
- `figure`: Existing Bokeh figure to add plot to (optional)
- `saveas`: Output filename base
- `copy_data`: If True, copy the data argument

Additional keyword arguments are passed to the Bokeh scatter function or used to configure plot parameters.

### Multi-Scatter Plot

`bokeh_multi_scatter(x, y=None, *, data=None, figure=None, xlabel='x', ylabel='y', title='', saveas='scatter', copy_data=False, set_default_legend=True, **kwargs)`

Creates an interactive scatter plot supporting multiple datasets.

**Parameters:**
- `x`: Array-like data or key for x-axis data
- `y`: Array-like data or key for y-axis data
- `data`: Data source (pandas DataFrame, ColumnDataSource, etc.)
- `xlabel`: Label for x-axis (default: 'x')
- `ylabel`: Label for y-axis (default: 'y')
- `title`: Figure title
- `figure`: Existing Bokeh figure to add plot to (optional)
- `saveas`: Output filename base
- `copy_data`: If True, copy the data argument
- `set_default_legend`: If True, use data names for legend labels

Additional keyword arguments are passed to the Bokeh scatter function or used to configure plot parameters.

## Data Processing

The module includes data processing utilities through `process_data_arguments()` which handles various data input formats including pandas DataFrames and Bokeh ColumnDataSource objects.

## Features

- Interactive plotting with hover tooltips
- Configurable plot parameters and defaults
- Support for multiple data formats
- Figure customization (labels, titles, limits)
- Export capabilities
- Legend management
- Straight line annotations
- Automatic tooltip generation

## Usage Example

```python
# Simple scatter plot
bokeh_scatter(x_data, y_data, xlabel='X Values', ylabel='Y Values', title='Scatter Plot')

# Multiple datasets with legend
bokeh_multi_scatter('x_column', ['y1', 'y2'], data=df, xlabel='X', ylabel='Y Values')
```

The plotting routines automatically handle data validation, figure preparation, and parameter application through the centralized `BokehPlotter` configuration system.
