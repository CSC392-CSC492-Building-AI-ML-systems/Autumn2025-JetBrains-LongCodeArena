GILDAS CLASS File Reader
===========================

This module provides functionality to read GILDAS CLASS files into a `pyspeckit.spectrum.ObsBlock` object. It supports multiple file versions and formats, handling both single and multiple spectra files with different encoding standards.

File Support
------------
The module recognizes the following file types and versions:

- **File Types**:
  - '1A  ': Multiple_IEEE
  - '1   ': Multiple_Vax
  - '1B  ': Multiple_EEEI
  - '2A  ': v2
  - '2   ': v2
  - '2B  ': v2
  - '9A  ': Single_IEEE
  - '9   ': Single_Vax
  - '9B  ': Single_EEEI

- **File Versions**:
  - '1A  ': v1
  - '2A  ': v2
  - '9A  ': v1 (untested)

Header Structure
----------------
The module parses various header sections identified by specific ID numbers:

- 0: USER CODE
- -1: COMMENT
- -2: GENERAL
- -3: POSITION
- -4: SPECTRO
- -5: BASELINE
- -6: HISTORY
- -7: UNKNOWN-APEX
- -9: GAUSSFIT
- -10: DRIFT
- -11: BEAMSWITCH
- -12: SHELLFIT
- -13: NH3FIT
- -14: CALIBRATION
- -18: ABSFIT

Key Functions
-------------
- `print_timing(func)`: A decorator that prints the execution time of the decorated function. Useful for monitoring read operations on large CLASS files.
- `ensure_bytes(string)`: Ensures the input string is in byte form, handling compatibility between Python 2 and 3.

Data Structure
--------------
The module defines data structures for various header sections:

**GENERAL Section (-2)**:
- UT: UT of observation (rad, float64[2])
- ST: LST of observation (rad, float64[2])
- AZ: Azimuth (rad, float32)
- EL: Elevation (rad, float32)
- TAU: Opacity (neper, float32)
- TSYS: System temperature (K, float32)
- TIME: Integration time (s, float32)

**COMMENT Section (-1)**:
- LTEXT: Length of comment (int32)
- CTEXT: Comment string (|S1024)

Dependencies
------------
- astropy
- numpy
- pyspeckit
- six (for Python 2/3 compatibility)

Note: This implementation includes fallbacks for missing optional dependencies like ProgressBar from astropy.utils.console.
