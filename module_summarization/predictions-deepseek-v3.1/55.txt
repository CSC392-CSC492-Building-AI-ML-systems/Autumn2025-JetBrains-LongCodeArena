`miscmodels.rst`
============

Linear Model with Student-t Distributed Errors
----------------------------------------------

This module provides maximum likelihood estimation for linear and ARMA models with Student-t distributed errors. The Student-t distribution, having heavier tails than the normal distribution, is particularly useful for modeling data with outliers or exhibiting fat-tailed behavior, leading to more robust parameter estimates.

Classes
-------

TLinearModel
~~~~~~~~~~~~

A class for estimating linear regression models with t-distributed errors using maximum likelihood estimation. Inherits from `GenericLikelihoodModel`.

**Parameters**
- `endog` : array_like
    The dependent variable.
- `exog` : array_like
    The independent variables.

**Attributes**
- `k_vars` : int
    Number of independent variables.
- `fix_df` : bool
    Flag indicating whether degrees of freedom are fixed (default is False).
- `k_params` : int
    Total number of parameters to estimate.
- `fixed_params` : array_like or None
    Fixed parameters if any.
- `fixed_paramsmask` : array_like or None
    Mask for fixed parameters.

**Methods**
- `initialize()`: Initializes the model parameters.
- `_set_start_params(start_params=None, use_kurtosis=False)`: Sets starting values for parameters, using OLS estimates by default.
- `loglike(params)`: Returns the log-likelihood.
- `nloglikeobs(params)`: Returns the negative log-likelihood per observation.
- `predict(params, exog=None)`: Makes predictions using the estimated parameters.

**Example**
::

    model = TLinearModel(y, X)
    result = model.fit()
    predictions = model.predict(result.params)

TArma
~~~~~

A class for estimating ARMA models with t-distributed errors. Inherits from `Arma`.

**Parameters**
- Inherits all parameters from `Arma`.

**Methods**
- `loglike(params)`: Returns the log-likelihood.
- `nloglikeobs(params)`: Returns the negative log-likelihood per observation.
- `fit_mle(order, start_params=None, method='nm', maxiter=5000, tol=1e-08, **kwds)`: Fits the model using maximum likelihood estimation.

**Example**
::

    model = TArma(y)
    result = model.fit_mle((1,1))

Notes
-----

- The t-distribution used is the standard (non-standardized) form; hence, the scale parameter does not equate to the standard deviation. The error variance is given by df/(df-2)*scaleÂ².
- Parameter restrictions and fixed parameters can be handled via `fixed_params`, though this may require further testing.
- Starting values for parameters are derived from OLS estimates where applicable.

References
----------

Kenneth L. Lange, Roderick J. A. Little, Jeremy M. G. Taylor (1989). *Robust Statistical Modeling Using the t Distribution*. Journal of the American Statistical Association, 84(408), 881-896.
