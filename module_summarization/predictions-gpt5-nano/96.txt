Generate Getting Started Tutorial
=================================

Introduction
------------

The Generate Getting Started Tutorial utility creates a complete getting-started guide from your codebase. It extracts module-level documentation, per-function/class docstrings, and embedded code examples to assemble a coherent reStructuredText document suitable for distribution and onboarding.

Prerequisites
-------------

- Python 3.x
- Access to the codebase you want to document
- Optional: a pre-existing set of docstrings and inline code examples

What it generates
-----------------

- Introduction and purpose of the library or module
- Quickstart and basic usage steps
- Code examples with explanations
- API references derived from docstrings
- Licensing, copyright headers, and attribution preserved where present

Usage
-----

Examples of how to run the generator:

.. code-block:: console

   $ python -m mitogen.docs.get_started --module mypackage.mymodule --output getting_started.rst

Options (labels may vary by implementation)
-------------------------------------------

- --module: Path to the module or package to document
- --output: Path to write the generated getting_started.rst
- --title: Optional title for the generated document
- --author: Optional author name
- --version: Optional version string

Output structure
----------------

- Introduction
- Prerequisites
- Quick Start
- Examples
- API Reference
- Troubleshooting
- License and Copyright

Notes
