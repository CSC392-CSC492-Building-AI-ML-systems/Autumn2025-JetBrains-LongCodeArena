Linear and Quadratic Discriminant Analysis
===========================================

Linear Discriminant Analysis (LDA) and Quadratic Discriminant Analysis (QDA) are supervised classification techniques that model class-conditional feature distributions as Gaussian densities and use Bayes’ rule to assign class labels.

Key ideas
---------

- LDA assumes that all classes share the same covariance matrix but have different class means.
- QDA allows each class to have its own covariance matrix.

Discriminant functions

- LDA: with shared covariance Σ, class means μk, and priors πk, the discriminant score for class k is
  delta_k(x) = x^T Σ^{-1} μk − 0.5 μk^T Σ^{-1} μk + log(πk).
  The predicted class is argmax_k delta_k(x). The decision boundaries are linear in x.

- QDA: with class-specific covariance Σk, the discriminant score is
  delta_k(x) = −0.5 x^T Σk^{-1} x + x^T Σk^{-1} μk − 0.5 μk^T Σk^{-1} μk + log(πk) − 0.5 log|Σk|.
  The predicted class is argmax_k delta_k(x). The decision boundaries are quadratic in x.

When to use
-----------

- Use LDA when class covariances are similar and a linear decision boundary is appropriate.
- Use QDA when class covariances differ substantially, allowing nonlinear (quadratic) decision boundaries.

In scikit-learn
----------------

- LinearDiscriminantAnalysis: implements LDA (linear decision boundaries).
- QuadraticDiscriminantAnalysis: implements QDA (quadratic decision boundaries).

Parameters

LinearDiscriminantAnalysis
- solver: {'svd', 'lsqr', 'eigen'}, default 'svd'
- shrinkage: float or {'auto'}, default None (shrinks covariance estimate; some solvers require it)
- priors: array-like of shape (n_classes,), optional
- n_components: int or None, optional
- store_covariance: bool, deprecated

QuadraticDiscriminantAnalysis
- priors: array-like of shape (n_classes,), optional
- reg_param: float, default 0.0 (regularization for covariance estimates)
- store_covariance: bool, deprecated

Examples
--------

.. code-block:: python

   from sklearn.discriminant_analysis import LinearDiscriminantAnalysis, QuadraticDiscriminantAnalysis

   # Prepare data: X, y

   # Linear Discriminant Analysis
   lda = LinearDiscriminantAnalysis()
   lda.fit(X, y)
   y_pred = lda.predict(X)

   # Quadratic Discriminant Analysis
   qda = QuadraticDiscriminantAnalysis()
   qda.fit(X, y)
   y_pred_qda = qda.predict(X)

Notes
------

- LDA performs dimensionality reduction consistent with the class structure when used with dimensionality reduction settings.
- QDA requires more parameters (covariance per class) and may require more data to estimate reliably.
- Regularization (via shrinkage or reg_param) can improve performance on small datasets or when features are highly collinear.
