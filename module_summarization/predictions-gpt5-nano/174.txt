Array API support
=================

This project provides Array APIâ€“style interoperability by using a NumPy-centric backend. Array operations accept array-like inputs and operate on NumPy-compatible arrays, enabling consistent behavior across array libraries that implement the Array API semantics.

Minimum versions and dependencies
---------------------------------

- numpy: 1.17.3 (1.19.2 on PyPy)
- scipy: 1.5.0
- joblib: 1.1.1

Notes
-----

- These minimum versions are used by CI to ensure consistent Array API compatibility.
- The dependency mapping is defined as dependent_packages and provides a per-package minimum version and associated tags (e.g., build, install, docs, tests).
- A simple reverse mapping (tag_to_packages) is used to group packages by their tags for CI workflows.
