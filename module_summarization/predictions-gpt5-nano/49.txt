Duration
==========

Module providing survival analysis utilities and cumulative incidence estimation under right-censoring. The implementation relies on numpy, pandas, scipy.stats (chi2, norm), and statsmodels.graphics.utils.

Public API
----------

Functions
---------

- _calc_survfunc_right(time, status, weights=None, entry=None, compress=True, retall=True)
  - Description: Calculate the survival function and its standard error for a single group.
  - Returns: sp, se, utime, rtime, n, d

- _calc_incidence_right(time, status, weights=None)
  - Description: Calculate the cumulative incidence function and its standard error.
  - Returns: ip, se, utime

- _checkargs(time, status, entry, freq_weights, exog)
  - Description: Validate input arguments; raises ValueError on invalid shapes or times.
