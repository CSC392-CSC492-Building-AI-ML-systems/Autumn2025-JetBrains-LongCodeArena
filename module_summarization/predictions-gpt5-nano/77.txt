Module: torchvision.transforms._functional_pil and _functional_tensor utilities

Overview
- This module provides low-level image utilities used by torchvision.transforms. It supports PIL Images, NumPy arrays, and Torch Tensors. It includes an InterpolationMode enum, mappings between enum values and PIL modes, and helpers to convert images to tensors.

InterpolationMode
- Enum of supported interpolation modes:
  - NEAREST = "nearest"
  - NEAREST_EXACT = "nearest-exact"
  - BILINEAR = "bilinear"
  - BICUBIC = "bicubic"
  - BOX = "box" (for PIL compatibility)
  - HAMMING = "hamming"
  - LANCZOS = "lanczos"

Internal helpers and mappings
- _interpolation_modes_from_int(i: int) -> InterpolationMode
  - Converts an integer to the corresponding InterpolationMode.
- pil_modes_mapping
  - Maps InterpolationMode values to PIL mode indices.
- _is_pil_image
  - Internal helper to check if an object is a PIL image (aliased from F_pil._is_pil_image).
- _is_numpy(img: Any) -> bool
  - Internal helper to check if input is a NumPy array.
- _is_numpy_image(img: Any) -> bool
  - Internal helper to check if a NumPy image has 2 or 3 dimensions.

Public helpers
- get_dimensions(img: Tensor) -> List[int]
  - Returns the dimensions of an image as [channels, height, width].
  - Args:
    - img (PIL Image or Tensor): The image to be checked.
  - Returns:
    - List[int]: The image dimensions.
- get_image_size(img: Tensor) -> List[int]
  - Returns the size of an image as [width, height].
  - Args:
    - img (PIL Image or Tensor)
  - Returns:
    - List[int]: The image size.
- get_image_num_channels(img: Tensor) -> int
  - Returns the number of channels of an image.
  - Args:
    - img (PIL Image or Tensor)
  - Returns:
    - int: The number of channels.

- to_tensor(pic) -> Tensor
  - Convert a PIL Image or numpy.ndarray to a tensor.
  - This function does not support torchscript.
  - See ToTensor for more details.
  - Args:
    - pic (PIL Image or numpy.ndarray): Image to be converted to tensor.
  - Returns:
    - Tensor: Converted image.

- pil_to_tensor(pic: Any) -> Tensor
  - Convert a PIL Image to a tensor of the same type.
  - This function does not support torchscript.
  - See PILToTensor for more details.
  - Note: A deep copy of the underlying array is performed.
  - Args:
    - pic (PIL Image): Image to be converted to tensor.
  - Returns:
    - Tensor: Converted image.

Examples
- Convert a PIL Image to a float32 tensor in [0, 1] range:
  from PIL import Image
  from torchvision.transforms import functional as F
  img = Image.new("RGB", (4, 4), color="red")
  t = F.to_tensor(img)
  print(t.shape)  # torch.Size([3, 4, 4])
  print(t.dtype)    # torch.float32

- Convert a NumPy array to a tensor:
  import numpy as np
  from torchvision.transforms import functional as F
  arr = np.zeros((3, 8, 8), dtype=np.uint8)
  t = F.to_tensor(arr)
  print(t.shape)  # torch.Size([3, 8, 8])

- Get image dimensions and size:
  dim = F.get_dimensions(img)     # e.g., [3, 4, 4]
  size = F.get_image_size(img)      # e.g., [4, 4]

- Get number of channels:
  channels = F.get_image_num_channels(img)

- Convert a PIL Image to a tensor using pil_to_tensor:
  t2 = F.pil_to_tensor(img)
