Nextcord Introduction
=====================

Nextcord is a Python library for interacting with the Discord API. It provides an object-oriented interface that mirrors Discord's entities and operations, with a focus on simplicity and speed.

Widget support
--------------

Nextcord includes lightweight representations for the Guild widget data. A widget is a read-only view of a guild suitable for displaying activity and presence information.

WidgetChannel
~~~~~~~~~~~~~

Represents a partial channel in a widget.

Attributes
~~~~~~~~~~

- id: int — The channel's ID.
- name: str — The channel's name.
- position: int — The channel's position.

Methods and properties
~~~~~~~~~~~~~~~~~~~~~~

- __str__(): returns the channel's name.
- __repr__(): returns a readable representation.
- mention: str — The string to mention the channel (e.g. <#12345>).
- created_at: datetime — The channel's creation time in UTC (derived from the ID).

WidgetMember
~~~~~~~~~~~~

Represents a partial member in the widget's guild.

Attributes
~~~~~~~~~~

- id, name, discriminator, bot, status, nick, avatar, activity, deafened, muted, suppress, connected_channel
- display_name: str — Returns the member's display name (nick if available, else username).

Notes
~~~~~

- Inherits from BaseUser.
- activity is created from the widget payload if present.

Widget
~~~~~~

Represents a Guild widget.

Attributes
~~~~~~~~~~

- id: int — The guild's ID.
- name: str — The guild's name.
- channels: List[WidgetChannel] — The accessible channels in the guild.
- members: List[WidgetMember] — The online members visible in the widget.

Operations
~~~~~~~~~~

- Equality / inequality checks.
- str(widget) returns the widget's JSON URL (as described in its docstring).
