Using lists
===========

Lists are a fundamental Python built-in container type. They are ordered, mutable sequences that can hold items of any type.

Creating lists
--------------

- Create an empty list: empty = []
- Create a list with initial items: nums = [1, 2, 3, 4]
- Convert an iterable to a list: items = list(range(5))

Accessing and mutating elements
--------------------------------

- Access by index: first = nums[0]
- Slicing: sub = nums[1:3]
- Replace a slice: nums[1:3] = [9, 9]

Common list methods
-------------------

- append(x): add a single item to the end
- extend(iterable): add multiple items
- insert(i, x): insert x at position i
- remove(x): remove first occurrence of x
- pop([i]): remove and return item at i (default last)
- clear(): remove all items
- index(x): return index of first x
- count(x): number of occurrences of x
- sort(key=None, reverse=False): sort in place
- reverse(): reverse in place
- copy(): shallow copy (list[:], list.copy())

List operations and patterns
---------------------------

- Length: len(l)
- Membership: x in l
- Concatenation: a + b creates a new list
- Repetition: a * n repeats the list

List comprehensions
-------------------

A concise way to build lists:

- squares = [n*n for n in range(10)]
- evens = [n for n in range(20) if n % 2 == 0]

Using lists with other modules
------------------------------

- itertools for advanced iteration patterns
- collections for specialized list-like structures (e.g., deque)

Performance tips
----------------

- Prefer list.append() in loops over repeated string or list concatenation.
- When building a large list, collect items in a list and extend or use comprehensions when possible.

Example snippets
----------------

- Transform a list of names to uppercase:

  ::
  
     names = ['alice', 'bob', 'charlie']
     upper = [n.upper() for n in names]

- Filter a list of numbers:

  ::
  
     nums = [1, 2, 3, 4, 5]
     evens = [n for n in nums if n % 2 == 0]

- Combine two lists:

  ::
  
     a = [1, 2, 3]
     b = [4, 5]
     combined = a + b  # creates a new list
     
- Copying lists:

  ::
  
     original = [1, 2, 3]
     shallow = original.copy()
     # or: shallow = original[:]
