Duration
========

Overview
--------
This module provides utilities for estimating and inferring the cumulative incidence function (CIF) under right-censoring, including support for competing risks. The primary focus is on calculating the all-cause survival function and then deriving cause-specific CIFs along with their standard errors. The implementation includes both internal helper routines and a class-based interface.

Public API
----------

Note: The following items are the public-facing interfaces documented here. Some helpers are intended for internal use within the module (indicated by leading underscores).

- _calc_survfunc_right(time, status, weights=None, entry=None, compress=True, retall=True)
  - Purpose: Calculate the survival function and its standard error for a single group.
  - Parameters:
    - time (array_like): Times (censoring or event times).
    - status (array_like): Event indicators; typically 1 indicates an event, 0 indicates censoring.
    - weights (array_like, optional): Frequency or sampling weights.
    - entry (array_like, optional): Entry times for left-truncated data (delayed entry).
    - compress (bool, optional): If True, retain only times where an event occurred.
    - retall (bool, optional): If True, return all intermediate arrays (utime, rtime, n, d) in addition to the survival function; if False, return only the survival function and the intermediate arrays.
  - Returns:
    - When retall is True: (sp, se, utime, rtime, n, d)
      - sp: Survival probabilities at event times
      - se: Standard errors (Greenwood with weights=None; Tsaiatis with weights provided)
      - utime: Unique event times
      - rtime: Ranks corresponding to original times
      - n: Size of the risk set just prior to each event time
      - d: Number of deaths (events) at each unique time
    - When retall is False: (sp, utime, rtime, n, d)
  - Notes:
    - If entry is provided, left-truncated data are accommodated.
    - Greenwood's variance is used when weights is None; otherwise Tsaiatis’ variance formula is used.

- _calc_incidence_right(time, status, weights=None)
  - Purpose: Calculate the cumulative incidence function (CIF) and its standard error.
  - Parameters:
    - time (array_like): Times (censoring or event times).
    - status (array_like): Event indicators; 0 for censored, 1..J for competing risks.
    - weights (array_like, optional): Frequency or sampling weights.
  - Returns:
    - If weights is None: (ip, se, utime)
      - ip: List of CIF arrays, one per cause (indexed by event type j = 1..ngrp)
      - se: List of standard error arrays corresponding to ip
      - utime: Unique event times
    - If weights is not None: (ip, None, utime)
      - Standard errors are not computed when weights are provided.
  - Notes:
    - All-cause survival function is computed first, then cause-specific contributions are derived.
    - ip[k] corresponds to the k-th cause (k = 0..ngrp-1, which maps to event type k+1).

- _checkargs(time, status, entry, freq_weights, exog)
  - Purpose: Validate input arguments and raise informative errors for inconsistent inputs.
  - Parameters:
    - time (array_like)
    - status (array_like)
    - entry (array_like or None)
    - freq_weights (array_like or None)
    - exog (array_like or None)
  - Behavior:
    - Ensures time and status have the same length.
    - If entry is provided, it must have the same length as time and all entry times must precede event times.
    - If freq_weights is provided, its length must match time.
    - If exog is provided, its number of rows must match the length of time.

- CumIncidenceRight
  - Purpose: Estimation and inference for the cumulative incidence function under right censoring, with optional handling of competing risks and covariates.
  - Parameters (from the class docstring):
    - time (array_like): Times (censoring or event times).
    - status (array_like): If status >= 1, indicates which event occurred at time t; status = 0 indicates censoring.
    - title (str, optional): Title used for plots and summary output.
    - freq_weights (array_like, optional): Frequency weights.
    - exog (array_like, optional): Covariates used to account for potential violation of independent censoring. If provided, kernel-based estimation may utilize bw_factor.
    - bw_factor (float): Bandwidth multiplier for kernel-based estimation. Used when exog is provided.
    - dimred (bool): If True, enable dimension reduction or proportional hazards-style modeling components in estimation (as indicated by the intended broader implementation).
  - Notes:
    - The class is designed to integrate CIF estimation with optional covariate adjustments; detailed public methods (beyond initialization) are not shown in the provided code snippet. The class docstring indicates the purpose and typical usage in a broader survival-analysis workflow.

Usage examples
--------------

Basic usage with internal helpers (illustrative; private helpers):
```python
import numpy as np
from duration import _calc_survfunc_right, _calc_incidence_right

# Example data
time = np.array([5, 8, 3, 10, 7])
status = np.array([1, 0, 1, 2, 0])  # 1 or 2 indicate event types; 0 = censored

# Survival function
sp, se, utime, rtime, n, d = _calc_survfunc_right(time, status, compress=True, retall=True)

# Cumulative incidence function (for competing risks)
ip, se_ip, utime_ip = _calc_incidence_right(time, status)
```

Public usage via CumIncidenceRight (illustrative):
```python
from duration import CumIncidenceRight
import numpy as np

time = np.array([5, 8, 3, 10, 7])
status = np.array([1, 0, 1, 2, 0])

# Initialize estimator
cuminc = CumIncidenceRight(time, status, title="CIF Example")

# Note: The actual public API (e.g., fit/compute/plot methods) is not shown in this snippet.
# This demonstrates intended usage pattern and integration within a larger package.
```

Notes
-----

- The functions _calc_survfunc_right and _calc_incidence_right are helper utilities for internal computations of survival and CIF. They are exposed here for completeness and for demonstrations or advanced usage.
- The implementation uses Greenwood’s formula for standard errors when no weights are supplied, and Tsaiatis’ formula when weights are provided.
- Entry times allow left-truncated data handling, adjusting the risk set accordingly.
