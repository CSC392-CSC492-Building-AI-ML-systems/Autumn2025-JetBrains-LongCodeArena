Isotonic Regression
===================

Overview
--------

This module provides utilities to perform isotonic regression, which enforces a monotonic relationship (increasing or decreasing) in a 1D regression setting. It exposes:

- check_increasing(x, y): Determine whether y is monotonically correlated with x using a Spearman test.
- isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True): Compute the isotonic fit of a 1D target y with optional weights and bounds.
- IsotonicRegression: A estimator/transformer compatible with scikit-learn’s API to fit and predict isotonic monotone functions with optional bounds and handling for out-of-bounds inputs.

Key ideas
---------

- Monotone fitting along a 1D order (the input order is interpreted as the order of X).
- Weights allow giving different importance to samples.
- Optional bounds (y_min, y_max) constrain the isotonic fit.
- The IsotonicRegression class provides a convenient, reusable estimator with fit/predict (and transform) semantics.

API reference
-------------

check_increasing(x, y)
~~~~~~~~~~~~~~~~~~~~~~~

Determine whether y is monotonically correlated with x.

Parameters
- x: array-like of shape (n_samples,)
  Training data.
- y: array-like of shape (n_samples,)
  Training target.

Returns
- increasing_bool: bool
  Whether the relationship is increasing (True) or decreasing (False).

Notes
- Uses Spearman correlation to assess monotonic relationship.
- If the 95% confidence interval of the Spearman estimate spans zero, a warning may be issued.

References
- Fisher transformation (for CI): Wikipedia

isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True)
-------------------------------------------------------------------------------------

Solve the isotonic regression model.

Parameters
- y: array-like of shape (n_samples,)
  The data to fit.
- sample_weight: array-like or None, default=None
  Weights on each point. If None, weights are all 1.
- y_min: float or None, default=None
  Lower bound on the lowest predicted value. If None, defaults to -inf.
- y_max: float or None, default=None
  Upper bound on the highest predicted value. If None, defaults to +inf.
- increasing: bool, default=True
  If True, fit an increasing sequence; if False, fit a decreasing sequence.

Returns
- y_: ndarray of shape (n_samples,)
  Isotonic fit of y.

Notes
- The implementation uses an in-place, contiguous isotonic regression algorithm.
- If bounds are supplied, the result is clipped to [y_min, y_max].

References
- Best, M. J., & Chakravarti, N. (Active set algorithms for isotonic regression)

IsotonicRegression
------------------

Isotonic regression model.

Version
- .. versionadded:: 0.13

Parameters
- y_min: float, default=None
  Lower bound on the lowest predicted value (minimum value may be higher). Defaults to -inf if not set.

- y_max: float, default=None
  Upper bound on the highest predicted value (maximum may be lower). Defaults to +inf if not set.

- increasing: bool or 'auto', default=True
  Determines whether predictions should increase with X. 'auto' will decide based on the sign of the Spearman correlation estimate.

- out_of_bounds: {'nan', 'clip', 'raise'}, default='nan'
  How to handle X values outside the training domain during prediction:
  - 'nan': predictions will be NaN
  - 'clip': predictions are clipped to the nearest train interval endpoint
  - 'raise': raise a ValueError

Attributes
- X_min_: float
  Minimum value of input array X_ for the left bound.

- X_max_: float
  Maximum value of input array X_ for the right bound.

- X_thresholds_: ndarray of shape (n_thresholds,)
  Unique ascending X values used to interpolate the monotonic function y = f(X).

  .. versionadded:: 0.24

- y_thresholds_: ndarray
  Values corresponding to the thresholds (monotone fit values).

Usage examples
--------------

Basic isotonic regression using the functional API
-------------------------------------------------

Compute an increasing isotonic fit for a 1D target y.

.. code-block:: python

    import numpy as np
    from sklearn.isotonic import isotonic_regression

    y = np.array([3.0, 1.0, 2.0, 4.0])
    y_iso = isotonic_regression(y, increasing=True)

    print("Isotonic fit:", y_iso)

With sample weights and bounds
--------------------------------

.. code-block:: python

    import numpy as np
    from sklearn.isotonic import isotonic_regression

    y = np.array([3.0, 1.0, 2.0, 4.0])
    w = np.array([1.0, 2.0, 1.0, 1.0])

    y_iso = isotonic_regression(y, sample_weight=w, y_min=-1.0, y_max=3.0)
    print("Isotonic fit with bounds:", y_iso)

IsotonicRegression estimator usage
----------------------------------

Use the estimator API to fit and predict monotone functions from 1D or 2D input X.

.. code-block:: python

    import numpy as np
    from sklearn.isotonic import IsotonicRegression

    # Example data
    X = np.array([[0.0], [1.0], [2.0], [3.0]])
    y = np.array([3.0, 1.0, 2.0, 4.0])

    # Create the model (auto-detect monotonic direction)
    model = IsotonicRegression(increasing='auto', y_min=None, y_max=None)

    # Fit and predict
    model.fit(X, y)
    y_pred = model.predict(X)

    print("Predicted:", y_pred)

Notes
-----

- check_increasing is a utility to assess whether the relationship between x and y appears increasing or decreasing based on Spearman correlation. It can be useful to initialize IsotonicRegression with increasing=False when the data are negatively correlated.
- The isotonic regression implementation relies on an optimized internal routine (_inplace_contiguous_isotonic_regression) for efficiency.

Developer notes
---------------

- This module exposes a minimal, NumPy/scipy-backed isotonic regression suitable for 1D monotone fits, with optional bounds and out-of-bounds handling during prediction.
- It is designed to be compatible with scikit-learn’s API, including fit/transform/predict semantics via BaseEstimator and mixins.
