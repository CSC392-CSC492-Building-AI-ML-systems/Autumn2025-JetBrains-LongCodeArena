Optimization
===============
This module provides general-purpose optimization utilities that can be used with any model fitting task. The goal is to decouple optimization routines from a specific LikelihoodModel so the same solvers and interfaces can be reused across different modeling contexts.

Overview
--------
- General fit infrastructure for any model with an objective function.
- Decoupled from LikelihoodModel to maximize reusability.
- Wraps around SciPy optimization routines while exposing a consistent interface.
- Supports multiple solvers (e.g., Newton, Nelder-Mead, BFGS, CG, Basin-hopping, and a general minimize wrapper).

Public API
----------

_check_method(method, methods)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Validate that a given optimization method is supported.

Parameters
- method: str
  The name of the optimization method to validate.
- methods: iterable of str
  The set of supported method names.

Returns
- None

Raises
- ValueError
  If method is not in the supported methods.

Optimizer
---------
A lightweight helper that provides a generic _fit interface for fitting any model with an objective function.

_fit(objective, gradient, start_params, fargs, kwargs, hessian=None, method='newton',
     maxiter=100, full_output=True, disp=True, callback=None, retall=False)
--------------------------------------------------------------------------------------------------
Fit a model with an objective function using a chosen optimization solver.

Description
- This method performs a generic optimization/fitting procedure for models described by an objective function.
- The design purpose is to be reusable for any model, not tied to a particular LikelihoodModel implementation.
- It delegates the actual optimization to SciPy solvers (via an internal wrapper) and returns the optimum parameters along with solver output.

Parameters
- objective: callable
  The objective function to be optimized (typically the negative log-likelihood or similar).
- gradient: callable
  The gradient (Jacobian) of the objective function.
- start_params: array_like, optional
  Initial guess for the parameters.
- fargs: tuple
  Extra arguments passed to the objective function: loglike(x, *fargs).
- kwargs: dict
  Additional keyword arguments passed to the objective function.
- hessian: callable, optional
  Hessian (or Hessian-vector product) information if available; may be None.
- method: str, default 'newton'
  Optimization method to use (e.g., 'newton', 'nm', 'bfgs', 'powell', 'cg',
  'ncg', 'basinhopping', or 'minimize').
- maxiter: int, default 100
  Maximum number of iterations (where supported by the solver).
- full_output: bool, default True
  If True, return all available solver output in the results object.
- disp: bool, default True
  If True, print convergence messages.
- callback: callable, optional
  Called after each iteration with the current parameter vector.
- retall: bool, default False
  If True, return the list of parameter vectors for each iteration.

Returns
- xopt: array
  The optimized parameter vector.
- retvals: dict or None
  If full_output is True, a dictionary with solver-specific information; otherwise None.
- optim_settings: dict
  A dictionary containing the settings/arguments passed to the solver.

Notes
- The basinhopping solver ignores maxiter, retall, and full_output explicit arguments.
- The exact explicit arguments available depend on the chosen solver; refer to SciPy optimize documentation for details.
- Supported solvers include:
  - 'newton' for Newton-Raphson
  - 'nm' for Nelder-Mead
  - 'bfgs' for BFGS
  - 'powell' for Powell's method
  - 'cg' for conjugate gradient
  - 'ncg' for Newton-conjugate gradient
  - 'basinhopping' for global basin-hopping
  - 'minimize' as a wrapper around scipy.optimize.minimize

Implementation Notes
- This module exposes a simple, solver-agnostic interface suitable for reuse across different models.
- The _fit method is designed to be flexible and allow plugging in different objective/gradient definitions, as well as optional Hessian information.

Examples
--------

Example 1: Simple quadratic objective using Newton method
- Goal: minimize f(x) = (x - 3)^2
- Gradient: f'(x) = 2*(x - 3)

.. code-block:: python

   import numpy as np
   from optimization import Optimizer

   def objective(x, a=None):
       # Example: square loss with a shift
       return np.sum((x - 3.0)**2)

   def gradient(x, a=None):
       return 2.0 * (x - 3.0)

   opt = Optimizer()
   start = np.array([0.0])
   xopt, retvals, settings = opt._fit(
       objective=objective,
       gradient=gradient,
       start_params=start,
       fargs=(),
       kwargs={}
   )

   print("Optimal x:", xopt)
   print("Solver settings:", settings)

Example 2: Using minimize wrapper
- Demonstrates usage with a generic minimize-based approach.

.. code-block:: python

   import numpy as np
   from optimization import Optimizer

   def objective(x, data=None):
       return (x - 2.0)**2

   def gradient(x, data=None):
       return 2.0*(x - 2.0)

   opt = Optimizer()
   start = np.array([0.0])
   xopt, retvals, settings = opt._fit(
       objective=objective,
       gradient=gradient,
       start_params=start,
       fargs=(),
       kwargs={}
       , method='minimize'
   )

   print("Optimal x:", xopt)
