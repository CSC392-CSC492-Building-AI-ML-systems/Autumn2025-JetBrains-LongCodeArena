Random Projection transformers
=============================

A simple and computationally efficient way to reduce the dimensionality of the data
-------------------------------------------------------------------------------

Random Projections provide a lightweight approach to dimensionality reduction by trading a controlled amount of accuracy for faster processing and smaller model sizes. The projection preserves pairwise distances between samples with high probability, enabling efficient downstream tasks such as clustering, nearest-neighbor search, and classification.

Key idea
--------
- Build a random projection matrix P with controlled distribution (e.g., Gaussian or sparse Achlioptas-style).
- Project data X (shape: n_samples × n_features) to Y = X · Pᵀ (shape: n_samples × n_components).
- By the Johnson-Lindenstrauss lemma, the distances between samples are preserved up to a factor (1 ± ε) with high probability when n_components is large enough.

Theory
------
The Johnson-Lindenstrauss lemma guarantees that a small set of points in high-dimensional space can be embedded into a lower-dimensional space so that pairwise distances are nearly preserved. The projection can be implemented via a random matrix with entries drawn from a suitable distribution. The number of components required depends on the number of samples and the desired distortion ε.

Formula (informal)
- For a dataset with n_samples and target distortion ε ∈ (0, 1), a projection to
  n_components ≥ 4 log(n_samples) / (ε² / 2 − ε³ / 3)
  suffices to achieve an ε-embedding with good probability.

API overview
------------
The module exposes the following public API:

- GaussianRandomProjection
  - A dense Gaussian random projection matrix with entries drawn from N(0, 1 / n_components).
  - Suitable for general-purpose dimensionality reduction with good distance preservation properties.

- SparseRandomProjection
  - A sparse random projection matrix with controlled density (default or user-specified).
  - More memory- and compute-efficient, especially for very high-dimensional data.

- johnson_lindenstrauss_min_dim(n_samples, eps)
  - Utility to compute a safe minimum number of components for a given dataset size and distortion.
  - Returns an int or an array of ints when inputs are array-like.

Key parameters
-------------
- n_components
  - Target dimensionality after projection. Must be a positive integer.

- density (for SparseRandomProjection)
  - Controls the sparsity of the random projection matrix.
  - Can be a float in (0, 1] or the string "auto".
  - "auto" defaults to 1 / sqrt(n_features), providing a principled sparsity level.

- random_state
  - Seed or RandomState for reproducibility of the random projection.

- eps (for johnson_lindenstrauss_min_dim)
  - Maximum distortion in the range (0, 1).
  - Smaller eps yields higher n_components.

- n_samples (for johnson_lindenstrauss_min_dim)
  - Number of samples in the dataset to be embedded.
  - Can be an int or an array-like of ints for batch computation.

Usage examples
--------------

Example 1: Compute a safe number of components with the JL bound
--------------------------------------------------------------
.. code-block:: python

   from sklearn.random_projection import johnson_lindenstrauss_min_dim

   # Single value
   n_components = johnson_lindenstrauss_min_dim(1_000_000, eps=0.5)
   print(n_components)  # e.g., 663

   # Array-like eps
   n_components = johnson_lindenstrauss_min_dim(1_000_000, eps=[0.5, 0.1, 0.01])
   print(n_components)  # e.g., [663, 11841, 1112658]

   # Array of sample counts
   n_components = johnson_lindenstrauss_min_dim([10_000, 100_000, 1_000_000], eps=0.1)
   print(n_components)  # e.g., [7894, 9868, 11841]

Example 2: Apply Gaussian random projection
--------------------------------------------
.. code-block:: python

   from sklearn.random_projection import GaussianRandomProjection
   import numpy as np

   X = np.random.randn(1000, 500)  # 1000 samples, 500 features
   rp = GaussianRandomProjection(n_components=200, random_state=42)
   X_rp = rp.fit_transform(X)

   print(X_rp.shape)  # (1000, 200)

Example 3: Apply Sparse random projection
-----------------------------------------
.. code-block:: python

   from sklearn.random_projection import SparseRandomProjection
   import numpy as np

   X = np.random.randn(1000, 500)  # 1000 samples, 500 features
   rp = SparseRandomProjection(n_components=200, density="auto", random_state=0)
   X_rp = rp.fit_transform(X)

   print(X_rp.shape)  # (1000, 200)

Notes and best practices
------------------------
- Random projections are especially beneficial when:
  - The original feature space is very high-dimensional.
  - You need fast, memory-efficient dimensionality reduction with acceptable loss of information.
- For very large datasets, SparseRandomProjection can offer substantial speedups and memory savings due to sparsity in the projection matrix.
- The number of components required grows with the dataset size; the JL bound provides a data-dependent guidance.
