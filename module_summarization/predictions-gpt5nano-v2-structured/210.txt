Empirical likelihood for descriptive statistics
-----------------------------------------------


Overview
========

This module provides empirical likelihood (EL) inference for descriptive statistics.
It enables hypothesis testing and confidence interval construction for:

- mean
- variance
- skewness
- kurtosis
- correlation

In addition, if matplotlib is available, the package can generate:

- multivariate confidence region plots, and
- mean–variance contour plots

Key idea
--------

Empirical likelihood uses estimating equations to form a nonparametric likelihood
based on observed data. It optimizes the Lagrange-multiplier (eta) that enforces
the estimating equations under a weighted empirical distribution. The core numeric
engine solves for eta using a modified Newton method, computing the EL-based
test statistics and confidence regions without assuming a specific parametric form
for the data-generating process.

Public API
----------

DescStat(endog)
    Factory that builds an inference object for descriptive statistics via
    empirical likelihood.

    Parameters
    ----------
    endog : ndarray
        Data array. If 1-D, a univariate analysis is assumed and a DescStatUV
        object is returned. If multi-dimensional (k > 1), a DescStatMV is
        returned for multivariate analysis.

    Returns
    -------
    DescStat instance
        - If endog is 1-D, returns a univariate instance (DescStatUV).
        - If endog has more than one column, returns a multivariate instance (DescStatMV).

Internal optimization utilities
-------------------------------

The module implements an internal helper class _OptFuncts that encapsulates
the optimization machinery used by empirical likelihood. It is intended for
advanced usage or for educational purposes; end users should rely on the
DescStat factory and the public analysis routines it exposes.

Class
-----
_OptFuncts
    A container for functions that set up and solve the estimating equations
    via empirical likelihood.

    Methods (selected)

    - _log_star(eta, est_vect, weights, nobs)
        Transforms the log of observation probabilities in terms of the Lagrange
        multiplier eta to a log-star form used in the optimization. This is a
        intermediate representation; the function is primarily used by the solver.

    - _hess(eta, est_vect, weights, nobs)
        Hessian of the weighted empirical likelihood problem with respect to eta.
        Used by the Newton-based solver.

    - _grad(eta, est_vect, weights, nobs)
        Gradient of the weighted empirical likelihood problem with respect to
        eta. Used by the solver.

    - _modif_newton(eta, est_vect, weights)
        Modified Newton method to maximize the log-star objective. This routine
        finds the maximizing eta (Lagrange multiplier) given the estimating equations
        and weights.

    - _find_eta(eta)
        Placeholder hook that can be used to locate eta as part of the solver
        workflow (internal use).

Notes
-----

- The methods rely on scipy.optimize and statsmodels’ Newton-based solver
  (via _fit_newton) for robust convergence.
- The design follows the standard EL framework for estimating equations:
  maximize the empirical likelihood subject to the estimating equations holding
  in the weighted empirical distribution.

Usage examples
--------------

Example 1: Basic univariate EL optimization (internal utility)

This illustrates how to exercise the internal optimization routine with a small
synthetic dataset. Note that in typical usage you would interact with the
DescStat interface rather than the internal _OptFuncts directly.

```python
import numpy as np
from emplike import _OptFuncts

# Sample data
x = np.array([1.0, 2.0, 3.0])
weights = np.ones_like(x)

# Estimating equations vector (example: centered data)
# Shape: (n, k) where k is number of estimating equations
est_vect = (x - x.mean()).reshape(-1, 1)

# Build optimizer and run modified Newton to get eta
opt = _OptFuncts(x.reshape(-1, 1))
eta0 = np.zeros((1, 1))

eta_opt = opt._modif_newton(eta0, est_vect, weights)
print("Optimized eta:", eta_opt)
```

Example 2: Basic usage pattern (conceptual)

1. Prepare data (univariate or multivariate).
2. Use DescStat(endog) to obtain an inference object for the descriptive statistic of interest.
3. Use the EL framework to compute likelihood-ratio statistics and derive
   confidence intervals or hypothesis tests for the statistic of interest.
4. If available, generate plots (requires matplotlib) for multivariate regions
   or mean–variance contours.

Note: The public API focused on end-user workflows is exposed via DescStat and
its accompanying high-level methods. The _OptFuncts class and its methods are
internal utilities used by those workflows.

Implementation details
======================

Key concepts
------------

- Estimating equations: For a target descriptive statistic (mean, variance,
  skewness, kurtosis, correlation), the EL approach constructs a vector of
  estimating equations that must be satisfied by the reweighted empirical
  distribution.
- Lagrange multipliers: The optimization problem introduces a Lagrange multiplier
  vector eta that enforces the estimating equations under the empirical likelihood.
- Log-star transformation: A numerically stable reformulation of the log-likelihood
  used inside the optimization routines.
- Modified Newton optimization: A Newton-based solver (_modif_newton) refines eta
  to maximize the log-likelihood subject to the estimating equations.
