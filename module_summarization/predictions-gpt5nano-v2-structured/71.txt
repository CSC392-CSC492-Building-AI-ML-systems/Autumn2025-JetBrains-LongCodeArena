Kernel Ridge Regression
=======================

Module: sklearn.kernel_ridge
----------------------------

This module implements kernel ridge regression (KRR), which combines ridge regression (L2-regularized least squares) with the kernel trick. KRR learns a linear function in the feature space induced by the chosen kernel, enabling non-linear relationships in the original space.

Key features
- Handles multi-output regression (y can be a 1D or 2D array)
- Supports a variety of kernels (linear, polynomial, RBF, etc.) or a user-supplied callable
- Optional precomputed kernel matrix via kernel="precomputed"
- Closed-form solution for efficient fitting on medium-sized datasets

Getting started
---------------

Basic usage (single target)
.. code-block:: python

    from sklearn.kernel_ridge import KernelRidge
    import numpy as np

    rng = np.random.RandomState(0)
    X = rng.randn(100, 5)
    y = rng.randn(100)

    krr = KernelRidge(alpha=1.0, kernel="rbf", gamma=0.1)
    krr.fit(X, y)
    y_pred = krr.predict(X)

Multi-target regression
.. code-block:: python

    Y = rng.randn(100, 3)  # 3 targets
    krr = KernelRidge(alpha=[1.0, 0.5, 0.2], kernel="rbf", gamma=0.1)
    krr.fit(X, Y)
    Y_pred = krr.predict(X)

Using a precomputed kernel
.. code-block:: python

    # Compute the kernel matrix in advance
    from sklearn.metrics.pairwise import pairwise_kernels
    K = pairwise_kernels(X, X, metric="rbf", gamma=0.1)

    krr = KernelRidge(alpha=1.0, kernel="precomputed")
    krr.fit(K, y)
    y_pred = krr.predict(K)

Parameters
-----------

The KernelRidge class exposes the following parameters (as documented in the class docstring):

- alpha: float or array-like of shape (n_targets,), default=1.0
  Regularization strength; controls the trade-off between fitting the data and keeping the model coefficients small.

- kernel: str or callable, default="linear"
  Kernel mapping used internally. If a string, it must be one of the PAIRWISE_KERNEL_FUNCTIONS or "precomputed".
  If "precomputed", X is treated as a kernel matrix. If a callable, it is invoked on each pair of samples.

- gamma: float, default=None
  Kernel coefficient for certain kernels (e.g., RBF, polynomial, sigmoid). Ignored by other kernels.

- degree: int, default=3
  Degree of the polynomial kernel. Ignored by other kernels.

- coef0: float, default=1
  Independent term in polynomial and sigmoid kernels.

- kernel_params: dict, default=None
  Additional keyword arguments passed to the kernel function when kernel is a callable or a string key.

Notes
-----

- The estimator supports multi-output regression. If y is 2D, a separate set of dual coefficients is learned for each target.
- For kernel="precomputed", the fit matrix must be the training kernel matrix of shape (n_samples, n_samples).
