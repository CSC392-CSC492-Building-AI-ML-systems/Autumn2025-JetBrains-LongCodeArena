Isotonic regression
===================

Isotonic regression fits a monotone (non-decreasing or non-increasing) function to data under monotonic constraints. It is useful when the target is known to follow a monotonic relationship with the input, but the exact functional form is unknown. This module provides a simple, robust implementation that supports optional weighting and bounds on the predicted values.

Key ideas
---------

- Non-decreasing or non-increasing fits are produced by solving an isotonic regression problem with an efficient active-set algorithm.
- Bounds on predictions can be imposed via y_min and y_max.
- Predictions outside the training domain can be handled in three ways via out_of_bounds: nan, clip, or raise.
- A lightweight helper function is provided to check the monotonic direction using Spearman correlation.

Functions
---------

check_increasing
~~~~~~~~~~~~~~~

Determine whether y is monotonically correlated with x.

- Purpose: Infer whether the relationship between x and y is increasing or decreasing using a Spearman correlation test.
- Parameters:
  - x: array-like of shape (n_samples,)
  - y: array-like of shape (n_samples,)
- Returns:
  - increasing_bool: boolean indicating whether the relationship is increasing (True) or decreasing (False)
- Notes:
  - The sign of the Spearman rho estimate is used to determine the direction.
  - If the 95% confidence interval for Spearman's rho spans zero, a warning is emitted.

isotonic_regression
~~~~~~~~~~~~~~~~~~~

Solve the isotonic regression problem for a 1-D response.

- Purpose: Compute an isotonic fit of y with optional weighting and bounds.
- Parameters:
  - y: array-like of shape (n_samples,)
    The data to fit.
  - sample_weight: array-like of shape (n_samples,) or None
    Weights for each point. If None, equal weights are used.
  - y_min: float or None
    Lower bound on the lowest predicted value. If None, defaults to -inf.
  - y_max: float or None
    Upper bound on the highest predicted value. If None, defaults to +inf.
  - increasing: bool, default True
    If True, compute a non-decreasing fit; if False, compute a non-increasing fit.
- Returns:
  - y_: ndarray of shape (n_samples,)
    Isotonic fit of y.
- Notes:
  - Internally, the function reorders the data to enforce the desired direction, performs the isotonic fit in-place, and then clips to bounds if provided.

IsotonicRegression
~~~~~~~~~~~~~~~~~~

Isotonic regression model (estimator).

- Purpose: Learn a monotone transformation of input X that best fits y under optional bounds and out-of-bounds handling.
- Parameters:
  - y_min: float, default=None
    Lower bound on the lowest predicted value. If not set, defaults to -inf.
  - y_max: float, default=None
    Upper bound on the highest predicted value. If not set, defaults to +inf.
  - increasing: bool or 'auto', default True
    Determines whether the predictions should be constrained to increase or decrease with X.
    - True: non-decreasing predictions
    - False: non-increasing predictions
    - 'auto': decide based on the sign of the Spearman correlation estimate (via check_increasing)
  - out_of_bounds: {'nan', 'clip', 'raise'}, default 'nan'
    How to handle X values outside the training domain during prediction:
    - 'nan': predictions for out-of-domain X are NaN
    - 'clip': predictions are set to the nearest interval endpoint value
    - 'raise': a ValueError is raised for out-of-domain X
- Attributes (examples):
  - X_min_: float
    Minimum value of input X_ used for the left bound.
  - X_max_: float
    Maximum value of input X_ used for the right bound.
  - X_thresholds_: ndarray of shape (n_thresholds,)
    Unique ascending X values used to interpolate the monotone function.
  - y_thresholds_: ndarray
    Thresholds corresponding to the monotone function across the X domain.
- Notes:
  - The estimator follows the standard scikit-learn API: fit(X, y) and predict(X) (and fit_transform/transform via the mixins).
  - The internal implementation uses an in-place, contiguous isotonic regression routine for efficiency.

Example usage
-------------

Quick single-dimensional isotonic regression

.. code-block:: python

   import numpy as np
   from sklearn.isotonic import isotonic_regression

   y = np.array([0.5, 2.0, 1.5, 3.0, 2.5], dtype=float)

   # Non-decreasing isotonic fit
   y_fit = isotonic_regression(y, increasing=True)
   print(y_fit)

Monotone regression with weights and bounds

.. code-block:: python

   import numpy as np
   from sklearn.isotonic import isotonic_regression

   y = np.array([0.2, 0.9, 0.4, 1.2, 0.8], dtype=float)
   w = np.array([1, 2, 1, 1, 1], dtype=float)

   y_fit = isotonic_regression(
       y, sample_weight=w, y_min=0.0, y_max=1.5, increasing=True
   )
   print(y_fit)

Isotonic regression as a model
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: python

   import numpy as np
   from sklearn.isotonic import IsotonicRegression

   X = np.array([0.0, 1.0, 2.0, 3.0, 4.0]).reshape(-1, 1)
   y = np.array([0.5, 1.7, 1.2, 2.5, 2.1])

   model = IsotonicRegression(increasing='auto', out_of_bounds='clip')
   model.fit(X, y)
   y_pred = model.predict(X)
   print(y_pred)

Notes
-----

- The isotonic regression implementation uses an active-set algorithm for efficiency, suitable for 1-D monotone fits.
- The check_increasing function provides a convenient way to determine the monotone direction from data, using Spearman correlation and an optional Fisher transform confidence interval check.
- Bounds (y_min, y_max) can be used to constrain predictions to a desired range.

References
----------

- Best, M. J., & Chakravarti, N. (1990). Algorithms for isotonic regression and related problems.
