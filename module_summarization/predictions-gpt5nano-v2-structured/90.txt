=====

AutoAugment: Predefined Image and Video Augmentations for DALI
--------------------------------------------------------------

Overview
--------

This module provides access to NVIDIA DALI's automatic augmentation (AutoAugment) functionality. It applies one of several predefined augmentation policies to a batch of images or videos, enabling automated and reproducible data augmentation suitable for training computer vision models.

Key concepts
- Policies: Predefined augmentation strategies inspired by the AutoAugment paper (e.g., image_net, reduced_image_net, svhn, reduced_cifar10).
- Translation and shape: Some policies depend on the spatial shape of the input; others do not. The magnitude of certain augmentations can be controlled via shape-related parameters.
- DataNode: All operations operate on DALI DataNode batches. Supported input formats are images in HWC layout and videos in FHWC layout with dtype uint8.
- Determinism: A seed can be provided to make augmentation random sampling deterministic across runs.

API
---

Functions
- auto_augment(data, policy_name='image_net', shape=None, fill_value=128, interp_type=None, max_translate_abs=None, max_translate_rel=None, seed=None) -> _DataNode

  Applies one of the predefined AutoAugment policies to the provided batch.

  Parameters
  - data (DataNode): Batch of samples. Images (HWC) or videos (FHWC). Type uint8.
  - policy_name (str): One of 'image_net', 'reduced_image_net', 'svhn', 'reduced_cifar10'. Defaults to 'image_net'.
  - shape (DataNode or Tuple[int, int], optional): Target height and width. If provided, translation magnitudes are scaled by the shape and max_translate_rel. If omitted, magnitudes use max_translate_abs.
  - fill_value (int, optional): Padding value for warp-based transforms (translation, rotation, shear). If None, border padding is clamped.
  - interp_type (types.DALIInterpType, optional): Interpolation method for warp-based ops. Typically INTERP_LINEAR or INTERP_NN.
  - max_translate_abs (int or (int, int), optional): Maximal pixel shift for translation when shape is not provided. Tuple specifies (height, width).
  - max_translate_rel (float or (float, float), optional): Maximal relative shift for translation when shape is provided. Tuple specifies (height, width).
  - seed (int, optional): Seed for random sampling of operations and magnitudes.

  Returns
  - DataNode: Transformed batch.

  Notes
  - If policy_name corresponds to a translation-free policy (e.g., 'reduced_image_net'), shape-related arguments must not be provided.
  - Invalid policy_name raises an exception with guidance on allowed values.

- auto_augment_image_net(data, shape=None, fill_value=128, interp_type=None, max_translate_abs=None, max_translate_rel=None, seed=None) -> _DataNode

  Convenience wrapper that applies the 'image_net' policy explicitly. Equivalent to:
  auto_augment(data, 'image_net', shape, fill_value, interp_type, max_translate_abs, max_translate_rel, seed)

  Parameters and return type are identical to auto_augment, with policy_name fixed to 'image_net'.

Usage examples
--------------

Basic usage with the default image_net policy
- This applies the standard AutoAugment image_net policy to a batch of images or videos.

.. code-block:: python

    from nvidia.dali import fn
    from nvidia.dali.auto_aug import auto_augment

    # Within a DALI pipeline, assuming 'images' is a data node of HWC uint8 images
    augmented = auto_augment(images, policy_name='image_net')

Deterministic augmentation with a seed
- Use a fixed seed to get deterministic results across runs.

.. code-block:: python

    augmented = auto_augment(images, policy_name='image_net', seed=42)

Using shape-dependent magnitudes
- Enable translation magnitudes that scale with the image shape.

.. code-block:: python

    augmented = auto_augment(images, policy_name='image_net', shape=(224, 224),
                           max_translate_rel=0.5, fill_value=128, seed=1)

Using a policy without translation
- The reduced_image_net policy does not rely on image shape for translation.

.. code-block:: python

    augmented = auto_augment(images, policy_name='reduced_image_net', shape=None,
                           max_translate_abs=None)

Convenience wrapper for image_net
- Equivalent to calling auto_augment with policy_name='image_net'.

.. code-block:: python

    augmented = auto_augment_image_net(images, shape=(256, 256), seed=7)

Supported policies
------------------

- image_net: Standard AutoAugment policy for ImageNet-scale data.
- reduced_image_net: A lighter policy with fewer translation-based augmentations.
- svhn: AutoAugment policy suitable for SVHN-like data.
- reduced_cifar10: A reduced policy tailored for CIFAR-10-like data.

Notes and constraints
---------------------

- Input must be a DataNode batch of uint8 images (HWC) or videos (FHWC).
- If policy_name does not use translation, do not supply shape, max_translate_abs, or max_translate_rel; otherwise an exception will be raised.
- The available policies are defined within the module and may be extended in future releases.

Appendix: Implementation Highlights (for maintainers)
-----------------------------------------------------

- The auto_augment function maps policy_name to builder functions for the corresponding policy.
- Translation-based magnitudes are controlled via shape or absolute values depending on policy.
- Validation enforces policy-shape compatibility and provides descriptive errors for invalid configurations.

This documentation provides a concise reference for using the AutoAugment module to apply predefined augmentation policies to batches of images or videos within NVIDIA DALI.
