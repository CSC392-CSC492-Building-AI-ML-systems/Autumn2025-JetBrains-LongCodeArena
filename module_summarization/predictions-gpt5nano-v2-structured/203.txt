Transforming the prediction target
===================================

Overview
--------
In regression tasks, transforming the target variable y can stabilize variance, improve model fit, or better meet model assumptions. The TransformedTargetRegressor wrapper applies a user-defined transformation to the target during fitting and automatically inverts the transformation for predictions. This allows you to train a model in a more favorable target space while returning predictions in the original scale.

Key features
------------
- Apply a forward transformation to the target y during training
- Automatically apply the inverse transformation to predictions
- Works with any regressor (e.g., LinearRegression, RandomForestRegressor)
- Can be used inside a Pipeline to combine with feature transformers
- Supports multi-output regression (transforms each target independently)
- Optional check_inverse to validate the invertibility of the transformation

Usage
-----
- Basic usage with a log transform to stabilize skewness in the target

.. code-block:: python

    from sklearn.datasets import make_regression
    from sklearn.linear_model import LinearRegression
    from sklearn.compose import TransformedTargetRegressor
    import numpy as np

    # generate data
    X, y = make_regression(n_samples=200, n_features=2, noise=0.1, random_state=42)

    # wrap a regressor with a target transformation
    reg = TransformedTargetRegressor(
        regressor=LinearRegression(),
        func=np.log1p,           # forward transform on y
        inverse_func=np.expm1,    # inverse transform for predictions
        check_inverse=True
    )

    reg.fit(X, y)
    y_pred = reg.predict(X)

- Inverse transform is applied to the regressor's predictions, so y_pred is in the original scale.
- The transformation is only applied to the target, not to the input features X.

- Using with a Pipeline (optional)
  You can place a TransformedTargetRegressor inside a Pipeline alongside feature transformers.

.. code-block:: python

    from sklearn.pipeline import Pipeline
    from sklearn.preprocessing import StandardScaler
    from sklearn.ensemble import RandomForestRegressor
    import numpy as np

    pipe = Pipeline([
        ('scaler', StandardScaler()),
        ('trans_target', TransformedTargetRegressor(
            regressor=RandomForestRegressor(n_estimators=200, random_state=0),
            func=np.sqrt, inverse_func=np.square
        ))
    ])

    pipe.fit(X, y)
    y_pred = pipe.predict(X)

Examples
--------
- When to use:
  - Target y exhibits skewness or heteroscedasticity that can be mitigated by a monotonic transform
  - You prefer modeling in a transformed space but need predictions in the original scale

- Target types:
  - Works for single-target and multi-target regression
  - Each target dimension is transformed independently

Notes
-----
- The chosen func and inverse_func must be invertible over the target domain; non-invertible transformations can lead to undefined predictions.
- If the target contains negatives or zeros, ensure the transformation is defined for those values, or use an appropriate transform (e.g., a shift or a different function).
- If check_inverse is enabled, the implementation will validate that applying the inverse to the transformed output approximately recovers the original y.

API reference (TransformedTargetRegressor)
------------------------------------------
- regressor: estimator implementing fit(X, y)
- func: callable, transformation applied to y during fitting
- inverse_func: callable, inverse transformation applied to predictions
- check_inverse: bool, whether to check that inverse_func(func(y)) ≈ y
- Other standard estimator methods (fit, predict, score, get_params, set_params)
