Imputation of missing values
============================

This module provides tools for filling in missing values in numerical and categorical data. The primary entry point is SimpleImputer, which supports several common strategies and can optionally add an indicator mask describing where imputations occurred.

Overview
--------

Imputation is the process of replacing missing values with substituted values. SimpleImputer operates on 2D arrays (or array-like structures) and imputes values per feature (column). It supports numeric and non-numeric data, with strategy choices that determine how missing values are filled.

Key concepts
- Per-feature imputation: Each column is imputed independently.
- Flexible missing marker: missing_values can be np.nan by default, but other placeholders (including pandas’ NA) are supported.
- Strategies:
  - mean: replace missing values with the column mean (numeric data only)
  - median: replace missing values with the column median (numeric data only)
  - most_frequent: replace missing values with the most frequent value (works with strings or numbers)
  - constant: replace missing values with a constant value (fill_value)
- Optional indicator: add_indicator to produce a parallel indicator mask that marks where imputations occurred. The indicator can be concatenated to the imputed data.

Estimator API
-------------

SimpleImputer
- missing_values: The placeholder for the missing values. Default: np.nan
- strategy: Imputation strategy. One of 'mean', 'median', 'most_frequent', 'constant'. Default: 'mean'
- fill_value: Used when strategy='constant'. If None, defaults to 0 for numeric data and "missing_value" for non-numeric data.
- copy: If True, do not modify the input array; if False, may perform imputation in place when possible. Default: True
- add_indicator: If True, a MissingIndicator is learned and used to indicate which values were imputed. Default: False
- keep_empty_features: If True, features consisting entirely of missing values are kept in the output. Default: False

Notes
- Input validation: If X is numeric and missing_values is not numeric, a ValueError is raised. For pandas dataframes with nullable dtypes, missing_values can be np.nan or pd.NA.
- The imputer computes the statistic per feature and applies it independently to each column.
- When add_indicator is enabled, the imputer stores an internal MissingIndicator and can transform inputs to return the augmented array consisting of the imputed data and the indicator features.

Usage patterns
--------------

- Basic numeric imputation with mean:
  - Replace missing values in numeric data with the mean of each column.

  Code example:
  .. code-block:: python

     import numpy as np
     from sklearn.impute import SimpleImputer

     X = np.array([[1.0, 2.0], [np.nan, 3.0], [7.0, np.nan]])
     imputer = SimpleImputer(strategy='mean')
     X_imputed = imputer.fit_transform(X)
     # X_imputed -> array with NaNs replaced by column means

- Categorical data imputation with most_frequent:
  - Replace missing values with the most frequent value in each column.

  Code example:
  .. code-block:: python

     import numpy as np
     from sklearn.impute import SimpleImputer

     X = np.array([['cat', 'red'], ['dog', None], ['cat', 'blue']])
     imputer = SimpleImputer(strategy='most_frequent')
     X_imputed = imputer.fit_transform(X)

- Constant imputation:
  - Fill missing values with a constant value.

  Code example:
  .. code-block:: python

     import numpy as np
     from sklearn.impute import SimpleImputer

     X = np.array([[1, 2], [np.nan, 4], [3, np.nan]])
     imputer = SimpleImputer(strategy='constant', fill_value=-1)
     X_imputed = imputer.fit_transform(X)

- Using an indicator to flag imputations:
  - When add_indicator=True, an additional set of features indicates which entries were imputed.

  Code example:
  .. code-block:: python

     import numpy as np
     from sklearn.impute import SimpleImputer

     X = np.array([[1.0, np.nan], [np.nan, 3.0]])
     imputer = SimpleImputer(strategy='mean', add_indicator=True)
     X_out = imputer.fit_transform(X)
     # X_out contains the imputed data and an appended indicator array

API reference (high level)
--------------------------

- SimpleImputer is a Transformer that can be used directly or inside a Pipeline or ColumnTransformer.
- It supports fit, transform, and fit_transform methods.
- When add_indicator is enabled, you can retrieve the feature names for the augmented output via get_feature_names_out.

Compatibility and notes
-----------------------

- Works with NumPy arrays and common array-like structures.
- For pandas with nullable integer dtypes, missing_values can be np.nan or pd.NA.
- The per-feature approach makes SimpleImputer suitable for a wide range of datasets, including those with mixed numeric and categorical data (using appropriate strategies per feature).

See Also
