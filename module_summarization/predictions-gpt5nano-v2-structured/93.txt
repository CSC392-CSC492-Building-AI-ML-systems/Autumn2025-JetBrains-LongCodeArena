# basics.indexing

Collection of utilities to manipulate structured arrays.

This module provides helpers to create, modify, and query NumPy structured and nested arrays. Originally implemented by John Hunter for Matplotlib and later rewritten and extended for convenience. The utilities support both flat and nested field structures, enabling operations such as field insertion, renaming, dropping, joining, and transformations between structured and unstructured forms.

## Quick start

Importing and basic usage:

```python
import numpy as np
from numpy.lib import recfunctions as rfn

# Example: a simple structured array
a = np.array([(1, 10.), (2, 20.)],
             dtype=[('A', np.int64), ('B', np.float64)])

# Flatten dtype descriptions
flattened = rfn.flatten_descr(a.dtype)

# Get field names (nested support)
names = rfn.get_names(a.dtype)

# Recursively fill an output array with input fields
b = np.zeros((3,), dtype=a.dtype)
rfn.recursive_fill_fields(a, b)
```

Code examples for common use cases are provided in the function docstrings embedded in the source.

## API Reference

The module exposes a collection of utilities to manipulate structured arrays. Public functions (as re-exported in `__all__`) include:

- append_fields
  - Append new fields to a structured array, returning a new array with the additional fields.
- apply_along_fields
  - Apply a function along the fields of a structured array.
- assign_fields_by_name
  - Assign values to fields by name from another array-like source.
- drop_fields
  - Remove specified fields from a structured array.
- find_duplicates
  - Identify duplicate field combinations in a structured array.
- flatten_descr
  - Flatten a nested dtype description into a flat sequence of (name, dtype) pairs.
  - Example:
    - Input: dtype([('a', '<i4'), ('b', [('ba', '<f8'), ('bb', '<i4')])])
    - Output: (('a', dtype('int32')), ('ba', dtype('float64')), ('bb', dtype('int32')))
- get_fieldstructure
  - Retrieve the field structure (useful for introspection of nested dtypes).
- get_names
  - Return field names as a nested tuple reflecting the dtype’s structure.
  - Example: for a dtype with a nested field, returns ('a', ('b', ('ba', 'bb')))
- get_names_flat
  - Return flattened field names as a 1-D tuple, including nested fields.
  - Example: ('a', 'b', 'ba', 'bb')
- join_by
  - Join two arrays by common key fields.
- merge_arrays
  - Merge multiple arrays based on common fields.
- rec_append_fields
  - Append new fields to a record array from separate data.
- rec_drop_fields
  - Drop fields from a record array.
- rec_join
  - Join two recarrays on key fields.
- recursive_fill_fields
  - Fill fields from an input array into an output array, supporting nested structures.
  - Example (from docstring):
    - a = array([(1, 10.), (2, 20.)], dtype=[('A', int64), ('B', float64)])
    - b = zeros(3, dtype=a.dtype)
    - recursive_fill_fields(a, b)  # fills (0,0) for missing row
- rename_fields
  - Rename fields in a structured array.
- repack_fields
  - Repack fields into a new dtype, potentially reorganizing layout.
- require_fields
  - Ensure that certain fields exist (or raise an error if not present).
- stack_arrays
  - Stack multiple arrays along a new axis to form a single structured array.
- structured_to_unstructured
  - Convert a structured array to a plain unstructured (regular) ndarray.
- unstructured_to_structured
  - Convert an unstructured ndarray into a structured array with the specified dtype.

> Note: Some utilities are private or internal helpers (e.g., `_get_fieldspec`, `_recursive_fill_fields_dispatcher`). They are exposed for internal consistency or advanced use but are not intended for casual user interaction.

## Examples

### Flatten a nested dtype
```python
import numpy as np
from numpy.lib import recfunctions as rfn

ndtype = np.dtype([('a', '<i4'), ('b', [('ba', '<f8'), ('bb', '<i4')])])
flattened = rfn.flatten_descr(ndtype)
# Result: (('a', dtype('int32')), ('ba', dtype('float64')), ('bb', dtype('int32')))
```

### Nested field names
```python
adtype = np.dtype([('a', int), ('b', [('ba', int), ('bb', int)])])
names = rfn.get_names(adtype)
# Result: ('a', ('b', ('ba', 'bb')))
```

### Flattened field names
```python
adtype = np.dtype([('a', int), ('b', [('ba', int), ('bb', int)])])
flat_names = rfn.get_names_flat(adtype)
# Result: ('a', 'b', 'ba', 'bb')
```

### Recursive fill of fields
```python
a = np.array([(1, 10.), (2, 20.)], dtype=[('A', np.int64), ('B', np.float64)])
b = np.zeros((3,), dtype=a.dtype)
rfn.recursive_fill_fields(a, b)
# b now contains [(1, 10.), (2, 20.), (0, 0.)]
```

## Notes

- These utilities are designed to work with NumPy structured arrays, including nested dtypes. They help perform common data wrangling tasks without manual, low-level dtype plumbing.
- For performance-sensitive workflows, prefer batch operations that minimize dtype transformations and copies.
- Internal helpers (prefixed with an underscore) are intended for internal use; rely on public API for stable behavior.
