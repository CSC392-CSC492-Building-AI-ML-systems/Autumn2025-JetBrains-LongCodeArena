Miner (web3.miner)
===================

Overview
--------
The Miner class provides a Python wrapper around common Ethereum miner RPC calls exposed via the Web3.py library. It enables programmatic control of mining operations and retrieval of mining-related information.

The Miner class expects an existing Web3 instance (web3) and uses its request manager to issue JSON-RPC calls to the node.

API Reference
-------------

Class: Miner
------------

Constructor
-----------
def __init__(self, web3)

- web3: A Web3 instance. The Miner will use web3._requestManager to issue RPC calls.

Attributes
----------
hashrate (property)
- Description: Current mining hashrate reported by the node.
- Decorator: @apply_formatters_to_return(to_decimal)
- Returns: Decimal (converted from the node response by to_decimal)

Methods
-------

makeDAG(number)
- Signature: def makeDAG(self, number)
- Parameters:
  - number (int): The DAG epoch/number to generate.
- RPC: miner_makeDag
- Returns: The result of the RPC call (node-dependent)

setExtra(extra)
- Signature: def setExtra(self, extra)
- Parameters:
  - extra (str): Extra data to embed in blocks.
- RPC: miner_setExtra
- Returns: The result of the RPC call

setEtherBase(etherbase)
- Signature: def setEtherBase(self, etherbase)
- Parameters:
  - etherbase (str): Address to receive mining rewards.
- RPC: miner_setEtherbase
- Returns: The result of the RPC call

setGasPrice(gas_price)
- Signature: def setGasPrice(self, gas_price)
- Parameters:
  - gas_price (int): Gas price to use for mining (in wei).
- RPC: miner_setGasPrice
- Returns: The result of the RPC call

start(num_threads)
- Signature: def start(self, num_threads)
- Parameters:
  - num_threads (int): Number of mining threads to use.
- RPC: miner_start
- Returns: The result of the RPC call

stop()
- Signature: def stop(self)
- Parameters: None
- RPC: miner_stop
- Returns: The result of the RPC call

startAutoDAG()
- Signature: def startAutoDAG(self)
- Parameters: None
- RPC: miner_startAutoDag
- Returns: The result of the RPC call

stopAutoDAG()
- Signature: def stopAutoDAG(self)
- Parameters: None
- RPC: miner_stopAutoDag
- Returns: The result of the RPC call

Usage Examples
--------------

Basic usage
-----------
// Assuming a running Web3 provider and a node with mining capabilities

from web3 import Web3

w3 = Web3(Web3.HTTPProvider('http://localhost:8545'))
miner = w3.miner

# Check current hashrate (as Decimal)
current_hashrate = miner.hashrate
print("Hashrate:", current_hashrate)

Control mining
--------------

# Start mining with 4 threads
miner.start(4)

# Stop mining
miner.stop()

Manage DAG and extras
---------------------

# Generate DAG for a specific epoch
miner.makeDAG(123)

# Set extra data for blocks
miner.setExtra("0x1234")

# Set Etherbase (mining payout address)
miner.setEtherBase("0xYourEtherbaseAddress")

# Set gas price (in wei)
miner.setGasPrice(2000000000)

Auto DAG management
-------------------

# Enable automatic DAG generation
miner.startAutoDAG()

# Disable automatic DAG generation
miner.stopAutoDAG()

Notes
-----
- All methods delegate to the underlying RPC endpoints exposed by the node (ETH/miner namespace). The exact return value depends on the node’s response.
- The hashrate property is formatted via to_decimal to provide a precise Decimal value.
