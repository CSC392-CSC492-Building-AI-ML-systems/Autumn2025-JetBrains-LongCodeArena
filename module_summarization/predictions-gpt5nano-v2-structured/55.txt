# miscmodels.rst

Module description
------------------

This module provides robust linear and ARMA models using a Student-t distribution
for the observation errors. The heavier tails of the t distribution make these
models more robust to outliers and heavy-tailed data compared to Gaussian
error models.

Key components
--------------

- TLinearModel: Maximum Likelihood Estimation of a linear model with t-distributed
  errors. This class extends GenericLikelihoodModel and demonstrates how to set up
  a likelihood-based estimation with an optional degrees-of-freedom (df) parameter
  and a scale parameter.

- TArma: Univariate ARMA model with t-distributed errors. This class extends
  statsmodels.tsa.arma_mle.Arma and overrides the log-likelihood computations to use
  a t-distribution for the innovations. The implied variance is not simply scale^2;
  the df and scale parameters appear in the likelihood.

Overview of the models
----------------------

- Student-t errors for robustness
  - The t-distribution has heavier tails than the normal distribution, providing
    robustness to outliers and model misspecification in the error terms.

- Parameters
  - beta: regression coefficients (for TLinearModel)
  - df: degrees of freedom of the t-distribution
  - scale: scale (absolute value to ensure positivity)
  - When fix_df is False (default), df is estimated along with scale. When
    fix_df is set to a numeric value, df is fixed and only scale (and coefficients)
    are estimated.

- Special notes
  - The scale parameter does not necessarily equate to the standard deviation of the
    errors; for the t-distribution, the relationship between scale and variance is
    df-dependent.
  - TArma computes the per-observation log-likelihood using the t-density of the
    residuals scaled by the scale parameter.

Usage examples
--------------

Linear model with t-distributed errors

```python
import numpy as np
import statsmodels.api as sm
from statsmodels.miscmodels import TLinearModel

# Example data
np.random.seed(0)
n = 100
X = sm.add_constant(np.random.randn(n, 1))
beta_true = np.array([1.0, 0.5])
y = X @ beta_true + np.random.standard_t(5, size=n)

# Fit the model
model = TLinearModel(y, X)
res = model.fit()  # uses default optimization method
print(res.summary())
```

Univariate ARMA model with t-distributed errors

```python
import numpy as np
from statsmodels.api import add_constant
from statsmodels.miscmodels import TArma

# Example data
np.random.seed(0)
y = np.random.randn(200)

# Fit an AR(1) with t-distributed errors
model = TArma(y, order=(1, 0))  # ARMA(p,q) specification
res = model.fit_mle(order=(1, 0), maxiter=1000)
print(res.summary())
```

Notes on model configuration
----------------------------

- Degrees of freedom (df)
  - By default, df is estimated along with other parameters.
  - To fix df to a constant value, set fix_df to that value (e.g., model.fix_df = 5).
  - When df is fixed, only scale (and the regression/ARMA parameters) are estimated.

- Scale
  - The last parameter in the parameter vector represents scale (absolute value is used).

- Fixed parameters support
  - The implementation supports fixing a subset of parameters via fixed_params and
    fixed_paramsmask (inherited machinery from GenericLikelihoodModel).

API reference (highlights)
--------------------------

TLinearModel
- initialize(): Prepare model dimensions, parameter configuration, and names.
- _set_start_params(): Compute sensible starting values (OLS-based) when possible.
- loglike(params): Scalar log-likelihood for the given parameter vector.
- nloglikeobs(params): Per-observation negative log-likelihood to enable robustness.
- predict(params, exog=None): Predict the mean values given parameters.

TArma
- loglike(params): Log-likelihood of the ARMA model with t-distributed errors.
- nloglikeobs(params): Per-observation negative log-likelihood for t-errors.
- fit_mle(order, ...): Maximum likelihood estimation for the specified ARMA order.
  - Note: This method mirrors the underlying Arma class but uses the t-distribution
    for residuals.

Design notes
------------

- This module demonstrates how to implement robust likelihood-based estimation
  for both regression and time-series contexts using the Student-t distribution.
- The emphasis is on illustrating generic MLE workflows via a t-likelihood
  implementation, with attention to parameter constraints and starting values.
