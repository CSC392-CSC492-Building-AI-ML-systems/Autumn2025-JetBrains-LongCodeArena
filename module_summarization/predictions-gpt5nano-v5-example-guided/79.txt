GILDAS CLASS File Reader (classfiles)

Overview
--------
This module provides utilities and data structures used by the GILDAS CLASS file reader to interpret CLASS files and prepare them for ingestion into pyspeckit ObsBlock objects. It defines mappings for file types and versions, header and record lengths, and the layout of CLASS header blocks. It also includes small helper utilities used during parsing, such as a timing decorator and a bytes-normalization function.

Key Components
-------------
- print_timing: Decorator to measure and log execution time of a function.
- ensure_bytes(string): Normalize strings to bytes for Python 2/3 compatibility.
- filetype_dict: Mapping from CLASS filetype codes to human-friendly names (with both str and bytes keys supported via ensure_bytes).
- fileversion_dict: Mapping from CLASS fileversion codes to version identifiers (with both str and bytes keys supported).
- record_lengths: Expected header block lengths for different CLASS record types.
- header_id_numbers: Mapping from header section codes to human-readable section names.
- header_id_lengths: Specified lengths for certain header sections (where fixed length applies).
- filedescv2_nw1: Constant related to the CLASS v2 descriptor (set to 14).
- keys_lengths: Structure describing the layout of header blocks, including:
  - UNKNOWN (a general observation header)
  - COMMENT (comment block)
  - GENERAL (general observational metadata)

The module references the CLASS specification:
Specification: http://iram.fr/IRAMFR/GILDAS/doc/html/class-html/node58.html

Usage
-----
This module is used by the CLASS file reader to interpret CLASS headers. You typically access its mappings and structures to understand or debug how a CLASS file is parsed.

Example:
- Access file type and version mappings
  from classfiles import filetype_dict, fileversion_dict, ensure_bytes
  t = filetype_dict[ensure_bytes('9A  ')]
  v = fileversion_dict[ensure_bytes('1A  ')]
  print(t, v)

- Inspect header block layouts
  from classfiles import keys_lengths
  print(keys_lengths['GENERAL'])

Code references (selected)
--------------------------
- Timing decorator
  def print_timing(func):
      """
      Prints execution time of decorated function.
      Included here because CLASS files can take a little while to read;
      this should probably be replaced with a progressbar
      """
      def wrapper(*arg,**kwargs):
          t1 = time.time()
          res = func(*arg,**kwargs)
          t2 = time.time()
          log.info('%s took %0.5g s' % (func.__name__, (t2-t1)))
          return res
      wrapper.__doc__ = func.__doc__
      return wrapper

- Byte normalization helper
  def ensure_bytes(string):
      """
      Ensure a given string is in byte form
      """
      if six.PY3:
          return bytes(string, 'utf-8')
      else:
          return str(string)

Notes
-----
- The definitions here reflect the CLASS file structure as described in the embedded Fortran-like comments and the external CLASS specification.
- The module is focused on structural metadata and does not itself implement the full file parsing logic in the excerpt provided. The constants and helpers are intended for use by the actual reader implementation that consumes CLASS files.
