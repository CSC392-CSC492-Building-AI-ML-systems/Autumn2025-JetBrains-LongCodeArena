# Developing scikit-learn estimators

## Overview
This module implements multioutput regression and classification meta-estimators. The estimators provided are meta-estimators: they require a base estimator in their constructor and extend single-output estimators to multioutput estimators.

Publicly exposed meta-estimators include:
- MultiOutputRegressor
- MultiOutputClassifier
- ClassifierChain
- RegressorChain

These meta-estimators leverage helper utilities to manage fitting, partial fitting, and conditional availability of sub-estimators.

## Key Components
- _fit_estimator
  - Clones a given estimator and fits it to X, y (with optional sample_weight and fit_params).
  - Example:
    def _fit_estimator(estimator, X, y, sample_weight=None, **fit_params):
        estimator = clone(estimator)
        if sample_weight is not None:
            estimator.fit(X, y, sample_weight=sample_weight, **fit_params)
        else:
            estimator.fit(X, y, **fit_params)
        return estimator

- _partial_fit_estimator
  - Handles partial_fit for estimators, supporting an initial clone on first_time and optional classes for incremental learning.
  - Example:
    def _partial_fit_estimator(
        estimator, X, y, classes=None, partial_fit_params=None, first_time=True
    ):
        partial_fit_params = {} if partial_fit_params is None else partial_fit_params
        if first_time:
            estimator = clone(estimator)

        if classes is not None:
            estimator.partial_fit(X, y, classes=classes, **partial_fit_params)
        else:
            estimator.partial_fit(X, y, **partial_fit_params)
        return estimator

- _available_if_estimator_has
  - Returns a function suitable for use with available_if to check whether sub-estimator(s) have a given attribute.
  - Helper for Chain implementations.
  - Example:
    def _available_if_estimator_has(attr):
        def _check(self):
            if hasattr(self, "estimators_"):
                return all(hasattr(est, attr) for est in self.estimators_)
            if hasattr(self.estimator, attr):
                return True
            return False
        return available_if(_check)

- _MultiOutputEstimator
  - Abstract base class for multioutput meta-estimators.
  - Defines parameter constraints and initialization.
  - Example:
    class _MultiOutputEstimator(MetaEstimatorMixin, BaseEstimator, metaclass=ABCMeta):
        _parameter_constraints: dict = {
            "estimator": [HasMethods(["fit", "predict"])],
            "n_jobs": [Integral, None],
        }

        @abstractmethod
        def __init__(self, estimator, *, n_jobs=None):
            self.estimator = estimator
            self.n_jobs = n_jobs

- Public API
  - The module exposes the following public classes via __all__: 
    - "MultiOutputRegressor",
    - "MultiOutputClassifier",
    - "ClassifierChain",
    - "RegressorChain"

## Usage
- Basic usage with a base estimator:
  - Example:
    from sklearn.linear_model import LinearRegression
    from sklearn.multioutput import MultiOutputRegressor

    base = LinearRegression()
    mor = MultiOutputRegressor(base)
    mor.fit(X, y)

- Internal helper usage (conceptual):
  - The _fit_estimator helper is used to clone and fit base estimators for each target, ensuring isolated copies per output.

- Partial fitting scenario (conceptual):
  - The _partial_fit_estimator helper enables incremental learning with optional first-time initialization and class handling when necessary.

## Details
- These meta-estimators enable extending single-output estimators to multioutput tasks by creating and managing copies of the base estimator for multiple targets.
- They rely on standard scikit-learn utilities such as clone, available_if, and HasMethods to ensure correct behavior and parameter validation.
- The design supports both batch fitting and incremental/partial fitting workflows where applicable, as illustrated by the _fit_estimator and _partial_fit_estimator helpers.
- The public API includes MetaEstimator-based wrappers like MultiOutputRegressor and MultiOutputClassifier, as well as chain-based meta-estimators such as ClassifierChain and RegressorChain for specific multioutput strategies.
