# emplike.rst — Empirical likelihood

Overview
- This module implements empirical likelihood inference for descriptive statistics.
- It supports hypothesis tests and confidence intervals for mean, variance, skewness, kurtosis, and correlation.
- When matplotlib is available, it can render multivariate confidence regions and mean–variance contour plots.
- Core optimization details and the optimization variable definitions are described in the _OptFuncts docstring.

- Purpose: Create an inference object for descriptive statistics using empirical likelihood.

Usage

- Typical workflow:
  - Create a DescStat instance from your data:
    - data = np.asarray([...])
    - stat = DescStat(data)
  - The object will be either univariate (DescStatUV) or multivariate (DescStatMV) depending on data dimensionality.
  - Internally, _OptFuncts provides the optimization routines that drive empirical likelihood computations via Lagrange multipliers.

Notes and References

- The empirical likelihood framework here follows the approach described by estimating equations and Lagrange multipliers (eta), maximizing a profile likelihood under moment constraints.
- General reference for the methodology:
  - Owen, A. (2001). "Empirical Likelihood." Chapman and Hall.
- Dependencies:
  - numpy, scipy (optimize, stats), and statsmodels (optimizer utilities, plotting extensions if available).
- Plotting:
