# Quantile Regression (regression.rst)

## Overview
This module implements a Quantile Regression model using Iteratively Reweighted Least Squares (IRLS). The asymptotic covariance of the estimated parameters is obtained via kernel density estimation. The implementation supports estimating coefficients for any specified quantile and provides options for different kernels and bandwidth selection. The Least Absolute Deviation (LAD) estimator is a special case when the quantile q is 0.5.

Key details:
- Author: Vincent Arel-Bundock
- License: BSD-3
- Original IRLS inspiration and translations are documented in the source code header.
- Core class: QuantReg, which extends a general RegressionModel.

References and theoretical notes are embedded in the implementation, including Greene (2008), Koenker (2005), and related econometrics literature.

## Key Components
- QuantReg (class)
  - Inherits from RegressionModel
  - Methods:
    - whiten(self, data): No-op whitening that returns data unchanged.
    - fit(self, q=0.5, vcov='robust', kernel='epa', bandwidth='hsheather', max_iter=1000, p_tol=1e-6, **kwargs)
      - Core fitting procedure using IRLS to estimate quantile regression coefficients.
      - Parameters:
        - q: float, target quantile in [0, 1]. Default 0.5 (LAD).
        - vcov: str, covariance estimator ('robust' by default; 'iid' is also supported).
        - kernel: str, kernel for asymptotic covariance estimation. Options include:
          - biw, cos, epa, gau, par (mapped to internal kernel implementations)
        - bandwidth: str, bandwidth selection method for kernel density estimation. Options:
          - hsheather (Hall-Sheather), bofinger, chamberlain
        - max_iter: int, maximum IRLS iterations (default 1000)
        - p_tol: float, convergence tolerance for parameter updates (default 1e-6)
      - Internal workflow:
        - Initialize and iteratively update parameter estimates via weighted least squares.
        - Transform residuals and weights according to the specified quantile.
        - Update the design matrix transformation and check convergence.
      - Returns: A RegressionResultsWrapper containing estimated coefficients, standard errors, and related statistics.

Notes:
- The implementation emphasizes general quantile estimation and robust covariance estimation, with explicit handling for convergence and iteration warnings as part of the integration with the Statsmodels framework.

## Usage
Example usage with typical data:

- Prepare data (endog and exog)
- Build and fit the model
- Inspect results

Code example:

- Basic quantile regression at the median (LAD):
  - from statsmodels.regression.quantile_regression import QuantReg
  - model = QuantReg(endog, exog)
  - res = model.fit(q=0.5)  # LAD
  - print(res.summary())

- Quantile regression at a non-median quantile with kernel-based covariance estimation:
  - res = model.fit(q=0.25, kernel='epa', bandwidth='hsheather')
  - print(res.params)
  - print(res.bse)

- Accessing results:
  - res.params: estimated coefficients
  - res.bse: standard errors
  - res.tvalues, res.pvalues: statistical tests (as provided by the underlying RegressionResults)

Code snippet (illustrative):

- import numpy as np
- import statsmodels.api as sm
- endog = np.asarray([...])
- exog = sm.add_constant(np.asarray([...]).T)
- model = QuantReg(endog, exog)
- res = model.fit(q=0.5, kernel='epa', bandwidth='hsheather')
- print(res.summary())

Note: The actual import path may vary depending on the Statsmodels version, but QuantReg is designed to integrate with the standard results framework (RegressionResults, RegressionResultsWrapper).

## Details
- The mode uses IRLS: the coefficients are updated in each iteration by solving a weighted least squares problem, with weights derived from the current residuals relative to the target quantile.
- Asymptotic covariance estimation relies on kernel density estimation:
  - Kernel choices (internal mapping): biw, cos, epa, gau, par
  - Bandwidth selection methods:
    - hsheather: Hall-Sheather (1988)
    - bofinger: Bofinger (1975)
    - chamberlain: Chamberlain (1994)
- The q parameter determines which conditional quantile of the response variable given the predictors is modeled.
  - q = 0.5 corresponds to LAD regression, a robust alternative to least squares.
- Convergence and iteration:
  - The algorithm iterates until either the maximum number of iterations is reached or the parameter change falls below p_tol.
  - Warnings and iteration limits are surfaced via standard regression result handling.

Reference materials embedded in the source (and useful for deeper understanding):
- Greene (2008), Econometric Analysis, for asymptotic covariance derivation
- Koenker (2005), Quantile Regression
- LeSage (1999), Applied Econometrics Using MATLAB (translated implementation)
