Overview
--------
This module provides utility wrappers to simplify spectral modeling and fitting
within the pyspeckit framework.

Key components include:
- SimpleFitter: a lightweight wrapper with a helper to obtain spectral moments.
- vheightmodel: a wrapper to promote a base (zero-height) model to a model with a
  variable background (height) component.
- SpectralModel: a comprehensive wrapper for generic spectral models, enabling
  multi-component modeling, parameter management, integrals, and annotations.
  It integrates with mpfit (and optionally lmfit) and relies on the Parinfo data
  structures for parameter handling.

- vheightmodel (variable-height wrapper)
  - Purpose: Wraps a base (zero-height) model to add an adjustable height (background)
    component.
  - How it works:
    - The wrapper returns a function vhm(xax, *pars, **kwargs) where the
      parameter order is: height, amplitude, shift, width.
    - If vheight is True (default, controllable via kwargs['vheight']), the
      function returns zeroheightmodel(xax, *pars[1:], **kwargs) + pars[0].
      If vheight is False, it returns zeroheightmodel(xax, *pars[1:], **kwargs) only.
    - The wrapper also appends the docstring of the underlying zero-height model.
  - Example usage:
    - Given a base model
      def zeroheightmodel(x, A, x0, sigma):
          return A * numpy.exp(-0.5 * ((x - x0) / sigma) ** 2)
    - Create a height-enabled version
      vhm = vheightmodel(zeroheightmodel)
    - Compute with height included:
      x = numpy.linspace(-5, 5, 100)
      y = vhm(x, 0.5, 2.0, 0.0, 1.0)  # height=0.5, A=2.0, x0=0.0, sigma=1.0

  - Source excerpt (core logic):
    - def vheightmodel(zeroheightmodel):
        def vhm(xax, *pars,**kwargs):
            """
            Wrapper function vhm to set variable height.
            Parameter order: height, amplitude, shift, width
            """
            vheight=True
            if 'vheight' in kwargs:
                vheight = kwargs.pop('vheight')
            if vheight:
                return zeroheightmodel(xax, *pars[1:],**kwargs) + pars[0]
            else:
                return zeroheightmodel(xax, *pars[1:],**kwargs)
        vhm.__doc__ += zeroheightmodel.__doc__
        return vhm

      integral_func=None,
      use_lmfit=False,

      **kwargs

  - Core behavior and attributes:

    - self.modelfunc_kwargs: additional keyword arguments passed to the model
    - self.use_lmfit: flag to choose lmfit vs mpfit
    - self.centroid_par: parameter representing the centroid of the profile
    - self.fwhm_func, self.fwhm_pars: FWHM handling for the model

  - Wrap a base model to add an adjustable background height.
  - Example:
