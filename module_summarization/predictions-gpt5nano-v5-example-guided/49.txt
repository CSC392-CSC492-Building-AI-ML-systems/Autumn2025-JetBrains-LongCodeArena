# duration

Duration utilities for survival and cumulative incidence calculations

## Overview

This module provides internal helpers for survival analysis and cumulative incidence estimation. It includes:

- _calc_survfunc_right: computes the survival function and its standard error for a single group.
- _calc_incidence_right: computes the cumulative incidence functions (one per event type) and their standard errors.
- _checkargs: validates input arguments for time-to-event analyses.
- CumIncidenceRight: a class interface for estimation and inference of cumulative incidence functions.

These components support right-censoring scenarios, with optional frequency weights and, in some cases, exposure or covariate adjustments.

## Key Components

- _calc_survfunc_right(time, status, weights=None, entry=None, compress=True, retall=True)
  - Purpose: Calculate the survival function S(t) and its standard error for a single group.
  - Parameters:
    - time: array-like of event or censoring times.
    - status: array-like; typical convention is 1 for event, 0 for censored.
    - weights: optional weights for each observation.
    - entry: optional entry times for left-truncation handling.
    - compress: if True, retain only times at which an event occurred.
    - retall: if True, return full intermediate results; otherwise return a reduced set.
  - Returns:
    - sp: survival probabilities at event times.
    - se: standard errors of the survival probabilities.
    - utime, rtime, n, d: internal arrays describing unique times and event counts/risk set sizes.
  - Notes: Uses Greenwood’s formula for the standard error when weights are absent; uses Tsiatis’ formula when weights are provided.

- _calc_incidence_right(time, status, weights=None)
  - Purpose: Calculate the cumulative incidence function(s) for one or more event types and their standard errors.
  - Parameters:
    - time: array-like of times.
    - status: array-like; event indicators where 0 indicates censored and positive integers indicate event types.
    - weights: optional weights for observations.
  - Returns:
    - ip: list of cumulative incidence functions (one per event type).
    - se: list of standard errors (or None if weights are provided).
    - utime: unique event times used for the calculation.
  - Notes: Internally computes the all-cause survival function, then derives cause-specific incidences.

- _checkargs(time, status, entry, freq_weights, exog)
  - Purpose: Validate inputs before analysis.
  - Checks:
    - time and status have the same length.
    - entry, if provided, matches time length and entry times precede events.
    - freq_weights, if provided, match time length.
    - exog, if provided, has the same number of rows as time.

- CumIncidenceRight
  - Purpose: Estimation and inference for a cumulative incidence function (CIF) for competing risks.
  - Concept:
    - If J = 1, 2, ..., the CIF for cause J is I(t, J) = P(T ≤ t and J = J).
    - Supports only right censoring. Frequency weights are supported, in which case the point estimate is returned without a standard error.
  - Parameters (inferred from the constructor signature and docstring):
    - time: array-like of times (censoring or event times).
    - status: array-like; indicates which event occurred at time t (0 = censored; 1,2,... = event types).
    - title: optional string used for plots or summary outputs.
    - freq_weights: optional frequency weights.
    - exog: optional external data used to account for potential dependent censoring.
    - bw_factor: bandwidth multiplier for kernel-based estimation (used if exog is provided).
    - dimred: flag for potential dimensionality reduction in modeling (e.g., to enable PH-type modeling).
  - Notes: The class supports CIF estimation in the presence of competing risks and may integrate with other parts of a larger survival-analysis framework.

## Usage

- Example: compute survival function for a single group

```python
import numpy as np
# 1 indicates event, 0 indicates censoring
time = np.array([5.0, 3.0, 6.0, 2.0, 4.0])
status = np.array([1, 0, 1, 0, 1])

sp, se, utime, rtime, n, d = _calc_survfunc_right(time, status)
print("Survival at event times:", sp)
print("Standard errors:", se)
```

- Example: compute cumulative incidence for competing risks

```python
# status: 0 = censored, 1 = event type 1, 2 = event type 2
time = np.array([1.0, 2.0, 2.0, 3.0, 4.0])
status = np.array([1, 0, 2, 0, 1])

ip, se, utime = _calc_incidence_right(time, status)
print("CIF for each event type:", ip)
```

- Example: validate inputs before analysis

```python
try:
    _checkargs([1,2,3], [0,1,0], None, None, None)
    print("Arguments are valid")
except ValueError as e:
    print("Invalid arguments:", e)
```

- Example: use CumIncidenceRight (conceptual)

```python
# time, status as described above
ci = CumIncidenceRight(time, status, title=" CIF example ")
# Further usage would depend on the class's public methods (not shown here)
```

## Details

- Right censoring is assumed by design.
- When frequency weights are provided, the point estimates are returned without standard errors (seeking to reflect weighting adjustments only if supported).
- Input validation is centralized in _checkargs to ensure consistency across analyses.
- The internal functions use numpy operations (bincount, unique, etc.) for efficiency and to handle varying censoring patterns.
