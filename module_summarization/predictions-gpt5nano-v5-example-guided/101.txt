# web3.miner.rst - Miner API Reference

## Overview
This module provides the Miner API for interacting with Ethereum node mining operations via the Web3.py library. It exposes a Miner class with methods to query and control mining-related settings and actions, such as hashrate, DAG creation, etherbase configuration, gas price, and DAG auto-management.

## Key Components
- Miner class: Encapsulates mining-related RPC calls and exposes convenient methods.
  - hashrate (property): Retrieves the current mining hashrate, formatted as a decimal.
  - makeDAG(number): Generates a DAG for the specified epoch number.
  - setExtra(extra): Sets extra data to attach to empty blocks.
  - setEtherBase(etherbase): Sets the etherbase (miner's address) for block rewards.
  - setGasPrice(gas_price): Sets the gas price for mining transactions.
  - start(num_threads): Starts mining with the specified number of threads.
  - stop(): Stops mining.
  - startAutoDAG(): Enables automatic DAG generation.
  - stopAutoDAG(): Disables automatic DAG generation.

## Usage
- Basic setup (example):
  - from web3 import Web3
  - w3 = Web3(Web3.HTTPProvider("http://localhost:8545"))
  - miner = w3.miner

- Check hashrate (decimal-formatted):
  - current_hashrate = miner.hashrate

- Create a DAG for a specific epoch:
  - miner.makeDAG(0)

- Set extra data for blocks:
  - miner.setExtra("0x68656c6c6f")  # example hex data

- Set etherbase (miner's address):
  - miner.setEtherBase("0xYourEtherbaseAddress")

- Adjust gas price for mining:
  - miner.setGasPrice(1)

- Start mining with a given number of threads:
  - miner.start(1)

- Stop mining:
  - miner.stop()

- Manage Auto DAG generation:
  - miner.startAutoDAG()
  - miner.stopAutoDAG()

## Details
- The hashrate property uses a formatter to convert the returned value to a decimal representation for readability.
- All methods issue synchronous (blocking) JSON-RPC calls via the internal request manager:
  - self.web3._requestManager.request_blocking(method, params)
- The available RPC methods map to common mining operations supported by Ethereum nodes:
  - eth_hashrate
  - miner_makeDag
  - miner_setExtra
  - miner_setEtherbase
  - miner_setGasPrice
  - miner_start
  - miner_stop
  - miner_startAutoDag
  - miner_stopAutoDag
