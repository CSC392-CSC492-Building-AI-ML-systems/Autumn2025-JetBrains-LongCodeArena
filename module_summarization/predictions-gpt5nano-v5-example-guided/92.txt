# distutils - Generate documentation for this module

## Overview
This module provides a collection of utilities used by build and installation processes. It includes:
- A lightweight container for installable libraries
- Temporary directory management with automatic cleanup
- Helpers for path manipulation and frame-based path discovery
- A mechanism to determine the number of parallel build jobs
- Deprecation-aware utilities and convenience functions

Key components include a small InstallableLib container, atexit-based temporary-directory cleanup, and several path and job-management helpers.

## Key Components

- InstallableLib
  - A simple container to hold information about an installable library.
  - Parameters
    - name (str): Name of the installed library.
    - build_info (dict): Dictionary with build-related information.
    - target_dir (str): Absolute path where the library should be installed.
  - Usage:
    ```
    lib = InstallableLib("mylib", build_info={"sources": ["mylib.c"]}, target_dir="/usr/local/lib")
    ```

- Temporary directory management
  - _tdata: Thread-local storage to track per-thread temporary data.
  - _tmpdirs: List of all created temporary directories for cleanup.
  - clean_up_temporary_directory(): Removes all tracked temporary directories (best-effort).
  - The cleanup is registered with atexit to run on program exit.

- get_num_build_jobs()
  - Determines the number of parallel build jobs to use.
  - Factors:
    - CPU count (with a cap of 8)
    - Environment variable NPY_NUM_BUILD_JOBS
    - Optional parallel settings on build commands (build, build_ext, build_clib)
  - Example:
    ```
    n = get_num_build_jobs()
    print(n)
    ```

- quote_args(args)
  - Quote a list of arguments. Note: this function is deprecated (deprecated since 1.22).
  - Example:
    ```
    # Deprecated usage example
    quoted = quote_args(['-O2', '-Wall'])
    ```

- allpath(name)
  - Convert a slash-separated path to the OS-specific separator.
  - Example:
    ```
    path = allpath('src/include')
    ```

- rel_path(path, parent_path)
  - Return the path of path relative to parent_path, with careful handling of symlinks.
  - Example:
    ```
    rel = rel_path('/abs/path/src/module', '/abs/path')
    ```

- get_path_from_frame(frame, parent_path=None)
  - Return the path of the module corresponding to a given frame object.
  - If parent_path is provided, the result is made relative to it.
  - Example:
    ```
    import inspect
    frame = inspect.currentframe()
    path = get_path_from_frame(frame)
    ```

- njoin(*path)
  - Robustly join path components, handling both sequences and individual components.
  - Features:
    - Accepts multiple arguments or a single sequence
    - Converts slash-separated paths to OS-specific separators
    - Resolves .. and .
  - Example:
    ```
    joined = njoin('a', ['b', 'c'], 'd')  # -> 'a/b/c/d' (platform-dependent)
    ```

- __all__
  - Public API export list including: Configuration, get_numpy_include_dirs, default_config_dict, dict_append, appendpath, generate_config_py, get_cmd, allpath, get_mathlibs, terminal_has_colors, red_text, green_text, yellow_text, blue_text, cyan_text, cyg2win32, mingw32, all_strings, has_f_sources, has_cxx_sources, filter_sources, get_dependencies, is_local_src_dir, get_ext_source_files, get_script_files, get_lib_source_files, get_data_files, dot_join, get_frame, minrelpath, njoin, is_sequence, is_string, as_list, gpaths, get_language, get_build_architecture, get_info, get_pkg_info, get_num_build_jobs, sanitize_cxx_flags, exec_mod_from_location

## Usage

- Determine the number of build jobs (respecting environment and command settings):
  ```
  n = get_num_build_jobs()
  print(n)
  ```

- Join and normalize paths:
  ```
  p = allpath('src/include')
  print(p)  # OS-specific path
  ```

- Compute a path relative to another:
  ```
  rel = rel_path('/abs/path/src/module', '/abs/path')
  print(rel)
  ```

- Resolve a module path from a frame:
  ```
  import inspect
  frame = inspect.currentframe()
  path = get_path_from_frame(frame)
  print(path)
  ```

- Use the InstallableLib container:
  ```
  lib = InstallableLib("mylib", {"sources": ["mylib.c"]}, "/usr/local/lib")
  print(lib.name, lib.target_dir)
  ```

- Join complex paths:
  ```
  joined = njoin('a', ['b', 'c'], 'd')
  print(joined)
  ```

- Deprecated argument quoting:
  ```
  quoted = quote_args(['-O2', '-Wall'])
  print(quoted)
  ```

## Details

- Thread-local and global cleanup
  - Temporary directories are tracked per thread via _tdata and globally via _tmpdirs.
  - The clean_up_temporary_directory function attempts to remove all tracked directories on exit.
  - The cleanup is registered with atexit to ensure cleanup occurs when the program terminates.

- Build jobs configuration
  - get_num_build_jobs uses:
    - The system’s CPU count (with safeguards for affinity on platforms that support it)
    - A maximum cap of 8 to avoid overloading the system
    - The environment variable NPY_NUM_BUILD_JOBS
    - The largest non-None parallel value from build, build_ext, and build_clib command objects if present

- Deprecation notice
  - quote_args is deprecated; prefer constructing arguments directly or using updated utilities as recommended in the broader project guidelines.

- Public API surface
  - The __all__ list defines the intended public API for external use, guiding which symbols are considered stable for import.

Note: The documented module combines various helpers used in build and installation workflows. Some internal behavior depends on the surrounding packaging tools (e.g., numpy.distutils) and the runtime environment.
