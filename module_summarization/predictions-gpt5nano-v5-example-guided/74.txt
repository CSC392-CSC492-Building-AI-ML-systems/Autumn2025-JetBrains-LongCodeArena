# Random Projection transformers

## Overview
This module implements Random Projection transformers that reduce data dimensionality by projecting onto a lower-dimensional space using random matrices. The approach trades a controlled amount of accuracy (variance) for faster processing and smaller model sizes, while preserving pairwise distances with high probability. The theoretical basis is the Johnson-Lindenstrauss lemma, which shows that a small set of points in high-dimensional space can be embedded into a lower-dimensional space with distances preserved within a factor (1 ± eps).

Key concepts:
- Projection matrices drawn from Gaussian or sparse distributions
- The minimal number of components required to achieve a given distortion eps
- Density control for sparse random projections

See the module docstring for the underlying theory and references.

## Key Components
- johnson_lindenstrauss_min_dim: Compute a safe minimum number of components needed to achieve an eps-embedding for a given number of samples.
- _check_density: Validate and normalize the projection density parameter (supports "auto" and numeric densities).
- _check_input_size: Validate positive n_components and n_features.
- _gaussian_random_matrix: Generate a dense Gaussian random matrix (N(0, 1) scaled by 1/n_components).
- GaussianRandomProjection: Public transformer using Gaussian random projection.
- SparseRandomProjection: Public transformer using sparse random projection.
- __all__: Public API exposure: SparseRandomProjection, GaussianRandomProjection, johnson_lindenstrauss_min_dim

Public API surface (typical usage):
- class-based transformers: GaussianRandomProjection, SparseRandomProjection
- helper function: johnson_lindenstrauss_min_dim

## Usage
Basic usage example:

```python
from sklearn.random_projection import johnson_lindenstrauss_min_dim

# Compute a safe number of components for 1,000,000 samples with eps=0.5
print(johnson_lindenstrauss_min_dim(1e6, eps=0.5))  # 663

# For multiple eps values
print(johnson_lindenstrauss_min_dim(1e6, eps=[0.5, 0.1, 0.01]))
# array([    663,   11841, 1112658])

# For multiple n_samples with a fixed eps
print(johnson_lindenstrauss_min_dim([1e4, 1e5, 1e6], eps=0.1))
# array([ 7894,  9868, 11841])
```

Notes:
- The function accepts int or array-like inputs and returns an int or an ndarray of ints.
- eps must be in (0, 1); n_samples must be > 0.

Usage with transformers (conceptual, API resembles scikit-learn):

```python
from sklearn.random_projection import GaussianRandomProjection

# Create a Gaussian random projection transformer
grp = GaussianRandomProjection(n_components=500, random_state=42)

# Fit and transform data
X_proj = grp.fit_transform(X)  # X: (n_samples, n_features)
```

## Details

- Johnson-Lindenstrauss lemma
  -: The distortion of pairwise distances under a random projection is bounded: (1 − eps) ||u − v||^2 < ||p(u) − p(v)||^2 < (1 + eps) ||u − v||^2 for a suitable random projection p.
  - The minimal number of components guaranteeing an eps-embedding is:
    n_components ≥ 4 log(n_samples) / (eps^2 / 2 − eps^3 / 3)
  - The required dimensionality depends on the number of samples, not the original feature count.

- _check_density(density, n_features)
  - If density is "auto", it is set to 1 / sqrt(n_features).
  - If density is not in (0, 1], it raises a ValueError.
  - Returns a normalized density value.

- _check_input_size(n_components, n_features)
  - Validates that n_components > 0 and n_features > 0.
  - Raises ValueError on invalid sizes.

- _gaussian_random_matrix(n_components, n_features, random_state=None)
  - Generates a dense Gaussian random matrix with entries drawn from N(0, 1) scaled by 1 / n_components.
  - Used in Gaussian random projection to map data to the target dimensionality.

- References
  - Johnson-Lindenstrauss lemma (general description and implications)
  - Dasgupta & Gupta, 1999, for elementary proof of the lemma
  - See the user guide sections and the Wikipedia page for deeper reading:
    - https://en.wikipedia.org/wiki/Johnson%E2%80%93Lindenstrauss_lemma

## Details about API and behavior

- Public classes:
  - GaussianRandomProjection: Uses Gaussian random projection to reduce dimensionality.
  - SparseRandomProjection: Uses sparse random projection to reduce dimensionality with controlled density.
- Public helper:
  - johnson_lindenstrauss_min_dim(n_samples, eps=0.1): Computes the safe minimum number of components.
