## Overview
Collection of utilities to manipulate structured arrays. Most of these functions were initially implemented by John Hunter for matplotlib and have been rewritten and extended for convenience.

This module provides tools for inspecting, reshaping, and combining structured (record) arrays, including handling nested field structures and subarrays.

## Key Components
- recursive_fill_fields(input, output)
  - Fills fields from input into output, with support for nested structures.
  - Notes: output should be at least the same size as input.
  - Example:
    - From the source:
      - a = [(1, 10.), (2, 20.)] with dtype [("A", int64), ("B", float64)]
      - b = zeros(3, dtype=a.dtype)
      - recursive_fill_fields(a, b) -> [(1, 10.), (2, 20.), (0, 0.)]
- _get_fieldspec(dtype)
  - Internal helper.
  - Produces a list of (name, dtype) pairs for the given dtype, handling subarray dtypes and preserving field titles when present.
- get_names(adtype)
  - Returns the field names of the input dtype as a tuple.
  - Supports nested structures; nested names are returned as tuples.
  - Examples:
    - get_names(np.empty((1,), dtype=[("A", int)]).dtype) -> ('A',)
    - get_names(np.empty((1,), dtype=[("A", int), ("B", float)]).dtype) -> ('A', 'B')
    - Nested example:
      - adtype = np.dtype([('a', int), ('b', [('ba', int), ('bb', int)])])
      - get_names(adtype) -> ('a', ('b', ('ba', 'bb')))
- get_names_flat(adtype)
  - Returns flattened field names as a tuple.
  - Iteratively traverses nested structures to yield a flat sequence of names.
  - Examples:
    - get_names_flat(np.empty((1,), dtype=[("A", int)]).dtype) -> ('A',)
    - get_names_flat(np.empty((1,), dtype=[("A", int), ("B", str)]).dtype) -> ('A', 'B')
    - Nested example:
      - adtype = np.dtype([('a', int), ('b', [('ba', int), ('bb', int)])])
      - get_names_flat(adtype) -> ('a', 'b', 'ba', 'bb')
- flatten_descr(ndtype)
  - Flatten a structured data-type description.
  - Returns a tuple of (name, dtype) pairs with all nested fields flattened.
  - Example:
    - ndtype = np.dtype([('a', '<i4'), ('b', [('ba', '<f8'), ('bb', '<i4')])])
    - flatten_descr(ndtype) -> (('a', dtype('int32')), ('ba', dtype('float64')), ('bb', dtype('int32')))
- _zip_dtype(seqarrays, flatten=False)
  - Internal helper to assemble a new dtype from a sequence of arrays.
  - If flatten is True, uses flatten_descr on each array’s dtype; otherwise preserves the field structure.
  - (Internal utility; details omitted in public docs.)

## Usage
- Example: recursive_fill_fields
  - from numpy.lib import recfunctions as rfn
  - a = np.array([(1, 10.), (2, 20.)], dtype=[('A', np.int64), ('B', np.float64)])
  - b = np.zeros((3,), dtype=a.dtype)
  - rfn.recursive_fill_fields(a, b)
  - Result: array([(1, 10.), (2, 20.), (0,  0.)], dtype=[('A', '<i8'), ('B', '<f8')])

- Example: get_names
  - from numpy.lib import recfunctions as rfn
  - rfn.get_names(np.empty((1,), dtype=[('A', int)]).dtype)
  - -> ('A',)

  - rfn.get_names(np.empty((1,), dtype=[('A', int), ('B', float)]).dtype)
  - -> ('A', 'B')

  - adtype = np.dtype([('a', int), ('b', [('ba', int), ('bb', int)])])
  - rfn.get_names(adtype)
  - -> ('a', ('b', ('ba', 'bb')))

- Example: get_names_flat
  - from numpy.lib import recfunctions as rfn
  - rfn.get_names_flat(np.empty((1,), dtype=[('A', int)]).dtype)  # -> ('A',)
  - rfn.get_names_flat(np.empty((1,), dtype=[('A', int), ('B', str)]).dtype)
  - -> ('A', 'B')
  - adtype = np.dtype([('a', int), ('b', [('ba', int), ('bb', int)])])
  - rfn.get_names_flat(adtype)
  - -> ('a', 'b', 'ba', 'bb')

- Example: flatten_descr
  - ndtype = np.dtype([('a', '<i4'), ('b', [('ba', '<f8'), ('bb', '<i4')])])
  - rfn.flatten_descr(ndtype)
  - -> (('a', dtype('int32')), ('ba', dtype('float64')), ('bb', dtype('int32')))

## Details
- The module focuses on nested and structured data manipulations for NumPy arrays.
- Internal helpers (e.g., _get_fieldspec, _zip_dtype) support the public utilities by providing mechanisms to inspect and rebuild dtypes, especially for nested or subarray fields.
- The examples illustrate how nested field structures are represented and how names are extracted in both hierarchical and flattened forms.
