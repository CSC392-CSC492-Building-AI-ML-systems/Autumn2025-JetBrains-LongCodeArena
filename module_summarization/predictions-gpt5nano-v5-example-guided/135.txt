# Work with images or data cubes related to Cherenkov Cameras

## Overview
This module provides support for representing and handling subarrays of Cherenkov-telescope cameras. It centers on collecting telescope descriptions and ground positions for a subarray, enabling work with images or data cubes by mapping telescope IDs to positions, descriptions, and index-based representations.

Key ideas:
- Represent a subarray as a collection of telescope descriptions plus their ground coordinates.
- Convert telescope position data into a convenient GroundFrame sky coordinate object.
- Provide indexing utilities to map telescope IDs to fixed-length arrays useful for data cubes and image processing.

## Key Components

- UnknownTelescopeID: Exception raised when an unknown telescope id is encountered.

- _group_consecutives, _range_extraction: Helpers used to summarize telescope IDs in a human-friendly form (e.g., in info output).

- SubarrayDescription: Central class that aggregates telescope descriptions and their ground positions.
  - Attributes
    - name: Subarray name.
    - positions: Dict[int, np.ndarray]
      Ground-positions of telescopes by tel_id (x, y, z in a common frame, converted to meters internally).
    - tels: Dict[int, TelescopeDescription]
      Mapping from tel_id to TelescopeDescription for each telescope in the subarray.
    - reference_location: EarthLocation
      Optional EarthLocation representing the array reference position for the coordinate system used in tel_positions.
  - Important properties
    - tel_coords: SkyCoord in the GroundFrame representing telescope positions.
    - tel_ids: Array of telescope IDs in the subarray.
    - tel_indices: Dict[int, int] mapping tel_id -> index (0-based) for ordering in fixed-length arrays.
    - tel_index_array: Expanded array mapping tel_id to its index; uses -1 for missing entries when IDs are non-contiguous.
  - Key methods
    - tel_ids_to_indices(tel_ids): Map a telescope ID (or array of IDs) to the corresponding flat index/indices.
    - tel_ids_to_mask(tel_ids): Convert a list of telescope IDs to a boolean mask of length n_tels indicating presence.
    - info(printer=print): Print a descriptive summary of the subarray, including per-telescope-type counts and ID ranges.
  - Special methods
    - __init__(name, tel_positions=None, tel_descriptions=None, reference_location=None)
      Initializes the subarray. Ensures that the telescope IDs present in tel_positions match those in tel_descriptions; otherwise raises ValueError.
    - __str__ and __repr__: Provide readable string representations for debugging and logging.

- CURRENT_TAB_VERSION and COMPATIBLE_VERSIONS: Versioning metadata relevant for serialization/deserialization of SubarrayDescription data.

## Usage

Examples (illustrative; adapt TelescopeDescription instances as needed for your context):

- Create a SubarrayDescription with telescope positions and descriptions:
  - Note: Telescopes must have matching IDs in both positions and descriptions.

  - Code example:
    SubarrayDescription(
        name="Subarray-Example",
        tel_positions={1: np.array([0.0, 0.0, 0.0]) * u.m,
                       2: np.array([10.0, 0.0, 0.0]) * u.m},
        tel_descriptions={
            1: TelescopeDescription(),  # replace with actual constructor
            2: TelescopeDescription(),
        },
        reference_location=None,
    )

- Access and inspect information:
  - Print the subarray's string representation:
    print(subarray)

  - Print a detailed summary:
    subarray.info()

- Work with coordinates and indexing:
  - Telescope coordinates on the ground (convertible to a SkyCoord in the GroundFrame):
    coords = subarray.tel_coords

  - Get telescope IDs:
    ids = subarray.tel_ids

  - Get a mapping to fixed-length array indices (tel_id -> index):
    index_map = subarray.tel_indices

  - Expand to a per-telescope index array (maps tel_id to a flat index; missing IDs get -1):
    index_array = subarray.tel_index_array

  - Map IDs to indices (supporting single ID or list of IDs):
    indices = subarray.tel_ids_to_indices([1, 2])

  - Create a boolean mask for a set of IDs (length = number of telescopes in subarray):
    mask = subarray.tel_ids_to_mask([1, 2])

## Details

- Coordinate handling
  - Telescopes’ ground positions are stored as positions by tel_id and are converted to meters via to_value(u.m) when constructing tel_coords.
  - tel_coords uses the GroundFrame to provide positions in a geophysical reference frame suitable for mapping to images or data cubes.

- Data integrity
  - The SubarrayDescription enforces consistency between tel_positions and tel_descriptions. If the keys (tel_ids) do not match, a ValueError is raised.

- Versioning
  - The class uses CURRENT_TAB_VERSION = "2.0" and COMPATIBLE_VERSIONS = {"2.0"} to facilitate compatibility checks when reading or writing serialized descriptions.

- Printing and summaries
  - The info() method aggregates per-telescope-type statistics and prints a table with Type, Count, and Tel IDs (grouped where appropriate). This leverages internal helpers to format ranges of IDs.

- Related concepts
  - UnknownTelescopeID is raised in contexts where a requested tel_id is not present in the subarray’s description set.
  - The tel_index_array and tel_ids_to_indices utilities enable efficient data-cube construction by providing deterministic flat indexing across telescopes.
