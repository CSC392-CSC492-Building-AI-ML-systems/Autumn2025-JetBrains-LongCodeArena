# Plothist Command

## Overview
The plothist command is a WestPA tool for plotting histogram data stored in an HDF5 file produced by WESTPA. It renders 1D or 2D histograms as heatmaps (with optional contour overlays for 2D plots), supports various scales (energy, linear, log10), and allows range clipping. It can also save the plotted data to an HDF5 file and apply a user-specified post-processing function before saving.

## Key Components
- sum_except_along(array, axes)
  Utility to reduce an array by summing over all axes except the ones specified. This helper is used to collapse histogram arrays along undesired dimensions.

  ```python
  def sum_except_along(array, axes):
      '''Reduce the given array by addition over all axes except those listed in the scalar or
      iterable ``axes``'''

      try:
          iter(axes)
      except TypeError:
          axes = [axes]

      kept = set(axes)
      summed = list(set(range(array.ndim)) - kept)

      # Reorder axes so that the kept axes are first, and in the order they
      # were given
      array = np.transpose(array, list(axes) + summed).copy()

      # Now, the last len(summed) axes are summed over
      for _ in range(len(summed)):
          array = np.add.reduce(array, axis=-1)

      return array
  ```

- PlotHistBase(WESTSubcommand)
  Base class implementing the plothist subcommand. It handles:
  - Argument parsing (add_args)
  - Argument processing and initialization (process_args)
  - Management of iteration ranges, dimensional specifications, and output filenames
  - Plot options including title, scale, energy zero-point, range, contour overlay, and post-processing

  The class exposes attributes for: input_h5, opmode, plotscale, enerzero, plotrange, plottitle, postprocess_function, plot_contour, dimension specs, and output filenames (plot, text, and HDF5).

- Important integrations
  - h5io: Reading iteration ranges and histogram metadata from the input HDF5
  - normhistnd: Normalized histogram utilities used in plotting
  - get_object: Dynamic object loading for post-processing functions
  - WESTMasterCommand / WESTSubcommand: WestPA command framework integration

Usage
-----
Basic usage:
```
west plothist input.h5 0
```

Advanced usage:
```
west plothist input.h5 0 --plot-output hist.pdf --title "Histogram" --energy
west plothist input.h5 1 --range "0.0,10.0" --plot-contour
```

Arguments and options
---------------------
- input: HDF5 file containing histogram data
- firstdim: Dimension to plot, specified as DIMENSION, LB:UB, LABEL
- -o / --output / --plot-output: Output plot file name (default: hist.pdf). Use an empty string to suppress plotting.
- --hdf5-output: Store plot data in the HDF5 file specified here
- --plot-contour: Overlay a contour on 2D plots
- --title: Top-of-graph title
- --linear / --energy / --log10 / --zero-energy: Plot scale options
  - Default: energy
  - --zero-energy: Set the energy zero point to a value, "min", or "max"
- --range: Plot range as LB,UB
- --postprocess-function: Function path (module.function) to call just before saving the plot. The function will be invoked as postprocess(hist, midpoints, binbounds) and should modify the plot in place.

Notes
-----
- The command opens the input HDF5 file and determines the available iteration range for plotting.
- If no dimension is specified, a default for dimension 0 with label "dimension 0" is used.
- Energy zero-point handling supports numeric values as well as "min" or "max".
