Isotonic Regression
===================

Overview
--------
This module provides utilities to fit isotonic regression models, ensuring that the fitted values are monotone with respect to the input. It includes helpers to determine monotonicity direction, a function to compute an isotonic fit with optional weighting and bounds, and a class that encapsulates the isotonic regression model for fitting and predicting.

Key Components
--------------
- check_increasing(x, y)
  - Determines whether y is monotonically correlated with x using a Spearman correlation test.
  - Returns a boolean indicating if the relationship is increasing (True) or decreasing (False).
  - Warns if the 95% confidence interval of the Spearman rho spans zero.
  - Reference: Spearman correlation, Fisher transformation CI.

  Example
  .. code-block:: python

     import numpy as np
     from sklearn.isotonic import check_increasing

     x = np.array([1, 2, 3, 4])
     y = np.array([1.1, 2.0, 1.9, 4.2])
     increasing = check_increasing(x, y)

- isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True)
  - Solves the isotonic regression problem for a 1D array y.
  - Optional weights via sample_weight; y_min/y_max provide bounds on the fitted values.
  - increasing controls the direction (True: increasing, False: decreasing).
  - Uses an in-place algorithm (_inplace_contiguous_isotonic_regression) and optionally clips to bounds.

  Example
  .. code-block:: python

     import numpy as np
     from sklearn.isotonic import isotonic_regression

     y = np.array([0.5, 0.2, 0.9, 1.2], dtype=float)
     y_fit = isotonic_regression(y, increasing=True)

- IsotonicRegression
  - A model that encapsulates isotonic regression with fit/transform (and related) API.
  - Parameters:
    - y_min: Lower bound on the lowest predicted value.
    - y_max: Upper bound on the highest predicted value.
    - increasing: True (increasing), False (decreasing), or 'auto' to infer from data (via check_increasing).
    - out_of_bounds: How predictions outside the training domain are handled ('nan', 'clip', 'raise').
  - Attributes (examples):
    - X_min_ and X_max_: bounds of the training input (X) used for left/right limits.
    - X_thresholds_: unique ascending X values used to interpolate the monotonic function.
    - y_thresholds_: corresponding fitted values or thresholds (details depend on implementation and version).

  Example
  .. code-block:: python

     import numpy as np
     from sklearn.isotonic import IsotonicRegression

     X = np.array([0.0, 1.0, 2.0, 3.0])
     y = np.array([0.5, 1.2, 0.9, 1.8])

     ir = IsotonicRegression(increasing=True)
     ir.fit(X.reshape(-1, 1), y)
     y_pred = ir.predict(X.reshape(-1, 1))

Usage
-----
- Core idea: isotonic regression produces a monotone fit to the data by projecting onto the set of monotone functions.
- The isotonic_regression function provides a lower-level interface for computing the monotone fit with optional weights and bounds.
- The IsotonicRegression class offers a higher-level interface suitable for pipelines and integrated workflows, including handling out-of-bounds predictions.

Details
-------
- check_increasing uses Spearman's rho to assess monotonicity direction. If rho is not ±1 and the sample size is large enough, a Fisher transform-based CI is computed. If the CI spans zero, a warning is emitted.
- isotonic_regression sorts data according to the desired monotonic direction (increasing by default) and applies an in-place isotonic regression algorithm, with an optional clipping step to enforce y_min and y_max.
- IsotonicRegression supports a flexible handling of predictions outside the training domain via the out_of_bounds parameter.
- The module relies on internal routines such as _inplace_contiguous_isotonic_regression and _make_unique to perform the core computations.

References
----------
- Best, M. J., & Chakravarti, N. (Active-set algorithms for isotonic regression) – framework used for the isotonic regression solution.
- Originating implementation in scikit-learn’s isotonic regression utilities.

Notes
-----
- The IsotonicRegression API is designed to integrate with scikit-learn’s estimator ecosystem, including compatibility with fit/transform workflows and common validation utilities.
