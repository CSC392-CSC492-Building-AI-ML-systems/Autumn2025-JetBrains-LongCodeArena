```rst
web3.miner
==========

.. py:class:: web3.miner.Miner

    The ``web3.miner`` object exposes methods to interact with the miner functionality
    of an Ethereum node.

    .. py:method:: hashrate

        Returns the current hash rate of the miner.

        :returns: The current hash rate as a decimal number
        :rtype: decimal.Decimal

    .. py:method:: makeDAG(number)

        Generates the DAG (Directed Acyclic Graph) for the specified block number.

        :param int number: The block number for which to generate the DAG
        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: setExtra(extra)

        Sets the extra data field for mined blocks.

        :param str extra: The extra data to include in mined blocks
        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: setEtherBase(etherbase)

        Sets the etherbase (coinbase) address for mining rewards.

        :param str etherbase: The address to receive mining rewards
        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: setGasPrice(gas_price)

        Sets the minimum gas price for mining transactions.

        :param int gas_price: The minimum gas price in wei
        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: start(num_threads)

        Starts the mining process with the specified number of threads.

        :param int num_threads: The number of threads to use for mining
        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: stop()

        Stops the mining process.

        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: startAutoDAG()

        Starts automatic DAG generation.

        :returns: True if successful, False otherwise
        :rtype: bool

    .. py:method:: stopAutoDAG()

        Stops automatic DAG generation.

        :returns: True if successful, False otherwise
        :rtype: bool
```
