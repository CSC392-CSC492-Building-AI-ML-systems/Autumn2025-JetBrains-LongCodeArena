Quantile Regression Module
==========================

The ``QuantReg`` class implements quantile regression, a form of regression analysis used to estimate conditional quantiles of the response variable. Unlike ordinary least squares regression which estimates the conditional mean, quantile regression provides a more complete view of the relationship between variables by estimating different quantiles of the conditional distribution.

Model Estimation
----------------

The model parameters are estimated using the Iterated Reweighted Least Squares (IRLS) algorithm. This iterative procedure solves the quantile regression optimization problem by transforming it into a series of weighted least squares problems.

Key Features
------------

- **Flexible Quantile Estimation**: Estimate conditional quantiles at any level between 0 and 1
- **Robust Estimation**: Less sensitive to outliers compared to mean regression
- **Multiple Variance Estimation Methods**: Support for both robust (heteroskedasticity-consistent) and iid standard errors
- **Kernel Density Estimation**: Asymptotic covariance matrix estimation using various kernel functions

Special Cases
-------------

The Least Absolute Deviation (LAD) estimator is a special case of quantile regression where the quantile parameter is set to 0.5.

Parameters
----------

endog : array_like
    Endogenous or response variable
exog : array_like
    Exogenous or explanatory variables

Methods
-------

fit(q=0.5, vcov='robust', kernel='epa', bandwidth='hsheather', max_iter=1000, p_tol=1e-6)
    Estimate the quantile regression model

    Parameters:
        q : float, optional (default=0.5)
            Quantile to estimate (must be between 0 and 1)
        
        vcov : str, optional (default='robust')
            Method for variance-covariance estimation:
            - 'robust': Heteroskedasticity robust standard errors
            - 'iid': Independent and identically distributed errors
        
        kernel : str, optional (default='epa')
            Kernel function for density estimation:
            - 'epa': Epanechnikov kernel
            - 'cos': Cosine kernel
            - 'gau': Gaussian kernel
            - 'par': Parzen kernel
        
        bandwidth : str, optional (default='hsheather')
            Bandwidth selection method:
            - 'hsheather': Hall-Sheather method
            - 'bofinger': Bofinger method
            - 'chamberlain': Chamberlain method
        
        max_iter : int, optional (default=1000)
            Maximum number of iterations for IRLS algorithm
        
        p_tol : float, optional (default=1e-6)
            Convergence tolerance

Notes
-----

The implementation follows the methodology described in Koenker (2005) and Greene (2008). The asymptotic covariance matrix estimation uses kernel density methods as outlined in Hall and Sheather (1988).
