.. _feature_extraction_ref:

==================
Feature extraction
==================

.. currentmodule:: sklearn.feature_extraction

The :mod:`sklearn.feature_extraction` module provides utilities to extract
feature vectors from raw data. This is particularly useful for transforming
text documents or images into numerical representations that can be used
by machine learning algorithms.

.. _text_feature_extraction:

Text feature extraction
=======================

.. currentmodule:: sklearn.feature_extraction.text

The :mod:`sklearn.feature_extraction.text` submodule implements utilities
to build feature vectors from text documents. It provides various text
processing tools and vectorizers that convert text data into numerical
representations suitable for machine learning.
