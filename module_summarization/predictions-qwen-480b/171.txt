Calculator Interface
====================

The calculator interface provides a set of tools for working with mathematical models and parameters. This interface allows users to list and examine parameters used across various models, facilitating model comparison and analysis.

Listing Parameters
------------------

The primary functionality of the calculator interface is to list all parameters used across different models. This can be done using the following command:

.. code-block:: bash

    python -m sasmodels.list_pars [-v]

The optional ``-v`` flag enables verbose mode, which lists not only the parameter names but also the models that contain each parameter.

Parameter Categories
--------------------

Parameters can be filtered by type using the following categories:

- ``volume``: Volume parameters
- ``orientation``: Orientation parameters  
- ``sld``: Scattering length density parameters
- ``none``: Parameters with no specific type
- ``any``: All parameters (default)

To filter by parameter type, use:

.. code-block:: bash

    python -m sasmodels.list_pars [volume|orientation|sld|none|any]

Functionality
-------------

The calculator interface provides the following core functions:

``find_pars(kind=None)``
    Finds all parameters in all models and returns a reference table mapping parameter names to the list of models that use them.

``list_pars(names_only=True, kind=None)``
    Prints all parameters in all models. When ``names_only`` is True (default), only parameter names are printed. When False, models containing each parameter are also listed.

``main()``
    Main program entry point that handles command-line argument parsing and executes the parameter listing functionality.

The interface supports both simple parameter name listing and detailed analysis showing which models use each parameter, making it a valuable tool for understanding model parameter relationships and dependencies.
