Delayed API Reference
=====================

.. currentmodule:: dask.delayed

The ``dask.delayed`` interface provides a convenient way to parallelize
code by delaying function calls and building up a task graph that can
be executed later.

Examples
--------

Basic Usage
~~~~~~~~~~~

.. code-block:: python

    from dask import delayed

    @delayed
    def add(a, b):
        return a + b

    @delayed
    def mul(a, b):
        return a * b

    x = delayed(1)
    y = delayed(2)
    z = delayed(3)

    a = add(x, y)
    b = mul(a, z)

    # Compute the result
    result = b.compute()
    print(result)  # Output: 9

Working with Collections
~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: python

    from dask import delayed

    @delayed
    def process_list(items):
        return [x * 2 for x in items]

    data = delayed([1, 2, 3, 4])
    processed = process_list(data)
    result = processed.compute()
    print(result)  # Output: [2, 4, 6, 8]

Control Flow
~~~~~~~~~~~~

.. code-block:: python

    from dask import delayed

    @delayed
    def conditional(value):
        if value > 0:
            return "positive"
        else:
            return "non-positive"

    x = delayed(-5)
    result = conditional(x)
    print(result.compute())  # Output: "non-positive"
