Auto Augment
============

.. currentmodule:: nvidia.dali.auto_aug.auto_augment

The Auto Augment module provides functionality to apply predefined augmentation policies from the AutoAugment paper (https://arxiv.org/abs/1805.09501) to image and video data.

Functions
---------

.. autofunction:: auto_augment

.. autofunction:: auto_augment_image_net

.. autofunction:: apply_auto_augment

Predefined Policies
-------------------

The following predefined policies are available:

- ``image_net``: Policy for ImageNet dataset
- ``reduced_image_net``: Reduced policy for ImageNet dataset
- ``svhn``: Policy for SVHN dataset
- ``reduced_cifar10``: Reduced policy for CIFAR-10 dataset

Usage
-----

The main function ``auto_augment`` applies one of the predefined policies to a batch of samples. It supports both images (HWC layout) and videos (FHWC layout) with uint8 data type.

Example usage:

.. code-block:: python

    from nvidia.dali.auto_aug import auto_augment
    
    # Apply ImageNet policy
    augmented_data = auto_augment(data, policy_name='image_net')
    
    # Apply with custom parameters
    augmented_data = auto_augment(
        data, 
        policy_name='image_net',
        fill_value=0,
        interp_type=types.INTERP_LINEAR
    )

The ``auto_augment_image_net`` function is a convenience wrapper that directly applies the ImageNet policy.

Policy Details
--------------

Each policy consists of a set of augmentation operations with specific probabilities and magnitude ranges. The augmentations include:

- Translation
- Rotation
- Shear
- Contrast
- Brightness
- Saturation
- Hue
- Sharpness
- Equalize
- Solarize
- Posterize
- AutoContrast
- Color
- Brightness

The magnitude of transformations is controlled by the policy and can be adjusted through the function parameters.
