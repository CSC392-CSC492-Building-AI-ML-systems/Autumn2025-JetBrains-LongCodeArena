Ploterr
=======

The ``ploterr`` command is used to generate plots of expectation values with confidence intervals from WESTPA simulation data. It supports both built-in plotting capabilities and custom dataset plotting from HDF5 files.

Subcommands
-----------

generic
~~~~~~~

Plots generic expectation/CI data from an arbitrary HDF5 file and dataset. The dataset must contain the following fields:

- ``iter_start``: Start iteration numbers
- ``iter_stop``: Stop iteration numbers  
- ``expected``: Expected values
- ``ci_lbound``: Lower confidence interval bounds
- ``ci_ubound``: Upper confidence interval bounds

The path to the HDF5 file and dataset must be provided in the format ``FILENAME/PATH[SLICE]``. If the dataset is not a vector (one-dimensional), a slice must be provided.

**Example**: To plot state 0 to state 1 rate evolution from ``w_kinavg`` output:
```
kinavg.h5/rate_evolution[:,0,1]
```

Command-line Arguments
----------------------

Plot Options
~~~~~~~~~~~~

--xscale {linear,log,symlog}
    Use "linear", "log", or "symlog" scaling for the x axis. (Default: linear)

--yscale {linear,log,symlog}  
    Use "linear", "log", or "symlog" scaling for the y axis. (Default: linear)

--xrange XRANGE
    Restrict X range to XRANGE, formatted as "xmin,xmax". (Default: determined by input data)

--yrange YRANGE
    Restrict Y range to YRANGE, formatted as "ymin,ymax". (Default: determined by input data)

--xlabel XLABEL
    Use XLABEL for the x-axis label. (Default: "Iteration")

--ylabel YLABEL
    Use YLABEL for the y-axis label. (Default: varies)

--title TITLE
    Use TITLE for the plot title. (Default: varies)

--terminal, -t
    Plot output in terminal instead of using matplotlib GUI.

Input/Output Options
~~~~~~~~~~~~~~~~~~~~

-o OUTPUT, --output OUTPUT
    Write plot to OUTPUT (default: errbars.pdf). Format determined by filename extension.

dsspec
    Use data located at DSSPEC, formatted as FILENAME/PATH[SLICE]. FILENAME is the HDF5 file to read, PATH is the HDF5 path to the dataset, and SLICE (if provided) must be a Numpy-style slice selecting a vector of data.
