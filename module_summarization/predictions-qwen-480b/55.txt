T-distributed Linear Models
===========================

The module provides maximum likelihood estimation for linear models with
t-distributed errors. The t-distribution has heavier tails than the normal
distribution, which makes it suitable for modeling data with outliers or
heavy-tailed error distributions. This leads to more robust parameter estimates
compared to standard linear regression models that assume normally distributed
errors.

The main class, :class:`TLinearModel`, inherits from
:class:`statsmodels.base.model.GenericLikelihoodModel`, and implements
a custom log-likelihood function based on the t-distribution. All inference
methods (standard errors, Hessian, etc.) are derived numerically.

Additionally, the module includes :class:`TArma`, which extends the ARMA
modeling framework to allow for t-distributed errors in time series contexts.
