Configuration
=============

This module provides functionality for managing configuration options related to import and export operations in PyAEDT. It includes classes and methods for controlling which elements of an AEDT design are exported or imported when working with JSON configuration files.

Classes
-------

ConfigurationsOptions
~~~~~~~~~~~~~~~~~~~~~

.. class:: ConfigurationsOptions

   Options class for the configurations. Users can enable or disable import/export components.

   This class allows fine-grained control over what aspects of an AEDT design are included during export operations and what is processed during import operations.

   **Attributes:**

   .. attribute:: object_mapping_tolerance

      Get/Set the tolerance value to be used in the object mapping (used e.g. for boundaries).

      :type: float
      :default: 1e-9

   .. attribute:: export_variables

      Define if the variables have to be exported into json file.

      :type: bool
      :default: True

      Example:
         >>> from pyaedt import Hfss
         >>> hfss = Hfss()
         >>> hfss.configurations.options.export_variables = False  # Disable the variables export

   .. attribute:: export_setups

      Define if the setups have to be exported to json file.

      :type: bool

Helper Functions
----------------

.. function:: _find_datasets(d, out_list)

   Recursively searches through a dictionary structure to find dataset references in string values.

   :param dict d: Dictionary to search through
   :param list out_list: List to append found dataset names to

.. function:: _arg2dict(args, dt)

   Converts arguments to dictionary format (imported from pyaedt.generic.DataHandlers).

.. function:: _create_json_file(dict_data, filepath)

   Creates a JSON file from dictionary data (imported from pyaedt.generic.general_methods).

.. function:: generate_unique_name(name, n=6)

.. function:: pyaedt_function_handler()

   Decorator for handling PyAEDT function calls (imported from pyaedt.generic.general_methods).

.. function:: generate_unique_folder_name(rootname="pyaedt")
