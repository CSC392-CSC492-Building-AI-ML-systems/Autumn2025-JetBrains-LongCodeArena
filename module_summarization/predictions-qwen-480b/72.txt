Pairwise Metrics and Distance Computations
==========================================

This module provides utilities for computing pairwise distances and metrics
between samples. These functions are used across scikit-learn for tasks such
as clustering, nearest neighbors, and kernel methods.

Distance Metrics
----------------

The following distance metrics are available for computing pairwise distances:

- Euclidean distance
- Manhattan distance
- Cosine distance
- Chi-squared distance
- and others

These metrics can be used with the pairwise distance computation functions.

Input Validation
----------------

The module includes comprehensive input validation to ensure that arrays
are properly formatted and compatible for distance computations. This includes:

- Checking for proper array dimensions
- Ensuring compatible data types
- Handling sparse matrix formats
- Validating finite values in arrays

The functions in this module are designed to work with both dense NumPy arrays
and sparse matrices, providing flexibility for different data representations.

Performance Considerations
--------------------------

The implementation leverages optimized routines for distance computations,
including specialized code for sparse matrices and parallel processing
capabilities where applicable. The module automatically selects appropriate
data types and computation strategies based on the input characteristics.
