# Cradmin Menu System

The Cradmin menu system provides a flexible and extensible way to create navigation menus in Django Cradmin applications. It uses a component-based approach where menu items are renderable objects that can be easily customized.

## Core Components

### AbstractMenuRenderable

Base class for rendering menus. This class manages a list of renderable menu items and provides methods for adding different types of menu elements.

Key features:
- Manages a collection of menu items through `renderable_list`
- Provides methods for creating different types of menu items
- Supports custom templates for complete HTML customization
- Integrates with Django's request and cradmin instance objects

### Menu Item Components

#### BaseMenuLinkRenderable

Base class for menu link items with the following characteristics:
- Uses BEM methodology for CSS class naming
- Supports active state styling
- Configurable parent BEM block for consistent styling
- Template: `django_cradmin/crmenu/menuitem/link.django.html`

Parameters:
- `label`: Text displayed in the menu
- `url`: Destination URL when clicked
- `is_active`: Boolean indicating if the item should be styled as active

#### NavLinkItemRenderable

Standard navigation link item that extends BaseMenuLinkRenderable. Use this for regular menu links in the main navigation.

#### NavLinkButtonItemRenderable

Button-styled navigation link that extends NavLinkItemRenderable. Renders as a button-style menu item.

#### ExpandableMenuItem

Menu item that supports expandable sub-menus. Features:
- Template: `django_cradmin/crmenu/menuitem/expandable-menu-item.django.html`
- Custom BEM block structure for expandable menus
- Supports nested menu items

#### ExpandableMenuSeparator

Visual separator for expandable menus:
- Template: `django_cradmin/crmenu/menuitem/expandable-menu-separator.django.html`
- Creates visual breaks between menu sections
- Uses BEM naming convention for consistent styling

#### MenuToggleItemItemRenderable

Toggle button for expandable menus:
- Template: `django_cradmin/crmenu/menuitem/menutoggle.django.html`
- Provides responsive menu toggle functionality
- Localized label ("Menu" by default)
- BEM-styled CSS classes for icon and label elements

## Configuration

### Default Expandable Menu

The system supports configurable default expandable menus through the `DJANGO_CRADMIN_DEFAULT_EXPANDABLE_MENU_CLASS` setting. Use the `get_default_expandable_menu_renderable()` function to retrieve the configured default menu.

## Usage

To create a custom menu:

1. Extend `AbstractMenuRenderable`
2. Define appropriate renderable classes for links, separators, and buttons
3. Add items to the menu using the provided methods
4. The menu automatically handles rendering of all contained items

The system provides flexibility for both simple navigation menus and complex hierarchical menus with expandable sections, all while maintaining consistent styling through the BEM methodology.
