Array API Support
=================

scikit-learn has experimental support for the Array API standard, which allows
using scikit-learn with array libraries other than NumPy, such as CuPy, JAX, and PyTorch.

To enable Array API support, you need to set the environment variable
``SKLEARN_ARRAY_API=True`` before importing scikit-learn.

.. code-block:: python

    import os
    os.environ["SKLEARN_ARRAY_API"] = "1"
    import sklearn

Alternatively, you can enable Array API support by calling
:func:`sklearn.set_config` with ``array_api_dispatch=True``.

.. code-block:: python

    import sklearn
    from sklearn import set_config
    set_config(array_api_dispatch=True)

Once enabled, scikit-learn will dispatch array operations to the array library
of the input arrays, provided that the library implements the Array API standard.

Note that Array API support is experimental and may not work with all
scikit-learn estimators or array libraries. Please report any issues you
encounter on the scikit-learn GitHub repository.
