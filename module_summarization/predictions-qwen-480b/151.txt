# General Plotting Routines

This documentation provides an overview of the general plotting routines available in the Masci-tools package, specifically focusing on the Bokeh backend. These routines allow users to create interactive plots with customizable parameters and styles.

## Helper Functions

### `set_bokeh_plot_defaults(**kwargs)`
Set default parameters for the Bokeh backend according to the given keyword arguments. These defaults will be applied to all subsequent plots unless explicitly overridden.

### `reset_bokeh_plot_defaults()`
Reset the Bokeh backend defaults to their hardcoded values.

### `show_bokeh_plot_defaults()`
Display the currently set defaults for the Bokeh backend in a readable format.

### `get_bokeh_help(key)`
Print the description of a specific parameter key used in the Bokeh backend.

### `load_bokeh_defaults(filename='plot_bokeh_defaults.json')`
Load Bokeh plotting defaults from a JSON file. This allows users to save and reuse their preferred plotting configurations.

**Parameters:**
- `filename`: Name of the file from which to load the defaults (default: 'plot_bokeh_defaults.json')

### `save_bokeh_defaults(filename='plot_bokeh_defaults.json', save_complete=False)`
Save the current Bokeh plotting defaults to a JSON file.

**Parameters:**
- `filename`: Name of the file where defaults should be stored (default: 'plot_bokeh_defaults.json')
- `save_complete`: If True, saves both user-modified and unmodified hardcoded defaults (default: False)

## Plotting Functions

### `bokeh_scatter(x, y=None, *, xlabel='x', ylabel='y', title='', figure=None, data=None, saveas='scatter', copy_data=False, **kwargs)`
Create an interactive scatter plot using the Bokeh backend.

**Parameters:**
- `x`: Array-like data or key for the x-axis data
- `y`: Array-like data or key for the y-axis data
- `data`: Data source (e.g., pandas DataFrame) for the plot
- `xlabel`: Label for the x-axis (default: 'x')
- `ylabel`: Label for the y-axis (default: 'y')
- `title`: Title of the figure (default: '')
- `figure`: Existing Bokeh figure to add the plot to (optional)
- `saveas`: Filename for saving the plot (default: 'scatter')
- `copy_data`: If True, creates a copy of the data argument (default: False)

Additional keyword arguments are passed to the `BokehPlotter` or directly to Bokeh's scatter function.

### `bokeh_multi_scatter(x, y=None, *, data=None, figure=None, xlabel='x', ylabel='y', title='', saveas='scatter', copy_data=False, set_default_legend=True, **kwargs)`
Create an interactive scatter plot with multiple datasets using the Bokeh backend.

**Parameters:**
- `x`: Array-like data or key for the x-axis data
- `y`: Array-like data or key for the y-axis data
- `data`: Data source (e.g., pandas DataFrame) for the plot
- `xlabel`: Label for the x-axis (default: 'x')
- `ylabel`: Label for the y-axis (default: 'y')
- `title`: Title of the figure (default: '')
- `figure`: Existing Bokeh figure to add the plot to (optional)
- `saveas`: Filename for saving the plot (default: 'scatter')
- `copy_data`: If True, creates a copy of the data argument (default: False)
- `set_default_legend`: If True, uses data names to generate default legend labels (default: True)

Additional keyword arguments are passed to the `BokehPlotter` or directly to Bokeh's scatter function.

## Usage Notes

- All plotting functions support interactive features through the Bokeh backend
- Default parameters can be customized globally using the helper functions
- Data can be provided in various formats including pandas DataFrames and Bokeh ColumnDataSources
- Plots can be saved automatically using the `saveas` parameter
- Tooltips and legends are automatically handled based on the data structure
- Straight lines and axis limits can be configured through plot parameters

These general plotting routines provide a foundation for creating customizable, interactive visualizations suitable for scientific data analysis and presentation.
