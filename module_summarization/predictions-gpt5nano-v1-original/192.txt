Manifold learning for non-linear dimensionality reduction
=========================================================

Overview
--------

Manifold learning seeks to uncover low-dimensional structures embedded in high-dimensional data. While many approaches aim to flatten data onto a lower-dimensional space, methods based on preserving local or global geometric relationships can reveal meaningful non-linear manifolds. Multidimensional Scaling (MDS) is a classical technique in this family that embeds objects in a low-dimensional space so that pairwise dissimilarities are well represented by Euclidean distances in the embedding. The SMACOF (Scaling by Majorizing a Complicated Function) algorithm provides an efficient, iterative method for performing MDS, including both metric and non-metric variants.

SMACOF-based multidimensional scaling (MDS)
-------------------------------------------

SMACOF minimizes a stress objective that measures the discrepancy between observed dissimilarities and distances in the embedding. The algorithm proceeds iteratively:

- Start from a random or user-provided configuration in the target dimension.
- Compute the current pairwise distances in the embedding.
- If metric MDS is used, treat the observed dissimilarities as the target distances (disparities equal to dissimilarities). If non-metric MDS is used, derive disparities via monotonic regression to preserve the rank order of dissimilarities.
- Compute the stress (fit quality) and update the embedding coordinates with the Guttman transform.
- Repeat until convergence according to a tolerance criterion.

In non-metric MDS, isotonic regression is used to transform the observed dissimilarities into monotone disparities, enabling a faithful representation of the rank structure while still aiming to preserve distances in the embedding.

Parameter highlights
--------------------

Core parameters control the SMACOF MDS run:

- dissimilarities: array-like with shape (n_samples, n_samples). Must be symmetric.
- metric: boolean flag. True for metric MDS; False for non-metric MDS.
- n_components: int, number of embedding dimensions.
- init: array-like of shape (n_samples, n_components) or None. Starting configuration; if None, a random embedding is used.
- max_iter: int, maximum number of SMACOF iterations per run.
- verbose: int, controls verbosity of progress output.
- eps: float, convergence tolerance for the stress change.
- random_state: int, RandomState instance or None, used to initialize the embedding if init is None.
- normalized_stress: bool, if True and metric is False, returns a normalized stress (Stress-1) value. Only supported for non-metric MDS.

Returns
-------

- X: ndarray of shape (n_samples, n_components). Coordinates of the embedded points.
- stress: float. The final value of the stress (a measure of misfit between dissimilarities and distances in the embedding). If normalized_stress is used, returns a normalized stress.
- n_iter: int. The number of iterations performed for the best stress.

Implementation notes
--------------------

- The algorithm ensures the dissimilarities are symmetric.
- Distances in the embedding are computed at each iteration; zeros are avoided in updates to maintain numerical stability.
- Non-metric MDS uses IsotonicRegression to obtain disparities that preserve the rank ordering of the original dissimilarities.
- The approach is rooted in majorization optimization, with references detailing the theoretical foundations and practical performance.
