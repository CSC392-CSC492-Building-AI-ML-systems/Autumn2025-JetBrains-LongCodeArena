Isotonic regression
===================

This module implements isotonic regression and helper utilities to determine monotonicity and to fit isotonic models. It provides a lightweight, dependency-light API for solving isotonic regression problems and for exposing a reusable estimator class.

Functions
---------

check_increasing(x, y)
^^^^^^^^^^^^^^^^^^^^^^^
Determine whether y is monotonically correlated with x.

- Parameters
  - x: array-like of shape (n_samples,)
    Training data.
  - y: array-like of shape (n_samples,)
    Training target.

- Returns
  - increasing_bool: bool
    Whether the relationship is increasing (True) or decreasing (False).

- Details
  The function uses a Spearman correlation test to assess monotonicity. The sign of the Spearman rho determines the result. If the 95% confidence interval (via a Fisher transform) spans zero (and the sample size is sufficient), a warning is issued.

- Reference
  Fisher transformation. See Wikipedia for background.

isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
Solve the isotonic regression model for a one-dimensional target.

- Parameters
  - y: array-like of shape (n_samples,)
    The data to fit.
  - sample_weight: array-like of shape (n_samples,), default=None
    Weights for each sample. If None, weights are set to 1.
  - y_min: float or None
    Lower bound on the minimum predicted value. If None, defaults to -inf.
  - y_max: float or None
    Upper bound on the maximum predicted value. If None, defaults to +inf.
  - increasing: bool, default True
    Whether the fitted function should be increasing (True) or decreasing (False).

- Returns
  - y_: ndarray of shape (n_samples,)
    The isotonic fit of y, respecting the optional bounds.

- Notes
  The implementation uses an in-place isotonic regression routine and applies clipping if bounds are provided.

Class
-----

IsotonicRegression
^^^^^^^^^^^^^^^^^^^
Isotonic regression model with optional bounds and out-of-bounds handling.

- Inheritance
  RegressorMixin, TransformerMixin, BaseEstimator

- Parameters
  - y_min: float, default=None
    Lower bound on the lowest predicted value.
  - y_max: float, default=None
    Upper bound on the highest predicted value.
  - increasing: bool or 'auto', default=True
    Determines whether predictions should be constrained to increase. If 'auto', the decision is based on the sign of the Spearman correlation estimate.
  - out_of_bounds: {'nan', 'clip', 'raise'}, default='nan'
    How to handle input values outside the training domain during prediction:
    - 'nan': predictions will be NaN.
    - 'clip': predictions will be clipped to the nearest interval endpoint.
    - 'raise': a ValueError is raised.

- Attributes
  - X_min_: float
    Minimum value of the input array X_ for the left bound.
  - X_max_: float
    Maximum value of the input array X_ for the right bound.
  - X_thresholds_: ndarray of shape (n_thresholds,)
    Unique ascending X values used to interpolate the monotonic function.
    versionadded: 0.24
  - y_thresholds_: ndarray of ...
    Thresholded y values corresponding to the monotonic fit. (Description provided in the original docstring; see source for details.)

- Examples
  Typical usage involves fitting the model to training data X and y, then transforming or predicting on new X with monotonic constraints. See the User Guide for isotonic regression for full examples.

Notes
-----

- Read more in the User Guide: :ref:`User Guide <isotonic>`.
- The monotonicity constraint is applied with optional bounds (y_min, y_max) and optional out-of-bounds handling during prediction.
- The decision of increasing vs decreasing can be automatically chosen when increasing='auto' based on the sign of the Spearman correlation estimate.

References

Examples
--------

- Basic check_increasing usage:
  .. code-block:: python

     import numpy as np
     x = np.array([0, 1, 2, 3])
     y = np.array([0, 1, 1.5, 3])
     from isotonic import check_increasing
     print(check_increasing(x, y))  # True (increasing)

- Simple isotonic regression:
  .. code-block:: python

     import numpy as np
     y = np.array([3.0, 1.0, 4.0, 2.0])
     from isotonic import isotonic_regression
     y_fit = isotonic_regression(y)

- Using the IsotonicRegression estimator:
  .. code-block:: python

     import numpy as np
     from isotonic import IsotonicRegression
     X = np.linspace(0, 1, 5).reshape(-1, 1)
     y = np.array([0.0, 1.0, 0.5, 1.5, 2.0])
     model = IsotonicRegression(increasing=True)
     model.fit(X, y)
     y_pred = model.predict(X)  # monotonic predictions honoring bounds as configured
