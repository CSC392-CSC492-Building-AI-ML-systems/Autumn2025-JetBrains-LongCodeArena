Ploterr
=======

Ploterr plots generic expectation values and confidence-interval data from an HDF5 dataset.

Subcommands
-----------

generic
    Plots a generic HDF5 dataset containing expectation and CI data.

Overview
--------

The generic subcommand reads a dataset from an HDF5 file in the form FILENAME/PATH[SLICE]. The dataset must be a 1-dimensional (after applying the slice) structured array with the following fields:

- iter_start
- iter_stop
- expected
- ci_lbound
- ci_ubound

The plot shows three lines over iteration:
- expected (black)
- ci_lbound (gray)
- ci_ubound (gray)

The x-axis is derived from iter_stop minus 1. The plot can be produced either as a matplotlib figure (default when a display is available) or as text/terminal output when plotting in a headless environment or when forced with a flag.

Usage
-----

ploterr generic DSSPEC [options]

DSSPEC format: FILENAME/PATH[SLICE]
- FILENAME: path to the HDF5 file
- PATH: dataset path inside the HDF5 file
- SLICE: optional NumPy-style slice (including brackets) to select a vector of data

Example
-------

ploterr generic kinavg.h5/rate_evolution[:,0,1] -o errbars.pdf
ploterr generic data.h5/some_dataset[100:200] -o plot.png --title "Rate evolution" --xlabel "Iteration" --ylabel "Rate"

Options
-------

-o, --output OUTPUT
    Write plot to OUTPUT. The format is inferred from the file extension.
    Default: errbars.pdf

-t, --terminal
    Plot output in the terminal (text interface). Overrides the matplotlib GUI if requested or if no DISPLAY is available.

--xscale {linear, log, symlog}
    Scaling for the x-axis. Default: linear

--yscale {linear, log, symlog}
    Scaling for the y-axis. Default: linear

--xrange XRANGE
    Restrict the x-axis range to XRANGE, formatted as "xmin,xmax". Default: determined by input data.

--yrange YRANGE
    Restrict the y-axis range to YRANGE, formatted as "ymin,ymax". Default: determined by input data.

--xlabel XLABEL
    Label for the x-axis. Default varies.

--ylabel YLABEL
    Label for the y-axis. Default varies.

--title TITLE
    Title for the plot. Default varies.

Notes
-----

- By default, Ploterr uses matplotlib to render plots when a display is available. If no display is available or if -t/--terminal is specified, the output is produced in a text-based interface.
- The output file is created by saving the generated plot to the specified path/filename.
- Data must be a 1D dataset after applying any slice, and must contain the fields iter_start, iter_stop, expected, ci_lbound, and ci_ubound for plotting.
