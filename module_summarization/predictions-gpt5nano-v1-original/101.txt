web3.miner
===========

Overview
--------
This module provides a Pythonic wrapper around the Ethereum node's mining-related JSON-RPC calls. It exposes a Miner class that performs RPC requests via Web3's request manager.

Class: Miner
------------

Constructor
^^^^^^^^^^^
- __init__(web3)
  - web3: Web3 instance used to send RPC requests.

Hashrate
^^^^^^^^
- hashrate (property)
  - Description: The current mining hashrate reported by eth_hashrate, returned as a decimal value.
  - Notes: The value is formatted to decimal using to_decimal.

Methods
^^^^^^^

- makeDAG(number)
  - number: int
  - Returns: RPC result from miner_makeDag for the specified DAG epoch.

- setExtra(extra)
  - extra: str
  - Returns: RPC result from miner_setExtra.

- setEtherBase(etherbase)
  - etherbase: str
  - Returns: RPC result from miner_setEtherbase.

- setGasPrice(gas_price)
  - gas_price: int
  - Returns: RPC result from miner_setGasPrice.

- start(num_threads)
  - num_threads: int
  - Returns: RPC result from miner_start.

- stop()
  - Returns: RPC result from miner_stop.

- startAutoDAG()
  - Returns: RPC result from miner_startAutoDag.

- stopAutoDAG()
  - Returns: RPC result from miner_stopAutoDag.

Usage example
-------------
.. code-block:: python

    from web3 import Web3
    w3 = Web3(Web3.HTTPProvider("http://localhost:8545"))
    miner = w3.miner

    # Check current hashrate
    h = miner.hashrate

    # Start mining with 2 threads
    miner.start(2)

    # Set the etherbase
    miner.setEtherBase("0xYourAddress")

    # Generate DAG for epoch 0
    miner.makeDAG(0)

Notes
-----
- All methods delegate to the node via the Web3 request manager and return the raw RPC response.
- The hashrate property applies a formatter to convert the value to decimal.
