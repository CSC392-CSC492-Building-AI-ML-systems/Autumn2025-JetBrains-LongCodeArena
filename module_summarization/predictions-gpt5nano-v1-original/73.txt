Naive Bayes
===========

The :mod:`sklearn.naive_bayes` module implements Naive Bayes classifiers.
Naive Bayes classifiers are supervised learning methods based on applying Bayes'
theorem with strong (naive) feature independence assumptions.

Overview
--------

Naive Bayes classifiers model the probability of each class given the input
features by combining a prior on the class with the likelihood of the features
under that class. Despite the strong independence assumption, these models are
fast, often effective baseline classifiers, and they can be updated online via
`partial_fit`.

Estimators
----------

The module provides the following naive Bayes estimators:

- BernoulliNB
- GaussianNB
- MultinomialNB
- ComplementNB
- CategoricalNB

Each estimator exposes the standard scikit-learn estimator interface, including:

- fit(X, y): Train the model.
- predict(X): Predict class labels for X.
- predict_proba(X): Predict class probabilities for X.
- predict_log_proba(X): Predict log-probabilities for X.
- predict_joint_log_proba(X): Predict joint log-probabilities log P(x, y).

Base class
----------

The internal abstract base class :class:`_BaseNB` defines common behavior and
the API surface for all Naive Bayes estimators:

- _joint_log_likelihood(X): Compute the unnormalized posterior log-probability for X.
  Implemented by concrete estimators.
- _check_X(X): Validation and feature checks performed by predict methods.
- predict_joint_log_proba(X): Return joint log probabilities for X and each class.
- predict(X): Return predicted class labels.
- predict_log_proba(X): Return log-probabilities for each class.
- predict_proba(X): Return probabilities for each class.

Gaussian Naive Bayes
--------------------

GaussianNB implements the Gaussian Naive Bayes classifier.

Summary
-------

- A probabilistic classifier that assumes features are conditionally Gaussian
  given the class.
- Supports online updates via :meth:`partial_fit`.
- Suitable for continuous data.

Parameters
----------

- priors : array-like of shape (n_classes,), default=None
  Prior probabilities of the classes. If given, priors are not adjusted from the data.

- var_smoothing : float, default=1e-9
  Portion of the largest variance of all features that is added to variances for
  calculation stability.

Attributes
----------

- class_count_ : ndarray of shape (n_classes,)
  Number of training samples observed in each class.

- class_prior_ : ndarray of shape (n_classes,)
  Probability of each class.

- classes_ : ndarray of shape (n_classes,)
  Class labels known to the classifier.

- epsilon_ : float
  Absolute additive value to variances.

- n_features_in_ : int
  Number of features seen during fit.

- feature_names_in_ : ndarray of shape (n_features_in_,), optional
  Names of features seen during fit. Defined only if X has feature names that are
  all strings.

- var_ : ndarray of shape (n_classes, n_features)
  Variance of each feature per class.

- theta_ : ndarray of shape (n_classes, n_features)
  Mean of each feature per class.

See Also
--------

- BernoulliNB
- MultinomialNB

Examples
--------

Basic usage:

.. code-block:: python

   from sklearn.naive_bayes import GaussianNB
   clf = GaussianNB()
   clf.fit(X_train, y_train)
   y_pred = clf.predict(X_test)
   proba = clf.predict_proba(X_test)

Notes
-----

- Naive Bayes classifiers are simple and fast and can perform well with small
  datasets. Their independence assumption is often violated in practice, but they
  still provide competitive performance as baseline models.

See Also
--------

- The concrete estimator implementations (BernoulliNB, MultinomialNB, ComplementNB,
