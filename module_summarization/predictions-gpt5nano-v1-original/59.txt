Quantile Regression (QuantReg)
==============================

Overview
--------
QuantReg implements a quantile regression model estimated via iterative reweighted least squares (IRLS). The asymptotic covariance matrix of the parameter estimates is computed using kernel density estimation.

This is a wrapper compatible with StatsModels regression models and follows the Quantile Regression framework described by Koenker and others. The implementation supports estimation of coefficients for a given quantile q and provides robust covariance estimation for inference.

Class
-----
QuantReg
  - Inherits from: RegressionModel
  - Purpose: Estimate a quantile regression model using IRLS.

Parameters
----------
endog : array-like or DataFrame
    Endogenous (response) variable.
exog : array-like or DataFrame
    Explanatory (design) matrix.

Notes
-----
- The Least Absolute Deviation (LAD) estimator corresponds to q = 0.5.
- The asymptotic covariance matrix is estimated via kernel density estimation (kernel specified by the fit method).
- This module includes references to foundational texts in quantile regression and its kernels.

fit method
----------
fit(q=0.5, vcov='robust', kernel='epa', bandwidth='hsheather',
    max_iter=1000, p_tol=1e-6, **kwargs)

Parameters
----------
q : float, default 0.5
    Quantile to estimate (0 <= q <= 1).
vcov : {'robust', 'iid'}, default 'robust'
    Method to compute the variance-covariance matrix of the parameters.
    - 'robust': heteroskedasticity-robust standard errors (Greene framework).
    - 'iid': iid errors standard errors (as in some econometric implementations).
kernel : str, default 'epa'
    Kernel function used in the kernel density estimation for the covariance.
    Supported options include: 'biw', 'cos', 'epa', 'gau', 'par' (as mapped internally).
bandwidth : {'hsheather', 'bofinger', 'chamberlain'}, default 'hsheather'
    Bandwidth selection method for the kernel density estimation.
max_iter : int, default 1000
    Maximum number of IRLS iterations.
p_tol : float, default 1e-6
    Convergence tolerance for the IRLS iterations.

Details
-------
- The model is configured by the endog/exog data provided at construction.
- The implementation starts from an initial OLS-like solution and iterates
  until convergence or until max_iter is reached.
- The method returns a RegressionResults-like object containing estimated
  coefficients and inference information.

Usage example
-------------
.. code-block:: python

from statsmodels.regression.quant_reg import QuantReg

model = QuantReg(endog, exog)
res = model.fit(q=0.25, vcov='robust', kernel='gau', bandwidth='hsheather')
print(res.summary())

References
----------
- Greene, W. H. Econometric Analysis (6th Edition).
- Koenker, R. Quantile Regression.
- LeSage, J. P. Applied Econometrics Using MATLAB.
- Hall, P. & Sheather, S. (1988). On the distribution of the Studentized quantile.
- Bofinger, E. (1975). Estimation of a density function using order statistics.
- Chamberlain, G. (1994). Quantile regression, censoring, and wages.

Notes
-----
The module originates from a Python port of the Matlab implementation and
leverages the StatsModels regression framework for integration with
the broader modeling ecosystem. The end-to-end workflow involves selecting
a quantile, choosing a kernel and bandwidth for covariance estimation,
and interpreting the resulting coefficient estimates and their standard errors.
