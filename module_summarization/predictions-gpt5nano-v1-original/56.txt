MixedGLM
==========
Bayesian inference for generalized linear mixed models.

Overview
--------
This module implements Bayesian inference for generalized linear mixed models (GLMMs). Currently only families without additional scale or shape parameters are supported (binomial and Poisson). Two estimation approaches are supported: Laplace approximation (maximum a posteriori) and variational Bayes (mean field approximation to the posterior distribution). All realizations of random effects are modeled to be mutually independent in this implementation.

The design matrix for the random effects is provided via exog_vc. Each column of exog_vc corresponds to a distinct random effect realization. The random effects have mean zero with an unknown standard deviation. The standard deviation parameters are constrained to be equal within subsets of the columns, determined by ident. When formulas are used, columns derived from a common formula are constrained to share the same standard deviation.

In many applications, exog_vc will be sparse. A sparse matrix may be passed when constructing a model class; if a dense matrix is passed, it will be converted internally to a sparse matrix. There is currently no way to avoid creating a temporary dense version of exog_vc when using formulas.

Model and parameterization
--------------------------
The joint density of data and parameters factors as:

.. math::

    p(y | vc, fep) p(vc | vcp) p(vcp) p(fe)

The terms p(vcp) and p(fe) are Gaussian priors (the vcp parameters are log standard deviations, so their priors are log-normal). The random effects distribution p(vc | vcp) is independent Gaussian (random effect realizations are independent within and between values of ident). The model p(y | vc, fep) depends on the specific GLM being fit.

Notes
-----
- The class implements two fitting approaches:
  - fit_map: Laplace approximation to the posterior (maximum a posteriori).
  - fit_vb: Variational Bayes approximation to the posterior (mean-field).
  See the corresponding method docstrings for details.
- The random effects are grouped by ident. Two random effect columns with the same ident value share the same standard deviation.
- When using formulas, ident is constructed internally (each element of vc_formulas yields a distinct label in ident).
- The exog_vc matrix can be passed as a SciPy sparse matrix; internal conversion to sparse is performed if necessary.
- If formulas are used, a temporary dense version of exog_vc may be created during processing.

Parameters
----------
endog : array_like
    Vector of response values.
exog : array_like
    Covariates for the fixed effects part of the mean structure.
exog_vc : array_like
    Covariates for the random part of the model. May be SciPy sparse; otherwise, it will be converted to sparse internally.
ident : array_like
    Integer labels showing which random terms (columns of exog_vc) share a common variance.
vcp_p : float
    Prior standard deviation for variance component parameters (the prior for the log of the standard deviation of a random effect has standard deviation vcp_p).
fe_p : float
    Prior standard deviation for fixed effects parameters.
family : statsmodels.genmod.families instance
    The GLM family (e.g., Binomial, Poisson).
fep_names : list of strings
    Names of the fixed effects parameters (corresponding to columns of exog). If None, default names are constructed.
vcp_names : list of strings
    Names of the variance component parameters (corresponding to distinct labels in ident). If None, default names are constructed.
vc_names : list of strings
    Names of the random effect realizations.

Returns
-------
MixedGLMResults object

Remarks
-------
There are three types of values in the posterior distribution:
- fep: fixed effects parameters (corresponding to exog columns)
- vc: random effects realizations (corresponding to exog_vc columns)
- vcp: standard deviations of the random effect realizations (corresponding to unique labels in ident)

All random effects are modeled as independent Gaussian values (given the variance structure parameters). Each column of exog_vc has its own realized random effect used to form the linear predictors. The elements of ident determine the distinct variance structure parameters. Two random effect realizations with the same ident value share the same variance. When fitting with a formula, ident is constructed internally (each element of vc_formulas yields a distinct label in ident).

The random effect standard deviation parameters (vcp) have log-normal priors with mean 0 and standard deviation vcp_p.

Note that for some families (e.g., Binomial), the posterior mode may be difficult to locate numerically if vcp_p is set too large. A value around 0.5 often works well.

The prior for the fixed effects parameters is Gaussian with mean 0 and standard deviation fe_p.
