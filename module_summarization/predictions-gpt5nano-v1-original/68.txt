Gaussian Process Kernels
========================

Kernels for Gaussian process regression and classification.

This module provides kernels that can be combined to form expressive kernel-engineering
expressions. Kernels can be added, multiplied, or exponentiated with scalars, and scalar
values are automatically converted to a constant kernel. All kernels support analytic
gradient-based hyperparameter optimization. Hyperparameters can be restricted by bounds, or
declared fixed (excluded from optimization).

Public API
----------

- Hyperparameter
  - A specification for a kernel hyperparameter.
  - Fields: name, value_type, bounds, n_elements, fixed.
  - Bounds can be a pair of floats (or arrays for vector-valued hyperparameters) or the
    string "fixed" (to indicate the parameter is not tunable).
  - Example usage (as described in the class docstring):
    - Hyperparameter(name='constant_value', value_type='numeric',
                    bounds=array([[0., 10.]]), n_elements=1, fixed=False)

- Kernel
  - Abstract base class for all kernels.
  - Kernels can be combined via the standard arithmetic operators:
    - Addition for summing kernels
    - Multiplication for product kernels
    - Exponentiation with a scalar to scale a kernel (or to form a constant-exponent kernel)
  - All kernels expose get_params for parameter inspection and compatibility with hyperparameter optimization
  - (The implementation supports cloning and parameter introspection via get_params.)

Internal helpers
----------------

- _check_length_scale(X, length_scale)
  - Validates and sanitizes the length_scale parameter.
  - Ensures anisotropic length scales (if provided) match the data dimensionality.
  - Returns a 1D array of length scales.

Notes
-----

- The module is designed to be compatible with gradient-based hyperparameter optimization.
- It draws inspiration from established kernel implementations and integrates with the surrounding GP framework.

Usage overview (conceptual)
---------------------------

- Define kernels and combine them:
  - K = RBF(length_scale=1.0) + WhiteKernel(noise_level=1e-2)
  - K = (SEKernel(length_scale=[1.0, 2.0]) * PeriodKernel(period=3.0))
  - K_scaled = K ** 0.5  # scale by a scalar via exponentiation
- Configure hyperparameters with bounds or mark as fixed to control optimization.
- Retrieve parameters with get_params for inspection or integration with optimization routines.
