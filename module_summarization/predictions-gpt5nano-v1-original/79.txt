GILDAS CLASS file reader
=========================

Read a CLASS file into an :class:`pyspeckit.spectrum.ObsBlock`.

Overview
--------

This module provides facilities to read a GILDAS CLASS file and convert its contents
into a PySpeckit ObsBlock for further analysis. It includes mappings for CLASS file
types and versions, header field definitions, and utilities to assist with byte/string
handling and timing.

Public API
----------

Functions
---------

- ``print_timing(func)``  
  Decorator that prints execution time of the decorated function. Useful for long CLASS file reads.

- ``ensure_bytes(string)``  
  Ensure a given string is represented as bytes (compatible with Python 2/3).

Constants
---------

- ``filetype_dict``  
  Mapping from CLASS filetype codes to a string label describing the file type
  (e.g., 'Multiple_IEEE', 'Single_Vax', etc.). Populated for both byte and text
  keys to support Python 2/3 compatibility.

- ``fileversion_dict``  
  Mapping from CLASS file version codes to a normalized version name
  (e.g., 'v1', 'v2'). Also extended to support both byte and text keys.

- ``record_lengths``  
  Dictionary of fixed record lengths per CLASS record type (e.g., '1A', '2A').

- ``header_id_numbers``  
  Mapping of numeric header IDs to human-readable section names
  (e.g., 0: 'USER CODE', -1: 'COMMENT', -2: 'GENERAL', -4: 'SPECTRO', etc.).

- ``header_id_lengths``  
  Mapping of certain header IDs to their field lengths. Note that some IDs have
  variable-length headers.

- ``filedescv2_nw1``  
  Constant related to the CLASS v2 file descriptor structure (value set to 14).

- ``keys_lengths``  
  Dictionary describing the layout of header keywords for different sections.
  Keys include:
  - 'unknown' (with a list of fields like 'NUM', 'VER', 'TELES', 'DOBS', etc.)
  - 'COMMENT' (fields for the comment text)
  - 'GENERAL' (fields for time, position, and other general metadata)
  The values specify (field_name, length, data_type) triplets.

Notes
-----

- The module includes The SPECIFICATION block for CLASS headers, including a
  GENERAL section description with UT, ST, AZ, EL, TAU, TSYS, TIME, etc.
- It is designed to assist in parsing CLASS files and converting to ObsBlock
  objects for use with pyspeckit.

Usage example (conceptual)
--------------------------

1. Import and read a CLASS file:
   - Use the module to parse a CLASS file and convert its contents into an
     ObsBlock for further spectral analysis.

2. Access resulting metadata and spectra via standard ObsBlock interfaces.
