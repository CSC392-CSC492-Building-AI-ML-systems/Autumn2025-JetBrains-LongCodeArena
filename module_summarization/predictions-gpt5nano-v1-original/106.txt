Hyperexpand
===========

Expand Hypergeometric (and Meijer G) functions into named special functions.

Overview
- This module programmatically constructs a Sphinx-ready docstring that renders a collection of hypergeometric formulae as mathematical blocks.

What it does
- Imports the core building blocks (Eq), the hypergeometric function representation (hyper), and a LaTeX pretty-printer (latex).
- Uses FormulaCollection to obtain a set of known formulae.
- For each formula, builds an equation of the form:
  Eq(hyper(ap, bq, z), closed_form(z).rewrite('nonrepsmall'))
  and renders it as a LaTeX math block, appending it to the module docstring.
- Assigns the accumulated content to __doc__, so importing the module yields a ready-to-display set of mathematical expressions in Sphinx.

Public API (highlights)
- FormulaCollection: collection of known hypergeometric representations used to generate documentation.
- hyper: representation of a hypergeometric function.
- latex: converts an expression to LaTeX for inclusion in Sphinx math blocks.
- Eq: symbolic equation used to express equality between hypergeometric forms and their closed forms.

How it works (high level)
- Iterate over c.formulae from FormulaCollection.
- For each f, construct an equality between a hyper function and its closed-form representation.
- Convert that equality to LaTeX and wrap it in a Sphinx math directive (.. math::).
- Concatenate all blocks into the module’s __doc__ for display in documentation.

Notes and references
- The approach is inspired by Meijer G function representations and related hypergeometric transformations.
- References to the Roach (1997) paper on Meijer G function representations are relevant for the underlying theory.

Usage example
- Import the module (as part of the documentation workflow, not for runtime use in normal code):
  from hyperexpand import FormulaCollection
  c = FormulaCollection()
  doc = ''
  for f in c.formulae:
      obj = Eq(hyper(f.func.ap, f.func.bq, f.z),
               f.closed_form.rewrite('nonrepsmall'))
      doc += f'.. math::\n  {latex(obj)}\n'
  __doc__ = doc
