Plothist
=========

PlotHist command for WestPA histogram data

Overview
--------

PlotHist reads histogram data from a WestPA HDF5 file and renders 1-D or 2-D plots of the specified histogram dimension. It supports various scaling and range options, optional contour overlay for 2-D plots, and an optional post-processing step before saving.

Usage
-----

plothist [options] input [FIRSTDIM]

Where:
- input: HDF5 file containing histogram data
- FIRSTDIM: optional dimension specification (see Dimensions)

Dimensions
-----------

Plot for the given DIMENSION, specified as INT[:[LB,UB]:LABEL], where
INT is a zero-based integer identifying the dimension in the histogram,
LB and UB are lower and upper bounds for plotting, and LABEL is the label for
the plot axis. (Default: dimension 0, full range.)

Options
-------

Input options
- input: HDF5 file containing histogram data
- FIRSTDIM (optional): dimension specification described above
  - If not provided, defaults to dimension 0 with the full range

Output options
- -o, --output, --plot-output: path to store the plot file (default: hist.pdf)
  - If set to an empty string, plotting is suppressed
  - The output format is inferred from the filename extension (default PDF)
- --hdf5-output: store plot data in the specified HDF5 file
- --plot-contour: overlay a contour plot on top of the heatmap for 2-D objects

Plot options
- --title: include TITLE as the top-of-graph title
- --linear: plot the histogram on a linear scale
- --energy: plot on an inverted natural-log scale (default; corresponds to free energy)
- --zero-energy: set the zero of energy; value can be a scalar or "min"/"max" (default: "min")
- --log10: plot the histogram on a base-10 log scale
- --range: plotting range, specified as "LB,UB"
  - For 1-D plots, this is the Y axis
  - For 2-D plots, this is the colorbar axis
  - Default is the full range
- --postprocess-function: name of a function (module.function) to be called just before saving the plot
  - Function signature: postprocess(hist, midpoints, binbounds)
  - The function may modify the histogram in place using the pyplot interface

Defaults
--------

- Plotscale defaults to energy (via plotscale='energy')
- Enerzero defaults to min (via enerzero='min')

Notes
-----

- The command uses the WESTPA plotting framework and Matplotlib for rendering.
- Iteration ranges for averaging/evolution and iteration-specific plots are supported internally (not detailed here).
- Energy-based scaling uses an inverted natural-log interpretation aligned with free-energy plots.

Examples
--------

- Basic: plothist input.h5 0
- With a label and restricted range: plothist input.h5 0:[-5,5]:Density --range "-5,5" --title "Density (0th dim)"
- Overlay contour on 2-D: plothist input.h5 1 --plot-contour --output heatmap_with_contour.png
- Save to HDF5 and suppress the plot: plothist input.h5 0 --plot-output='' --hdf5-output results.h5

See Also
--------

- h5io and normhistnd utilities in WestPA for histogram data handling
- WESTMasterCommand and WESTSubcommand infrastructure for WestPA CLI tools
