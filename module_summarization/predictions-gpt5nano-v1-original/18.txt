w_bins
======

Name
----
w_bins - Display information and statistics about binning in a WEST simulation, or modify the binning for the current iteration of a WEST simulation.

Synopsis
--------
w_bins info [options]
w_bins rebin [options]

Description
-----------
w_bins uses the WestPA BinMappingComponent to report on current binning or to rebuild the current iteration with a revised binning. It reads simulation data through WESTDataReader and can print detailed per-bin information as well as summary statistics.

Subcommands
-----------

info
    Display information about binning.

    Options:
      -n, --n-iter N_ITER
          Consider initial points of segment N_ITER (default: current iteration).

      --detail
          Display detailed per-bin information in addition to summary information.

      Bin mapping options
          Additional options provided by the BinMappingComponent for specifying binning (e.g., bins, from file, etc.).

rebin
    Rebuild current iteration with new binning.

    Options:
      --confirm
          Commit the revised iteration to HDF5; without this option, the effects of the new binning are only calculated and printed.

      --detail
          Display detailed per-bin information in addition to summary information.

      -n, --n-iter N_ITER
          Consider initial points of segment N_ITER (default: current iteration).

      Bin mapping options
          BinMappingComponent options for configuring the new binning; for this command, the option to read bins from a file is suppressed.

      --target-counts
          (Provided by BinMappingComponent) Configure target bin counts for the new binning.

Notes
-----
- When rebining, target counts may be required.
- The rebin operation can print information before optionally writing changes to the HDF5 store if --confirm is used.

Behavior
--------
- info prints the binning information for the specified iteration and writes per-bin details if --detail is provided.
- rebin rebuilds the current iteration using the (possibly updated) binning, computes new weights and assignments, and can persist changes to disk with --confirm.

Examples
--------
- Display basic binning information for the current iteration:
  w_bins info

- Display detailed binning information for iteration 5:
  w_bins info -n 5 --detail

- Rebin the current iteration with new binning and show details:
  w_bins rebin --detail -n 5

- Rebin and commit changes for iteration 5:
  w_bins rebin --confirm -n 5 --detail

See Also
--------
westpa, w_init, and other WEST tool commands.
