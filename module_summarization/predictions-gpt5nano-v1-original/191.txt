Validation curves
=================

Validation curves: plotting scores to evaluate models
----------------------------------------------------

Overview
~~~~~~~~
Validation curves help you understand how a model’s performance changes as a specific hyperparameter is varied. By plotting both training and cross-validated scores across a range of parameter values, you can diagnose issues such as overfitting, underfitting, or whether a parameter has little effect on performance.

Computing validation curves
~~~~~~~~~~~~~~~~~~~~~~~~~~
The validation_curve function computes training and cross-validated scores for a range of parameter values.

Syntax
~~~~~~~
sklearn.model_selection.validation_curve(
    estimator,
    X,
    y,
    param_name,
    param_range,
    *,
    scoring=None,
    cv=None,
    n_jobs=None,
    verbose=0,
    return_train_score=True
)

Parameters
~~~~~~~~~~
- estimator: Estimator object implementing fit and predict.
- X, y: Training data.
- param_name: The parameter of the estimator to be varied.
- param_range: The values of the parameter to evaluate.
- scoring: A string or callable to evaluate the predictions on the validation set. If None, uses the estimator’s default scorer.
- cv: Cross-validation strategy. If None, 5-fold cross-validation is used by default.
- n_jobs: Number of jobs to run in parallel.
- verbose: Controls verbosity during execution.
- return_train_score: Whether to return training scores along with validation scores.

Returns
~~~~~~~
train_scores, test_scores: Arrays of shape (n_param_values, n_cv_folds) containing training and validation scores respectively.

Plotting and displaying
~~~~~~~~~~~~~~~~~~~~~~
You can plot the mean (and optionally the standard deviation) of the training and cross-validated scores across param_range. The sklearn plotting utilities also provide a ValidationCurveDisplay to visualize these results conveniently.

Examples
--------
1) Basic validation curve with a decision tree classifier

.. code:: python

    from sklearn.datasets import load_iris
    from sklearn.tree import DecisionTreeClassifier
    from sklearn.model_selection import validation_curve
    import numpy as np
    import matplotlib.pyplot as plt

    X, y = load_iris(return_X_y=True)
    param_range = np.arange(1, 11)

    train_scores, test_scores = validation_curve(
        DecisionTreeClassifier(), X, y,
        param_name="max_depth",
        param_range=param_range,
        cv=5,
        scoring="accuracy",
        n_jobs=-1,
    )

    train_mean = train_scores.mean(axis=1)
    train_std = train_scores.std(axis=1)
    test_mean = test_scores.mean(axis=1)
    test_std = test_scores.std(axis=1)

    plt.plot(param_range, train_mean, label="Training score")
    plt.fill_between(param_range, train_mean - train_std, train_mean + train_std, alpha=0.2)

    plt.plot(param_range, test_mean, label="Cross-validation score")
    plt.fill_between(param_range, test_mean - test_std, test_mean + test_std, alpha=0.2)

    plt.xlabel("max_depth")
    plt.ylabel("Accuracy")
    plt.legend(loc="best")
    plt.title("Validation Curve for DecisionTreeClassifier")
    plt.show()

2) Using ValidationCurveDisplay for a higher-level approach

.. code:: python

    from sklearn.datasets import load_iris
    from sklearn.tree import DecisionTreeClassifier
    from sklearn.model_selection import ValidationCurveDisplay
    import numpy as np

    X, y = load_iris(return_X_y=True)
    param_range = np.arange(1, 11)

    display = ValidationCurveDisplay.from_estimator(
        DecisionTreeClassifier(),
        X, y,
        param_name="max_depth",
        param_range=param_range,
        cv=5,
        scoring="accuracy",
        n_jobs=-1,
        xlabel="max_depth",
        ylabel="Score",
        name=None,
    )
    display.plot()
    display.ax_.legend(loc="best")
    display.ax_.set_title("Validation Curve for DecisionTreeClassifier")
    plt.show()

Interpreting the curves
~~~~~~~~~~~~~~~~~~~~~~
- If the training score remains high and the cross-validated score is low across parameter values, the model may be underfitting.
- If the training score is high but the cross-validated score decreases as the parameter grows, the model may be overfitting or becoming too complex.
- The ideal parameter values often lie where the cross-validated score is maximized with reasonable training performance.

~~~~~~~~
