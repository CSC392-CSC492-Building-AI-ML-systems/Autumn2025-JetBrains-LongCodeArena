Isotonic regression
===================

Isotonic regression fits a non-decreasing (or non-increasing) function to data under optional bounds. It is implemented as a standalone function and as a small estimator class with a familiar scikit-learn interface.

check_increasing
----------------

Determine whether y is monotonically correlated with x using a Spearman correlation test.

Parameters
- x: array-like of shape (n_samples,)
  Training data.
- y: array-like of shape (n_samples,)
  Training target.

Returns
- increasing_bool: bool
  Whether the relationship is increasing (True) or decreasing (False).

Notes
- The Spearman correlation coefficient is estimated from the data, and the sign of the estimate is used as the result.
- If the Fisher transform-based 95% confidence interval spans zero (and there are more than 3 samples), a warning is issued:
  “Confidence interval of the Spearman correlation coefficient spans zero. Determination of ``increasing`` may be suspect.”

References
- Fisher transformation. Wikipedia. https://en.wikipedia.org/wiki/Fisher_transformation

isotonic_regression
-------------------

Solve the isotonic regression model for y.

Parameters
- y: array-like of shape (n_samples,)
  The data.
- sample_weight: array-like of shape (n_samples,), default=None
  Weights on each point of the regression. If None, weights are equal.
- y_min: float or None, default=None
  Lower bound on the lowest predicted value (minimum value may be higher). If None, defaults to -inf.
- y_max: float or None, default=None
  Upper bound on the highest predicted value (maximum may be lower). If None, defaults to +inf.
- increasing: bool, default=True
  Whether the fitted y_ should be increasing (True) or decreasing (False).

Returns
- y_: ndarray of shape (n_samples,)
  Isotonic fit of y.

Notes
- If y_min or y_max are provided, the fitted values are clipped to these bounds after fitting.
- This function performs the isotonic regression in the specified order (increasing or decreasing).

References
- Best, M. J., & Chakravarti, N. (Section 3). Active set algorithms for isotonic regression; a unifying framework.

IsotonicRegression
------------------

Isotonic regression estimator with a scikit-learn-like interface.

Parameters
- y_min: float, default=None
  Lower bound on the lowest predicted value (minimum may still be higher). If not set, defaults to -inf.

- y_max: float, default=None
  Upper bound on the highest predicted value (maximum may still be lower). If not set, defaults to +inf.

- increasing: bool or 'auto', default=True
  Determines whether the predictions should be constrained to increase with X.
  If 'auto', the decision is based on the sign of the Spearman correlation estimate.

- out_of_bounds: {'nan', 'clip', 'raise'}, default='nan'
  How to handle X values outside the training domain during prediction:
  - 'nan': predictions will be NaN.
  - 'clip': predictions are set to the nearest train interval endpoint.
  - 'raise': a ValueError is raised.

Attributes
- X_min_: float
  Minimum value of the input array X_ for the left bound.

- X_max_: float
  Maximum value of the input array X_ for the right bound.

- X_thresholds_: ndarray of shape (n_thresholds,)
  Unique ascending X values used to interpolate the monotonic function y = f(X).
  (Added in a later version; used to describe the decision thresholds for prediction.)

- y_thresholds_: ndarray
  Predicted values corresponding to X_thresholds_ that define the isotonic fit.

Notes
- The estimator supports fitting with a 1-D input X and 1-D target y, with optional weights and bounds, and can predict on new X values using the monotonic fit.
- The 'auto' mode for increasing selects the monotonic direction automatically based on the data.

See also
- isotonic_regression for a lightweight, non-estimator isotonic fit with optional bounds.

References
- Best, M. J., & Chakravarti, N. Active set algorithms for isotonic regression; A unifying framework.
