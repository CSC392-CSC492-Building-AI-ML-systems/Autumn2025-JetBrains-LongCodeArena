MapPlot
-------

Make plots of the cube and interactively connect them to spectrum plotting. This is an interactive component of the package and is intended for user analysis rather than publication-quality plots. The MapPlotter uses APLpy when available to generate publication-quality color maps, but remains usable without it for quick exploration.

Dependencies
- matplotlib
- numpy
- astropy.wcs (or pywcs) and astropy.io.fits (or pyfits)
- APLpy (optional; enables enhanced plotting with FITS figures)
- The cubes package (for header handling)

Overview
- The MapPlotter class provides an interactive map viewer for a 3D data cube (a "cube"). It can display a 2D map derived from a plane of the cube and allow interactive operations to inspect spectra, overplot additional spectra, or measure circular regions.

Class: MapPlotter
- Purpose: Create and manage an interactive map figure from a cube.
- Key attributes:
  - Cube: The input 3D data cube (or cube-like object). If provided, its header is flattened and used to set up WCS.
  - header: A flattened FITS header derived from the cube.
  - wcs: World Coordinate System information (if available).
  - figure / axis: Matplotlib figure and axis used for plotting.
  - FITSFigure: An APLpy figure wrapper when APLpy is available.
  - plane: The 2D data plane selected from the cube for plotting.
  - _origin: WCS origin (0 for FITS convention, 1 for PIXEL convention if applicable).
  - Interaction state: Click marks, overplotted spectra, etc.

Initialization
- __init__(self, Cube=None, figure=None, doplot=False, **kwargs)
  - Cube: 3D data cube (or cube-like object) with a header attribute.
  - figure: Matplotlib Figure, an integer fig number, or None (auto-create).
  - doplot: If True, immediately generate and display the map via mapplot.
  - Other keyword arguments are stored for later use by mapplot.

Public API
- __call__(self, **kwargs)
  - Alias for mapplot; allows the object to be called as a function to display/update the map.

- mapplot(self, convention='calabretta', colorbar=True, useaplpy=True,
            vmin=None, vmax=None, cmap=None, plotkwargs={}, **kwargs)
  - Purpose: Plot a map derived from the input cube.
  - Parameters:
    - convention: Projection convention for Galactic data ('calabretta' or 'griesen').
    - colorbar: Show a colorbar alongside the map.
    - useaplpy: If True and a FITS header is available, use APLpy for plotting (preferred when available).
    - vmin, vmax: Color scale limits. If None, determined from the data.
    - cmap: Colormap for the map.
    - plotkwargs: Additional keyword arguments passed to APLpy’s show_colorscale or matplotlib.imshow.
    - **kwargs: Passed to the map creation routine (e.g., estimator, or other options).
  - Behavior:
    - Creates or reuses a figure, builds the 2D plane from the cube, and renders it.
    - Enables interactive features: click to inspect spectra, overplot spectra, and interact with a circular aperture.
    - If APLpy is available and selected, uses its FITSFigure interface; otherwise falls back to matplotlib imshow.
  - Returns: None (modifies internal state and displays the map).

- _connect(self)
  - Sets up event connections for mouse clicks, releases, and key presses to enable interactivity (spectrum plotting, overplotting, and region selection).

- _disconnect(self)
  - Removes event connections to gracefully disable interactivity.

Notes
- The module handles optional dependencies gracefully. If APLpy or certain FITS/WCS libraries are unavailable, it falls back to a standard Matplotlib-based plotting path.
- The Cube parameter is expected to provide a header attribute compatible with the code path that flattens and uses the header for WCS handling.
- The MapPlotter is designed as an interactive helper for exploratory data analysis of spectral cubes, rather than for final publication plots.

Example
- Basic usage (assuming a cube object named mycube exists and contains a 3D data array with a header):
  - from cubes import MapPlotter
  - mp = MapPlotter(Cube=mycube, figure=1, doplot=True)
  - mp()  # or mp.mapplot()

- Without APLpy (fallback path):
  - mp = MapPlotter(Cube=mycube, figure=1, doplot=True)
  - mp.mapplot(useaplpy=False)

References
- The implementation relies on the cubes package for header handling (e.g., cubes.flatten_header) and on optional APLpy and Astropy components for advanced plotting and WCS support.
