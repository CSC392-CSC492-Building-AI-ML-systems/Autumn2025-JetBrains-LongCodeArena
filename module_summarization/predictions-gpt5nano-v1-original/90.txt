AutoAugment (nvidia.dali.auto_aug.auto_augment)
===============================================

Overview
--------
This module provides AutoAugment policies for data augmentation within NVIDIA DALI.
It exposes a high-level function auto_augment that applies one of several predefined
policies to a batch of samples (images or videos) and a convenience wrapper
auto_augment_image_net for the common image_net policy.

The AutoAugment policies are inspired by the AutoAugment paper
“AutoAugment: Learning Augmentation Policies from Data” (arXiv:1805.09501).

API
---

auto_augment
------------

Signature
- auto_augment(data: DataNode,
               policy_name: str = 'image_net',
               shape: Optional[Union[DataNode, Tuple[int, int]]] = None,
               fill_value: Optional[int] = 128,
               interp_type: Optional[types.DALIInterpType] = None,
               max_translate_abs: Optional[int] = None,
               max_translate_rel: Optional[float] = None,
               seed: Optional[int] = None) -> DataNode

Description
Applies one of the predefined policies from the AutoAugment paper to the provided batch of samples.

Parameters
- data: DataNode
  A batch of samples to be processed. Supported: images with HWC layout and videos with FHWC layout, dtype uint8.

- policy_name: str
  Name of the predefined policy. Acceptable values: 'image_net', 'reduced_image_net',
  'svhn', 'reduced_cifar10'. Defaults to 'image_net'.

- shape: DataNode or Tuple[int, int], optional
  The target height/width for the images/frames. If provided, translation magnitudes
  depend on the shape and span from 0 to max_translate_rel * shape. Otherwise,
  magnitudes are bounded by max_translate_abs.

- fill_value: int, optional
  Value used for padding in warp_affine operations (translation, shear, rotate). If None,
  padding uses the border value (clamped).

- interp_type: types.DALIInterpType, optional
  Interpolation method for warp_affine ops. Supported: types.INTERP_LINEAR, types.INTERP_NN.

- max_translate_abs: int or (int, int), optional
  Maximum translation when shape is not provided. If a tuple, first component is height, second is width.
  Default behavior uses a magnitude up to 250.

- max_translate_rel: float or (float, float), optional
  Maximum translation as a fraction of the image shape when shape is provided. If a tuple, first
  component corresponds to height, second to width. Default is 1 (can shift out of canvas).

- seed: int, optional
  Seed for random operations, ensuring reproducibility.

Returns
- DataNode
  A batch of transformed samples.

Notes
- The policy_name must be one of the supported values. If a policy without translation is chosen
  (currently 'reduced_image_net'), providing shape-related arguments will raise an error.
- If numpy is not available, a runtime error is raised (numpy is required by the examples).

Policies
--------
- image_net
- reduced_image_net
- svhn
- reduced_cifar10

Policies without translation
- reduced_image_net (no translation-based augmentations)

auto_augment_image_net
----------------------
Signature
- auto_augment_image_net(data: DataNode,
                         shape: Optional[Union[DataNode, Tuple[int, int]]] = None,
                         fill_value: Optional[int] = 128,
                         interp_type: Optional[types.DALIInterpType] = None,
                         max_translate_abs: Optional[int] = None,
                         max_translate_rel: Optional[float] = None,
                         seed: Optional[int] = None) -> DataNode

Description
Convenience wrapper that applies the image_net policy via auto_augment.

Equivalent to:
auto_augment(data, policy_name='image_net', shape, fill_value, interp_type,
             max_translate_abs, max_translate_rel, seed)

Examples
--------

Basic usage:

.. code-block:: python

   import nvidia.dali as dali
   from nvidia.dali.auto_aug.auto_augment import auto_augment

   # data is a DataNode containing a batch of images (HWC, uint8)
   transformed = auto_augment(data, policy_name='image_net')

Using shape to scale translation magnitudes:

.. code-block:: python

   transformed = auto_augment(data,
                              policy_name='image_net',
                              shape=(224, 224),
                              max_translate_rel=0.5,
                              seed=42)

Note on dependencies
- This module depends on numpy. If numpy is not installed, a RuntimeError is raised at import time.
