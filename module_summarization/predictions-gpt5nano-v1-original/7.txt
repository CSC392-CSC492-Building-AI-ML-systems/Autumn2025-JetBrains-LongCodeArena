Using lists
===========

This topic describes how to work with Python lists, including creation, manipulation, and common idioms. In practice, lists are often used to collect identifiers, batch operations, and pass sequences to API calls.

Creating lists
- List literals: a = [1, 2, 3]
- Empty list: empty = []
- Copying a list: b = a[:], c = list(a)

Common operations
- Add items: a.append(item)
- Extend with multiple items: a.extend(iterable)
- Insert at position: a.insert(index, item)
- Remove a value: a.remove(item)
- Remove by index (pop): last = a.pop(), specific = a.pop(index)
- Clear all items: a.clear()
- Find index or count: idx = a.index(item); cnt = a.count(item)
- Sort and reverse (in place): a.sort(), a.sort(key=key, reverse=True), a.reverse()

Slicing and indexing
- Access: a[0], a[-1]
- Slices: a[1:3], a[:5], a[::2], a[::-1] (reversed)

List comprehensions
- Short, expressive transformations: squares = [x*x for x in range(10)]
- Filtering during construction: evens = [n for n in numbers if n % 2 == 0]
- Mapping with conditions: results = [func(x) for x in data if condition(x)]

Working with lists in practice
- Collecting IDs or attributes: ids = [e.id for e in entities]
- Filtering and transforming: names = [e.name for e in entities if e.active]
- Flattening nested lists: flat = [item for sub in nested for item in sub]
- Grouping or batching: for batch_start in range(0, len(ids), batch_size): batch = ids[batch_start:batch_start+batch_size]

Joining and combining
- Concatenate lists: combined = list1 + list2
- Build from iterables: combined = [*list1, *list2]
- Flatten with itertools: import itertools; flat = list(itertools.chain.from_iterable(nested_lists))

Performance and memory tips
- Use generator expressions when possible to avoid creating intermediate lists
- For large data, process in chunks (batches) rather than loading everything at once
- Prefer list comprehensions over manual loops for clarity and speed

Examples
- Create a list of active entity IDs:
  .. code-block:: python
     ids = [e.id for e in entities if e.active]

- Process in batches:
  .. code-block:: python
     batch_size = 100
     for i in range(0, len(ids), batch_size):
         batch_ids = ids[i:i+batch_size]
         process(batch_ids)
