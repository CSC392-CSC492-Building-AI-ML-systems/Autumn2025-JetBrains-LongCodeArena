RandAugment (module rand_augment)

Overview
- This module implements RandAugment augmentation as described in
  https://arxiv.org/abs/1909.13719. It applies a random set of n augmentation operations
  to each sample in a batch, with a configurable magnitude m. Magnitude is quantized
  into num_magnitude_bins and can be influenced by a shape parameter. The suite of
  augmentations can be customized by excluding specific operations or by providing
  a custom set of augmentations.

Functions

rand_augment
- Purpose
  Apply RandAugment to a batch of samples using a randomly selected subset of
  augmentations.

- Signature
  rand_augment(
      data: _DataNode,
      n: int,
      m: int,
      num_magnitude_bins: int = 31,
      shape: Optional[Union[_DataNode, Tuple[int, int]]] = None,
      fill_value: Optional[int] = 128,
      interp_type: Optional[types.DALIInterpType] = None,
      max_translate_abs: Optional[int] = None,
      max_translate_rel: Optional[float] = None,
      seed: Optional[int] = None,
      monotonic_mag: bool = True,
      excluded: Optional[List[str]] = None,
  ) -> _DataNode

- Parameters
  data
    A batch of samples to process. Supports images in HWC layout and videos in FHWC layout
    with uint8 data type.

  n
    Number of randomly sampled operations to apply to each sample.

  m
    Magnitude (strength) for each operation. Must be an integer in [0, num_magnitude_bins - 1].

  num_magnitude_bins
    Number of bins used to discretize magnitudes. Default is 31.

  shape
    Size (height, width) of the image or video frames. If provided, magnitudes for
    translation-related operations depend on this shape and span from 0 to
    max_translate_rel * shape. Otherwise magnitudes span from 0 to max_translate_abs.

  fill_value
    Padding value used by warp_affine operations (translation, shear, rotate). If None,
    padding uses border value repeated (clamped).

  interp_type
    Interpolation method used by warp_affine ops. Supported values include
    types.INTERP_LINEAR (default) and types.INTERP_NN.

  max_translate_abs
    Maximum translation in pixels when shape is not provided. If a tuple is given, the
    first element constrains height and the second constrains width.

  max_translate_rel
    Maximum translation as a fraction of the image shape when shape is provided. If a
    tuple is given, the first element constrains height and the second constrains width.

  seed
    Seed for random sampling of augmentations and for magnitudes.

  monotonic_mag
    If True (default), magnitude bins map monotonically to strength. If False, some
    color operations may vary non-monotonically in strength with magnitude.

  excluded
    Names of augmentations to exclude from the default RandAugment suite.

- Returns
  A batch of transformed samples as a _DataNode.

- Notes
  - If any names in excluded do not exist in the generated augmentation suite, an
    Exception is raised with a message listing available augmentations.

- See Also
  - get_rand_augment_suite, get_rand_augment_non_monotonic_suite
  - nvidia.dali.auto_aug.augmentations

apply_rand_augment
- Purpose
  Apply a provided list of augmentations to data by randomly selecting n of them
  and applying with magnitude m, forwarding additional keyword arguments to the
  augmentations (e.g., fill_value, interp_type, shape).

- Signature (inferred from usage)
  apply_rand_augment(
      augmentations: List[_Augmentation],
      data: _DataNode,
      n: int,
      m: int,
      num_magnitude_bins: int = 31,
      seed: Optional[int] = None,
      **kwargs
  ) -> _DataNode

- Parameters
  augmentations
    A list of augmentation objects to choose from.

  data
    A batch of samples to process.

  n
    Number of augmentations to apply to each sample.

  m
    Magnitude index for the augmentations.

  num_magnitude_bins
    Number of magnitude bins used to discretize magnitudes.

  seed
    Optional seed for deterministic randomness.

  kwargs
    Additional keyword arguments forwarded to augmentations, such as:
    - fill_value
    - interp_type
    - shape

- Returns
  A batch of transformed samples as a _DataNode.

- Notes
  - This function is used internally by rand_augment to perform the actual
    augmentation pipeline after selecting the subset of augmentations.

Usage example
- Example usage and typical workflows can be built by calling rand_augment with
  a batch of images and optionally excluding certain augmentations to tailor the
  augmentation suite.
