Optimization
============

Overview
--------
This module provides general-purpose optimization utilities for model fitting. The goal is to decouple optimization routines from LikelihoodModel so that they can be reused across different models and objectives.

Public API
----------

Functions
---------

- _check_method(method, methods)
  Validate that a given fit method is among the supported options. If the method is unknown, a ValueError is raised.

Classes
-------

- Optimizer
  A lightweight optimizer wrapper that can fit any model with an objective function.

  _fit(objective, gradient, start_params, fargs, kwargs, hessian=None, method='newton', maxiter=100, full_output=True, disp=True, callback=None, retall=False)

  Description
  -------------
  Fit function for any model with an objective function.

  Parameters
  ----------
  start_params : array_like, optional
      Initial guess of the solution for the loglikelihood maximization. Default is an array of zeros.
  method : str {'newton','nm','bfgs','powell','cg','ncg','basinhopping','minimize'}
      Solver to use. Options include Newton-Raphson ('newton'), Nelder-Mead ('nm'),
      BFGS ('bfgs'), Powell's method ('powell'), conjugate gradient ('cg'), Newton-conjugate
      gradient ('ncg'), global basin-hopping ('basinhopping'), or a generic 'minimize'
      wrapper for scipy.optimize.minimize.
      The explicit arguments are passed to the solver; the basin-hopping solver has its
      own set of supported arguments.
  maxiter : int
      Maximum number of iterations to perform.
  full_output : bool
      When True, return a dictionary of solver output in the result.
  disp : bool
      If True, print convergence messages.
  fargs : tuple
      Extra arguments passed to the objective function, i.e., loglike(x, *args).
  callback : callable, optional
      Called after each iteration as callback(xk), where xk is the current parameter vector.
  retall : bool
      If True, return a list of solutions at each iteration.

  Returns
  -------
  xopt : array
      The optimized parameters that maximize (or minimize) the objective.
  retvals : dict or None
      If full_output is True, a dictionary with solver-specific information. If
      full_output is False, this is None.
  optim_settings : dict
      Dictionary containing the parameters passed to the solver.

  Notes
  -----
  The 'basinhopping' solver ignores maxiter, retall, and full_output explicit arguments.

  Optional arguments for the solvers (available in Results.mle_settings):
  ---------------------------------------------------------------------
  'newton'
      tol : float
          Relative error in parameters acceptable for convergence.
  'nm' -- Nelder Mead
      xtol : float
          Relative error in parameters acceptable for convergence.
      ftol : float
          Relative error in loglike(params) acceptable for convergence.
      maxfun : int
          Maximum number of function evaluations.
  'bfgs'
      gtol : float
          Stop when the gradient norm is less than gtol.
      norm : float
          Order of the norm (np.Inf for max, -np.Inf for min).
      epsilon
          If the gradient is approximated, the step size used for the finite
          difference approximation.
  'lbfgs'
      m : int
          Maximum number of variable metric corrections.
      pgtol : float
          Terminate when the projected gradient is small enough.
      factr : float
          Termination tolerance for the objective; related to ftol.
      maxfun : int
          Maximum number of iterations.
      epsilon : float
          Step size for numerical gradient if approximate_grad is True.
      approx_grad : bool
          Whether to approximate the gradient numerically.
  'cg'
      gtol : float
          Stop when gradient norm is less than gtol.
      norm : float
          Order of the norm (np.Inf for max, -np.Inf for min).
      epsilon : float
          Step size for numerical gradient if approximate gradient is used.
  'ncg'
      fhess_p : callable
          Function to compute the Hessian-vector product, f'(x) * p, used when
          the Hessian is not provided.
      avextol : float
          Stop when the average relative minimizer error falls below this value.
      epsilon : float or ndarray
          Step size for numerical Hessian approximation if needed.
  'powell'
      xtol : float
          Line-search error tolerance.
      ftol : float
          Relative error in loglike(params) for convergence.
      maxfun : int
          Maximum number of function evaluations.
      start_direc : ndarray
          Initial direction set.
  'basinhopping'
      niter : int
          Number of basin-hopping iterations.
      niter_success : int
          Stop when the global minimum candidate remains the same for this many iterations.
      T : float
          Temperature parameter for the accept/reject criterion.
      stepsize : float
          Initial step size for the random displacement.
      interval : int
          Interval for updating the step size.
      minimizer : dict
          Extra keyword arguments to be passed to the minimizer scipy.optimize.minimize
          (e.g., method, tol). Other arguments map from the explicit arguments of fit:
          - args <- fargs
          - jac  <- score
          - hessian <- hessian (if applicable)

See Also
--------
- scipy.optimize for the underlying optimization solvers and their options.

Usage notes
-----------
- This Optimizer is designed to be reused with any objective function and its gradient,
  not tied to a specific LikelihoodModel.
- The behavior and available options depend on the chosen solver. Basinhopping has its own
  set of arguments and may ignore some higher-level options.
