SimpleFitter wrapper
====================

Adds a variable height (background) component to any model

Module API
^^^^^^^^^^

- SimpleFitter
  - __init__(self): Placeholder initializer (signature in code is empty).
  - moments(self, *args, **kwargs): Get spectral moments from the moments package (pyspeckit.spectrum.moments.moments).

- vheightmodel(zeroheightmodel)
  - Returns: a wrapper function vhm(xax, *pars, **kwargs).
  - Purpose: wrapper to set a variable height background for a given zero-height model.
  - Behavior:
    - The parameter order is: height, amplitude, shift, width.
    - If vheight is True (default) or not specified, the wrapper returns zeroheightmodel(xax, *pars[1:], **kwargs) + pars[0].
    - If vheight is False, it returns zeroheightmodel(xax, *pars[1:], **kwargs) without the additive height.
  - Notes: The vhm function copies the docstring from the zero-height model.

Constructor

__init__(self, modelfunc, npars,
         shortvarnames=("A","\\Delta x","\\sigma"),
         fitunit=None,
         centroid_par=None,
         fwhm_func=None,
         fwhm_pars=None,
         integral_func=None,
         use_lmfit=False,
         guess_types=('amplitude', 'center', 'width'),
         **kwargs)

- modelfunc: The model function to be fitted. It should take an X-axis input and then parameter values, returning an array with the same shape as the input X-axis.
- npars: The number of parameters required by the model.
- shortvarnames: TeX names for variables used when annotating.
- fitunit: Optional unit to convert the X-axis to before passing to the model.
- centroid_par: Parameter representing the centroid of the model.
- fwhm_func: Function to compute the full width at half maximum and its parameters.
- fwhm_pars: Parameters for the FWHM function.
- integral_func: Function to compute an analytic integral (if provided).
- use_lmfit: If True, use lmfit backend; otherwise use mpfit.
- guess_types: Tuple describing the types of parameters when guessing; defaults to ('amplitude', 'center', 'width').
- kwargs: Additional keyword arguments for parameter information (parinfo) and related configuration.
