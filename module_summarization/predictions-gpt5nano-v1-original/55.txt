Miscellaneous models
=====================

Linear Model with Student-t distributed errors
----------------------------------------------

Overview
--------
This module implements robust statistical models that assume the observation
noise follows a Student-t distribution rather than a normal distribution.
The heavier tails of the t distribution provide robustness to outliers and
heavy-tailed data, yielding more resilient estimators for linear models and
ARMA-type time series models.

Key ideas:
- The last two parameters of the models represent the degrees of freedom (df)
  and a scale parameter. When df is not fixed, both df and scale are estimated;
  otherwise, df may be fixed and only the scale is estimated.
- The likelihood is evaluated using the t-distribution, with gradients and
  Hessians obtained through numerical differentiation.
- The scale parameter is not generally the standard deviation of the errors; the
  variance for the standardized formulation is df/(df-2) * scale^2 for df > 2.

Module API
----------

TLinearModel
~~~~~~~~~~~~
Maximum Likelihood Estimation of Linear Model with t-distributed errors.

- Inherits from: GenericLikelihoodModel
- Purpose: Provide a generic MLE framework for a linear regression setup
  where the residuals are t-distributed.
- Key methods:
  - initialize(): sets up dimension counts, fixed-df options, and parameter names.
  - _set_start_params(): derives sensible starting values from ordinary least squares (OLS) when available; can incorporate kurtosis-based df initialization if requested.
  - loglike(params): returns the (negative) log-likelihood for the current parameters.
  - nloglikeobs(params): per-observation log-likelihood contributions; used internally by optimization routines.
  - predict(params, exog=None): predictions given a parameter vector.
- Notes:
  - The last two parameters in the parameter vector are df and scale.
  - If df is fixed, a subset of parameters becomes fixed and only the remaining parameters are estimated.
  - Uses the gammaln form for the t-distribution log-PDF and includes a correction for the scale.

TArma
~~~~~
Univariate ARMA Model with t-distributed errors.

- Inherits from: Arma
- Purpose: Extend the standard ARMA model to t-distributed innovations.
- Key methods:
  - loglike(params): user-visible interface returning the log-likelihood (via nloglikeobs).
  - nloglikeobs(params): per-observation log-likelihood for the t-distributed errors.
- Notes:
  - The ancillary parameter is assumed to be the last element of the parameter vector.
  - The implied error variance is scale^2 * df/(df-2) for df > 2.
  - The last two parameters are df and scale, with scale taken as absolute value to ensure non-negativity.
  - The implementation integrates with statsmodels’ existing Arma framework, using a t-distribution likelihood.

-------------
- Fit a linear model with t-distributed errors using TLinearModel.
- Fit a univariate ARMA model with t-distributed errors using TArma.
