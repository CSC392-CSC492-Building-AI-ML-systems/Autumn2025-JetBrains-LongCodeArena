w_succ
======

Description
-----------

w_succ lists segments which have successfully reached a target state (i.e., recycled segments) for each iteration up to the current one. The output shows, per recycled segment, the iteration, segment ID, weight, and the final value(s) of the progress coordinate (pcoord) at the end of the segment.

Usage
-----

.. code-block:: text

   w_succ [options]

Output
..........

By default, output is written to standard output. You can directing the output to a file with the -o/--output option.

Header and columns
~~~~~~~~~~~~~~~~~~

If headers are not suppressed, the tool prints a header describing the columns:

# successful (recycled) segments
# column 0:    iteration
# column 1:    seg_id
# column 2:    weight
# column>2:    final progress coordinate value

For each eligible recycled segment, a line is produced with:

- column 0: iteration number
- column 1: segment ID
- column 2: segment weight
- columns > 2: the final values of the progress coordinate (one value per pcoord dimension)

Notes on data formatting
~~~~~~~~~~~~~~~~~~~~~~~~

Final pcoord values are printed with formats chosen according to their data types. The formatting is designed to accommodate common integer and floating-point representations used for pcoord data.

Algorithm and data selection
~~~~~~~~~~~~~~~~~~~~~~~~~~~

- The command iterates over iterations from 1 up to (but not including) the current iteration.
- For each iteration, recycled segments are identified (segments whose endpoint_type is SEG_ENDPOINT_RECYCLED).
- If there are recycled segments, the final pcoord values (the last pcoord frame for each recycled segment) are retrieved and printed alongside the iteration, seg_id, and weight.

Options
-------

- -o, --output OUTPUT_FILE
  Write output to OUTPUT_FILE. If omitted, output is written to standard output.

- Additional common WestPA options provided by the WestPA runtime configuration (via westpa.rc) are available; see the WestPA documentation for details on global options.

Examples
--------

- Basic usage (print to stdout):
  w_succ

- Write output to a file:
  w_succ -o succ.txt
