Memory alignment
================

Overview
--------

Memory alignment refers to ensuring that data resides at memory addresses that are multiples of a given number of bytes. On modern CPUs, aligned memory accesses are faster and, for some vectorized operations, required. NumPy allocates and uses memory buffers with alignment appropriate for the platform and the data type being stored. The build process records CPU features and SIMD extensions, which influence the alignment assumptions used by optimized code paths.

Dtypes and alignment
---------------------

- The alignment of a data type is available as the dtype.alignment attribute.
- The alignment of an ndarray’s data buffer is the alignment of its dtype for typical contiguous arrays.

Example:
.. code-block:: python

   import numpy as np
   dt = np.dtype(np.float64)
   print("dtype alignment:", dt.alignment)

   a = np.empty(8, dtype=dt)
   print("buffer address alignment check:", a.ctypes.data % dt.alignment)

How alignment is used
---------------------

- NumPy’s kernels may take advantage of SIMD instructions (SSE, AVX, AVX-512, etc.) when memory is suitably aligned.
- If data is not aligned for the required path, NumPy will use alternative, possibly slower, unaligned paths.
- Build-time configuration records the SIMD capabilities detected on the host (e.g., through __cpu_features__ and __cpu_dispatch__), which guides the selection of optimized code paths and alignment assumptions.

Inspecting and ensuring alignment
---------------------------------

- Inspect alignment via dtype.alignment and verify that buffers satisfy the alignment for critical operations.
- Ensure arrays intended for vectorized operations are aligned by using dtype-based alignment and allocating compatible buffers.
- For external or user-provided buffers, verify alignment before processing to enable faster, vectorized code paths.

Practical guidelines
--------------------

- Prefer creating arrays with standard dtypes that have well-defined alignments.
- Where possible, rely on NumPy’s default aligned allocations by using regular dtypes and contiguous arrays.
- For performance-critical code, compile NumPy with support for the target CPU’s SIMD extensions; the runtime will then utilize aligned paths where possible.
