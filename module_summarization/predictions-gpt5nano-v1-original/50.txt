Generate example for GAM Spline Smoothers
==========================================

Overview
--------
This page demonstrates how to generate and use spline-based smoothers for Generalized Additive Models (GAM) with the classes and helpers provided in the codebase. It covers B-spline smoothers (BSplines) and cyclic cubic splines (CyclicCubicSplines), how to construct bases, and how to assemble and fit a GLMGam model with these smoothers. It also shows a simple cross-validation path for multivariate GAMs.

Example
-------
The following example generates synthetic data, constructs a B-spline basis for a single predictor, and fits a GAM using GLMGam with the spline smoother.

.. code-block:: python

    import numpy as np

    # Import the smoother bases and GAM model
    from generalized_additive_model import GLMGam
    from gam_cross_validation.gam_cross_validation import MultivariateGAMCVPath
    from smooth_basis import BSplines, CyclicCubicSplines

    # 1. Generate synthetic data
    np.random.seed(123)
    n = 200
    x = np.linspace(0, 1, n)
    y = np.sin(2 * np.pi * x) + 0.3 * np.random.randn(n)

    # 2. Create a B-spline basis for x
    # df = degrees of freedom for the spline, degree = spline degree
    bs = BSplines(x[:, None], df=[6], degree=[3])

    # 3. Fit a GAM with the spline smooth term
    # (Note: adjust the API to your actual import paths and exact constructor)
    gam = GLMGam(endog=y, smoother=[bs])
    res = gam.fit()

    # 4. (Optional) Cross-validate smoothing paths for multivariate GAM
    # cv = MultivariateGAMCVPath(gam)
    # cv_results = cv.run()

    print(res.summary())

Notes
-----
- The smoothing bases (BSplines, CyclicCubicSplines) are designed to be passed to GLMGam as the smoother argument.
- Knot placement is handled by the helper functions in the module (e.g., get_knots_bsplines, compute_all_knots) and supports both quantile-based and evenly spaced knot strategies.
- The MultivariateGAMCVPath utility provides a framework for cross-validation of GAM smoothing paths in multivariate settings.

API Highlights
--------------
- BSplines: Construct B-spline basis for one or more predictors.
- CyclicCubicSplines: Construct cyclic cubic spline bases for periodic data.
- _equally_spaced_knots, _R_compat_quantile: Internal helpers for knot generation and quantile-based knot placement.
- _eval_bspline_basis, compute_all_knots, make_bsplines_basis: Internal routines for evaluating and assembling spline bases.
- get_knots_bsplines: High-level function to obtain knots for B-splines with flexible spacing and boundaries.
