Duration module documentation
=============================

This module provides utilities for survival analysis with right-censoring, including calculation of survival functions, incidence functions, and related standard errors. It includes internal helpers and a class for estimation and inference of cumulative incidence.

Notes
-----

- The functions prefixed with an underscore (_calc_survfunc_right, _calc_incidence_right, _checkargs) are intended for internal use but are documented here for completeness.
- The CumIncidenceRight class provides estimation and inference for a cumulative incidence function under right censoring.

Dependencies
------------

- numpy
- pandas
- scipy (for distributions such as chi2 and norm)
- statsmodels (for graphics utilities)
