Kernel approximation
====================

The sklearn.kernel_approximation module implements several approximate kernel feature maps based on Fourier transforms and Count Sketches.

PolynomialCountSketch
---------------------

- Purpose
  - Polynomial kernel approximation via Tensor Sketch.

- Kernel
  - K(X, Y) = (gamma * <X, Y> + coef0)^degree

- Approach
  - Efficiently computes a Count Sketch of the outer product of a vector with itself using Fast Fourier Transforms (FFT).

- Parameters
  - gamma: float, default=1.0
  - degree: int, default=2
  - coef0: float, default=0
  - n_components: int, default=100
  - random_state: int, RandomState instance, default=None

- Attributes (after fit)
  - indexHash_: ndarray of shape (degree, n_features)
    Array of indices in [0, n_components) used for 2-wise independent hash functions.
  - bitHash_: ndarray of shape (degree, n_features)
    Array with random entries in {+1, -1} for the 2-wise independent hash functions.
  - _n_features_out_: int
    Dimensionality of the output feature space (equal to n_components).

- Example
  .. code-block:: python

     from sklearn.kernel_approximation import PolynomialCountSketch
     from sklearn.linear_model import SGDClassifier

     X = [[0, 0], [1, 1], [1, 0], [0, 1]]
     y = [0, 0, 1, 1]
     ps = PolynomialCountSketch(degree=3, random_state=1)
     X_features = ps.fit_transform(X)
     clf = SGDClassifier(max_iter=10, tol=1e-3)
     clf.fit(X_features, y)
     score = clf.score(X_features, y)
     print(score)  # 1.0 for separable data

RBFSampler
---------

- Purpose
  - Approximate the RBF (Gaussian) kernel feature map using random Fourier features.

Nystroem
--------

- Purpose
  - Approximates a kernel map using a subset of the training data.

AdditiveChi2Sampler
-------------------

- Purpose
  - Approximate feature map for the additive chi-squared kernel.

SkewedChi2Sampler
-----------------

- Purpose
  - Approximate feature map for the skewed chi-squared kernel.

Notes
-----

- The module leverages Fourier transforms and Count Sketch techniques to provide scalable kernel approximations for large datasets.
- The available classes offer different trade-offs between accuracy and computational efficiency, suitable for various kernel choices (polynomial, RBF, chi-squared, etc.).
