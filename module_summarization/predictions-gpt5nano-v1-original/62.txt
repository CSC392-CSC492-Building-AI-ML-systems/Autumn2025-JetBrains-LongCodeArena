Visitors
==========

Module: sql.visitors

Overview
--------
This module provides the visitor and traversal interface used by SQLAlchemy’s compilation and traversal machinery. It defines the base class for visitable objects, the internal traversal symbols, and a small collection of helper functions used to perform traversal and compilation dispatch.

Public API
----------
The public API exported by this module includes:
- iterate
- traverse_using
- traverse
- cloned_traverse
- replacement_traverse
- Visitable
- ExternalTraversal
- InternalTraversal
- anon_map

Key concepts
------------
Visitable
- Purpose: Serves as the base class for objects that participate in the SQL compilation and traversal process.
- Dynamic dispatch: A class can define a __visit_name__ which is used to generate a compiler dispatch function. The generated _compiler_dispatch looks for a visitor method named visit_<visit_name> and invokes it with the same keyword arguments. If such a method does not exist on the visitor, it falls back to the visitor’s visit_unsupported_compilation method.
- Compatibility: The implementation preserves an _original_compiler_dispatch to support overriding in derived classes (for example, extensions that override compilation behavior). If a class already defines _compiler_dispatch, that method is preserved as _original_compiler_dispatch and no new dispatcher is generated.
- Generics support: __class_getitem__ returns the class itself to support generic usage in Python 3.9+.

InternalTraversal
- Purpose: Defines visitor symbols used for internal traversal. These symbols are used by internal traversal logic (e.g., HasTraverseInternals) to describe what kind of data structure is being traversed.
- Members (examples):
  - dp_has_cache_key = "HC": Visit a HasCacheKey object.
  - dp_has_cache_key_list = "HL": Visit a list of HasCacheKey objects.
  - dp_clauseelement = "CE": Visit a ClauseElement object.
  - dp_fromclause_canonical_column_collection = "FC": Visit a FromClause object in the context of the columns collection (canonical column collection, typically for TableClause or Table objects).
  - dp_clauseelement_tuples = "CTS": Visit a list of tuples of ClauseElement objects.
- Usage: These tokens drive the internal traversal machinery to navigate and manipulate compiler internals and cache key generation.

anon_map
- Purpose: Provides a mapping of anonymous identifiers used during traversal and compilation.
- Implementation detail: Depending on availability of a cythonized extension, anon_map and its helpers may be provided by Python fallbacks or cython-extension modules. The module imports the appropriate implementation conditionally to maximize performance when a C/Cython extension is present.

Implementation notes
--------------------
- Typing and imports: The module uses typing.TYPE_CHECKING to avoid circular imports at runtime and to enable static type checking. It also uses HAS_CYEXTENSION to decide whether to import cython-backed utilities.
- Public surface: The __all__ list explicitly enumerates the public API, clarifying which symbols are intended for external use.

Documentation references
-----------------------

Notes
-----
This module is part of SQLAlchemy’s traversal and compilation infrastructure. It provides the foundational pieces that allow the compiler to dispatch to visitor methods and to describe internal traversal operations in a consistent and extensible manner.
