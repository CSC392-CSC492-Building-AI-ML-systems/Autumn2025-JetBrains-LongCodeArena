Empirical likelihood for descriptive statistics
===============================================

Overview
--------
This module provides empirical likelihood (EL) inference for descriptive statistics. It enables hypothesis tests and construction of confidence intervals for the mean, variance, skewness, kurtosis, and correlation. When matplotlib is available, the module can also produce multivariate confidence region plots and mean–variance contour plots.

General references
------------------
Owen, A. (2001). Empirical Likelihood. Chapman and Hall

API
---

DescStat(endog)
~~~~~~~~~~~~~~~
Returns an instance to conduct inference on descriptive statistics via empirical likelihood. See DescStatUV and DescStatMV for details.

Parameters
- endog : ndarray
    Array of data. If a one-dimensional array is provided, it is reshaped into a column vector.

Returns
- DescStat instance
    If k = 1, returns a univariate instance DescStatUV. If k > 1, returns a multivariate instance DescStatMV.

Notes
- This function delegates to univariate or multivariate implementations depending on the dimensionality of endog.

Optimum-Calculation Helpers
---------------------------

_OptFuncts class
^^^^^^^^^^^^^^^^^^^^
A container for functions used in the optimization/solution of the empirical likelihood problem. The general pattern is that methods starting with _opt_ (in this codebase described in the class docstring) build a vector of estimating equations est_vect such that p^T est_vect = 0, where p is the observation weight vector and est_vect is the n × k estimating equations matrix. The Lagrange multiplier (eta) is then solved via a Newton-based procedure to maximize the profile empirical likelihood.

In presence of nuisance parameters, these routines are intended to be used in a way that profiles out nuisance parameters.

Key methods (internal/private helpers):

- _log_star(eta, est_vect, weights, nobs)
  Purpose
  Transforms the log of observation probabilities into a log-star representation used in the optimization.

  Parameters
  - eta : float or array
      Lagrange multiplier
  - est_vect : ndarray (n, k)
      Estimating equations matrix
  - weights : 1D array
      Observation weights
  - nobs : int
      Number of observations

  Returns
  - data_star : ndarray
      Weighted log-star of the estimating equations

  Notes
  This is a placeholder for integration with the _fit_newton routine; the function value is not used directly in optimization.

- _hess(eta, est_vect, weights, nobs)
  Purpose
  Compute the Hessian of the weighted empirical likelihood with respect to eta.

  Parameters
  - eta : ndarray (1, m)
  - est_vect : ndarray (n, k)
  - weights : 1D array
  - nobs : int

  Returns
  - hess : ndarray
      The weighted Hessian used in the Newton step

- _grad(eta, est_vect, weights, nobs)
  Purpose
  Compute the gradient of the weighted empirical likelihood with respect to eta.

  Parameters
  - eta : ndarray (1, m)
  - est_vect : ndarray (n, k)
  - weights : 1D array
  - nobs : int

  Returns
  - gradient : ndarray (m, 1)

- _modif_newton(eta, est_vect, weights)
  Purpose
  Modified Newton method for maximizing the log-star equation. This routine uses _fit_newton to find the optimal eta.

  Parameters
  - eta : ndarray (1, m)
  - est_vect : ndarray (n, k)
  - weights : 1D array

  Returns
  - params : 1 × m array
      Lagrange multiplier that maximizes the log-likelihood

- _find_eta(eta)
  Placeholder for locating the eta vector (Lagrange multipliers). The implementation is not shown in the provided code excerpt.

Implementation notes
--------------------
- The optimization relies on a Newton-type solver from SciPy (_fit_newton) to maximize the log-likelihood with respect to the Lagrange multipliers.
- The log-star transformation and the gradient/hessian definitions follow from empirical likelihood theory for constrained estimating equations.

Notes
-----
- This module uses NumPy and SciPy for numerical routines, and integrates with Statsmodels' optimization tooling.
- If Matplotlib is installed, plotting utilities may be available to render multivariate confidence regions and mean–variance contours.
