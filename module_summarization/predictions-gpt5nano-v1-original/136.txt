Image Extraction
================

Overview
--------

This module implements a collection of charge extraction algorithms used to reduce camera waveforms to one value per pixel. The algorithms operate on per-pixel waveforms to produce a DL1-style image representation (per-pixel charge, and, for some methods, timing information).

API
---

Classes
  - ImageExtractor
  - FlashCamExtractor
  - FullWaveformSum
  - FixedWindowSum
  - GlobalPeakWindowSum
  - LocalPeakWindowSum
  - SlidingWindowMaxSum
  - NeighborPeakWindowSum
  - BaselineSubtractedNeighborPeakWindowSum
  - TwoPassWindowSum

Functions
  - extract_around_peak
      Computes per-pixel charge by summing a window around a detected peak and a pulse time by a weighted average of samples within the window.
      Inputs: waveforms (n_pix, n_samples), peak_index, width, shift, sampling_rate_ghz.
      Outputs: sum (per-pixel charge), peak_time (per-pixel pulse time in ns).

  - extract_sliding_window
      Finds the window with the largest sum of a given width and returns the corresponding charge and timing.
      Inputs: waveforms (n_pix, n_samples), width, sampling_rate_ghz.
      Outputs: sum (per-pixel charge), peak_time (per-pixel timing in ns).

  - neighbor_average_maximum
      (Description inferred from API: computes a neighborhood-based metric used by certain extractors.)

  - subtract_baseline
      (Description inferred from API: baseline subtraction utility used by some extraction options.)

  - integration_correction
      (Description inferred from API: applies a correction to the integrated charge.)

Notes
-----

- The implementations rely on performance-oriented tools such as NumPy, SciPy, and Numba (guvectorize and JIT). 
- The extraction methods operate on waveform data typically shaped as (n_pix, n_samples) and produce per-pixel scalars (and, for some methods, per-pixel timing). 
- The module integrates with CTApipe constructs (DL1CameraContainer, TelescopeComponent) and uses auxiliary utilities for cleaning, morphology, timing, and pixel handling.
