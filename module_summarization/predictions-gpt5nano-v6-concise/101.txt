# web3.miner.rst — Miner API Reference

Overview
- The Miner class is a thin, synchronous wrapper around Ethereum client miner-related JSON-RPC calls.
- It relies on Web3.py’s request manager to perform blocking RPC requests.
- The hashrate property uses a formatter to convert the raw result to a decimal representation for easier numeric handling.

Class and module context
- Module: web3.miner
- Core class: Miner
- Primary responsibility: expose miner-related RPC methods as Python methods/properties in a convenient, Pythonic form.

API surface

Constructor
- Miner(web3)
  - Description: Create a Miner helper bound to a Web3 instance.
  - Parameters:
    - web3: An authenticated Web3.py instance (e.g., Web3 or a compatible wrapper) with a _requestManager capable of request_blocking.
  - Returns: A Miner object bound to the provided web3 instance.

Property
- hashrate
  - Type: Decimal-like numeric
  - Decorator: @property
  - Formatter: apply_formatters_to_return(to_decimal)
  - RPC: eth_hashrate
  - Description: Retrieves the current miner hash rate (in H/s) reported by the connected Ethereum client.
  - Notes:
    - The result is passed through to_decimal to ensure consistent numeric type for downstream calculations.
    - This is a blocking call.

Methods
- makeDAG(number)
  - Signature: makeDAG(self, number)
  - RPC: miner_makeDag
  - Parameters:
    - number: int
  - Description: Generates (or regenerates) the DAG for the specified epoch/level. This corresponds to the DAG used by Ethash for mining.
  - Return: RPC response (type depends on the node; typically a success indicator or similar payload).
  - Notes: This is a blocking call.

- setExtra(extra)
  - Signature: setExtra(self, extra)
  - RPC: miner_setExtra
  - Parameters:
    - extra: str (typically a hex string or raw data to be embedded in the block header’s extra field)
  - Description: Sets the extra data field that gets included in mined blocks.
  - Return: RPC response (depends on the node; usually a success/failure indicator).

- setEtherBase(etherbase)
  - Signature: setEtherBase(self, etherbase)
  - RPC: miner_setEtherbase
  - Parameters:
    - etherbase: str (address)
  - Description: Sets the coinbase/etherbase address where mined rewards are sent.
  - Return: RPC response.

- setGasPrice(gas_price)
  - Signature: setGasPrice(self, gas_price)
  - RPC: miner_setGasPrice
  - Parameters:
    - gas_price: int or hex/int string representing the gas price (in wei)
  - Description: Configures the minimum gas price for the mining reward to be accepted by the network (when mining on block templates that include transactions).
  - Return: RPC response.

- start(num_threads)
  - Signature: start(self, num_threads)
  - RPC: miner_start
  - Parameters:
    - num_threads: int
  - Description: Starts the miner with the specified number of CPU mining threads. A value of 0 typically means “auto” or “use all available threads,” depending on the client.
  - Return: RPC response.

- stop()
  - Signature: stop(self)
  - RPC: miner_stop
  - Parameters: none
  - Description: Stops mining operations.
  - Return: RPC response.

- startAutoDAG()
  - Signature: startAutoDAG(self)
  - RPC: miner_startAutoDag
  - Parameters: none
  - Description: Enables automatic DAG generation as needed by the miner.
  - Return: RPC response.

- stopAutoDAG()
  - Signature: stopAutoDAG(self)
  - RPC: miner_stopAutoDag
  - Parameters: none
  - Description: Disables automatic DAG generation.
  - Return: RPC response.

Usage patterns

Basic usage
- Instantiate and query hashrate:
```python
from web3 import Web3
from web3.miner import Miner

w3 = Web3(Web3.HTTPProvider("http://localhost:8545"))
miner = Miner(w3)

# Get current hashrate (converted to Decimal)
current_hashrate = miner.hashrate
print(f"Hashrate: {current_hashrate} H/s")
```

Managing mining state
```python
# Start mining with 2 CPU threads
miner.start(2)

# Stop mining
miner.stop()
```

DAG and network settings
```python
# Generate DAG for epoch 0
miner.makeDAG(0)

# Change the extra field in blocks
miner.setExtra("0x1234567890abcdef")

# Set the etherbase (mining rewards destination)
miner.setEtherBase("0xYourEtherbaseAddress")

# Set the minimum gas price for mined blocks
miner.setGasPrice(20000000000)  # 20 Gwei (example)
```

Automatic DAG management
```python
# Enable auto-DAG generation
miner.startAutoDAG()

# Disable auto-DAG generation
miner.stopAutoDAG()
```

Design notes and considerations

- Blocking behavior: All Miner methods use request_blocking, making them synchronous. This is suitable for scripts and CLI tools but may block your event loop in asynchronous contexts.
- RPC surface parity: The methods map 1:1 to the corresponding JSON-RPC miner_* calls used by clients like Geth. If your node does not support a given RPC family, the call may raise an RPC error.
- Return values: The exact structure of RPC responses depends on the Ethereum client. The documentation here reflects typical behavior; refer to your node’s RPC responses for precise return formats.
- Error handling: RPC errors (e.g., unsupported methods, invalid parameters) will propagate as exceptions from the underlying request manager. Wrap usage with try/except as needed.

Implementation notes

- hashrate property:
  - Uses apply_formatters_to_return(to_decimal) to ensure numeric consistency across environments.
  - Accessed as a read-only property; it triggers an RPC call when accessed.
- RPC method names:
  - Internal mapping follows: eth_hashrate -> hashrate, miner_makeDag, miner_setExtra, miner_setEtherbase, miner_setGasPrice, miner_start, miner_stop, miner_startAutoDag, miner_stopAutoDag.
- Dependency:
  - Requires a functioning Web3.py instance with a valid request manager capable of request_blocking.

Versioning and compatibility

- This API is designed to align with common patterns in Web3.py miner wrappers and should be compatible with standard Ethereum clients that expose miner_* and eth_hashrate RPCs.
- If you upgrade the client or RPC interface, verify the availability of each method and parameter expectations.
