# isototonic.rst — Generate documentation for isototonic module

This module provides utilities and a scikit-learn compatible estimator for isotonic (monotone) regression. It includes a lightweight function-based API as well as a class-based estimator that can be used in a familiar fit/predict workflow.

## Overview

- check_increasing: Determine whether one 1D array is monotonically increasing or decreasing with another, using Spearman’s rank correlation.
- isotonic_regression: Solve a 1D isotonic regression problem (monotone fit) with optional weighting and value bounds.
- IsotonicRegression: A scikit-learn compatible estimator that learns a monotone function f(X) ≈ y, with options for direction, bounds, and out-of-domain handling.

The implementation relies on an in-place isotonic regression routine (_inplace_contiguous_isotonic_regression) and exposes a minimal yet complete API suitable for downstream modeling workflows.

## check_increasing(x, y)

Determine whether y is monotonically correlated with x based on Spearman correlation.

Parameters
- x: array-like, shape (n_samples,)
  Training data for the predictor.
- y: array-like, shape (n_samples,)
  Training target values.

Returns
- increasing_bool: bool
  True if the relationship is increasing (rho >= 0), False otherwise.

Notes
- The Spearman rho is estimated from the data; its sign is used to decide increasing vs decreasing.
- If the 95% confidence interval (computed via Fisher transform) spans zero (and n > 3), a warning is emitted because the determination may be suspect.

References
- Fisher transformation. Wikipedia. https://en.wikipedia.org/wiki/Fisher_transformation

Example
- Determine whether y grows with x:
  - increasing = check_increasing(x, y)

## isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True)

Solve the isotonic regression problem for a 1D sequence y.

Parameters
- y: array-like, shape (n_samples,)
  The data to fit.

- sample_weight: array-like or None, shape (n_samples,), default=None
  Weights for each data point. If None, equal weights are used.

- y_min: float or None, default=None
  Lower bound on the lowest predicted value. If None, defaults to -inf.

- y_max: float or None, default=None
  Upper bound on the highest predicted value. If None, defaults to +inf.

- increasing: bool, default=True
  If True, fit a nondecreasing function; if False, fit a nonincreasing function.

Returns
- y_ : ndarray, shape (n_samples,)
  Isotonic fit of y.

Notes
- The function reorders y according to the requested direction (increasing or decreasing).
- After the isotonic fit, values can be clipped to [y_min, y_max] if bounds are provided.

References
- Best, M. J., & Chakravarti, N. (Active set algorithms for isotonic regression; A unifying framework) (section 3).

Example
- Basic isotonic regression with equal weights:
  - y_fit = isotonic_regression(y)

- Isotonic regression with bounds and a nondecreasing constraint:
  - y_fit = isotonic_regression(y, y_min=0.0, y_max=10.0, increasing=True)

## IsotonicRegression (class)

A scikit-learn compatible estimator for isotonic regression with optional bounds and direction.

Parameters
- y_min: float, default=None
  Lower bound on the lowest predicted value (the minimum value may still be higher). If not set, defaults to -inf.

- y_max: float, default=None
  Upper bound on the highest predicted value (the maximum may still be lower). If not set, defaults to +inf.

- increasing: bool or 'auto', default=True
  Determines whether the predictions should be constrained to increase with X.
  If 'auto', the direction is determined automatically based on the sign of the Spearman correlation (via check_increasing).

- out_of_bounds: {'nan', 'clip', 'raise'}, default='nan'
  How to handle X values outside the training domain during prediction:
  - 'nan': predictions are NaN outside the training domain.
  - 'clip': predictions are clipped to the nearest train interval endpoint.
  - 'raise': a ValueError is raised for out-of-bounds queries.

Attributes
- X_min_: float
  Minimum value of the input array X_ used to define the left bound.

- X_max_: float
  Maximum value of the input array X_ used to define the right bound.

- X_thresholds_: ndarray of shape (n_thresholds,)
  Unique ascending X values used to interpolate the monotone function f(X).
  (Added in version 0.24.)

- y_thresholds_: ndarray
  Corresponding y-values (predicted) at the X_thresholds_ breakpoints.
  (Structure aligns with piecewise-constant isotonic fit.)

Notes
- The estimator is designed to be compatible with the scikit-learn API:
  - fit(X, y) learns the monotone mapping from X to y.
  - predict(X) returns the isotonic regression of y over the provided X.
  - fit_transform(X, y) and transform(X) semantics follow TransformerMixin conventions when applicable.

- The implementation uses an in-place algorithm and supports weighting via sample_weight.

Examples
- Basic usage with automatic direction:
  - ir = IsotonicRegression(increasing='auto')
  - ir.fit(X, y)
  - y_pred = ir.predict(X)

- Handling out-of-bounds inputs:
  - ir = IsotonicRegression(out_of_bounds='clip')
  - ir.fit(X, y)
  - y_pred = ir.predict(X_new)

- Bounding the predictions:
  - ir = IsotonicRegression(y_min=0.0, y_max=5.0)
  - ir.fit(X, y)
  - y_pred = ir.predict(X_new)
