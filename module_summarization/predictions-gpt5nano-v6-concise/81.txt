### vheightmodel

Description
- A higher-order wrapper that adds a variable-height (background) component to a
  base model. This lets you fit a model with a free constant background term.

Returned object
- vhm: a function wrapper with the following signature:
  - vhm(xax, *pars, **kwargs)
  - Parameter order (as documented in the code): height, amplitude, shift, width
  - If vheight is True (default), the wrapper adds a constant height to the base
    model: zeroheightmodel(xax, *pars[1:], **kwargs) + height
  - If vheight is False, the wrapper uses the zero-height model without the extra term.

Key behaviors
- Accepts a vheight keyword in kwargs to enable/disable the height term.
- Inherits the docstring of the underlying zero-height model.

Usage example
- Given a base model function zeroheightmodel(x, A, dx, sigma, ...), create a
  variable-height wrapper:
  - wrapped = vheightmodel(zeroheightmodel)
  - y = wrapped(x, height, A, dx, sigma)

### SpectralModel

Description
- A robust wrapper class for building and fitting spectral models. It supports:
  - Passing a model function and the number of parameters.
  - Optional integration/annotation utilities.
  - Different fitting backends (mpfit by default, with an option for lmfit).
  - Parameter naming, units, and guess-type handling for automatic fitting.

Key initialization parameters (highlights)

- modelfunc: function
  - The model function to be fitted. Signature should be (x, *pars, **kwargs) and
    must return an array with the same shape as x.
- npars: int
  - Number of free parameters for the model.
- shortvarnames: tuple[str]
  - TeX-friendly (or presentation-friendly) names for variables used in plots/annotations.
- fitunit: str|None
  - Units for the X-axis (used to convert X before passing to the model).
- centroid_par: any
  - Optional designation of which parameter represents the model centroid.
- fwhm_func, fwhm_pars: optional
  - Functions/parameters to compute the full width at half maximum, if needed.
- integral_func: optional
  - User-provided analytic integral for the model (used for analytical chi-squared or likelihood calculations).
- use_lmfit: bool
  - If True, switch from mpfit to the lmfit backend.
- guess_types: tuple[str]
  - Types for automatic initial guesses. Valid values include 'amplitude',
    'center', and 'width'.
- **kwargs: arbitrary keyword arguments
  - Passed to a helper `_make_parinfo` to construct default parameter info.
