vheightmodel
----------
Purpose
- Creates a variable-height background wrapper around an existing zero-height model.

API
- vheightmodel(zeroheightmodel)
  - Input:
    - zeroheightmodel: the base model function that returns the model values without background.
  - Output:
    - vhm: a wrapper function that adds a variable height (background) term to the base model.

- vhm(xax, *pars, **kwargs)
  - Behavior:

      zeroheightmodel(xax, *pars[1:], **kwargs)

- Practical notes
