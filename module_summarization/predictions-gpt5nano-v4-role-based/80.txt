Module: MapPlotter in cubes
===========================

Overview
--------
This module provides an interactive plotting component for visualizing a spectral cube
and exploring its spectrum interactively. The MapPlotter class is designed to:
- render a 2D map (a plane of a data cube)
- optionally plot using aplpy if a FITS header is available
- support interactive actions via mouse clicks and keyboard input to:
  - plot the selected pixel’s spectrum in a separate window
  - overplot additional spectra
  - disconnect the interactive viewer
  - define circular regions by click-and-drag (or by keyboard shortcuts)

Notes
- The MapPlotter is primarily an interactive tool meant for analysis rather than publication-ready plotting.
- If aplpy is installed and a FITS header is available, the plotting integrates with aplpy for potential publication-quality figures.
- Dependencies include NumPy, Matplotlib, Astropy (WCS, FITS), and optionally aplpy.

Public API
-----------

Class: MapPlotter
-----------------

Description
~~~~~~~~~~~
A helper class to plot a map from a spectral cube and enable interactive spectrum plotting
from the map viewer. The class wires up mouse and keyboard events to allow users to
inspect spectra and perform overplot operations.

Public methods
^^^^^^^^^^^^^^^

__init__(self, Cube=None, figure=None, doplot=False, **kwargs)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Create a MapPlotter instance and optionally render the initial map.

Parameters
- Cube: The data cube object. Typically a FITS-derived object with a header
  that describes the WCS. If provided, the header is flattened via
  cubes.flatten_header(self.Cube.header, delete=True) and used to initialize
  a WCS object (pywcs) when available.
- figure: An existing matplotlib.figure.Figure or an integer figure number.
  If None, a new figure is created when mapplot is called.
- doplot: If True, immediately generate the map by calling mapplot().
- kwargs: Additional arguments passed to the mapplot() call when doplot is True
  (e.g., plotting parameters or overrides).

Attributes set during initialization
- figure: The Matplotlib Figure used for plotting (or None until created).
- axis: The Axes object where the map is drawn.
- FITSFigure: If aplpy is available and used, holds the aplpy.FITSFigure instance.
- _click_marks, _circles: Internal lists used to manage interactive marks and regions.
- _clickX, _clickY: Internal coordinates of the most recent click.
- overplot_colorcycle: Iterator cycling through colors for overplotting spectra.
- overplot_linestyle: Default line style for overplotted spectra.
- Cube: The input cube provided.
- header: Flattened header extracted from the cube (if Cube is provided).
- wcs: astropy.wcs.WCS instance created from the header (if available).
- plane: The 2D plane (map) generated from the cube by makeplane() (set by mapplot()).
- canvas: The Matplotlib Canvas associated with the figure.
- colorbar: Colorbar object (if created).
- origin: Coordinate origin convention used for plotting (0 for FITS, 0 for standard).

__call__(self, **kwargs)
~~~~~~~~~~~~~~~~~~~~~~~~
Delegate to mapplot(**kwargs). Provides a convenient shorthand to re-render or update the map.

mapplot(self, convention='calabretta', colorbar=True, useaplpy=True,
        vmin=None, vmax=None, cmap=None, plotkwargs={}, **kwargs)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Generate and display the map from the cube. This is the core rendering method.

Parameters
- convention: Projection convention to use (default 'calabretta'). Relevant when
  using aplpy.
- colorbar: Whether to display a colorbar alongside the map.
- useaplpy: If True and a FITS header is available, use aplpy for plotting; otherwise
  fall back to a standard Matplotlib imshow.
- vmin, vmax: Data value range to display. If None, these are computed from the
  plane data.
- cmap: Colormap name or object to use for display.
- plotkwargs: Additional keyword arguments passed to the underlying plotting
  call (aplpy.show_colorscale or matplotlib.pyplot.imshow).
- kwargs: Extra keyword arguments forwarded to makeplane (e.g., estimator, etc.).
  Note: If 'estimator' is provided in kwargs, it will be removed before plotting
  the map (as it is used for plane extraction).

Behavior and workflow
- If the internal figure is not set, a new figure is created.
- The map (self.plane) is produced via self.makeplane(**kwargs). The exact
  implementation of makeplane is not shown here, but it is responsible for
  extracting a 2D map slice from the input cube.
- If vmin/vmax are not provided, they are inferred from the non-NaN plane values.
- If aplpy is available and useaplpy is True:
  - A FITSHeader-based image is prepared and plotted with aplpy.FITSFigure.
  - The color scale is shown with show_colorscale, and a colorbar may be added.
  - The axis is derived from the underlying aplpy axes (self.axis).
- If aplpy is unavailable or disabled:
  - A standard Matplotlib axes is created and the plane is displayed via imshow.
  - A colorbar is added if requested.
- The method stores the drawing canvas and connects interactive event handlers
  to support click/drag and keyboard interactions.
- After rendering, interactive event callbacks are connected via self._connect().

_connect(self)
^^^^^^^^^^^^^^
Attach event handlers for:
- button_press_event -> self.click
- button_release_event -> self.plot_spectrum
- key_press_event -> self.plot_spectrum

_disconnect(self)
^^^^^^^^^^^^^^
Detach the above event handlers to disable interactivity.

Notes on interactivity
- Left-click (Button 1) or key '1': plot the spectrum for the clicked pixel in a new window and mark the pixel with an 'x'.
- Middle-click or key 'o': overplot a second spectrum in the external window.
- Right-click (Button 3): disconnect the interactive viewer.
- Drag with Button 1 to select a circular region; press 'c' to set the circle center and 'r' to set the radius for region-averaging operations.

Public usage patterns
---------------------
- Basic usage with an existing cube and automatic plotting:
  mp = MapPlotter(Cube=my_cube, figure=1, doplot=True)
  # Interactive session starts automatically

- Manual control with explicit plotting:
  mp = MapPlotter(Cube=my_cube)
  mp.mapplot(convention='calabretta', colorbar=True)

- Interactive customization:
  mp.mapplot(useaplpy=True, vmin=None, vmax=None, cmap='viridis', plotkwargs={'origin':'lower'})

Dependencies and compatibility
------------------------------
- Core: NumPy, Matplotlib
- Astronomy: Astropy (WCS, FITS), astropy.io.fits (or PyFITS)
- Optional: aplpy (if available, enables publication-quality plotting and FITSFigure)
- The class gracefully falls back to Matplotlib-based plotting when aplpy or WCS support is unavailable.

Examples
--------

Basic example (assuming a cube variable is available):

.. code-block:: python

   from cubes import MapPlotter

   # cube should provide .data and .header attributes compatible with the API
   mp = MapPlotter(Cube=cube, figure=1, doplot=True)

   # Interactive session starts and allows pixel/region interactions

Custom plotting without automatic Doplot:

.. code-block:: python

   mp = MapPlotter(Cube=cube)
   mp.mapplot(convention='griesen', colorbar=True, useaplpy=False, vmin=0, vmax=100)

Related references
------------------
- cubes.flatten_header: utility to flatten a FITS header for WCS initialization.
- aplpy.FITSFigure: optional, for high-quality astronomical plotting.
- pywcs / pyfits: for WCS and FITS header handling.

Implementation notes
--------------------
- The MapPlotter class is designed to be a lightweight, interactive visualization tool
  within the cubes package. It relies on makeplane to extract a 2D map from the input cube
  and then renders the map either via aplpy (when available) or via Matplotlib’s imshow.
- Interactive features are implemented through Matplotlib event connections and are
  primarily intended for exploratory data analysis rather than automation.

Example usage scenarios (summary)
- Quick interactive inspection of a cube’s map with automatic plotting:
  mp = MapPlotter(Cube=cube, figure=1, doplot=True)

- Advanced plotting with explicit controls and no aplpy:
  mp = MapPlotter(Cube=cube)
  mp.mapplot(useaplpy=False, vmin=0, vmax=1, cmap='plasma')
