Isotonic Regression
====================

This module provides utilities for isotonic regression, including a pair of helper functions and a scikit-learn compatible estimator.

Public API
-----------

- check_increasing(x, y)
- isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True)
- IsotonicRegression

Overview
--------

Isotonic regression fits a monotone (non-decreasing or non-increasing) function to data. It is useful when the underlying relationship is known to be monotonic but not strictly linear. This implementation supports optional output bounds, sample weights, and an automatic mode for determining the monotone direction from data.

Functions
---------

check_increasing(x, y)
~~~~~~~~~~~~~~~~~~~~~~~

Determine whether y is monotonically correlated with x using Spearman correlation.

Parameters
- x : array-like of shape (n_samples,)
  Training data.

- y : array-like of shape (n_samples,)
  Training target.

Returns
- increasing_bool : bool
  True if the relationship is increasing (rho >= 0), otherwise False.

Notes
- The Spearman correlation coefficient is estimated from the data; its sign determines the result.
- If the 95% confidence interval of Spearman's rho spans zero (and there are more than 3 samples), a warning may be raised.

References
- Fisher transformation (used to compute a 95% CI for Spearman’s rho):
  https://en.wikipedia.org/wiki/Fisher_transformation

Example
---------

import numpy as np
from isotonic import check_increasing

x = np.array([1, 2, 3, 4])
y = np.array([1, 2, 2.5, 4])
check_increasing(x, y)  # True

isotonic_regression(y)
~~~~~~~~~~~~~~~~~~~~~~~

Solve the isotonic regression problem for a 1-D response y.

Parameters
- y : array-like of shape (n_samples,)
  The data to fit.

- sample_weight : array-like of shape (n_samples,), default=None
  Weights on each point. If None, uniform weights are used.

- y_min : float, default=None
  Lower bound on the lowest predicted value. If None, the bound is -inf.

- y_max : float, default=None
  Upper bound on the highest predicted value. If None, the bound is +inf.

- increasing : bool, default=True
  If True, fit an increasing function; if False, fit a decreasing function.

Returns
- y_ : ndarray of shape (n_samples,)
  The isotonic-fitted values.

Notes
- The function computes the isotonic regression in the specified order
  (increasing or decreasing), applies the fitted isotonic values, and
  clips to the provided bounds if given.
- This is a direct, in-place implementation that leverages a fast
  isotonic regression routine.

Example
--------

import numpy as np
from isotonic import isotonic_regression

y = np.array([3.0, 0.0, 2.0, 5.0, 1.0])
y_fit = isotonic_regression(y)  # isotonic fit (non-decreasing)

# With weights and bounds
weights = np.ones_like(y)
y_fit_bound = isotonic_regression(y, sample_weight=weights, y_min=0.0, y_max=4.0)

IsotonicRegression
------------------

Isotonic regression estimator compatible with scikit-learn’s estimator interface.

Parameters
- y_min : float, default=None
  Lower bound on the lowest predicted value (minimum may still be higher). If None, defaults to -inf.

- y_max : float, default=None
  Upper bound on the highest predicted value (maximum may still be lower). If None, defaults to +inf.

- increasing : bool or 'auto', default=True
  Determines whether the predictions should be constrained to be non-decreasing
  with respect to X. If 'auto', the direction is chosen based on the sign of the
  Spearman correlation estimate.

- out_of_bounds : {'nan', 'clip', 'raise'}, default='nan'
  How predictions are handled for X values outside of the training domain:
  - 'nan': predictions are NaN.
  - 'clip': predictions are clipped to the nearest endpoint.
  - 'raise': a ValueError is raised.

Attributes
- X_min_ : float
  Minimum value of the input array X_ used for the left bound.

- X_max_ : float
  Maximum value of the input array X_ used for the right bound.

- X_thresholds_ : ndarray of shape (n_thresholds,)
  Unique ascending X values used to interpolate the monotonic function
  (the thresholds defining the piecewise-constant/linear segments).

- y_thresholds_ : ndarray of shape (n_thresholds,)
  The corresponding y-values at the thresholds (defining the isotonic
  function).

Notes
- The estimator is designed to be used with a 1-D input X (or a column-like input)
  and a corresponding y, but it adheres to the standard scikit-learn estimator API.
- The 'auto' option for increasing leverages the internal check_increasing logic.

Examples
--------

from isotonic import IsotonicRegression
import numpy as np

# Simple isotonic regression on a 1-D feature
X = np.array([1.0, 2.0, 3.0, 4.0])
y = np.array([3.0, 0.0, 2.0, 5.0])

iso = IsotonicRegression()
y_pred = iso.fit_transform(X, y)

# With bounds and non-increasing constraint
iso_dec = IsotonicRegression(increasing=False, y_min=None, y_max=None)
y_pred_dec = iso_dec.fit_transform(X, y)

# Handling out-of-bounds predictions
iso_oob = IsotonicRegression(out_of_bounds='clip')
y_pred_oob = iso_oob.fit_transform(X, y)
