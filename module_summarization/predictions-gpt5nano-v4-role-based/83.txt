Cockroach Engine (cockroach_engine.rst)
========================================

Overview
--------
CockroachEngine is a CockroachDB-specific extension of Piccolo's standard PostgresEngine. It behaves like the PostgresEngine but adds a CockroachDB–specific initialization step to enable experimental column-type alteration features.

Key characteristics
-------------------
- Inherits from: PostgresEngine
- engine_type: "cockroach"
- min_version_number: 0 (placeholder; CockroachDB versioning differs and this value does not reflect version requirements)

Key Method: prep_database
-------------------------
.. code-block:: python
   async def prep_database(self):
       try:
           await self._run_in_new_connection(
               "SET CLUSTER SETTING sql.defaults.experimental_alter_column_type.enabled = true;"
           )
       except asyncpg.exceptions.InsufficientPrivilegeError:
           colored_warning(
               "=> Unable to set up Cockroach DB "
               "functionality may not behave as expected. Make sure "
               "your database user has permission to set cluster options.",
               level=Level.medium,
           )

Behavior and purpose
--------------------
- During prep_database, CockroachEngine attempts to enable CockroachDB's experimental feature to allow altering column types by setting a cluster setting.
- If the connected user lacks the required privileges to alter cluster settings, a non-fatal warning is emitted to inform the user that some functionality may not behave as expected.

Usage notes
-----------
- This behavior is specific to CockroachDB and is not required for standard PostgreSQL deployments.
- The warning is non-blocking; the engine will continue to operate, but certain operations related to altering column types may be restricted if privileges are insufficient.

Example usage
-------------
.. code-block:: python
   from piccolo.engine.cockroach_engine import CockroachEngine

   cfg = {
       "host": "localhost",
       "database": "mydb",
       "user": "dbuser",
       "password": "secret",
   }
   engine = CockroachEngine(config=cfg)
   await engine.prep_database()
