One-Parameter Exponential Family (GLM) — gee.rst

Overview
--------
The Family class is the base for one-parameter exponential family distributions used by generalized linear models (GLM) in this codebase. Each Family associates:
- a link function (transforming the mean mu to the linear predictor space)
- a variance function (Var(mu))

Key responsibilities include providing convenient helpers for IRLS (Iteratively Reweighted Least Squares) updates:
- starting_mu(y): sensible initial mu for IRLS
- weights(mu): IRLS weights given the current mean
- deviance(endog, mu, ...): deviance for model assessment
- resid_dev(endog, mu, ...): deviance residuals
- fitted(lin_pred): inverse-link transformation to obtain mu from a linear predictor

Design notes
- The link is stored as an instance implementing the required interface (notably deriv(mu) for g'(mu)).
- The variance must be an instance or callable providing Var(mu) for the current mean.
- Passing a link class (instead of an instance) to Family(...) triggers a deprecation warning and an automatic instantiation of the class; prefer passing a Link instance.
- This module is intentionally lightweight and designed to be extended by concrete families (e.g., Poisson, Binomial, Normal), each defining an appropriate variance function and link.

API reference (highlights)
- Class: Family
  - __init__(link, variance)
    - link: a link function instance (must implement deriv(mu))
    - variance: a variance function instance (callable on mu)
  - link (property): get/set the link instance
  - variance (attribute): the variance function instance
  - starting_mu(y): starting value for mu (IRLS)
  - weights(mu): IRLS weights for a given mu
  - deviance(endog, mu, var_weights=1., freq_weights=1., scale=1.): model deviance
  - resid_dev(endog, mu, var_weights=1., scale=1.): deviance residuals
  - fitted(lin_pred): inverse-link transformation to mu from a linear predictor

Example: Minimal usage with an identity link
--------------------------------------------
The following self-contained example demonstrates how to construct a minimal Family with an identity link and a constant variance function. This is suitable for illustrating the API and basic computations (starting_mu, weights, deviance, resid_dev, and fitted) without depending on external link/variance registries.

Code
-----
```python
import numpy as np
from gee import Family

# Minimal identity link implementation (g(mu) = mu)
class IdentityLink:
    def __call__(self, mu):
        # inverse link: mu -> mu (identity)
        return mu

    def deriv(self, mu):
        # derivative g'(mu) for weights: d/dmu of identity is 1
        return np.ones_like(mu)

# Minimal variance: Var(mu) = 1 (e.g., Normal with constant variance)
class ConstantVariance:
    def __call__(self, mu):
        return np.ones_like(mu)

# Construct a simple one-parameter exponential family
F = Family(link=IdentityLink(), variance=ConstantVariance())

# Example data
y = np.array([0.0, 2.0, 1.0, 4.0])
mu = np.array([0.5, 1.5, 1.0, 2.0])

# Starting value for mu in IRLS
mu0 = F.starting_mu(y)

# IRLS weights for given mu
w = F.weights(mu)

# Deviance (with default weights/scales)
D = F.deviance(y, mu)

# Deviance residuals
rd = F.resid_dev(y, mu)

# Fitted values from a linear predictor (identity inverse)
lin_pred = np.array([0.0, 0.5, 1.0, -0.2])
mu_from_lin = F.fitted(lin_pred)

print("starting_mu(y):", mu0)
print("weights(mu):", w)
print("deviance(y, mu):", D)
print("resid_dev(y, mu):", rd)
print("fitted(lin_pred):", mu_from_lin)
```

Expected behavior and notes
- The example shows the minimal surface of the API; real-world usage will typically substitute a concrete Poisson/Binomial/Normal family with their respective link and variance functions.
- The identity link is a convenient baseline for demonstrations and testing; other links require appropriate deriv(mu) implementations and inverse-link behavior supported by the linked codebase.
- For full GLM workflows, pair a suitable link with its canonical/appropriate variance (e.g., Poisson with log link and Var(mu) = mu).
