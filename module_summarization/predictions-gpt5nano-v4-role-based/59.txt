Quantile Regression (QuantReg)
================================

Overview
--------
QuantReg implements a quantile regression model estimated via Iteratively Reweighted Least Squares (IRLS). The model targets conditional quantiles q ∈ (0, 1) of the response variable given the predictors. The asymptotic covariance matrix of the estimated coefficients is obtained through kernel density estimation, with selectable kernel and bandwidth parameters.

Key points
- Special case: q = 0.5 corresponds to the Least Absolute Deviations (LAD) estimator.
- Variance-covariance estimation:
  - vcov = 'robust': heteroskedasticity-robust standard errors (Greene 6th ed.).
  - vcov = 'iid': independent and identically distributed errors (as in Stata).
  - Covariance estimation uses kernel density estimation with a kernel chosen by the user and a bandwidth selection method.
- Kernel options (kernel): 'biw' (Biweight), 'cos' (Cosine), 'epa' (Epanechnikov), 'gau' (Gaussian), 'par' (Parzen).
- Bandwidth options (bandwidth): 'hsheather' (Hall–Sheather), 'bofinger', or 'chamberlain'.
- The algorithm iteratively solves a weighted least squares problem with weights derived from the quantile check function ρ_q.

API
----
Classes
- QuantReg(endog, exog, **kwargs)
  - Inherits from RegressionModel.
  - Core method: fit(...)

Methods
- fit(q=0.5, vcov='robust', kernel='epa', bandwidth='hsheather', max_iter=1000, p_tol=1e-6, **kwargs)
  - Solves the quantile regression by IRLS.
  - Parameters:
    - q: Target quantile (0 < q < 1).
    - vcov: Variance-covariance estimator ('robust' or 'iid').
    - kernel: Kernel for the asymptotic covariance estimation ('biw', 'cos', 'epa', 'gau', 'par').
    - bandwidth: Bandwidth selection method for kernel density estimation ('hsheather', 'bofinger', 'chamberlain').
    - max_iter: Maximum number of IRLS iterations.
    - p_tol: Convergence tolerance for parameter changes.
  - Returns: A RegressionResults wrapper containing estimated coefficients, standard errors, and diagnostic information.

- whiten(data)
  - Model whitening step; for this implementation, returns data unchanged.

Notes
-----
- The underlying estimation is based on IRLS with adaptive weights derived from the quantile check function. The update step uses a preliminary OLS-like solve on a transformed design matrix xstar.
- The initial parameter vector is set to ones for the rank of the design matrix, and the routine starts from an OLS-like solution with xstar = exog / resid.
- Convergence is determined by the maximum absolute change in coefficients (diff) falling below p_tol, or reaching max_iter iterations, or encountering a convergence cycle.

Algorithmic details
-------------------
- Given endog (y) and exog (X), iterative steps compute weights from residuals to steer the update toward the q-quantile.
- At each iteration:
  - Compute beta from a weighted least squares step using xstar.
  - Compute residuals: resid = y - X beta.
  - Construct weights with a quantile-adjusted transformation:
    - Handle near-zero residuals by a tiny offset.
    - Transform resid using q or (1-q) depending on sign.
    - Use absolute value of the transformed residuals to form xstar = X / resid.
  - Update beta by solving (Xᵗ X)⁻¹ Xᵗ y with the current xstar.
  - Check convergence criteria.
- The process continues until convergence, or the maximum number of iterations is reached.

Usage Example
-------------
 Basic usage with a small dataset:

.. code-block:: python

   import numpy as np
   import statsmodels.api as sm

   # Example data
   n = 100
   X = np.random.randn(n, 2)
   X = sm.add_constant(X)  # add intercept
   beta_true = np.array([1.0, -2.0, 0.5])
   y = X @ beta_true + np.random.randn(n)

   # Fit Quantile Regression for the 25th percentile
   mod = sm.QuantReg(y, X)
   res = mod.fit(q=0.25)

   print(res.params)
   print(res.bse)
   print(res.summary())

 Notes on results:
 - res.params contains the estimated coefficients.
 - res.bse provides standard errors corresponding to the chosen vcov method.
 - res.summary() presents a comprehensive report of the fit.
