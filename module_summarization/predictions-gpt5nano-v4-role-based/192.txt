Manifold learning with MDS: A non-linear dimensionality reduction approach

Overview
- Multidimensional Scaling (MDS) is a manifold learning technique that seeks a low-dimensional embedding of data points such that the pairwise dissimilarities between points in the original space are preserved as closely as possible in the embedding.
- The SMACOF (Scaling by MAjorizing a COmplicated Function) algorithm provides a robust iterative method to minimize a stress objective, yielding a non-linear mapping that respects the given dissimilarities.
- This implementation supports both metric MDS (preserving actual dissimilarities) and non-metric MDS (preserving the rank order of dissimilarities via an isotonic regression transform).

Key ideas
- Input is a dissimilarity matrix D, where D[i, j] encodes how different item i is from item j.
- In the metric case, the embedding distances directly aim to match D.
- In the non-metric case, the algorithm first maps the dissimilarities to a monotone (isotonic) surrogate set of disparities that better align with the embedding distances.
- The core optimization uses the Guttman transform to iteratively update the embedding X in a way that reduces the stress (sum of squared differences between target dissimilarities and realized distances).

How SMACOF works (conceptual workflow)
- Initialize X in a low-dimensional space (n_components).
- Compute pairwise Euclidean distances in the current embedding.
- If metric, set disparities equal to the input dissimilarities.
- If non-metric, obtain disparities by isotonic regression of the input dissimilarities against the current distances.
- Compute stress: a measure of misfit between disparities and current distances.
- Update the embedding X using the Guttman transform: X = (1/n) B X, where B is built from the ratio disparities / distances.
- Repeat until convergence, dictated by a tolerance eps or a maximum number of iterations.
- For non-metric MDS, normalized_stress (Stress-1) can be returned to provide a scale-invariant goodness-of-fit metric.

Input and output
- Input:
  - dissimilarities: ndarray of shape (n_samples, n_samples). Must be symmetric.
  - metric: boolean. True for metric MDS, False for non-metric MDS.
  - n_components: int. Target embedding dimensionality.
  - init: optional array-like of shape (n_samples, n_components). If provided, overrides n_components.
  - max_iter: int. Maximum iterations per run.
  - verbose: int. Verbosity level.
  - eps: float. Convergence tolerance for the stress criterion.
  - random_state: int or RandomState. Seed for initialization (reproducibility).
  - normalized_stress: bool. If True and metric=False, return Stress-1 (normalized stress).
- Output:
  - X: ndarray of shape (n_samples, n_components). Embedding coordinates.
  - stress: float. Final stress value (or Stress-1 if normalized_stress is True and non-metric).
  - n_iter: int. Number of iterations performed (until convergence).

Parameter reference (highlights)
- dissimilarities (array-like): Symmetric matrix of pairwise dissimilarities.
- metric (bool): If True, perform metric MDS; if False, perform non-metric MDS.
- n_components (int): Target embedding dimensionality. If init is provided, this is inferred from init.
- init (array-like or None): Starting configuration. If None, a random start is used.
- max_iter (int): Maximum number of iterations per run.
- verbose (int): Verbosity level.
- eps (float): Relative tolerance for convergence based on stress improvement.
- random_state (random_state): Seed for random initialization to ensure reproducibility.
- return_n_iter: (contextual in surrounding estimator) whether to return the number of iterations.
- normalized_stress (bool): If True with non-metric MDS, return Stress-1 rather than raw stress.
- n_jobs: Optional. Number of parallel jobs (if supported by the surrounding estimator).

Implementation notes
- Symmetry check: dissimilarities are validated to be symmetric.
- Non-metric handling: for non-metric MDS, zeros in the dissimilarities are treated as missing values during disparity estimation and are not forced to map to zero in the embedding.
- Isotonic regression: non-metric MDS uses isotonic regression to derive monotone relations between the input dissimilarities and the embedding distances, followed by rescaling to preserve overall normalization.
- Normalized stress (Stress-1): available for non-metric MDS to provide a scale-invariant metric of fit.
- Convergence: the algorithm stops when the improvement in stress (normalized) falls below eps, or when max_iter is reached.
- References: the approach follows classical non-metric and metric MDS theories:
  - Kruskal, J. (1964). Nonmetric multidimensional scaling: a numerical method.
  - Kruskal, J. (1964). Multidimensional scaling by optimizing goodness of fit to a nonmetric hypothesis.
  - Borg, I., & Groenen, P. (1997). Modern Multidimensional Scaling – Theory and Applications.

Practical guidance
- When you have a precomputed dissimilarity matrix (e.g., from domain-specific measures, psychometrics, or distance-based features), you can apply MDS to obtain a 2D or 3D visualization that preserves the relationships captured by those dissimilarities.
- Metric MDS is appropriate when the dissimilarities reflect actual distances in the original space you wish to preserve.
- Non-metric MDS is useful when only the order of dissimilarities matters (e.g., subjective or ordinal similarity scores).
- Normalized_stress provides a more interpretable goodness-of-fit measure in non-metric contexts.
- Computational considerations: SMACOF scales with the number of samples n and embedding dimension n_components; larger datasets require more computation per iteration.

Examples
- Basic metric MDS with precomputed dissimilarities
  - Given a symmetric dissimilarity matrix D (shape (n_samples, n_samples)):
    - from sklearn.manifold import MDS
    - mds = MDS(metric=True, dissimilarities='precomputed', n_components=2, random_state=0)
    - X = mds.fit_transform(D)

- Basic non-metric MDS with precomputed dissimilarities
  - mds = MDS(metric=False, dissimilarities='precomputed', n_components=2, random_state=0, normalized_stress=True)
  - X, stress = mds.fit_transform(D, return_stress=True)  # depending on API surface in the estimator
