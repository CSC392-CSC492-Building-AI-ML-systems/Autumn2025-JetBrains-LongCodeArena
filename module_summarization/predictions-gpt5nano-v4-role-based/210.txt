Empirical likelihood (emplike)
===============================

Overview
--------

This module provides empirical likelihood (EL) inference for descriptive statistics. It supports hypothesis testing and confidence interval construction for descriptive quantities such as the mean, variance, skewness, kurtosis and correlation. If matplotlib is available, it can also produce multivariate confidence region plots and mean–variance contour plots.

Key ideas
---------

- EL constructs likelihood-based tests and intervals without making parametric distributional assumptions beyond the estimating equations.
- Inference is driven by estimating equations that describe the descriptive statistic of interest. The model seeks a discrete probability weights vector p over observations that maximize a profile likelihood subject to the estimating equations.
- The optimization is performed in terms of a Lagrange multiplier (eta) that enforces the estimating equations under the weighted empirical distribution. The core machinery is encapsulated in the internal class _OptFuncts.
- The implementation uses a log-star transformation of observation probabilities and a modified Newton method (via scipy’s _fit_newton) to solve for eta.

Public API
----------

Note: The public API is exposed through DescStat, which dispatches to univariate or multivariate implementations. Internal optimization is performed by _OptFuncts; the latter is not intended for direct use in typical user workflows.

- DescStat(endog)
  - Description: Factory function that returns an inference object for descriptive statistics.
  - Parameters:
    - endog : ndarray
      Data array. If 1-D, it is reshaped to a column vector. If it has a single column, a univariate descriptor (DescStatUV) is returned; if multiple columns, a multivariate descriptor (DescStatMV) is returned.
  - Returns:
    - DescStatUV when endog has a single variable.
    - DescStatMV when endog contains multiple variables.

- _OptFuncts(endog)
  - Description: Internal container for optimization helpers used by the empirical likelihood machinery.
  - Important: This class is intended for internal use; users should interact with DescStat and its un/ multivariate variants.

Internal optimization utilities (used by the public API)
--------------------------------------------------------

The _OptFuncts class implements the optimization backbone for empirical likelihood. The following methods are defined:

- _log_star(eta, est_vect, weights, nobs)
  - Purpose: Compute the transformed log-probabilities (log-star) used in the Newton-based optimization for the Lagrange multipliers.
  - Parameters:
    - eta : array_like, shape (1, m)
    - est_vect : ndarray, shape (n, k)
      Estimating equations evaluated at each observation.
    - weights : 1-D array, length n
      Observation weights.
    - nobs : int
      Number of observations.
  - Returns:
    - data_star : ndarray
      Transformed log-probabilities (used internally for optimization). The function is a placeholder for the _fit_newton process; the resulting values are not directly used for the final likelihood ratio calculation beyond the optimization step.

- _hess(eta, est_vect, weights, nobs)
  - Purpose: Compute the Hessian (second derivative) of the weighted EL objective with respect to eta.
  - Returns:
    - hess : ndarray
      Hessian matrix (m x m).

- _grad(eta, est_vect, weights, nobs)
  - Purpose: Compute the gradient of the weighted EL objective with respect to eta.
  - Returns:
    - gradient : ndarray
      Gradient vector (m, 1).

- _modif_newton(eta, est_vect, weights)
  - Purpose: Solve for the optimal eta by applying a modified Newton method to maximize the log-star objective.
  - Parameters:
    - eta : ndarray, shape (1, m)
    - est_vect : ndarray, shape (n, k)
    - weights : 1-D array, length n
  - Returns:
    - params : 1 x m array
      The Lagrange multipliers that maximize the (log-star) objective.

- _find_eta(eta)
  - Description: Placeholder for a higher-level entry point to locate the optimal eta.
  - Note: The provided excerpt shows this method but its implementation is not included. It is part of the internal mechanism that determines the EL weights/parameters.

Notes on usage and workflow
---------------------------

- Endogenous data shape governs the inference type:
  - 1-D input is reshaped to a (n, 1) column vector and treated as univariate.
  - 2-D input with one column yields a univariate descriptor (DescStatUV).
  - 2-D input with multiple columns yields a multivariate descriptor (DescStatMV).
- The public DescStat interface delegates to DescStatUV or DescStatMV, which expose the user-facing API for hypothesis tests and confidence intervals.
- The optimization core relies on:
  - Estimating equations est_vect: a matrix of dimension (n, k) capturing the constraints to be satisfied by the EL probabilities.
  - Weights: observation weights (often uniform by default).
  - A Lagrange multiplier vector eta that enforces the estimating equations in the EL framework.
  - The modified Newton routine (_fit_newton) drives the search for eta by optimizing the log-star objective.

Caveats and details
-------------------

- The _log_star function and related gradient/hessian calculations are internal and tailored to the empirical likelihood solver. They are not intended for direct use by typical end users.
- The module relies on SciPy for optimization (scipy.optimize._fit_newton) and scipy.stats for asymptotic references (chi-square, skewness, kurtosis).
- If matplotlib is installed, additional plotting capabilities are available for multivariate confidence regions and mean–variance contours.

Algorithmic background
----------------------

- Empirical likelihood constructs nonparametric likelihoods using weights p_i on observations, subject to estimating equations E_p[g(X_i, theta)] = 0.
- The EL ratio is formed by maximizing the nonparametric likelihood under these constraints, leading to a profile likelihood that depends on a Lagrange multiplier eta.
- Inference (e.g., confidence regions) is based on the log-likelihood ratio statistic, which asymptotically follows a chi-square distribution under regularity conditions.
