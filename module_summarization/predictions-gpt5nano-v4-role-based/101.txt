web3.miner.rst

Overview
- The Miner API is a lightweight wrapper around the Ethereum JSON-RPC miner namespace. It allows you to query and control the node’s mining state through a Web3 instance.
- Access is via web3.miner on a compatible Web3 object. All calls are performed through the node’s request manager (request_blocking), and the hashrate property is formatted for readability.

Class: Miner
- Purpose: Encapsulates miner-related RPC methods and exposes them as convenient Python methods.

Constructor
- Miner(web3)
  - web3: A Web3 instance. The Miner wrapper uses the node’s request manager via web3._requestManager to perform RPC calls.
  - Example:
    - web3 = Web3(HTTPProvider("http://localhost:8545"))
    - miner = web3.miner

Properties
hashrate
- Description: Current mining hash rate reported by the node.
- Implementation detail: The RPC response is passed through a formatter (to_decimal) via apply_formatters_to_return, so the value is returned as a decimal integer rather than a hex string.
- RPC method: eth_hashrate
- Returns: int (hash rate in H/s), formatted to decimal.
- Access: web3.miner.hashrate
- Example:
  - h = web3.miner.hashrate
  - print(h)  # e.g., 12345678

Methods
makeDAG(number)
- Purpose: Create or regenerate the mining DAG for the specified epoch.
- Parameters:
  - number: int — the DAG epoch number to build.
- RPC method: miner_makeDag
- Returns: The RPC result (as returned by the node).
- Example:
  - web3.miner.makeDAG(1)

setExtra(extra)
- Purpose: Set the extra data field to be included in mined blocks.
- Parameters:
  - extra: str — extra data to append (typically a hex string or ASCII-encoded data).
- RPC method: miner_setExtra
- Returns: The RPC result.
- Example:
  - web3.miner.setExtra("0xabcdef")

setEtherBase(etherbase)
- Purpose: Set the etherbase (coinbase) address for mined blocks.
- Parameters:
  - etherbase: str — Ethereum address to receive mining rewards.
- RPC method: miner_setEtherbase
- Returns: The RPC result.
- Example:
  - web3.miner.setEtherBase("0xYourEtherbaseAddress")

setGasPrice(gas_price)
- Purpose: Configure the gas price used by miners.
- Parameters:
  - gas_price: int — gas price in wei (e.g., 2000000000 for 2 Gwei).
- RPC method: miner_setGasPrice
- Returns: The RPC result.
- Example:
  - web3.miner.setGasPrice(2000000000)

start(num_threads)
- Purpose: Start mining using the specified number of CPU threads.
- Parameters:
  - num_threads: int — number of mining threads to use.
- RPC method: miner_start
- Returns: The RPC result.
- Example:
  - web3.miner.start(2)

stop()
- Purpose: Stop mining.
- Parameters: None
- RPC method: miner_stop
- Returns: The RPC result.
- Example:
  - web3.miner.stop()

startAutoDAG()
- Purpose: Enable automatic DAG generation as needed.
- Parameters: None
- RPC method: miner_startAutoDag
- Returns: The RPC result.
- Example:
  - web3.miner.startAutoDAG()

stopAutoDAG()
- Purpose: Disable automatic DAG generation.
- Parameters: None
- RPC method: miner_stopAutoDag
- Returns: The RPC result.
- Example:
  - web3.miner.stopAutoDAG()

Implementation notes
- All methods delegate to the node via web3._requestManager.request_blocking and return the raw JSON-RPC result (except for hashrate, which is formatted to decimal by to_decimal).
- The hashrate property is read-only; it fetches data from the node each time it is accessed.

Usage example
- Basic workflow illustrating common operations:
  - web3 = Web3(HTTPProvider("http://localhost:8545"))
  - miner = web3.miner
  - print("Current hashrate:", miner.hashrate)
  - miner.setGasPrice(2000000000)
  - miner.start(1)
  - miner.makeDAG(1)
  - miner.setExtra("0x1234")
  - miner.setEtherBase("0xYourMinerAddress")
  - miner.stop()
