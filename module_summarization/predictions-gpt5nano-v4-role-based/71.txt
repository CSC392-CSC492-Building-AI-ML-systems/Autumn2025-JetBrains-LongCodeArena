KernelRidge
===========

Module: sklearn.kernel_ridge

Overview
--------
Kernel ridge regression (KRR) combines ridge regression (linear least squares with L2 regularization)
with the kernel trick. It learns a linear function in the feature space induced by the chosen kernel.
For non-linear kernels, this corresponds to a non-linear function in the original input space.

The model learned by KRR has the same functional form as Support Vector Regression (SVR) with a squared
error loss, but it uses L2 regularization and a closed-form solution. This makes fitting faster than SVR
on medium-sized datasets, at the cost of non-sparsity in the resulting model. KRR supports multi-output
regression (y can be 1D or 2D: shape [n_samples, n_targets]).

This estimator is documented here as part of the scikit-learn API and follows the same conventions as other
regressors in the library. See the User Guide for kernel ridge specifics.

Key references:
- Kevin P. Murphy, "Machine Learning: A Probabilistic Perspective", ch. 14.4.3

API
---
The primary estimator is KernelRidge.

class KernelRidge(MultiOutputMixin, RegressorMixin, BaseEstimator)

Initialization parameters
-------------------------
alpha : float or array-like of shape (n_targets,), default=1.0
    Regularization strength (positive). Larger values imply stronger regularization.
    If an array is provided, penalties apply per target. Equivalent to 1 / (2C) in other linear models.

kernel : str or callable, default="linear"
    Kernel mapping used internally. Passed to pairwise_kernels. If a string, it must be one of the
    supported kernels or "precomputed". If "precomputed", X is treated as a kernel matrix.
    If a callable, it should be a function that accepts two 1D arrays (two samples) and returns a
    scalar kernel value. Note: when using a callable kernel, parameters can be provided via
    kernel_params.

gamma : float, default=None
    Gamma parameter for certain kernels (e.g., RBF, polynomial, sigmoid). Its interpretation depends on the
    kernel in use. Ignored by kernels that do not use gamma.

degree : int, default=3
    Degree of the polynomial kernel. Ignored by other kernels.

coef0 : float, default=1
    Zero coefficient for polynomial and sigmoid kernels. Ignored by other kernels.

kernel_params : dict, default=None
    Additional keyword arguments for the kernel function when kernel is a callable.

Attributes
----------
dual_coef_ : ndarray of shape (n_samples,) or (n_samples, n_targets)
    Representation of the weights in kernel space (the dual coefficients).

X_fit_ : ndarray or sparse matrix of shape (n_samples, n_features)
    The data used to fit the model. If kernel == "precomputed", this contains the training kernel
    matrix of shape (n_samples, n_samples).

n_features_in_ : int
    Number of features seen during fit.

feature_names_in_ : ndarray of shape (n_features_in_,)
    Names of features seen during fit. Defined only if X has feature names that are all strings.
    Versionadded: 1.0

Notes
-----
- KernelRidge can handle multi-target regression via the y array having shape (n_samples, n_targets).
- When kernel == "precomputed", the input X to fit/predict is treated as a kernel matrix.
- The internal solution corresponds to solving (K + alpha I) alpha = y, where K is the kernel matrix.
- The model is non-sparse, which can affect prediction speed on very large datasets.

Examples
--------
Basic usage with a standard kernel

.. code-block:: python

    from sklearn.kernel_ridge import KernelRidge
    import numpy as np

    # Generate some data
    rng = np.random.RandomState(0)
    n_samples, n_features = 50, 5
    X = rng.randn(n_samples, n_features)
    y = rng.randn(n_samples)

    # Create and fit a KernelRidge model
    krr = KernelRidge(alpha=1.0, kernel="rbf", gamma=0.5)
    krr.fit(X, y)

    # Predict on new data
    y_pred = krr.predict(X)

Precomputed kernel matrix

.. code-block:: python

    from sklearn.kernel_ridge import KernelRidge
    import numpy as np

    rng = np.random.RandomState(0)
    X = rng.randn(40, 4)
    K = (X @ X.T)  # example kernel: linear
    krr = KernelRidge(alpha=1.0, kernel="precomputed")
    krr.fit(K, rng.randn(40))
    y_pred = krr.predict(K)  # predictions on the same kernel matrix

Custom kernel function

.. code-block:: python

    import numpy as np
    from sklearn.kernel_ridge import KernelRidge

    def my_kernel(a, b):
        # Simple example: dot product plus a bias term
        return a.dot(b) + 1.0

    X = np.random.RandomState(0).randn(15, 3)
    y = np.random.RandomState(0).randn(15)

    krr = KernelRidge(alpha=0.5, kernel=my_kernel)
    krr.fit(X, y)
    pred = krr.predict(X)

Multi-output regression example

.. code-block:: python

    import numpy as np
    from sklearn.kernel_ridge import KernelRidge

    rng = np.random.RandomState(0)
    X = rng.randn(100, 4)
    Y = rng.randn(100, 3)  # three targets

    krr = KernelRidge(alpha=1.0, kernel="linear")
    krr.fit(X, Y)
    Y_pred = krr.predict(X)
