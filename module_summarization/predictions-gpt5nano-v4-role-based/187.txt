Isotonic regression
===================

Isotonic regression learns a monotone (nondecreasing or nonincreasing) function that best fits the given data in a least-squares sense. It can also apply optional bounds to the predicted values and control how predictions are produced for inputs outside the training domain.

Key concepts
------------
- Monotone fitting: find a function f such that f(x1) <= f(x2) whenever x1 <= x2 (for nondecreasing) or the opposite for nonincreasing.
- Weighted fitting: optional sample weights can be supplied to emphasize certain samples.
- Bounds: optional y_min and y_max to constrain the fitted values within a fixed range.
- Auto direction: the IsotonicRegression model can automatically decide whether to fit a nondecreasing or nonincreasing function by inspecting the sign of the Spearman correlation between X and y.

API reference
-------------

check_increasing
  Determine whether y is monotonically correlated with x.

  Signature
  - check_increasing(x, y)

  Parameters
  - x: array-like, shape (n_samples,)
    Training data.
  - y: array-like, shape (n_samples,)
    Training target.

  Returns
  - increasing_bool: bool
    True if the relationship is increasing with x, False if decreasing.

  Notes
  - The sign of Spearman's rho is used to decide increasing vs. decreasing.
  - A Fisher transform-based 95% confidence interval for rho is computed; if the interval spans zero, a warning is emitted.

  See also
  - Spearman correlation (scipy.stats.spearmanr)

isotonic_regression
  Compute isotonic regression for a 1D target y, with optional weights and bounds.

  Signature
  - isotonic_regression(y, *, sample_weight=None, y_min=None, y_max=None, increasing=True)

  Parameters
  - y: array-like of shape (n_samples,)
    The data to fit.
  - sample_weight: array-like of shape (n_samples,), default=None
    Weights for each observation. If None, weights are set to 1.
  - y_min: float or None
    Lower bound on the predicted values. If None, defaults to -inf.
  - y_max: float or None
    Upper bound on the predicted values. If None, defaults to +inf.
  - increasing: bool, default True
    If True, fit a nondecreasing function; if False, fit a nonincreasing function.

  Returns
  - y_: ndarray of shape (n_samples,)
    Isotonic fit of y, respecting the specified direction and bounds.

  Notes
  - The implementation performs an in-place, contiguous isotonic regression.
  - After fitting, values are clipped to [y_min, y_max] if bounds are provided.
  - If increasing is False, the input y is processed in reverse order to enforce nonincreasing behavior, and the result is reordered back.

  References
  - Best & Chakravarti, “Active set algorithms for isotonic regression; A unifying framework,” section 3.

IsotonicRegression
  Isotonic regression model that maps real-valued inputs to a monotone function of X, with optional bounds and out-of-bounds handling.

  Version added
  - 0.13

  Parameters
  - y_min: float, default=None
    Lower bound on the lowest predicted value (minimum value may still be higher). If not set, defaults to -inf.
  - y_max: float, default=None
    Upper bound on the highest predicted value (maximum may still be lower). If not set, defaults to +inf.
  - increasing: bool or 'auto', default True
    Determines whether predictions should be constrained to increase with X.
    - True: nondecreasing
    - False: nonincreasing
    - 'auto': decide based on the sign of the Spearman correlation estimate (via check_increasing)

  out_of_bounds: {'nan', 'clip', 'raise'}, default 'nan'
    How predictions for X outside the training domain are handled during prediction:
    - 'nan': predictions are NaN
    - 'clip': predictions are set to the value corresponding to the nearest training interval endpoint
    - 'raise': a ValueError is raised

  Attributes
  - X_min_: float
    Minimum value of input array X_ used for the left bound of interpolation.

  - X_max_: float
    Maximum value of input array X_ used for the right bound of interpolation.

  - X_thresholds_: ndarray, shape (n_thresholds,)
    Unique ascending X values used to interpolate the monotone function y = f(X).

    Note: This represents the "knots" of the fitted monotone function when applied to X.

  - y_thresholds_: ndarray
    Fitted monotone function values corresponding to X_thresholds_.

  Examples
  Basic usage
  - Fit a simple isotonic regression on a 1D response y and obtain a monotone fit.

  .. code-block:: python

    from sklearn.isotonic import isotonic_regression

    y = [3.0, 1.0, 4.0, 2.0, 5.0]
    y_fit = isotonic_regression(y)
    # y_fit is monotone nondecreasing and close to y in L2 sense

  With bounds
  - Fit with bounds to constrain the predictions.

  .. code-block:: python

    y = [0.5, 2.0, 1.0, 3.5]
    y_fit = isotonic_regression(y, y_min=0.0, y_max=2.5)

  Using the IsotonicRegression class
  - Create a model that automatically decides the direction and can handle out-of-bounds.

  .. code-block:: python

    from sklearn.isotonic import IsotonicRegression
    import numpy as np

    X = np.array([0.1, 0.4, 0.2, 0.8, 0.5])
    y = np.array([1.0, 2.5, 1.5, 3.0, 2.0])

    model = IsotonicRegression(increasing='auto', out_of_bounds='clip')
    model.fit(X, y)
    y_pred = model.predict(X)

Notes
-----
- The core algorithm is an in-place, contiguous isotonic regression algorithm, based on the pool-adjacent-violators framework.
- If increasing is set to 'auto', the direction of monotonicity is chosen according to the sign of the Spearman correlation between X and y (via check_increasing).
- Bounds y_min and y_max are applied after the isotonic fit via clipping. If either bound is None, the corresponding bound is effectively infinite.
- For predictions with inputs outside the training domain, the out_of_bounds setting controls the behavior:
  - 'nan': return NaN
  - 'clip': interpolate to the nearest boundary value
  - 'raise': raise a ValueError

--------------------
- The isotonic_regression function relies on _inplace_contiguous_isotonic_regression and _make_unique from the internal _isotonic module.
- check_increasing uses spearmanr and Fisher transform to assess monotonicity and emit a warning if the confidence interval spans zero.
