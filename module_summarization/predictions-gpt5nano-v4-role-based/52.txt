Module: gmm.rst

Overview
--------
This module implements an instrumental variables estimator using Two-Stage Least Squares (2SLS) and provides a lightweight scaffolding for Generalized Method of Moments (GMM) style estimation. The core class, IV2SLS, inherits from LikelihoodModel and performs the standard 2SLS procedure using matrix algebra. Results are returned as a RegressionResults wrapper, with an additional OLS second-stage model available for diagnostic purposes.

Key concepts
------------
- Endogenous equation: y ~ X
- Instruments: Z (which must include any exogenous variables intended for instrumenting)
- Two-stage procedure:
  1. First stage: regress X on Z to obtain the predicted endogenous variables Xhat
  2. Second stage: regress y on Xhat to obtain parameter estimates
- Covariance: a simple, analytical form is computed from the second-stage cross-products (using the inverted cross-product matrices)
- Results wrapping: IV2SLS returns a RegressionResultsWrapper around an IVRegressionResults object, complemented by an auxiliary OLS fit of y on Xhat

Core functionality
------------------
- Class IV2SLS(LikelihoodModel)
  - Purpose: estimate a linear model with endogenous regressors using instrumental variables (2SLS)
  - Parameters
    - endog: endogenous dependent variable (nobs x 1)
    - exog: explanatory variables (nobs x k)
    - instrument: instruments for exog (nobs x m). Must include non-instrumented exog variables as well
  - Derived attributes
    - df_resid: degrees of freedom for residuals
    - df_model: degrees of freedom for the model
    - wendog, wexog:working copies used internally (initialized to endog and exog)
  - Key methods
    - initialize(self)
      - Initializes working copies (wendog, wexog)
    - whiten(self, X)
      - Placeholder: not implemented
    - fit(self)
      - Executes the 2SLS estimation
      - Steps:
        1) Compute Z^T Z and Z^T X
        2) Solve for xhatparams = (Z^T Z)^{-1} (Z^T X)
        3) Form F = Z XhatParams; compute FtF = F^T F
        4) Compute F^T y (Fty) and F^T X (Ftx)
        5) Solve for params = (F^T F)^{-1} (F^T y)
        6) Compute the inverse of F^T X (Ftxinv) and construct normalized_cov_params = Ftxinv^T FtF Ftxinv
        7) Create an IVRegressionResults instance with params and normalized_cov_params
        8) Store xhatparams and xhat (the predicted X from the first stage) for diagnostics
        9) Compute a secondary OLS fit of y on xhat and store it as _results_ols2nd
      - Returns: RegressionResultsWrapper(lfit), where lfit is an IVRegressionResults instance
    - predict(self, params, exog=None)
      - Returns the linear predictions given a parameter vector
      - If exog is not provided, uses self.exog
- Class IVRegressionResults(RegressionResults)
  - Subclass of RegressionResults tailored for the 2SLS IV results
  - Inherits standard result statistics from RegressionResults
  - Serves as the primary results container for IV2SLS

Usage example (conceptual)
--------------------------
from gmm import IV2SLS

# y, X, Z are numpy arrays (nobs x 1), (nobs x k), (nobs x m) respectively
model = IV2SLS(endog=y, exog=X, instrument=Z)
results = model.fit()

# Access parameter estimates
params = results.params

# Access second-stage OLS fit on Xhat
second_stage = model._results_ols2nd
print(second_stage.summary())

Notes and limitations
---------------------
- Whitening or robust weighting options are not implemented in this core 2SLS path.
- The whiten method is a placeholder (Not implemented) and can be extended for generalized whitened estimators.
- The implementation follows textbook 2SLS calculations and relies on numpy.linalg.solve for linear solves.
- The class is designed to be compatible with Statsmodels' result handling by returning a RegressionResultsWrapper.
- The module documents basic 2SLS behavior; extensions toward full GMM with arbitrary moment conditions are intended but not included in the visible code path.
- A secondary OLS fit on the first-stage predicted X (xhat) is provided for diagnostic or comparison purposes.

Implementation notes
--------------------
- Core calculation:
  - First stage: xhatparams = (Z^T Z)^{-1} Z^T X
  - Predicted X: F = Z XhatParams
  - Second stage: params = (F^T F)^{-1} F^T y
  - Covariance: normalized_cov_params = (F^T X)^{-1,T} (F^T F) (F^T X)^{-1}
- Results:
  - IVRegressionResults stores the two-stage estimates
  - _results_ols2nd provides access to a traditional OLS fit of y on Xhat
  - predict() is implemented to produce fitted values from the given parameters

Source references in code
-------------------------
- IV2SLS is implemented as a subclass of LikelihoodModel
- Uses OLS from statsmodels.regression.linear_model for the second-stage fit
- Returns a RegressionResultsWrapper to align with existing results interfaces
