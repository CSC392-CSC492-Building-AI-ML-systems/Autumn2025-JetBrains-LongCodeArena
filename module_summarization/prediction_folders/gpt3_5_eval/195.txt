Learn Gaussian Mixture Models
=============================

Base class for mixture models.
------------------------------

This abstract class specifies an interface for all mixture classes and provides basic common methods for mixture models.

Parameters
----------
- n_components : int
    Number of mixture components.
- tol : float
    Tolerance for convergence.
- reg_covar : float
    Non-negative regularization added to the diagonal of covariance.
- max_iter : int
    Maximum number of EM iterations.
- n_init : int
    Number of initializations to perform.
- init_params : str
    Method for initialization, options are "kmeans", "random", "random_from_data", "k-means++".
- random_state : int, RandomState instance or None
    Controls the random seed.
- warm_start : bool
    Whether to reuse the solution of the previous call to fit as initialization.
- verbose : bool
    Enable verbose output.
- verbose_interval : int
    Number of iterations between verbose output.

Methods
-------
- _check_parameters(X)
    Check initial parameters of the derived class.
- _initialize_parameters(X, random_state)
    Initialize the model parameters.
- _initialize(X, resp)
    Initialize the model parameters of the derived class.
- fit(X, y=None)
    Estimate model parameters with the EM algorithm.
- fit_predict(X, y=None)
    Estimate model parameters using X and predict the labels for X.

Raises
------
- ValueError
    If the parameter shape is incorrect.
- ConvergenceWarning
    If the algorithm does not converge.
