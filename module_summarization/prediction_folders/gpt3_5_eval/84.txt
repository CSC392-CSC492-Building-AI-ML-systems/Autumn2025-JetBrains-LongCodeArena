Generate documentation for this module
=======================================

This module provides functionality for generating documentation for Piccolo apps. It includes classes and functions for managing app configurations, migrations, tables, and commands.

MigrationModule Class
---------------------
- `ID`: A string attribute representing the ID of the migration module.
- `VERSION`: A string attribute representing the version of the migration module.
- `DESCRIPTION`: A string attribute representing the description of the migration module.
- `forwards()`: A static asynchronous method that performs migration forwards.

PiccoloAppModule Class
----------------------
- `APP_CONFIG`: An attribute representing the AppConfig for the Piccolo app.

table_finder Function
---------------------
- `table_finder(modules, include_tags=None, exclude_tags=None, exclude_imported=False)`: 
  A function that imports Table subclasses from specified modules based on include and exclude tags.

Command Class
-------------
- `callable`: A callable attribute representing the function or coroutine.
- `aliases`: A list attribute containing aliases for the command.

AppConfig Class
---------------
- `app_name`: A string attribute representing the name of the app.
- `migrations_folder_path`: A string attribute representing the path to the app's migration files.
- `table_classes`: A list attribute containing registered Table classes.
- `migration_dependencies`: A list attribute containing dependencies for app migrations.
- `commands`: A list attribute containing functions and coroutines registered with the Piccolo CLI.
- `register_table(table_class)`: A method to register a Table class.
- `migration_dependency_app_configs`: A property to get AppConfig instances from migration dependencies.
- `get_table_with_name(table_class_name)`: A method to get a Table subclass by name.

AppRegistry Class
-----------------
- `apps`: A list attribute containing paths to Piccolo apps.
- `app_configs`: A dictionary attribute mapping app names to AppConfig instances.
