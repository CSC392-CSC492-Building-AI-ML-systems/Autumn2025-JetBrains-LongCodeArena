Generate documentation for the `rand_augment` module
===================================================

The `rand_augment` module provides functionality to apply the RandAugment augmentation scheme to a batch of samples. This scheme is based on the paper [RandAugment: Practical automated data augmentation with a reduced search space](https://arxiv.org/abs/1909.13719).

Functions
---------

.. function:: rand_augment(data, n, m, num_magnitude_bins=31, shape=None, fill_value=128, interp_type=None, max_translate_abs=None, max_translate_rel=None, seed=None, monotonic_mag=True, excluded=None) -> DataNode

    Applies RandAugment augmentation scheme to the provided batch of samples.

    Parameters
    ~~~~~~~~~~

    - ``data`` (:class:`DataNode`): A batch of samples to be processed.
    - ``n`` (int): The number of randomly sampled operations to be applied to a sample.
    - ``m`` (int): A magnitude (strength) of each operation to be applied.
    - ``num_magnitude_bins`` (int, optional): The number of bins to divide the magnitude ranges into.
    - ``shape`` (DataNode or Tuple[int, int], optional): The size of the image or frames in the video sequence.
    - ``fill_value`` (int, optional): A value to be used as padding for transformed images/frames.
    - ``interp_type`` (:class:`types.DALIInterpType`, optional): Interpolation method used by warp_affine ops.
    - ``max_translate_abs`` (int or (int, int), optional): Maximal shift in pixels in the translation augmentation.
    - ``max_translate_rel`` (float or (float, float), optional): Maximal shift as a fraction of image shape.
    - ``seed`` (int, optional): Seed to be used for random sampling operations.
    - ``monotonic_mag`` (bool, optional): Flag to control the strength of operations.
    - ``excluded`` (List[str], optional): List of names of operations to be excluded.

    Returns
    ~~~~~~~~

    - :class:`DataNode`: A batch of transformed samples.
