Unified Renderable Interface
============================

This module defines an interface for rendering components in a unified way. The core class in this interface is the `AbstractRenderable` class, which provides methods for rendering templates and managing context data.

AbstractRenderable Class
------------------------

The `AbstractRenderable` class is an abstract base class that implements the renderable interface. It provides the following methods:

- `get_template_names`: Returns the template name(s) to be used for rendering.
- `get_context_data`: Returns the context data to be passed to the template.
- `render`: Renders the template with the provided context data.

AbstractRenderableWithCss Class
-------------------------------

The `AbstractRenderableWithCss` class extends the `AbstractRenderable` class by adding methods for managing CSS classes. It provides the following additional methods:

- `get_base_css_classes_list`: Returns the base CSS classes for the component.
- `get_extra_css_classes_list`: Returns any extra CSS classes for the component.
- `get_css_classes_list`: Returns a list of CSS classes for the component.
- `css_classes`: Property that joins the CSS classes into a string.

AbstractBemRenderable Class
---------------------------

The `AbstractBemRenderable` class is an alternative to `AbstractRenderableWithCss` that uses the BEM (Block Element Modifier) naming convention for CSS classes. It provides methods for managing BEM block, element, and variants. The following methods are available:

- `get_bem_block`: Returns the BEM block string.
- `get_bem_element`: Returns the BEM element string.
- `bem_block_or_element`: Property that returns the BEM block or element.
