Generate documentation for this module
======================================

This module provides functionality for generating documentation for Piccolo projects. It includes classes and functions for managing Piccolo apps, migrations, tables, and commands.

MigrationModule
---------------
- Represents a migration module with attributes ID, VERSION, DESCRIPTION, and a method forwards().

PiccoloAppModule
----------------
- Represents a Piccolo app module with an attribute APP_CONFIG of type AppConfig.

table_finder
------------
- A function for finding and importing Table subclasses from specified modules based on include and exclude tags.

Command
-------
- A dataclass representing a command with a callable and a list of aliases.

AppConfig
---------
- A dataclass representing the configuration for a Piccolo app, including app name, migrations folder path, table classes, migration dependencies, and commands.

AppRegistry
-----------
- A class for recording Piccolo apps in a project, with a list of app paths.

This module provides a comprehensive set of tools for managing and documenting Piccolo projects, including defining app configurations, registering tables, and handling migrations and commands.
