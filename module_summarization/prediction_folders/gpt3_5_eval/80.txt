MapPlot
-------

Make plots of the cube and interactively connect them to spectrum plotting.
This is an interactive component of the package designed for user interactive analysis. The plotter utilizes `APLpy <https://github.com/aplpy/aplpy>`_ for creating publication-quality plots.

:author: Adam Ginsburg
:date: 03/17/2011

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

MapPlotter Class
----------------

The `MapPlotter` class is used to plot a spectrum. It provides an interactive environment for plotting and analyzing data.

Initialization
~~~~~~~~~~~~~~

The `MapPlotter` class can be initialized with the following parameters:

- `Cube`: Data cube to be plotted
- `figure`: Matplotlib figure to use for plotting
- `doplot`: Boolean flag to indicate whether to plot immediately
- `**kwargs`: Additional keyword arguments

Methods
~~~~~~~

- `mapplot`: Plot a map based on an input data cube. The plotted map is interactive, allowing users to click on it with different mouse buttons to perform various actions.
- `__call__`: Alias for `mapplot`
- `_connect`: Connect click, click up (release click), and key press events to the canvas
- `_disconnect`: Disconnect click, click up, and key press events from the canvas

Parameters
~~~~~~~~~~

- `convention`: Projection convention for Galactic data
- `colorbar`: Boolean flag to show a colorbar
- `plotkwargs`: Dictionary of keyword arguments for plotting
- `useaplpy`: Boolean flag to use APLpy if a FITS header is available
- `vmin`, `vmax`: Override values for the minimum and maximum values
- `cmap`: Colormap for the plot

Interactive Features
~~~~~~~~~~~~~~~~~~~~

- Button 1 or keyboard '1': Plot the selected pixel's spectrum in another window
- Button 2 or keyboard 'o': Overplot a second spectrum in the external plot window
- Button 3: Disconnect the interactive viewer
- Click-and-drag with button 1: Average over a circular region
