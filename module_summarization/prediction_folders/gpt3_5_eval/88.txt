This module provides utilities for using SQLite and asyncio effectively in Piccolo applications. It includes adapters and converters to ensure consistent data handling between SQLite and the Postgres engine used by Piccolo.

## Adapters

### Input Adapters

- `convert_numeric_in(value)`: Converts Decimal values to floats.
- `convert_uuid_in(value)`: Converts UUID values to strings.
- `convert_time_in(value)`: Converts time values to strings.
- `convert_date_in(value)`: Converts date values to strings.
- `convert_datetime_in(value)`: Converts datetime values to strings, converting to UTC if timezone aware.
- `convert_timedelta_in(value)`: Converts timedelta values to total seconds.
- `convert_array_in(value)`: Converts list values to strings.

### Output Adapters

- `convert_numeric_out(value)`: Converts float values to Decimals.
- `convert_int_out(value)`: Ensures integer values are of type int.
- `convert_uuid_out(value)`: Converts UUID values to `uuid.UUID` instances.
- `convert_date_out(value)`: Converts date values from bytes to `datetime.date`.
- `convert_time_out(value)`: Converts time values from bytes to `datetime.time`.
- `convert_seconds_out(value)`: Converts seconds values from bytes to `datetime.timedelta`.
- `convert_boolean_out(value)`: Converts boolean values from bytes to bool.
- `convert_timestamp_out(value)`: Converts timestamp values from bytes to `datetime.datetime`.
- `convert_timestamptz_out(value)`: Converts timestamptz values from bytes to `datetime.datetime` with UTC timezone.
- `convert_array_out(value)`: Deserializes array column values from bytes back into a list.
- `convert_M2M_out(value)`: Splits values from M2M columns into a list.

## Registering Adapters

The module registers the above adapters with SQLite using `sqlite3.register_adapter()` and `sqlite3.register_converter()` functions to ensure proper data conversion during interactions with the database.

## AsyncBatch Class

The `AsyncBatch` class extends `Batch` and provides asynchronous batch processing capabilities for executing queries in an asyncio environment. It includes methods for fetching data in batches, iterating asynchronously, and managing the execution context.

### Properties

- `connection`: The SQLite connection object.
- `query`: The query object to be executed.
- `batch_size`: The size of each batch.
- `_cursor`: Internal cursor object for fetching data.

### Methods

- `next()`: Asynchronously fetches the next batch of data.
- `__aiter__()`: Returns an asynchronous iterator.
- `__anext__()`: Asynchronously fetches the next batch of data for iteration.
- `__aenter__()`: Asynchronously sets up the batch execution context.
- `__aexit__()`: Asynchronously cleans up the batch execution context.
