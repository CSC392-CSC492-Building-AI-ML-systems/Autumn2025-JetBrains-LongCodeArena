Configuration Options
======================

The `ConfigurationsOptions` class provides a set of options for configuring the behavior of the code. Users can enable or disable various import and export components using the properties provided by this class.

Object Mapping Tolerance
------------------------
- **Description:** Get/Set the tolerance value to be used in the object mapping (used e.g. for boundaries).
- **Type:** float

Export Variables
----------------
- **Description:** Define if the variables have to be exported into a JSON file.
- **Default Value:** True
- **Type:** bool

Export Setups
-------------
- **Description:** Define if the setups have to be exported to a JSON file.
- **Default Value:** True
- **Type:** bool

Export Optimizations
---------------------
- **Description:** Define if the optimizations have to be exported to a JSON file.
- **Default Value:** True
- **Type:** bool

Export Parametrics
-------------------
- **Description:** Define if the parametrics have to be exported to a JSON file.
- **Default Value:** True
- **Type:** bool

Export Boundaries
-----------------
- **Description:** Define if the boundaries have to be exported to a JSON file.
- **Default Value:** True
- **Type:** bool

Import Datasets
---------------
- **Description:** Define if datasets have to be imported from a JSON file.
- **Default Value:** True
- **Type:** bool

Export Datasets
---------------
- **Description:** Define if datasets have to be exported to a JSON file.
- **Default Value:** True
- **Type:** bool

Export Mesh Operations
-----------------------
- **Description:** Define if mesh operations have to be exported to a JSON file.
