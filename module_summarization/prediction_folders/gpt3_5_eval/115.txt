Methods for getting and setting ASDF global configuration options.

AsdfConfig Class
----------------
Container for ASDF configuration options. Users are not intended to construct this object directly; instead, use the `asdf.get_config` and `asdf.config_context` module methods.

Attributes:
- resource_mappings: Get the list of registered resource mapping instances.
- resource_manager: Get the `asdf.resource.ResourceManager` instance.
- extensions: Get the list of registered extensions.
- default_version: Get or set the default ASDF Standard version used for new files.
- io_block_size: Get or set the block size used when reading and writing files.

Methods:
- add_resource_mapping(mapping): Register a new resource mapping.
- remove_resource_mapping(mapping=None, package=None): Remove a registered resource mapping.
- reset_resources(): Reset registered resource mappings to the default list provided as entry points.
- add_extension(extension): Register a new extension.
- remove_extension(extension=None, package=None): Remove a registered extension.
- reset_extensions(): Reset extensions to the default list registered via entry points.
