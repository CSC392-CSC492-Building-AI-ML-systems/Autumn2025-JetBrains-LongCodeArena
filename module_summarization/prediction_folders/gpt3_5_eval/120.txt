Channel Options
===============

The following are the available channel options for specifying visual encoding in Vega-Lite:

- ``Aggregate``: Aggregate data values.
- ``AggregateOp``: The aggregation operation to apply.
- ``AggregateTransform``: A transform that summarizes data.
- ``AggregatedFieldDef``: A field definition for aggregated data.
- ``Align``: The horizontal alignment of text.
- ``AllSortString``: A special string value that sorts all other values.
- ``Angle``: The angle to rotate the text, in degrees.
- ``AngleDatum``: A datum value for the angle.
- ``AngleValue``: A constant value for the angle.
- ``AnyMark``: A mark type or a range plot type.
- ``AnyMarkConfig``: A configuration object for any mark type.
- ``AreaConfig``: A configuration object for area marks.
- ``ArgmaxDef``: A definition for the argmax transform.
- ``ArgminDef``: A definition for the argmin transform.
- ``AutoSizeParams``: Parameters for automatically determining the size of a view.
- ``AutosizeType``: The type of autosize layout.
- ``Axis``: The properties of an axis.
- ``AxisConfig``: A configuration object for axes.
- ``AxisOrient``: The orientation of an axis.
- ``AxisResolveMap``: A mapping of axis properties to resolve scale domain conflicts.
- ``BarConfig``: A configuration object for bar marks.
- ``BaseTitleNoValueRefs``: A base title without value references.
- ``Baseline``: The baseline for text.
- ``Bin``: A binning definition.
- ``BinExtent``: The extent of the binning range.
- ``BinParams``: Parameters for binning data.
- ``BinTransform``: A transform that bins data values.
- ``BindCheckbox``: A binding for a checkbox input element.
- ``BindDirect``: A direct binding definition.
- ``BindInput``: A binding for an input element.
- ``BindRadioSelect``: A binding for a radio select input element.
- ``BindRange``: A binding for a range input element.
- ``Binding``: A binding definition.
- ``Blend``: A blend transform.
- ``BoxPlot``: A box plot specification.
- ``BoxPlotConfig``: A configuration object for box plots.
- ``BoxPlotDef``: A definition for a box plot.
- ``BrushConfig``: A configuration object for brush marks.
- ``CalculateTransform``: A transform that calculates new data values.
- ``Categorical``: A categorical data type.
- ``Chart``: The properties of a chart.
- ``Color``: The color of a mark.
- ``ColorDatum``: A datum value for color.
- ``ColorDef``: A field definition for color.
- ``ColorName``: A named color value.
- ``ColorScheme``: A color scheme for the visualization.
- ``ColorValue``: A constant color value.
- ``Column``: The column position of a mark.
- ``CompositeMark``: A composite mark type.
- ``CompositeMarkDef``: A definition for a composite mark.
- ``CompositionConfig``: A configuration object for composite marks.
- ``ConcatChart``: A concatenated chart specification.
- ``ConcatSpecGenericSpec``: A generic specification for concatenated charts.
- ``ConditionalAxisColor``: A conditional axis color property.
- ``ConditionalAxisLabelAlign``: A conditional axis label alignment property.
- ``ConditionalAxisLabelBaseline``: A conditional axis label baseline property.
- ``ConditionalAxisLabelFontStyle``: A conditional axis label font style property.
- ``ConditionalAxisLabelFontWeight``: A conditional axis label font weight property.
- ``ConditionalAxisNumber``: A conditional axis number property.
- ``ConditionalAxisNumberArray``: A conditional axis number array property.
- ``ConditionalAxisPropertyAlignnull``: A conditional axis property for alignment with null value.
- ``ConditionalAxisPropertyColornull``: A conditional axis property for color with null value.
- ``ConditionalAxisPropertyFontStylenull``: A conditional axis property for font style with null value.
- ``ConditionalAxisPropertyFontWeightnull``: A conditional axis property for font weight with null value.
- ``ConditionalAxisPropertyTextBaselinenull``: A conditional axis property for text baseline with null value.
- ``ConditionalAxisPropertynumberArraynull``: A conditional axis property for number array with null value.
- ``ConditionalAxisPropertynumbernull``: A conditional axis property for number with null value.
- ``ConditionalAxisPropertystringnull``: A conditional axis property for string with null value.
- ``ConditionalAxisString``: A conditional axis string property.
- ``ConditionalMarkPropFieldOrDatumDef``: A conditional mark property definition for field or datum.
- ``ConditionalMarkPropFieldOrDatumDefTypeForShape``: The type of conditional mark property definition for shape.
- ``ConditionalParameterMarkPropFieldOrDatumDef``: A conditional parameter mark property definition for field or datum.
- ``ConditionalParameterMarkPropFieldOrDatumDefTypeForShape``: The type of conditional parameter mark property definition for shape.
- ``ConditionalParameterStringFieldDef``: A conditional parameter string field definition.
- ``ConditionalParameterValueDefGradientstringnullExprRef``: A conditional parameter value definition for gradient with string or null value expression reference.
- ``ConditionalParameterValueDefTextExprRef``: A conditional parameter value definition for text with expression reference.
- ``ConditionalParameterValueDefnumber``: A conditional parameter value definition for number.
- ``ConditionalParameterValueDefnumberArrayExprRef``: A conditional parameter value definition for number array with expression reference.
- ``ConditionalParameterValueDefnumberExprRef``: A conditional parameter value definition for number with expression reference.
- ``ConditionalParameterValueDefstringExprRef``: A conditional parameter value definition for string with expression reference.
- ``ConditionalParameterValueDefstringnullExprRef``: A conditional parameter value definition for string or null value expression reference.
- ``ConditionalPredicateMarkPropFieldOrDatumDef``: A conditional predicate mark property definition for field or datum.
- ``ConditionalPredicateMarkPropFieldOrDatumDefTypeForShape``: The type of conditional predicate mark property definition for shape.
- ``ConditionalPredicateStringFieldDef``: A conditional predicate string field definition.
- ``ConditionalPredicateValueDefAlignnullExprRef``: A conditional predicate value definition for alignment with null value expression reference.
- ``ConditionalPredicateValueDefColornullExprRef``: A conditional predicate value definition for color with null value expression reference.
- ``ConditionalPredicateValueDefFontStylenullExprRef``: A conditional predicate value definition for font style with null value expression reference.
- ``ConditionalPredicateValueDefFontWeightnullExprRef``: A conditional predicate value definition for font weight with null value expression reference.
