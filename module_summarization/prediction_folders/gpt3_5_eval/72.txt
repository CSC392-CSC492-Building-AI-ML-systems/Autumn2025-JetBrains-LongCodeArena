The module contains utility functions for handling pairwise arrays and checking inputs for distance metrics calculations. It also includes functions for setting appropriate data types, ensuring array compatibility, and handling sparse matrices.

## Utility Functions

### `_return_float_dtype(X, Y)`

This function determines the appropriate data type for the input arrays `X` and `Y`. It checks if the arrays are sparse or not, and if the data type is `float32`, it returns `float32`; otherwise, it returns `float`.

### `check_pairwise_arrays(X, Y, *, precomputed=False, dtype=None, accept_sparse="csr", force_all_finite=True, copy=False)`

This function sets the input arrays `X` and `Y` appropriately and performs checks on the inputs. It ensures that both arrays are at least two-dimensional and contain float elements. It also verifies the compatibility of the array dimensions for distance calculations. Additionally, it handles various parameters such as data type, sparse matrix formats, and finite value constraints.

### `check_paired_arrays(X, Y)`

This function sets the input arrays `X` and `Y` appropriately and checks the inputs for paired distance metrics. It ensures that both arrays are at least two-dimensional and contain float elements.

These utility functions are essential for validating inputs and preparing data for pairwise distance calculations within the module.
