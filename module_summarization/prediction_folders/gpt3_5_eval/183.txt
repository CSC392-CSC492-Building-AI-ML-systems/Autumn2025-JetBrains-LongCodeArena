# Feature Selection / Dimensionality Reduction on Sample Sets

This module implements multioutput regression and classification using feature selection and dimensionality reduction techniques.

The estimators provided in this module are meta-estimators, which require a base estimator to be provided in their constructor. These meta-estimators extend single output estimators to multioutput estimators, allowing for feature selection and dimensionality reduction on sample sets.

The module includes the following classes:

- `MultiOutputRegressor`: A meta-estimator for multioutput regression tasks.
- `MultiOutputClassifier`: A meta-estimator for multioutput classification tasks.
- `ClassifierChain`: A meta-estimator implementing a chain of classifiers for multioutput classification.
- `RegressorChain`: A meta-estimator implementing a chain of regressors for multioutput regression.

These classes provide functionality for selecting features and reducing dimensionality in multioutput regression and classification problems.
