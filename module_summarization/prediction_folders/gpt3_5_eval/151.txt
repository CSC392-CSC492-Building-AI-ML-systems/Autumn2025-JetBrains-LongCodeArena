## General Plotting Routines

### `set_bokeh_plot_defaults(**kwargs)`
Set defaults for the Bokeh backend according to the given keyword arguments. Available defaults can be seen in `BokehPlotter`.

### `reset_bokeh_plot_defaults()`
Reset the defaults for the Bokeh backend to the hardcoded defaults. Available defaults can be seen in `BokehPlotter`.

### `show_bokeh_plot_defaults()`
Show the currently set defaults for the Bokeh backend. Available defaults can be seen in `BokehPlotter`.

### `get_bokeh_help(key)`
Print the description of the given key in the Bokeh backend. Available defaults can be seen in `BokehPlotter`.

### `load_bokeh_defaults(filename='plot_bokeh_defaults.json')`
Load defaults for the Bokeh backend from a JSON file. Specify the `filename` from where the defaults should be taken.

### `save_bokeh_defaults(filename='plot_bokeh_defaults.json', save_complete=False)`
Save the current defaults for the Bokeh backend to a JSON file. Specify the `filename` where the defaults should be stored. Set `save_complete=True` to store not only the overwritten user defaults but also the unmodified hardcoded defaults.

### `bokeh_scatter(x, y=None, *, xlabel='x', ylabel='y', title='', figure=None, data=None, saveas='scatter', copy_data=False, **kwargs)`
Create an interactive scatter plot with Bokeh. Parameters include `x` (arraylike or key for data for the x-axis), `y` (arraylike or key for data for the y-axis), `data` (source for the data of the plot), `xlabel` (label for the x-axis), `ylabel` (label for the y-axis), `title` (title of the figure), `figure` (Bokeh figure), `saveas` (filename of the output file), and `copy_data` (boolean to copy the data argument). Additional keyword arguments are passed on to `BokehPlotter`.

### `bokeh_multi_scatter(x, y=None, *, data=None, figure=None, xlabel='x', ylabel='y', title='', saveas='scatter', copy_data=False, set_default_legend=True, **kwargs)`
Create an interactive scatter plot with multiple data sets using Bokeh. Parameters include `x` (arraylike or key for data for the x-axis), `y` (arraylike or key for data for the y-axis), `data` (source for the data of the plot), `xlabel` (label for the x-axis), `ylabel` (label for the y-axis), `title` (title of the figure), `figure` (Bokeh figure), `saveas` (filename of the output file), `copy_data` (boolean to copy the data argument), and `set_default_legend` (boolean to generate default legend labels). Additional keyword arguments are passed on to `BokehPlotter`.
