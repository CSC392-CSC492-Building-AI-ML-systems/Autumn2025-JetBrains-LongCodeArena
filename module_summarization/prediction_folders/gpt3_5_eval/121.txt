Create Cradmin Menu System
===========================

The Create Cradmin Menu System provides a flexible way to create menus in Django Cradmin.

Default Expandable Menu Renderable
----------------------------------

The `get_default_expandable_menu_renderable` function retrieves the default expandable menu renderable object based on the settings defined in `DJANGO_CRADMIN_DEFAULT_EXPANDABLE_MENU_CLASS`.

BaseMenuLinkRenderable
----------------------

The `BaseMenuLinkRenderable` class represents a basic menu link item with properties such as label, URL, and active status. It can be extended to create custom menu link renderables.

NavLinkItemRenderable
----------------------

The `NavLinkItemRenderable` class is used to add links to the main menu. It can be used to create simple menu link items.

NavLinkButtonItemRenderable
---------------------------

The `NavLinkButtonItemRenderable` class is used to add links styled as buttons to the main menu. It extends the `NavLinkItemRenderable` class.

ExpandableMenuItem
------------------

The `ExpandableMenuItem` class is used to add expandable menu items to the main menu. It provides methods to customize the appearance and behavior of expandable menu items.

ExpandableMenuSeparator
-----------------------

The `ExpandableMenuSeparator` class represents a separator element for expandable menus. It can be used to visually separate menu items within an expandable menu.

MenuToggleItemItemRenderable
----------------------------

The `MenuToggleItemItemRenderable` class is used to add a menu toggle item to the menu. It provides methods to customize the label and appearance of the menu toggle item.

AbstractMenuRenderable
----------------------

The `AbstractMenuRenderable` class serves as the base class for rendering a menu. It provides methods to customize the rendering of menu items and allows for creating custom HTML menus by setting a custom template.
