To generate the schema for a database using Tortoise ORM, follow these steps:

1. Import the necessary modules:
   
   ```python
   import logging
   from typing import TYPE_CHECKING
   ```

2. Define the function `get_schema_sql` to generate the SQL schema for a given database client:

   ```python
   def get_schema_sql(client: "BaseDBAsyncClient", safe: bool) -> str:
       """
       Generates the SQL schema for the given client.

       :param client: The DB client to generate Schema SQL for
       :param safe: When set to true, creates the table only when it does not already exist.
       """
   ```

3. Define the function `generate_schema_for_client` to generate and apply the SQL schema directly to the given client:

   ```python
   async def generate_schema_for_client(client: "BaseDBAsyncClient", safe: bool) -> None:
       """
       Generates and applies the SQL schema directly to the given client.

       :param client: The DB client to generate Schema SQL for
       :param safe: When set to true, creates the table only when it does not already exist.
       """
   ```

4. Use the `generate_schema_for_client` function to create the schema for the database client.

Remember to replace `"BaseDBAsyncClient"` with the actual type of your database client.
