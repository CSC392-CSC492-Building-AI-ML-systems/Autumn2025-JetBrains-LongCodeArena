Imputation of missing values
=============================

The imputation of missing values is a crucial step in data preprocessing to ensure the quality and integrity of the dataset. The scikit-learn library provides a variety of tools for imputing missing values, including the `SimpleImputer` class.

SimpleImputer
-------------

The `SimpleImputer` class is a univariate imputer that completes missing values using simple strategies. It replaces missing values with descriptive statistics such as mean, median, most frequent value, or a constant value along each column of the dataset.

Parameters
~~~~~~~~~~

- `missing_values` (int, float, str, np.nan, None or pandas.NA, default=np.nan): The placeholder for missing values in the dataset.

- `strategy` (str, default='mean'): The imputation strategy to use. Options include:
    - "mean": Replace missing values with the mean along each column (numeric data only).
    - "median": Replace missing values with the median along each column (numeric data only).
    - "most_frequent": Replace missing values with the most frequent value along each column (strings or numeric data).
    - "constant": Replace missing values with a specified fill value.

- `fill_value` (str or numerical value, default=None): The value to use for imputation when `strategy` is set to "constant".

- `copy` (bool, default=True): If True, a copy of the dataset will be created; if False, imputation will be done in-place whenever possible.

Usage
~~~~~

To use the `SimpleImputer` class, instantiate an object with the desired parameters and then fit and transform the dataset using the `fit_transform` method. This will replace missing values in the dataset according to the specified strategy.

For more information on imputation of missing values and the `SimpleImputer` class, refer to the scikit-learn documentation.

.. versionadded:: 0.20
   The `SimpleImputer` class replaces the previous `sklearn.preprocessing.Imputer` estimator, which has been removed.
