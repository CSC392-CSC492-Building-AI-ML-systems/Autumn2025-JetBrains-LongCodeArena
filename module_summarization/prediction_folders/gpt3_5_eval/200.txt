# Novelty and Outlier Detection

This module implements multioutput regression and classification for detecting novelties and outliers in data. The estimators provided in this module are meta-estimators, which require a base estimator to be provided in their constructor. These meta-estimators extend single output estimators to multioutput estimators, allowing for the detection of novelties and outliers in various scenarios.

The module includes the following classes:

- `MultiOutputRegressor`: A meta-estimator for multioutput regression tasks.
- `MultiOutputClassifier`: A meta-estimator for multioutput classification tasks.
- `ClassifierChain`: A meta-estimator implementing a chain of classifiers for multioutput classification.
- `RegressorChain`: A meta-estimator implementing a chain of regressors for multioutput regression.

These classes provide functionalities for detecting novelties and outliers in datasets, enabling users to perform robust analysis and modeling in the presence of unusual or anomalous data points.
