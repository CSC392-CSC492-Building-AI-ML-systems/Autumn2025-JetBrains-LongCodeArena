# w_succ command

The `w_succ` command lists segments which successfully reach a target state.

## Usage

```bash
w_succ [-h] [-o OUTPUT_FILE]
```

## Description

The `w_succ` command is used to display information about segments that have successfully reached a target state. It provides details such as the iteration number, segment ID, weight, and final progress coordinate values for recycled segments.

## Options

- `-o OUTPUT_FILE, --output OUTPUT_FILE`: Specifies the output file to store the command's output. If not provided, the output will be written to the standard output.

## Output Format

The output format of the `w_succ` command includes the following columns:

1. Iteration number
2. Segment ID
3. Weight
4. Final progress coordinate values

Each row in the output represents a recycled segment that has successfully reached the target state.
