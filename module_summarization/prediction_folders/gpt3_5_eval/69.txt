Isotonic
========

Generate documentation for this module.

Authors: Fabian Pedregosa <fabian@fseoane.net>, Alexandre Gramfort <alexandre.gramfort@inria.fr>, Nelle Varoquaux <nelle.varoquaux@gmail.com>

License: BSD 3 clause

This module provides functionality for isotonic regression, a method for fitting a non-decreasing (or non-increasing) function to data points. Isotonic regression is useful for modeling relationships where the target variable should exhibit a monotonic behavior with respect to the input variable.

Functions
---------

check_increasing(x, y)
    Determine whether y is monotonically correlated with x based on a Spearman correlation test.

isotonic_regression(y, sample_weight=None, y_min=None, y_max=None, increasing=True)
    Solve the isotonic regression model to fit a non-decreasing (or non-increasing) function to the input data.

Classes
-------

IsotonicRegression
    Isotonic regression model that can be used for fitting and predicting non-decreasing (or non-increasing) functions to data points.

Attributes
----------

X_min_
    Minimum value of input array X_ for left bound.

X_max_
    Maximum value of input array X_ for right bound.

X_thresholds_
    Unique ascending X values used to interpolate the y = f(X) monotonic function.

y_thresholds_
    Array of thresholds for the isotonic regression model.

References
----------

- Fisher transformation. Wikipedia. https://en.wikipedia.org/wiki/Fisher_transformation
- "Active set algorithms for isotonic regression; A unifying framework" by Michael J. Best and Nilotpal Chakravarti, section 3.

Read more in the User Guide: https://scikit-learn.org/stable/modules/isotonic.html
