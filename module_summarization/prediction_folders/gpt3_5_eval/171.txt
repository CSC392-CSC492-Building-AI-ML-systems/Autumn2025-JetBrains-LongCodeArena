# Calculator Interface

The Calculator Interface provides functionality to list parameters used across all models. It includes the following functions:

- `find_pars(kind=None)`: Finds all parameters in all models and returns a reference table `{parameter: [model, model, ...]}`.
- `list_pars(names_only=True, kind=None)`: Prints all parameters in all models. If `names_only` is set to `True`, only the parameter names are printed, otherwise both the parameter name and the models it occurs in are displayed.
- `main()`: Program to list the parameters used across all models. It accepts command-line arguments to specify the kind of parameters to list and whether to list the models using those parameters.

To use the Calculator Interface, you can run the program using the command `python -m sasmodels.list_pars [-v]`. The `-v` flag can be used to list the models containing the parameter in addition to just the parameter name.

The interface also includes a function `ellipticJi(u, v, m)` that returns `[sn, cn, dn](u + iv|m)`, providing functionality to calculate elliptic functions.

Please refer to the source code for detailed information on the implementation and usage of the Calculator Interface.
