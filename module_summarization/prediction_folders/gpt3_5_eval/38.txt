Examples and Usage
------------------

Below are some examples and usage guidelines for working with the `Field` class in Tortoise:

1. Creating a Basic Field:

    To create a basic field, you can instantiate the `Field` class with the desired parameters. For example:

    .. code-block:: python

        from tortoise.fields import Field

        my_field = Field()

2. Specifying Field Attributes:

    You can specify various attributes for a field such as `source_field`, `generated`, `pk`, `null`, `default`, `unique`, `index`, and `description`. Here's an example:

    .. code-block:: python

        my_field = Field(
            source_field="my_source_field",
            generated=True,
            pk=True,
            null=False,
            default="default_value",
            unique=True,
            index=True,
            description="This is a custom field description."
        )

3. Converting Values:

    The `to_db_value` and `to_python_value` methods can be used to convert values between Python and the database type. Here's how you can use them:

    .. code-block:: python

        value = my_field.to_db_value("some_value", instance)
        python_value = my_field.to_python_value(db_value)

4. Checking Field Requirements:

    The `required` property can be used to check if a field is required. It returns `True` if the field is required. Here's an example:

    .. code-block:: python

        is_required = my_field.required
