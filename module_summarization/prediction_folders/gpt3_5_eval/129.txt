Table Data
==========

This module contains classes and methods related to handling table data.

VerifyError
-----------

Exception class for verification errors.

VerifyWarning
-------------

Warning class for verification warnings.

VERIFY_OPTIONS
--------------

List of options for verification, including "ignore", "warn", "exception", "fix", "silentfix", "fix+ignore", "fix+warn", "fix+exception", "silentfix+ignore", "silentfix+warn", and "silentfix+exception".

_Verify
-------

Shared methods for verification.

- ``run_option(option="warn", err_text="", fix_text="Fixed.", fix=None, fixable=True)``
  Execute the verification with the selected option.

- ``verify(option="warn")``
  Verify all values in the instance.

_ErrList
--------

List class for verification errors, with a nested structure constructed by error messages generated by verifications at different class levels.

NotifierMixin
-------------

Mixin class that provides services for registering listeners to changes on objects.

- ``_add_listener(listener)``
  Register a listener on an instance of the notifier.

- ``_notify('change_type', *args, **kwargs)``
  Update all listeners upon changes by calling the appropriate update method on each listener.
