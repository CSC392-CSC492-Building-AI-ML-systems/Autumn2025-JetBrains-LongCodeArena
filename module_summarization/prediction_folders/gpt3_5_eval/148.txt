# General HDF5 File Reader

The `HDF5` file reader in the Masci-tools package provides a collection of predefined transformations for the `HDF5Reader` class. These transformations are designed to handle various types of datasets and ensure smooth processing of data. The transformations are registered as functions using decorators and can be applied to datasets in different ways.

## Transformations

### `get_first_element`

- **Description**: Retrieves the first element of the dataset.
- **Parameters**:
  - `dataset`: Dataset to transform.
- **Returns**: The first element of the dataset.

### `index_dataset`

- **Description**: Retrieves the n-th element of the dataset.
- **Parameters**:
  - `dataset`: Dataset to transform.
  - `index`: Index of the element to retrieve.
- **Returns**: The n-th element of the dataset.

### `slice_dataset`

- **Description**: Slices the dataset with the given slice argument.
- **Parameters**:
  - `dataset`: Dataset to transform.
  - `slice_arg`: Slice to apply to the dataset.
- **Returns**: The sliced dataset.

### `get_shape`

- **Description**: Retrieves the shape of the dataset.
- **Parameters**:
  - `dataset`: Dataset to get the shape.
- **Returns**: The shape of the dataset.

### `get_name`

- **Description**: Retrieves the name of the dataset.
- **Parameters**:
  - `dataset`: Dataset to get the name.
  - `full_path`: If `True`, returns the full path to the dataset.
- **Returns**: The name of the dataset.

### `tile_array`

- **Description**: Repeats the array n-times using `numpy.tile`.
- **Parameters**:
  - `dataset`: Dataset to transform.
  - `n_repeats`: Number of times to repeat the array.
- **Returns**: The dataset repeated n-times.

### `repeat_array`

- **Description**: Repeats each element in the array n-times using `numpy.repeat`.
- **Parameters**:
  - `dataset`: Dataset to transform.
  - `n_repeats`: Number of times to repeat each element.
- **Returns**: The dataset with elements repeated n-times.

### `get_all_child_datasets`

- **Description**: Retrieves all datasets contained in the given group.
- **Parameters**:
  - `group`: `h5py` object to extract datasets from.
  - `ignore`: String or iterable of strings to ignore specific keys.
  - `contains`: String or iterable of strings that must be in the key.
- **Returns**: A dictionary with the contained datasets entered with their names as keys.

These transformations are essential for handling different types of datasets, including `h5py.Dataset`, `numpy` arrays, and dictionaries. They ensure flexibility and robustness in reading and processing HDF5 files.
