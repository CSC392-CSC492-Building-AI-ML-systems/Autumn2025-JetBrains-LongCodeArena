Executing notebooks
===================

This module contains a preprocessor that executes the code cells and updates outputs. It provides functionality to preprocess notebooks by executing each code cell.

HighlightMagicsPreprocessor
---------------------------

This preprocessor detects and tags code cells that use languages other than Python through magic extensions such as `%%R` or `%%octave`. It associates each magic language extension with a Pygments lexer for syntax highlighting.
