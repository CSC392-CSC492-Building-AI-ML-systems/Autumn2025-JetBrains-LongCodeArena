# FleurXMLModifier Class

The `FleurXMLModifier` class is designed for organizing and grouping changes to an input file of FLEUR in a robust manner. It serves as a low-level alternative to the `FleurinpModifier` in `aiida_fleur`.

## Overview

The class allows for collecting and performing multiple modifications to an input file simultaneously. It provides a structured approach to managing changes efficiently.

## Usage

To use the `FleurXMLModifier` class, follow these basic steps:

1. Create an instance of the class.
2. Add changes by calling the appropriate methods on the class instance. These methods correspond to the setting methods in the `xml_setters` modules.
3. Collect all the tasks to be performed.
4. Apply the changes to an input file using the `modify_xmlfile` method.

### Example

```python
from masci_tools.io.fleurxmlmodifier import FleurXMLModifier

fmode = FleurXMLModifier()

# Add changes by calling the methods on this class
fmode.set_inpchanges({'Kmax': 4.0})  # Set Kmax to 4.0
fmode.shift_value({'Gmax': 5.0})  # Add 5 to the current value of Gmax

# Set the local orbital configuration on all iron atoms to '3s 3p'
fmode.set_species('all-Fe', {'lo': [{'n': 3, 'l': 's', 'type': 'SCLO'},
                                    {'n': 3, 'l': 'p', 'type': 'SCLO'}]})

# To undo the last change, call undo
# fmode.undo()

# To reset all added tasks, use revert_all=True
# fmode.undo(revert_all=True)

# Apply the changes to an input file
new_xmltree, additional_files = fmode.modify_xmlfile('/path/to/input/file/inp.xml')
```
