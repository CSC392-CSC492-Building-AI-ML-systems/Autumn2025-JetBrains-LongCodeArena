# Computational Performance

The `computational_performance.rst` file contains documentation about the computational performance of the code provided. The code includes implementations for multioutput regression and classification using meta-estimators that extend single output estimators to multioutput estimators.

## Performance Enhancements

The code utilizes various techniques to enhance computational performance, including:
- **Parallelization**: Utilizing multiple cores for parallel processing using the `n_jobs` parameter.
- **Efficient Estimator Fitting**: Implementing functions like `_fit_estimator` and `_partial_fit_estimator` to optimize the fitting process.
- **Metadata Routing**: Leveraging metadata routing for efficient method mapping and parameter validation.
- **Method Availability Checks**: Ensuring availability of specific methods in sub-estimators for improved performance.
- **Random State Handling**: Utilizing random state handling for reproducibility and performance consistency.

## Estimators Provided

The code provides the following multioutput estimators:
- `MultiOutputRegressor`
- `MultiOutputClassifier`
- `ClassifierChain`
- `RegressorChain`

These meta-estimators extend single output estimators to support multioutput regression and classification tasks, enhancing the versatility and performance of the models.
