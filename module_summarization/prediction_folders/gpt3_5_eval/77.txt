# Generate documentation for transforms module

The `transforms` module provides functions for transforming images between different formats and representations. This module includes functions for converting images to tensors, getting image dimensions, sizes, and number of channels, as well as defining interpolation modes for image processing.

## Functions

### `get_dimensions(img: Tensor) -> List[int]`

Returns the dimensions of an image as a list `[channels, height, width]`.

- **Parameters:**
  - `img (PIL Image or Tensor)`: The image to be checked.

- **Returns:**
  - `List[int]`: The image dimensions.

### `get_image_size(img: Tensor) -> List[int]`

Returns the size of an image as a list `[width, height]`.

- **Parameters:**
  - `img (PIL Image or Tensor)`: The image to be checked.

- **Returns:**
  - `List[int]`: The image size.

### `get_image_num_channels(img: Tensor) -> int`

Returns the number of channels of an image.

- **Parameters:**
  - `img (PIL Image or Tensor)`: The image to be checked.

- **Returns:**
  - `int`: The number of channels.

### `to_tensor(pic) -> Tensor`

Converts a `PIL Image` or `numpy.ndarray` to a tensor.

- **Parameters:**
  - `pic (PIL Image or numpy.ndarray)`: Image to be converted to tensor.

- **Returns:**
  - `Tensor`: Converted image.

### `pil_to_tensor(pic: Any) -> Tensor`

Converts a `PIL Image` to a tensor of the same type.

- **Parameters:**
  - `pic (PIL Image)`: Image to be converted to tensor.

- **Returns:**
  - `Tensor`: Converted image.

## Enum

### `InterpolationMode`

An enumeration of interpolation modes for image processing.

Available interpolation methods are:
- `nearest`
- `nearest-exact`
- `bilinear`
- `bicubic`
- `box`
- `hamming`
- `lanczos`

## Utility Functions

### `_interpolation_modes_from_int(i: int) -> InterpolationMode`

Maps an integer to the corresponding `InterpolationMode`.

- **Parameters:**
  - `i (int)`: Integer representing the interpolation mode.

- **Returns:**
  - `InterpolationMode`: The corresponding interpolation mode.

## Notes

- The functions in this module do not support torchscript.
- A deep copy of the underlying array is performed when converting `PIL Image` to a tensor.

This module provides essential functionality for handling image transformations and processing in PyTorch.
