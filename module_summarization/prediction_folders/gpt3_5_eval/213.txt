# Nonparametric Methods

## Univariate Kernel Density Estimators

The Univariate Kernel Density Estimators module provides functionality for estimating the density of a univariate variable using kernel density estimation.

### References

- Racine, Jeff. (2008) "Nonparametric Econometrics: A Primer," Foundation and Trends in Econometrics: Vol 3: No 1, pp1-88. [Link to paper](http://dx.doi.org/10.1561/0800000009)
- [Wikipedia - Kernel (statistics)](https://en.wikipedia.org/wiki/Kernel_%28statistics%29)
- Silverman, B.W. Density Estimation for Statistics and Data Analysis.

### Classes

#### `KDEUnivariate`

The `KDEUnivariate` class represents a Univariate Kernel Density Estimator.

##### Parameters

- `endog` : array_like
  The variable for which the density estimate is desired.

##### Methods

- `fit()`
  Attach the density estimate to the `KDEUnivariate` class.

  - Parameters:
    - `kernel` : str
    - `bw` : str, float, callable
    - `fft` : bool
    - `weights` : None
    - `gridsize` : int
    - `cut` : float
    - `adjust` : float

  - Returns:
    - `KDEUnivariate`: The instance fit.

- `cdf()`
  Returns the cumulative distribution function evaluated at the support.

  Notes:
  Will not work if fit has not been called.

##### Examples

```python
import statsmodels.api as sm
import matplotlib.pyplot as plt

nobs = 300
np.random.seed(1234)  # Seed random generator
dens = sm.nonparametric.KDEUnivariate(np.random.normal(size=nobs))
dens.fit()
plt.plot(dens.cdf)
plt.show()
```

### Kernel Switch

The `kernel_switch` dictionary provides a mapping of kernel names to their respective kernel functions for estimators. The available kernels are:

- Gaussian
- Epanechnikov
- Uniform
- Triangular
- Biweight
- Triweight
- Cosine
- Cosine2
