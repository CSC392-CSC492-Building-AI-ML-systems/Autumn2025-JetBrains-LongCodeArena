Array API Support
=================

This module provides support for the Array API, allowing seamless integration with various array-like objects.

Classes
-------

MultiOutputRegressor
    A meta-estimator for multi-output regression tasks. Requires a base estimator to be provided in the constructor.

MultiOutputClassifier
    A meta-estimator for multi-output classification tasks. Extends single output classifiers to multi-output classifiers.

ClassifierChain
    A meta-estimator implementing a chain of classifiers for multi-output classification tasks.

RegressorChain
    A meta-estimator implementing a chain of regressors for multi-output regression tasks.

Functions
---------

_fit_estimator(estimator, X, y, sample_weight=None, **fit_params)
    Fits an estimator to the input data X and target y. Supports optional sample weights and additional fit parameters.

_partial_fit_estimator(estimator, X, y, classes=None, partial_fit_params=None, first_time=True)
    Partially fits an estimator to the input data X and target y. Supports incremental learning with optional class information and partial fit parameters.

_available_if_estimator_has(attr)
    Returns a function to check if the sub-estimator(s) has(have) a specific attribute. Useful for checking attributes in chain implementations.

Attributes
----------

_parameter_constraints
    A dictionary defining the parameter constraints for the multi-output estimators.

Methods
-------

__init__(estimator, *, n_jobs=None)
    Constructor for the multi-output estimator classes. Requires a base estimator to be provided and optionally accepts the number of parallel jobs to run.
