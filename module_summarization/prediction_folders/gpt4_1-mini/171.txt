Calculator Interface
====================

Overview
--------

The Calculator Interface provides a programmatic way to list and inspect parameters used across various models. It allows users to query all parameters, optionally filtering by parameter type, and to see which models use each parameter.

Usage
-----

The interface can be accessed via the command line using the following command:

```
python -m sasmodels.list_pars [-v] [kind]
```

- `-v`, `--verbose`: When specified, the output includes the list of models that use each parameter.
- `kind`: Optional positional argument to filter parameters by their kind. Valid choices are:
  - `volume`
  - `orientation`
  - `sld`
  - `none`
  - `any` (default)

If `kind` is omitted or set to `any`, all parameters are listed regardless of their type.

Functionality
-------------

### find_pars(kind=None)

Finds all parameters in all available models.

- **Parameters:**
  - `kind` (str or None): If specified, filters parameters by their type (e.g., 'volume', 'orientation', 'sld'). If `None`, all parameters are included.

- **Returns:**
  - A dictionary mapping parameter names to a list of model names that use the parameter:
    ```python
    {parameter_name: [model1, model2, ...], ...}
    ```

### list_pars(names_only=True, kind=None)

Prints all parameters found in the models.

- **Parameters:**
  - `names_only` (bool): If `True`, only parameter names are printed. If `False`, the models using each parameter are also listed.
  - `kind` (str or None): Filter parameters by kind, passed to `find_pars`.

- **Behavior:**
  - When `names_only` is `True`, the parameter names are printed in a columnized format.
  - When `names_only` is `False`, each parameter is printed along with the models that use it.

### main()

Entry point for the command line interface.

- Parses command line arguments.
- Calls `list_pars` with appropriate options based on the arguments.

Examples
--------

- List all parameter names:

  ```
  python -m sasmodels.list_pars
  ```

- List all parameters with the models that use them:

  ```
  python -m sasmodels.list_pars -v
  ```

- List only parameters of kind `volume`:

  ```
  python -m sasmodels.list_pars volume
  ```

- List parameters of kind `sld` with model names:

  ```
  python -m sasmodels.list_pars -v sld
  ```

Notes
-----

- The interface relies on the internal model registry and parameter metadata.
- Parameter kinds correspond to categories such as volume-related parameters, orientation parameters, scattering length density (sld), or none.
- The tool is useful for understanding parameter usage across models and for selecting parameters for fitting or analysis.
