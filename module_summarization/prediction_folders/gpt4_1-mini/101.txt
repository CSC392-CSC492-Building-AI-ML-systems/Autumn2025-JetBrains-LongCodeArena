Miner
=====

The `Miner` class provides an interface to interact with the Ethereum miner through the Web3.py library. It allows you to query miner information and control mining operations.

Initialization
--------------

`Miner(web3)`

- `web3`: An instance of the Web3 class used to make JSON-RPC requests.

Properties
----------

hashrate
~~~~~~~~

The current mining hashrate.

:type: int

Returns the current mining hashrate as a decimal number.

Example:

```python
current_hashrate = miner.hashrate
```

Methods
-------

makeDAG(number)
~~~~~~~~~~~~~~

Generate the DAG (Directed Acyclic Graph) for a given block number.

:param int number: The block number for which to generate the DAG.

:return: Result of the DAG generation request.

Example:

```python
miner.makeDAG(100000)
```

setExtra(extra)
~~~~~~~~~~~~~~

Set the extra data field for the miner.

:param str extra: The extra data to set.

:return: Result of the request to set extra data.

Example:

```python
miner.setExtra("0x1234abcd")
```

setEtherBase(etherbase)
~~~~~~~~~~~~~~~~~~~~~~

Set the etherbase (coinbase) address for the miner.

:param str etherbase: The address to set as etherbase.

:return: Result of the request to set the etherbase.

Example:

```python
miner.setEtherBase("0xabc123...")
```

setGasPrice(gas_price)
~~~~~~~~~~~~~~~~~~~~~~

Set the gas price for the miner.

:param int gas_price: The gas price to set.

:return: Result of the request to set the gas price.

Example:

```python
miner.setGasPrice(20000000000)
```

start(num_threads)
~~~~~~~~~~~~~~~~~~

Start the miner with the specified number of threads.

:param int num_threads: Number of threads to use for mining.

:return: Result of the request to start mining.

Example:

```python
miner.start(4)
```

stop()
~~~~~~

Stop the miner.

:return: Result of the request to stop mining.

Example:

```python
miner.stop()
```

startAutoDAG()
~~~~~~~~~~~~~

Start automatic DAG generation.

:return: Result of the request to start automatic DAG generation.

Example:

```python
miner.startAutoDAG()
```

stopAutoDAG()
~~~~~~~~~~~~

Stop automatic DAG generation.

:return: Result of the request to stop automatic DAG generation.

Example:

```python
miner.stopAutoDAG()
```
