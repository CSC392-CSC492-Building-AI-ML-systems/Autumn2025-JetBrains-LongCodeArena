Transforms Module
=================

This module provides utilities and functions for image transformations, including conversion between different image formats and handling various interpolation modes. It supports images represented as PIL Images, NumPy arrays, and PyTorch tensors.

InterpolationMode Enum
---------------------

The `InterpolationMode` enum defines the available interpolation methods for image transformations:

- `NEAREST`: Nearest neighbor interpolation.
- `NEAREST_EXACT`: Exact nearest neighbor interpolation.
- `BILINEAR`: Bilinear interpolation.
- `BICUBIC`: Bicubic interpolation.
- `BOX`: Box filter interpolation (for PIL compatibility).
- `HAMMING`: Hamming filter interpolation.
- `LANCZOS`: Lanczos filter interpolation.

These modes can be used to specify the interpolation method when resizing or transforming images.

Utility Functions
-----------------

### get_dimensions(img: Tensor) -> List[int]

Returns the dimensions of an image as `[channels, height, width]`.

- **Args:**
  - `img` (PIL Image or Tensor): The image to be checked.
- **Returns:**
  - `List[int]`: The image dimensions.

### get_image_size(img: Tensor) -> List[int]

Returns the size of an image as `[width, height]`.

- **Args:**
  - `img` (PIL Image or Tensor): The image to be checked.
- **Returns:**
  - `List[int]`: The image size.

### get_image_num_channels(img: Tensor) -> int

Returns the number of channels of an image.

- **Args:**
  - `img` (PIL Image or Tensor): The image to be checked.
- **Returns:**
  - `int`: The number of channels.

### to_tensor(pic) -> Tensor

Converts a `PIL Image` or `numpy.ndarray` to a PyTorch tensor. This function does not support TorchScript.

- **Args:**
  - `pic` (PIL Image or numpy.ndarray): Image to be converted to tensor.
- **Returns:**
  - `Tensor`: Converted image tensor with shape `[channels, height, width]` and float dtype normalized to [0, 1] if the input is a byte image.

- **Raises:**
  - `TypeError`: If the input is not a PIL Image or numpy ndarray.
  - `ValueError`: If the numpy array is not 2D or 3D.

This function handles different input types:
- For numpy arrays, it converts the array to a tensor and permutes dimensions to channel-first format.
- For PIL Images, it converts the image to a numpy array and then to a tensor.
- Supports `accimage` if available for faster image loading.

### pil_to_tensor(pic: PIL Image) -> Tensor

Converts a `PIL Image` to a tensor of the same type. This function does not support TorchScript.

- **Args:**
  - `pic` (PIL Image): Image to be converted to tensor.
- **Returns:**
  - `Tensor`: Converted image tensor.

- **Note:**
  - A deep copy of the underlying array is performed to ensure the tensor is independent of the original image.

Example Usage
-------------

```python
from PIL import Image
import numpy as np
import torch
from transforms import to_tensor, get_image_size, InterpolationMode

# Load a PIL image
pil_img = Image.open("example.jpg")

# Convert PIL image to tensor
tensor_img = to_tensor(pil_img)
print(f"Tensor shape: {tensor_img.shape}")

# Get image size
size = get_image_size(pil_img)
print(f"Image size (width, height): {size}")

# Use interpolation mode
mode = InterpolationMode.BILINEAR
print(f"Selected interpolation mode: {mode.value}")
```

This module is designed to facilitate image preprocessing and transformation workflows in PyTorch-based computer vision applications.
