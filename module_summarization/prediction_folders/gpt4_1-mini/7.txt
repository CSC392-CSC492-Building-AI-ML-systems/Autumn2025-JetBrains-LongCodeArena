Using Lists
===========

Lists are a fundamental data structure in Python that allow you to store an ordered collection of items. They are versatile and can hold elements of any data type, including other lists.

Creating Lists
--------------

You can create a list by placing comma-separated values inside square brackets:

.. code-block:: python

    my_list = [1, 2, 3, 4]
    fruits = ['apple', 'banana', 'cherry']

Accessing Elements
------------------

List elements are accessed by their index, starting at 0 for the first element:

.. code-block:: python

    first_fruit = fruits[0]  # 'apple'
    last_fruit = fruits[-1]  # 'cherry'

Modifying Lists
---------------

Lists are mutable, meaning you can change their content after creation:

.. code-block:: python

    fruits[1] = 'blueberry'  # ['apple', 'blueberry', 'cherry']
    fruits.append('date')    # Adds 'date' to the end of the list

Common List Operations
----------------------

- **Appending items**: Use `append()` to add an item to the end of the list.

- **Inserting items**: Use `insert(index, item)` to add an item at a specific position.

- **Removing items**: Use `remove(item)` to remove the first occurrence of an item.

- **Popping items**: Use `pop()` to remove and return the last item, or `pop(index)` to remove an item at a specific index.

- **Slicing**: Extract a subset of the list using slice notation:

  .. code-block:: python

      sublist = fruits[1:3]  # ['blueberry', 'cherry']

Iterating Over Lists
--------------------

You can loop through the elements of a list using a `for` loop:

.. code-block:: python

    for fruit in fruits:
        print(fruit)

List Comprehensions
-------------------

List comprehensions provide a concise way to create lists:

.. code-block:: python

    squares = [x**2 for x in range(10)]

Nested Lists
------------

Lists can contain other lists, allowing you to create multi-dimensional data structures:

.. code-block:: python

    matrix = [
        [1, 2, 3],
        [4, 5, 6],
        [7, 8, 9]
    ]

Useful Built-in Functions
-------------------------

- `len(list)`: Returns the number of items in the list.

- `sorted(list)`: Returns a new sorted list from the items.

- `list.count(item)`: Returns the number of occurrences of an item.

- `list.index(item)`: Returns the index of the first occurrence of an item.

Summary
-------

Lists are a powerful and flexible way to work with ordered collections of data in Python. They support a wide range of operations that make data manipulation straightforward and efficient.
