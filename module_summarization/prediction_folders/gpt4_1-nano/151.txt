# General Plotting Routines Documentation

This section provides an overview of the general plotting routines available for creating interactive visualizations using the Bokeh backend. These routines facilitate the creation of scatter plots and multi-data scatter plots with customizable options.

## Bokeh Plotting Functions

### set_bokeh_plot_defaults(**kwargs)
Set default parameters for Bokeh plots according to the provided keyword arguments.  
**Usage:** Customize default plot settings globally for subsequent plots.

### reset_bokeh_plot_defaults()
Reset Bokeh plot defaults to the hardcoded initial settings.  
**Usage:** Revert any customizations to default values.

### show_bokeh_plot_defaults()
Display the current default settings for Bokeh plots.  
**Usage:** Review the active default configuration.

### get_bokeh_help(key)
Print the description of a specific default parameter key in the Bokeh backend.  
**Usage:** Obtain detailed information about a particular setting.

### load_bokeh_defaults(filename='plot_bokeh_defaults.json')
Load Bokeh default settings from a JSON configuration file.  
**Usage:** Restore a saved configuration of plot defaults.

### save_bokeh_defaults(filename='plot_bokeh_defaults.json', save_complete=False)
Save the current Bokeh plot defaults to a JSON file.  
**Usage:** Persist current settings for future use. If `save_complete` is True, include unmodified hardcoded defaults.

## Plotting Routines

### bokeh_scatter(x, y=None, **kwargs)
Create an interactive scatter plot with Bokeh.

- **x:** Array-like data or key for data for the x-axis.
- **y:** Array-like data or key for data for the y-axis.
- **data:** Data source (e.g., pandas DataFrame) for the plot.
- **xlabel:** Label for the x-axis.
- **ylabel:** Label for the y-axis.
- **title:** Title of the plot.
- **figure:** Optional existing Bokeh figure to add the plot to.
- **saveas:** Filename for saving the plot.
- **copy_data:** If True, copies the data source.

**Description:**  
This routine generates an interactive scatter plot. It processes input data, sets default plot parameters, and adds tooltips and straight lines if specified. The plot can be customized via additional keyword arguments passed to the underlying Bokeh scatter function.

---

### bokeh_multi_scatter(x, y=None, **kwargs)
Create an interactive scatter plot with multiple data sets.

- **x:** Array-like data or key for data for the x-axis.
- **y:** Array-like data or key for data for the y-axis.
- **data:** Data source (e.g., pandas DataFrame).
- **xlabel:** Label for the x-axis.
- **ylabel:** Label for the y-axis.
- **title:** Plot title.
- **figure:** Optional existing Bokeh figure.
- **saveas:** Filename for saving.
- **set_default_legend:** If True, uses data names for legend labels.
- **copy_data:** If True, copies the data source.

**Description:**  
This routine allows plotting multiple data sets in a single scatter plot, each potentially with its own legend label. It processes the data, applies default or custom plot parameters, and supports multiple data series visualization.

---

## Additional Utilities

- **Data Processing:** The routines utilize `process_data_arguments` to handle various data input formats, ensuring consistent data handling.
- **Plot Customization:** Plot parameters can be adjusted globally or per plot, including titles, axis labels, and visual styles.
- **Interactivity:** Tooltips and straight lines can be added for enhanced data exploration.
- **Saving and Loading Defaults:** Default plot settings can be saved to or loaded from JSON files, enabling reproducibility and consistency across plots.

---

This collection of routines provides a flexible and powerful interface for creating interactive scatter plots and managing plot configurations efficiently within your data visualization workflows.
