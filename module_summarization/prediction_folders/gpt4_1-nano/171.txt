Calculator Interface
====================

The calculator module provides a set of functions and classes to perform various mathematical and model-related calculations. It is designed to support operations such as parameter listing, model comparison, and complex mathematical transformations, including elliptic functions and coordinate conversions.

Main Components
---------------

1. Parameter Listing and Model Analysis
----------------------------------------

- find_pars(kind=None):  
  Finds all parameters used across available models. Returns a dictionary mapping parameter names to the list of models that use them. You can filter parameters by their kind, such as 'volume', 'orientation', or 'sld'.

- list_pars(names_only=True, kind=None):  
  Prints all parameters found in models. If *names_only* is True, only parameter names are displayed; otherwise, parameter names along with the models that use them are shown. Filtering by *kind* is supported.

- main():  
  Command-line interface entry point that allows users to list parameters with optional verbosity and filtering by parameter kind. Supports arguments:
  - `-v` or `--verbose`: List models that use each parameter.
  - `kind`: Filter parameters by type ('volume', 'orientation', 'sld', 'none', or 'any').

Usage
-----

To list all parameters used in models:

```bash
python -m sasmodels.list_pars
```

To list parameters along with the models that use them:

```bash
python -m sasmodels.list_pars -v
```

To filter parameters by their kind:

```bash
python -m sasmodels.list_pars volume
```

2. Mathematical Functions and Transformations
---------------------------------------------

The module includes advanced mathematical functions, such as elliptic functions and coordinate transformations, supporting complex calculations required in modeling and data analysis.

- ellipticJi(u, v, m):  
  Computes the Jacobian elliptic functions [sn, cn, dn] for complex arguments u + iv with parameter m.

- _ellipticJi_real(u, m):  
  Computes elliptic functions for real u.

- _ellipticJi_imag(v, m):  
  Computes elliptic functions for imaginary v.

- Additional functions (not fully shown) support elliptic integrals, special functions, and coordinate conversions, enabling sophisticated mathematical modeling.

Usage Notes
-----------

- The parameter listing functions are primarily used for model analysis and parameter management.
- The mathematical functions are used internally for complex calculations, such as elliptic integrals and coordinate transformations.
- The command-line interface facilitates easy exploration of model parameters without modifying code.

Dependencies
------------

- `numpy`: For numerical operations and array handling.
- `scipy.special`: For special functions like elliptic integrals.

This interface is designed to be extensible and integrate seamlessly with other components of the modeling framework, supporting both programmatic and command-line workflows.
