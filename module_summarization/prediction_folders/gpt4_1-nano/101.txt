web3.miner
==========

Overview
--------
The `web3.miner` module provides a `Miner` class that allows interaction with the mining functionalities of an Ethereum node via JSON-RPC requests. It enables users to query mining status, control mining operations, and configure mining parameters programmatically.

Classes
-------
Miner
    A class to interface with the mining features of an Ethereum node.

Initialization
--------------
__init__(web3)
    Creates an instance of the `Miner` class.
    :param web3: An instance of the Web3 class used to communicate with the Ethereum node.

Properties
----------
hashrate (property)
    Retrieves the current mining hashrate in H/s.
    Uses the `eth_hashrate` JSON-RPC method.
    The returned value is formatted to a decimal number.

Methods
-------
makeDAG(number)
    Generates the Directed Acyclic Graph (DAG) for the specified block number.
    :param number: The block number for which to generate the DAG.
    :return: The result of the `miner_makeDag` JSON-RPC call.

setExtra(extra)
    Sets the extra data field in the block header.
    :param extra: A string containing extra data to include.
    :return: The result of the `miner_setExtra` JSON-RPC call.

setEtherBase(etherbase)
    Sets the coinbase (etherbase) address for mining rewards.
    :param etherbase: The address to set as the coinbase.
    :return: The result of the `miner_setEtherbase` JSON-RPC call.

setGasPrice(gas_price)
    Sets the gas price for mining transactions.
    :param gas_price: The gas price to set.
    :return: The result of the `miner_setGasPrice` JSON-RPC call.

start(num_threads)
    Starts the mining process with the specified number of threads.
    :param num_threads: The number of mining threads to use.
    :return: The result of the `miner_start` JSON-RPC call.

stop()
    Stops the mining process.
    :return: The result of the `miner_stop` JSON-RPC call.

startAutoDAG()
    Enables automatic DAG generation.
    :return: The result of the `miner_startAutoDag` JSON-RPC call.

stopAutoDAG()
    Disables automatic DAG generation.
    :return: The result of the `miner_stopAutoDag` JSON-RPC call.

Usage Example
-------------
```python
from web3 import Web3
from web3.miner import Miner

web3 = Web3(Web3.HTTPProvider('http://localhost:8545'))
miner = Miner(web3)

# Get current hashrate
current_hashrate = miner.hashrate

# Start mining with 4 threads
miner.start(4)

# Set extra data
miner.setExtra("My mining node")

# Stop mining
miner.stop()
```

Note
----
All methods communicate with the Ethereum node via JSON-RPC requests through the `request_blocking` method of the request manager associated with the `web3` instance. Ensure that the node supports mining RPC methods before using these functionalities.
