Create Cradmin Menu System
===========================

Overview
--------
The Cradmin menu system provides a flexible and customizable way to create navigation menus within Django applications. It includes various renderable classes to build menus with links, expandable items, separators, and toggle buttons, all styled with BEM (Block Element Modifier) methodology.

Getting Started
---------------
The menu system is built around renderable classes that generate HTML components for menus. These classes can be extended or customized to fit specific design requirements.

Default Expandable Menu
-----------------------
The function `get_default_expandable_menu_renderable()` retrieves the default expandable menu renderable based on the Django setting `DJANGO_CRADMIN_DEFAULT_EXPANDABLE_MENU_CLASS`. It dynamically imports and instantiates the specified class, allowing for flexible configuration.

Base Menu Link Renderable
-------------------------
`BaseMenuLinkRenderable` is the foundational class for menu links. It renders a link with a label and URL, and can be styled as active. It uses BEM variants for styling, such as 'active' when the link is active.

- **Attributes:**
  - `label`: The text shown in the menu.
  - `url`: The destination URL.
  - `is_active`: Boolean indicating if the link is active.
  - `parent_bem_block`: The BEM block name for styling.

- **Methods:**
  - `get_bem_element()`: Returns the BEM element name for the link.
  - `get_bem_variant_list()`: Returns a list of BEM variants, including 'active' if applicable.
  - `get_parent_bem_block()`: Determines the parent BEM block, defaulting to 'adminui-page-header'.

Specialized Link Renderables
----------------------------
- **NavLinkItemRenderable**: Used to add standard links to the main menu.
- **NavLinkButtonItemRenderable**: Extends `NavLinkItemRenderable` to style links as buttons, adding the 'button' BEM variant.

Expandable Menu Items
---------------------
`ExpandableMenuItem` extends `BaseMenuLinkRenderable` to create menu items that can expand to show sub-items. It uses a specific template and BEM block for styling.

Expandable Menu Separators
--------------------------
`ExpandableMenuSeparator` provides visual separation within expandable menus. It renders a separator element with a configurable parent BEM block.

Menu Toggle Item
----------------
`MenuToggleItemItemRenderable` adds a toggle button to expand or collapse menus. It provides methods to get labels, icons, and CSS classes, supporting localization with `pgettext_lazy`.

Abstract Menu Renderable
------------------------
`AbstractMenuRenderable` is the base class for rendering entire menus. It manages a list of renderables (links, separators, buttons) and provides methods to retrieve the appropriate renderable classes.

- **Attributes:**
  - `link_renderable_class`: Class used for link items.
  - `separator_renderable_class`: Class used for separators.
  - `button_renderable_class`: Class used for buttons.

- **Methods:**
  - `iter_renderables()`: Iterates over menu items.
  - `get_link_renderable_class()`: Retrieves the link renderable class, raising an error if not set.
  - `get_separator_renderable_class()`: Retrieves the separator class.
  - `get_button_renderable_class()`: Retrieves the button class.

Customization
-------------
Developers can extend or override these classes and methods to create custom menu structures, styles, and behaviors tailored to their application's needs. The system's modular design facilitates easy integration and extension within Django projects.
