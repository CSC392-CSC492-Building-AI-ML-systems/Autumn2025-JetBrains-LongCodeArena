Quick Start
===========

This guide provides a quick overview of how to use the toppra library for geometric path planning and trajectory interpolation.

1. Import Necessary Modules
---------------------------
```python
import numpy as np
from toppra import propose_gridpoints, AbstractGeometricPath, RaveTrajectoryWrapper
```

2. Define or Load a Geometric Path
----------------------------------
Create a class that inherits from `AbstractGeometricPath` and implement the required methods:
- `__call__(self, path_positions, order=0)`
- `dof(self)`
- `path_interval(self)`
- `waypoints(self)` (optional)

Example:
```python
class MyPath(AbstractGeometricPath):
    def __call__(self, path_positions, order=0):
        # Implement path evaluation
        pass

    @property
    def dof(self):
        # Return degrees of freedom
        return 6

    @property
    def path_interval(self):
        # Return start and end of path
        return np.array([0.0, 1.0])

    @property
    def waypoints(self):
        # Return waypoints if applicable
        return None
```

3. Generate Gridpoints for Path Coverage
-----------------------------------------
Use `propose_gridpoints()` to generate gridpoints that sufficiently cover your path:
```python
path = MyPath()
gridpoints = propose_gridpoints(path)
```

4. Wrap OpenRAVE Trajectories (Optional)
-----------------------------------------
If working with OpenRAVE trajectories, wrap them using `RaveTrajectoryWrapper`:
```python
import openravepy as orpy

traj = orpy.RaveTrajectory()
robot = ...  # Obtain an OpenRAVE robot instance
wrapped_traj = RaveTrajectoryWrapper(traj, robot)
```

5. Use the Path and Trajectory for Planning
-------------------------------------------
With the generated gridpoints and wrapped trajectories, proceed with your motion planning or trajectory optimization tasks.

Note: Ensure that your path class correctly implements the abstract methods for proper integration with toppra algorithms.

For more detailed usage and advanced features, refer to the full documentation.
