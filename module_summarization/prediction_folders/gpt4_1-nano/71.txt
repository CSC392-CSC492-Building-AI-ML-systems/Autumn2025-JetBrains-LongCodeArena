Kernel Ridge Regression (sklearn.kernel_ridge)
===============================================

Overview
--------
The `sklearn.kernel_ridge` module implements kernel ridge regression (KRR), a powerful supervised learning method that combines ridge regression with the kernel trick. This allows modeling of complex, non-linear relationships in data by implicitly mapping inputs into a high-dimensional feature space.

Kernel Ridge Regression
------------------------
Kernel ridge regression learns a linear function in the feature space induced by a specified kernel. It minimizes the squared error loss with an L2 regularization term, providing a closed-form solution that is computationally efficient for medium-sized datasets. Unlike support vector regression, KRR produces a non-sparse model, which can be slower at prediction time but easier to interpret and tune.

Main Class
----------
`KernelRidge`
: Implements the kernel ridge regression estimator with support for multi-target regression.

Parameters
----------
- `alpha` (float or array-like of shape (n_targets,)): Regularization strength; must be positive. Larger values increase regularization.
- `kernel` (str or callable, default="linear"): Kernel function used internally. Can be a string specifying a kernel in `pairwise.PAIRWISE_KERNEL_FUNCTIONS` or "precomputed". Alternatively, a callable that takes two samples and returns a kernel value.
- `gamma` (float, default=None): Kernel coefficient for RBF, Laplacian, polynomial, exponential chi2, and sigmoid kernels.
- `degree` (int, default=3): Degree of the polynomial kernel.
- `coef0` (float, default=1): Zero coefficient for polynomial and sigmoid kernels.
- `kernel_params` (dict, default=None): Additional parameters for the kernel function if it is a callable.

Attributes
----------
- `dual_coef_` (ndarray): Representation of the model in kernel space.
- `X_fit_` (ndarray or sparse matrix): Training data or precomputed kernel matrix.
- `n_features_in_` (int): Number of features seen during fit.
- `feature_names_in_` (ndarray): Names of features if available.

Methods
-------
- `_get_kernel(X, Y=None)`: Computes the kernel matrix between `X` and `Y` using the specified kernel function.
- `fit(X, y, sample_weight=None)`: Fits the model to data.
- `predict(X)`: Predicts target values for new data.
- `get_params(deep=True)`: Retrieves model parameters.
- `set_params(**params)`: Sets model parameters.

Usage Example
-------------
```python
from sklearn.kernel_ridge import KernelRidge
import numpy as np

# Generate sample data
n_samples, n_features = 10, 5
rng = np.random.RandomState(0)
X = rng.randn(n_samples, n_features)
y = rng.randn(n_samples)

# Initialize and fit the model
krr = KernelRidge(alpha=1.0)
krr.fit(X, y)

# Make predictions
predictions = krr.predict(X)
```
