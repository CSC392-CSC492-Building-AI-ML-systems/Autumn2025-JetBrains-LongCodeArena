Quantile Regression Module
==========================

Overview
--------
This module provides an implementation of Quantile Regression using iterative reweighted least squares (IRLS). Quantile regression estimates the conditional quantiles of a response variable, offering a comprehensive view of the potential outcomes and their distribution, which is particularly useful in heteroskedastic or non-normal error contexts.

Key Features
------------
- Estimation of quantile regression models for specified quantiles (e.g., median, quartiles).
- Utilizes IRLS for parameter estimation, ensuring robustness and efficiency.
- Supports calculation of asymptotic covariance matrices using kernel density estimation techniques.
- Provides options for different kernel functions and bandwidth selection methods to tailor the covariance estimation.
- Compatible with the statsmodels framework, integrating seamlessly with other regression tools.

Main Class
----------
QuantReg
    - Inherits from RegressionModel.
    - Facilitates the fitting of quantile regression models.
    - Parameters:
        - endog: array-like or DataFrame
            The endogenous (response) variable.
        - exog: array-like or DataFrame
            The exogenous (predictor) variables.
    - Methods:
        - whiten(data): Returns the data unchanged; placeholder for potential whitening procedures.
        - fit(q=0.5, vcov='robust', kernel='epa', bandwidth='hsheather', max_iter=1000, p_tol=1e-6, **kwargs):
            Fits the quantile regression model for a specified quantile.
            Parameters:
                - q: float, the quantile to estimate (between 0 and 1).
                - vcov: string, method for variance-covariance matrix estimation ('robust' or 'iid').
                - kernel: string, kernel function used in density estimation ('epa', 'cos', 'gau', 'par').
                - bandwidth: string, method for bandwidth selection ('hsheather', 'bofinger', 'chamberlain').
                - max_iter: int, maximum number of iterations for convergence.
                - p_tol: float, tolerance for convergence.
            Returns:
                - RegressionResultsWrapper object containing the estimated parameters and statistics.

References
----------
This implementation is based on foundational econometric and statistical literature, including works by Birkes and Dodge (1993), Greene (2008), Koenker (2005), and LeSage (1999). Kernel functions and bandwidth selection methods are adapted from established methods in density estimation literature.

Usage Notes
-----------
- Suitable for modeling conditional quantiles, providing a more complete statistical picture than mean regression.
- The model is robust to heteroskedasticity and outliers, especially when using the robust variance-covariance estimation.
- Users should select appropriate kernel and bandwidth parameters based on their data characteristics and analysis needs.
