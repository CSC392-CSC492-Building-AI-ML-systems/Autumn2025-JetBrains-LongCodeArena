Generate example for this module
===============================

This module implements the one-parameter exponential family base class used by GLM and GEE models. The Family class encapsulates a link function and a variance function and provides helpers used by fitting routines (starting values, IRLS weights, deviance and deviance residuals, and fitted values).

Example
-------

A concise example showing common operations with a Family instance:

.. code-block:: python

    import numpy as np
    from statsmodels.genmod.families import Family
    from statsmodels.genmod.families import links as L
    from statsmodels.genmod.families import varfuncs as V

    # Example: Poisson-like family with a log link and Poisson variance
    fam = Family(link=L.Log(), variance=V.poisson)

    # Observed response
    y = np.array([0, 1, 2, 3, 4], dtype=float)

    # 1) Starting values for mu used by IRLS
    mu0 = fam.starting_mu(y)

    # 2) Weights for IRLS given a mean vector mu
    weights = fam.weights(mu0)

    # 3) Deviance and deviance residuals
    dev = fam.deviance(endog=y, mu=mu0)            # scalar total deviance
    resid = fam.resid_dev(endog=y, mu=mu0)         # per-observation deviance residuals

    # 4) Convert between linear predictor and fitted mean:
    #    - link(mu) gives the linear predictor (eta = g(mu))
    #    - link.inverse(eta) gives fitted mean mu
    eta = fam.link(mu0)
    mu_fitted = fam.link.inverse(eta)

Notes
-----

- The Family class expects a link instance (subclass of links.Link) and a variance function that maps mu -> Var(mu).
- Methods shown above are used internally by GLM/GEE fitting algorithms (IRLS and deviance calculations) and are useful for diagnostics and custom fitting code.
