web3.miner
===========

Overview
--------
The web3.miner module exposes the Miner class, a lightweight wrapper around the Ethereum miner JSON-RPC APIs. It enables programmatic control of mining operations, DAG generation, and miner configuration through a Web3.py instance.

Class: Miner
-------------
The Miner class provides access to miner-related RPC calls via a Web3.py request manager.

- __init__(web3)
  - Description: Initialize with a Web3.py instance.
  - Parameters:
    - web3: Web3 instance used to perform RPC calls.

- hashrate (property)
  - Description: Current miner hashrate in hashes per second (H/s).
  - Decorator: apply_formatters_to_return(to_decimal)
  - Returns: Decimal value representing the hashrate. May be 0 if mining is not active.
  - RPC call: eth_hashrate with no parameters.

  Example:
  ```python
  from decimal import Decimal

  w3 = Web3(HTTPProvider("http://localhost:8545"))
  print("Hashrate:", w3.miner.hashrate)  # Decimal value
  ```

API Methods
-----------

makeDAG(number)
  - Description: Generate the DAG for the given epoch number (DAG index).
  - Parameters:
    - number: int — The DAG epoch/epoch number to generate.
  - Returns: The raw RPC response from miner_makeDag (depends on node version; typically indicates success or status).
  - Example:
    ```python
    w3.miner.makeDAG(3)
    ```

setExtra(extra)
  - Description: Set the extra data field in newly mined blocks.
  - Parameters:
    - extra: str — Extra data to include in the coinbase block.
  - Returns: RPC response (status/confirmation from miner_setExtra).
  - Example:
    ```python
    w3.miner.setExtra("0x1234abcd")
    ```

setEtherBase(etherbase)
  - Description: Set the etherbase (coinbase) address for mining rewards.
  - Parameters:
    - etherbase: str — Ethereum address to receive mining rewards.
  - Returns: RPC response.
  - Example:
    ```python
    w3.miner.setEtherBase("0xABCD...EF01")
    ```

setGasPrice(gas_price)
  - Description: Set the gas price that miners will use for new blocks.
  - Parameters:
    - gas_price: int — Gas price in wei.
  - Returns: RPC response.
  - Example:
    ```python
    w3.miner.setGasPrice(2000000000)  # 2 Gwei
    ```

start(num_threads)
  - Description: Start mining with a specified number of CPU threads.
  - Parameters:
    - num_threads: int — Number of mining threads to run.
  - Returns: RPC response.
  - Example:
    ```python
    w3.miner.start(4)
    ```

stop()
  - Description: Stop all mining activity.
  - Returns: RPC response.
  - Example:
    ```python
    w3.miner.stop()
    ```

startAutoDAG()
  - Description: Enable automatic DAG generation as needed.
  - Returns: RPC response.
  - Example:
    ```python
    w3.miner.startAutoDAG()
    ```

stopAutoDAG()
  - Description: Disable automatic DAG generation.
  - Returns: RPC response.
  - Example:
    ```python
    w3.miner.stopAutoDAG()
    ```

Usage Notes
-----------

- Prerequisites:
  - A running Ethereum node (e.g., Geth) with mining APIs enabled and accessible via the Web3.py provider.
  - The node must be configured to permit miner_* RPC calls for the connected user.
- Return values:
  - Most methods return the raw RPC response from the node. The exact type depends on node version; common results are booleans or status messages.
- Edge cases:
  - hashrate may be 0 if mining is not started or if the node is not connected to a mining network.
  - DAG generation (makeDAG) can be time-consuming and may block if run synchronously; consider using startAutoDAG to manage DAGs automatically.
  - Changing gas price or etherbase while mining is active will affect subsequent blocks but may require node-specific policies or reorg behavior.

Usage example
-------------
```python
from web3 import Web3, HTTPProvider

# Initialize Web3 and miner interface
w3 = Web3(HTTPProvider("http://localhost:8545"))
miner = w3.miner

# Check current hashrate
print("Hashrate:", miner.hashrate)

# Configure mining
miner.setEtherBase("0xYourEtherbaseAddress")
miner.setGasPrice(2_000_000_000)  # 2 Gwei
miner.setExtra("0x546f6b656e")    # Example extra data (hex)

# Start mining with 4 threads
miner.start(4)

# Ensure DAGs are managed automatically
miner.startAutoDAG()

# Later, stop mining
miner.stop()
miner.stopAutoDAG()
```
