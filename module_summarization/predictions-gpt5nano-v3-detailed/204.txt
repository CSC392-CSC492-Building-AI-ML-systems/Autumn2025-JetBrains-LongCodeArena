Introduction
------------
This module provides simple, computationally efficient transformers for reducing the dimensionality of data through random projections. By projecting high-dimensional data into a lower-dimensional space using a random matrix, these transformers trade a controlled amount of accuracy for faster processing and smaller models.

Overview
--------
- Random projections preserve pairwise distances between samples with high probability, as guaranteed by the Johnson-Lindenstrauss lemma.
- Two transformer variants are available:
  - GaussianRandomProjection: uses a dense Gaussian random matrix.
  - SparseRandomProjection: uses a sparse random matrix (controlled by density) for even faster computations on large feature spaces.

Core concepts
-------------
- Random projection matrices are typically drawn from a Gaussian distribution with entries in N(0, 1/n_components) or from a sparse distribution guided by the density parameter.
- The minimal number of components required for an eps-embedding (distortion up to eps) can be estimated using the Johnson-Lindenstrauss bound.

Key functions and classes
-------------------------

johnson_lindenstrauss_min_dim
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
Compute a safe, minimal number of components to guarantee an eps-embedding with the given number of samples.

Parameters
- n_samples: int or array-like of int
  Number of samples (must be > 0). If an array is provided, computation is performed element-wise.
- eps: float or array-like of shape (n_components,), default 0.1
  Maximum distortion (0, 1). If an array is provided, compute components array-wise.

Returns
- n_components: int or ndarray of int
  Minimal number of components guaranteeing an eps-embedding with n_samples.

Examples
--------
>>> from sklearn.random_projection import johnson_lindenstrauss_min_dim
>>> johnson_lindenstrauss_min_dim(1e6, eps=0.5)
663

>>> johnson_lindenstrauss_min_dim(1e6, eps=[0.5, 0.1, 0.01])
array([    663,   11841, 1112658])

pytest
>>> johnson_lindenstrauss_min_dim([1e4, 1e5, 1e6], eps=0.1)
array([ 7894,  9868, 11841])

GaussianRandomProjection
^^^^^^^^^^^^^^^^^^^^^^^^^^^
Transformer that projects data using a random dense Gaussian matrix.

Parameters
- n_components: int
  Dimensionality of the projected space. Must be > 0.
- random_state: int, RandomState instance or None, optional
  Seed or RNG used to generate the random matrix.
- dense_output: bool, default True
  If True, the transformed data is returned as a dense array. If False, a sparse representation may be produced when possible.

Notes
- The projection matrix entries are drawn from N(0, 1/n_components).
- Suitable for general-purpose dimensionality reduction where a dense projection is acceptable.

Example
-------
>>> from sklearn.random_projection import GaussianRandomProjection
>>> import numpy as np
>>> rng = np.random.RandomState(0)
>>> X = rng.randn(10, 50)
>>> rp = GaussianRandomProjection(n_components=20, random_state=0)
>>> X_rp = rp.fit_transform(X)
>>> X_rp.shape
(10, 20)

SparseRandomProjection
^^^^^^^^^^^^^^^^^^^^^^
Transformer that projects data using a sparse random matrix.

Parameters
- n_components: int
  Dimensionality of the projected space. Must be > 0.
- density: ("auto" or float], default="auto"
  Controls the density of the random projection matrix. When "auto", density is set to 1 / sqrt(n_features).
- random_state: int, RandomState instance or None, optional
  Seed or RNG used to generate the random matrix.
- dense_output: bool, default True
  If True, the transformed data is returned as a dense array. If False, a sparse representation may be produced when possible.

Density handling
- "auto": dynamic density based on the number of features.
- If a numeric value is provided, it must satisfy 0 < density <= 1.

Notes
- The matrix entries are drawn so that the projection preserves distances with high probability, akin to the Johnson-Lindenstrauss framework, but with a sparse structure to improve efficiency on high-dimensional data.

Example
-------
>>> from sklearn.random_projection import SparseRandomProjection
>>> import numpy as np
>>> rng = np.random.RandomState(0)
>>> X = rng.randn(100, 1000)
>>> rp = SparseRandomProjection(n_components=50, density="auto", random_state=0)
>>> X_rp = rp.fit_transform(X)
>>> X_rp.shape
(100, 50)

Usage and practical guidance
----------------------------
- When working with very high-dimensional data, SparseRandomProjection with density="auto" often offers a good balance between speed and accuracy.
- Use johnson_lindenstrauss_min_dim to estimate a safe target dimensionality before fitting a projection, especially when you have a target distortion eps.
- The minimal components required scale with the dataset size n_samples and the allowed distortion eps; larger datasets require more components to maintain the same embedding quality.

Important considerations and edge cases
---------------------------------------
- eps must be in (0, 1). Values outside this range raise ValueError.
- n_samples must be > 0. Non-positive values raise ValueError.
- n_components and n_features must be strictly positive.
- density must be in (0, 1] if specified; "auto" is supported to auto-compute a density based on the feature count.
- Dense vs. sparse output: use dense_output to control whether results are dense or can leverage sparse representations for efficiency.
- Theoretical guarantees depend on the Johnson-Lindenstrauss lemma; in practice, empirical performance depends on data structure and distribution.

References
----------
- Johnson-Lindenstrauss lemma: https://en.wikipedia.org/wiki/Johnson%E2%80%93Lindenstrauss_lemma
- Dasgupta, Gupta (1999): An elementary proof of the Johnson-Lindenstrauss Lemma

Usage examples
--------------
Basic Gaussian projection
------------------------
```python
from sklearn.random_projection import GaussianRandomProjection
import numpy as np

X = np.random.RandomState(0).randn(100, 500)
rp = GaussianRandomProjection(n_components=100, random_state=42)
X_projected = rp.fit_transform(X)
print(X_projected.shape)  # (100, 100)
```

Sparse projection with auto density
----------------------------------
```python
from sklearn.random_projection import SparseRandomProjection
import numpy as np

X = np.random.RandomState(0).randn(200, 1000)
rp = SparseRandomProjection(n_components=120, density="auto", random_state=42)
X_projected = rp.fit_transform(X)
print(X_projected.shape)  # (200, 120)
```

Minimal dimension calculation (JL bound)
---------------------------------------
```python
from sklearn.random_projection import johnson_lindenstrauss_min_dim

n_samples = 1_000_000
eps = 0.5
n_components = johnson_lindenstrauss_min_dim(n_samples, eps=eps)
print(n_components)  # e.g., 663 for eps=0.5
```
